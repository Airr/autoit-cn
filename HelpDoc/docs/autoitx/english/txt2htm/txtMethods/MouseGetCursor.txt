###Method###
MouseGetCursor

###Description###
Returns a cursor ID Number of the current Mouse Cursor.

###Syntax###
MouseGetCursor


###Parameters###
None.

###ReturnValue###
Returns a cursor ID Number:
	0 = UNKNOWN  (this includes pointing and grabbing hand icons)
	1 = APPSTARTING
	2 = ARROW
	3 = CROSS
	4 = HELP
	5 = IBEAM
	6 = ICON
	7 = NO
	8 = SIZE
	9 = SIZEALL
	10 = SIZENESW
	11 = SIZENS
	12 = SIZENWSE
	13 = SIZEWE
	14 = UPARROW
	15 = WAIT


###Remarks###
None.


###Related###
MouseGetPosX, MouseGetPosY
