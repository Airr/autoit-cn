###Function###
StringIsASCII

###Description###
Checks if a string contains only ASCII characters in the range 0x00 - 0x7f (0 - 127).

###Syntax###
StringIsASCII ( "string" )


###Parameters###
@@ParamTable@@
string
	The string to check
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	Returns 1.
Failure:	Returns 0 if string contains extended ASCII characters.
@@End@@


###Remarks###
None.
