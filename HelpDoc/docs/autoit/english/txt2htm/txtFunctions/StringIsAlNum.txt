###Function###
StringIsAlNum

###Description###
Checks if a string contains only alphanumeric characters.

###Syntax###
StringIsAlNum ( "string" )


###Parameters###
@@ParamTable@@
string
	The string to check
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	Returns 1.
Failure:	Returns 0 if string contains non-alphanumeric characters.
@@End@@


###Remarks###
Keep in mind that any whitespace will cause <a href="StringIsAlNum.htm">StringIsAlNum()</a> to return 0.
