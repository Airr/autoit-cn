###Function###
IsInt

###Description###
Checks if a variable or expression is an integer type.

###Syntax###
IsInt ( variable )


###Parameters###
@@ParamTable@@
variable
	The variable/expression to check.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	1.
Failure:	0 if expression is not integer.
@@End@@


###Remarks###
<a href="IsInt.htm">IsInt</a>(7.5 - 2.5) returns 1 (meaning true).

See <a href="..\intro\lang_datatypes.htm">language datatypes</a> for a detailed description.
