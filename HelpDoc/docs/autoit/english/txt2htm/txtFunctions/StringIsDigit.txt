###Function###
StringIsDigit

###Description###
Checks if a string contains only digit (0-9) characters.

###Syntax###
StringIsDigit ( "string" )


###Parameters###
@@ParamTable@@
string
	The string to check
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	Returns 1.
Failure:	Returns 0 if string contains non-digits.
@@End@@

See examples.
