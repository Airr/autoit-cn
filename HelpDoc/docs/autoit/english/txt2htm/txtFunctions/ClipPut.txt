###Function###
ClipPut

###Description###
Writes text to the clipboard.

###Syntax###
ClipPut ( "value" )


###Parameters###
@@ParamTable@@
value
	The text to write to the clipboard.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	1.
Failure:	0.
@@End@@


###Remarks###
Any existing clipboard contents are overwritten.
An empty string "" will empty the clipboard.
