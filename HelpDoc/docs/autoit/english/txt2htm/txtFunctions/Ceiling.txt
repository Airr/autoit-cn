###Function###
Ceiling

###Description###
Returns a number rounded up to the next integer.

###Syntax###
Ceiling ( expression )


###Parameters###
@@ParamTable@@
expression
	Any valid numeric expression.
@@End@@

###ReturnValue###
Returns the rounded number.

None.
