###Function###
StringReverse

###Description###
Reverses the contents of the specified string.

###Syntax###
StringReverse ( "string" [, flag = 0] )


###Parameters###
@@ParamTable@@
string
	The string to reverse.
flag
	[optional] Changes the way the string is reversed
	flag = 0 (default), reversed in full UTF-16 mode.
	flag = 1, a much faster method - only use if using UCS-2 text.
@@End@@

###ReturnValue###
Returns a reversed string.


###Remarks###
None.
