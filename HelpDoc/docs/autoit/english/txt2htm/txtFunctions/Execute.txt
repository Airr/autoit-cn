###Function###
Execute

###Description###
Execute an expression.

###Syntax###
Execute ( string )


###Parameters###
@@ParamTable@@
string
	string representing an expression to be evaluated.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	Returns the value of the evaluated expression.
Failure:	Returns "" (empty string) with @error set not to 0.
@@End@@


###Remarks###
None.
