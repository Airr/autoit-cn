###Function###
StringIsUpper

###Description###
Checks if a string contains only uppercase characters.

###Syntax###
StringIsUpper ( "string" )


###Parameters###
@@ParamTable@@
string
	The string to check
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	Returns 1.
Failure:	Returns 0 if string contains non-uppercase characters.
@@End@@


###Remarks###
Note that digits/punctuation/whitespace will cause <a href="StringIsUpper.htm">StringIsUpper()</a> to return 0.
