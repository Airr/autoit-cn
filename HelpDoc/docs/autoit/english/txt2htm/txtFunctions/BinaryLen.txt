###Function###
BinaryLen

###Description###
Returns the number of bytes in a binary variant.

###Syntax###
BinaryLen ( binary )


###Parameters###
@@ParamTable@@
binary
	The binary data to evaluate.
@@End@@

###ReturnValue###
Returns the length of the binary data in bytes.
