###Keyword###
#include-once

###Description###
Specifies that the current file may only be included once.

###Syntax###
<strong>#include-once</strong>


###Parameters###
None.

###Remarks###
If you include the same file containing a user-function more than once, you will get a "Duplicate function" error. When writing an include file that may be used in this way, make sure that the top line contains <strong>#include-once</strong> to prevent that file from being included more than once.
