###User Defined Function###
_GUICtrlSlider_GetLineSize

###Description###
Retrieves the number of logical positions the slider moves

###Syntax###
#include <GuiSlider.au3>
_GUICtrlSlider_GetLineSize ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
@@End@@

###ReturnValue###
Returns the line size for the slider.


###Remarks###
The default setting for the line size is 1
