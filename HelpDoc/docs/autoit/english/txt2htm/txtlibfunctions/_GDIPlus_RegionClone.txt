###User Defined Function###
_GDIPlus_RegionClone

###Description###
Clones a Region object

###Syntax###
#include <GDIPlus.au3>
_GDIPlus_RegionClone ( $hRegion )


###Parameters###
@@ParamTable@@
$hRegion
	Pointer to a Region object to be cloned
@@End@@

###ReturnValue###
Success: Pointer to a new cloned Region object
Failure: 0


###Remarks###
After you are done with the object, call _GDIPlus_RegionDispose to release the object resources
