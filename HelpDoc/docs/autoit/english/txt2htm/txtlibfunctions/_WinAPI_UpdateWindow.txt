###User Defined Function###
_WinAPI_UpdateWindow

###Description###
Updates the client area of a window by sending a WM_PAINT message to the window

###Syntax###
#include <WinAPI.au3>
_WinAPI_UpdateWindow ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Handle of window to update
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True
Failure:	False
@@End@@
