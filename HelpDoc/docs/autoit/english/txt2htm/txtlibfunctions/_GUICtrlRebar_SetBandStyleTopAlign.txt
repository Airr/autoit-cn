###User Defined Function###
_GUICtrlRebar_SetBandStyleTopAlign

###Description###
Set whether to keep band in top row

###Syntax###
#include <GuiReBar.au3>
_GUICtrlRebar_SetBandStyleTopAlign ( $hWnd, $iIndex [, $fEnabled = True] )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the rebar control
$iIndex
	0-based index of the band
$fEnabled
	[optional] If True the item state is set, otherwise it is not set
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True.
Failure:	False.
@@End@@


###Remarks###
None.
