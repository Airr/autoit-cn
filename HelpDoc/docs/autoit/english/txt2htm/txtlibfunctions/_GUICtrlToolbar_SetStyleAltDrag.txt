###User Defined Function###
_GUICtrlToolbar_SetStyleAltDrag

###Description###
Sets whether that the control allows buttons to be dragged

###Syntax###
#include <GuiToolbar.au3>
_GUICtrlToolbar_SetStyleAltDrag ( $hWnd [, $fState = True] )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$fState
	[optional] True to set, False to unset
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True
Failure:	False
@@End@@


###Remarks###
None.
