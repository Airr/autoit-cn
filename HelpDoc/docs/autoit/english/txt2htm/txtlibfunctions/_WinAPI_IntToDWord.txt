###User Defined Function###
_WinAPI_IntToDWord

###Description###
Converts a value of type INT to a value of type DWORD.

###Syntax###
#include <WinAPIMisc.au3>
_WinAPI_IntToDWord ( $iValue )


###Parameters###
@@ParamTable@@
$iValue
	The value to be converted.
@@End@@

###ReturnValue###
Returns the converted (DWORD) value.
