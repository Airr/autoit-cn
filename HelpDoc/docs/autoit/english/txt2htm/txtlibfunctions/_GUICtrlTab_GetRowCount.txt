###User Defined Function###
_GUICtrlTab_GetRowCount

###Description###
Retrieves the current number of rows of tabs

###Syntax###
#include <GuiTab.au3>
_GUICtrlTab_GetRowCount ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
@@End@@

###ReturnValue###
Returns the number of rows of tabs.


###Remarks###
Only tab controls that have the $TCS_MULTILINE style can have multiple rows of tabs.
