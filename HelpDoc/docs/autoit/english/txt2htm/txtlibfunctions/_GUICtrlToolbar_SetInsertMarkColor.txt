###User Defined Function###
_GUICtrlToolbar_SetInsertMarkColor

###Description###
Sets the color used to draw the insertion mark

###Syntax###
#include <GuiToolbar.au3>
_GUICtrlToolbar_SetInsertMarkColor ( $hWnd, $iColor )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$iColor
	Insertion mark color
@@End@@

###ReturnValue###
Success: Previous insertion mark color


###Remarks###
None.
