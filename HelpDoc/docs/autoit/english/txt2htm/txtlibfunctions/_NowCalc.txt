###User Defined Function###
_NowCalc

###Description###
Returns the current Date and Time in format YYYY/MM/DD HH:MM:SS for use in date calculations.

###Syntax###

_NowCalc (  )


###Parameters###
None.

###ReturnValue###
Returns the current Date and Time in format YYYY/MM/DD HH:MM:SS
