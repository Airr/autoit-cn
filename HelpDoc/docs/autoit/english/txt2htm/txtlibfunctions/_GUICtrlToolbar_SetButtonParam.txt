###User Defined Function###
_GUICtrlToolbar_SetButtonParam

###Description###
Sets the button param value

###Syntax###
#include <GuiToolbar.au3>
_GUICtrlToolbar_SetButtonParam ( $hWnd, $iCommandID, $iParam )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$iCommandID
	Button command ID
$iParam
	Application-defined value
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True.
Failure:	False.
@@End@@
