###User Defined Function###
_GUICtrlTab_SetImageList

###Description###
Sets the image list associated with a tab control

###Syntax###
#include <GuiTab.au3>
_GUICtrlTab_SetImageList ( $hWnd, $hImage )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
$hImage
	Handle to the image list to assign to the tab control
@@End@@

###ReturnValue###
Returns the handle to the previous image list.


###Remarks###
None.
