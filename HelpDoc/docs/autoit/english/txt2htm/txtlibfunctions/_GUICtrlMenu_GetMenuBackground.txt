###User Defined Function###
_GUICtrlMenu_GetMenuBackground

###Description###
Retrieves the brush to use for the menu's background

###Syntax###
#include <GuiMenu.au3>
_GUICtrlMenu_GetMenuBackground ( $hMenu )


###Parameters###
@@ParamTable@@
$hMenu
	Handle of the menu
@@End@@

###ReturnValue###
Returns the brush used for the menu's background.


###Remarks###
None.
