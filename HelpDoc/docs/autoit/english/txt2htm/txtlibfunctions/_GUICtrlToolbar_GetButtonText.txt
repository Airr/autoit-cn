###User Defined Function###
_GUICtrlToolbar_GetButtonText

###Description###
Retrieves the display text of a button

###Syntax###
#include <GuiToolbar.au3>
_GUICtrlToolbar_GetButtonText ( $hWnd, $iCommandID )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$iCommandID
	Button command ID
@@End@@

###ReturnValue###
Returns the button text.


###Remarks###
None.
