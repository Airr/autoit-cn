###User Defined Function###
_GDIPlus_MatrixSetElements

###Description###
Sets the elements of a matrix

###Syntax###
#include <GDIPlus.au3>
_GDIPlus_MatrixSetElements ( $hMatrix [, $iM11 = 1 [, $iM12 = 0 [, $iM21 = 0 [, $iM22 = 1 [, $iDX = 0 [, $iDY = 0]]]]]] )


###Parameters###
@@ParamTable@@
$hMatrix
	A handle value
$iM11
	[optional] A floating point number value. Default is 1
$iM12
	[optional] A floating point number value. Default is 0
$iM21
	[optional] A floating point number value. Default is 0
$iM22
	[optional] A floating point number value. Default is 1
$iDX
	[optional] A floating point number value. Default is 0
$iDY
	[optional] A floating point number value. Default is 0
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True.
Failure:	False and sets the @error flag to non-zero, @extended may contain GPSTATUS error code ($GPID_ERR*).
@@End@@


###Remarks###
None.
