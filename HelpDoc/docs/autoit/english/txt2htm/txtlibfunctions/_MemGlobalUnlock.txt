###User Defined Function###
_MemGlobalUnlock

###Description###
Decrements the lock count associated with a memory object that was allocated with GMEM_MOVEABLE

###Syntax###
#include <Memory.au3>
_MemGlobalUnlock ( $hMemory )


###Parameters###
@@ParamTable@@
$hMemory
	Handle to the global memory object
@@End@@

###ReturnValue###
None.


###Remarks###
None.
