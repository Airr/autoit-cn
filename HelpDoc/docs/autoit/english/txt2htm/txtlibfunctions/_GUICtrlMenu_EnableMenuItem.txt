###User Defined Function###
_GUICtrlMenu_EnableMenuItem

###Description###
Enables, disables, or grays the specified menu item

###Syntax###
#include <GuiMenu.au3>
_GUICtrlMenu_EnableMenuItem ( $hMenu, $iItem [, $iState = 0 [, $fByPos = True]] )


###Parameters###
@@ParamTable@@
$hMenu
	Menu handle
$iItem
	Identifier or position of the menu item
$iState
	[optional] Indicates whether the menu item is enabled, disabled, or grayed:
	0 - Enabled
	1 - Grayed
	2 - Disabled
$fByPos
	[optional] Menu identifier flag:
	True - $iItem is a zero based item position
	False - $iItem is a menu item identifier
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True
Failure:	False
@@End@@
