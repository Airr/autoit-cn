###User Defined Function###
_GUICtrlHeader_Destroy

###Description###
Delete the Header control

###Syntax###
#include <GuiHeader.au3>
_GUICtrlHeader_Destroy ( ByRef $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True, $hWnd is set to 0.
Failure:	False.
@@End@@


###Remarks###
Restricted to only be used on Header controls created with <a href="_GUICtrlHeader_Create.htm">_GUICtrlHeader_Create()</a>.
