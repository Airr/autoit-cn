###User Defined Function###
_GUICtrlListView_SetItemGroupID

###Description###
Sets the item group ID

###Syntax###
#include <GuiListView.au3>
_GUICtrlListView_SetItemGroupID ( $hWnd, $iIndex, $iGroupID )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$iIndex
	Zero based index of the item
$iGroupID
	Group ID
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True
Failure:	False
@@End@@

None.
