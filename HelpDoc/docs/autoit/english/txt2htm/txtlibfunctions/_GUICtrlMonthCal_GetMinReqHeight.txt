###User Defined Function###
_GUICtrlMonthCal_GetMinReqHeight

###Description###
Retrieves the minimum height required to display a full month

###Syntax###
#include <GuiMonthCal.au3>
_GUICtrlMonthCal_GetMinReqHeight ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
@@End@@

###ReturnValue###
Returns the minimum height, in pixels.


###Remarks###
None.
