###User Defined Function###
_GDIPlus_DecodersGetCount

###Description###
Get the number of available image decoders

###Syntax###
#include <GDIPlus.au3>
_GDIPlus_DecodersGetCount (  )


###Parameters###
None.

###ReturnValue###
@@ReturnTable@@
Success:	the number of image decoders.
Failure:	-1 and sets the @error flag to non-zero, @extended may contain GPSTATUS error code (GDIP_ERR*).
@@End@@
