###User Defined Function###
_GUICtrlHeader_AddItem

###Description###
Adds a new header item

###Syntax###
#include <GuiHeader.au3>
_GUICtrlHeader_AddItem ( $hWnd, $sText [, $iWidth = 50 [, $iAlign = 0 [, $iImage = -1 [, $fOnRight = False]]]] )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$sText
	Item text
$iWidth
	[optional] Item width
$iAlign
	[optional] Text alignment:
	0 - Text is left-aligned
	1 - Text is right-aligned
	2 - Text is centered
$iImage
	[optional] Zero based index of an image within the image list
$fOnRight
	[optional] If True, the column image appears to the right of text
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	The index of the new item
Failure:	-1
@@End@@


###Remarks###
None.
