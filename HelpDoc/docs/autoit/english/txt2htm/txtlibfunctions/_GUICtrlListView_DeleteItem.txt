###User Defined Function###
_GUICtrlListView_DeleteItem

###Description###
Removes an item from a list-view control

###Syntax###
#include <GuiListView.au3>
_GUICtrlListView_DeleteItem ( $hWnd, $iIndex )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
$iIndex
	Zero based index of the list-view item to delete
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True
Failure:	False
@@End@@


###Remarks###
None.
