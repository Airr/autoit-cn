###User Defined Function###
_GUICtrlRichEdit_Destroy

###Description###
Delete the Rich Edit control

###Syntax###
#include <GuiRichEdit.au3>
_GUICtrlRichEdit_Destroy ( ByRef $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	1, $hWnd is set to 0.
Failure:	0 and sets the @error flag to non-zero.
@error:	1 - attempt to destroy a control belonging to another application
@@End@@


###Remarks###
Restricted to only be used on Rich Edit controls created with <a href="_GUICtrlRichEdit_Create.htm">_GUICtrlRichEdit_Create()</a>.
