###User Defined Function###
_WinAPI_PathIsSameRoot

###Description###
Compares two paths to determine if they have a common root component.

###Syntax###
#include <WinAPIShPath.au3>
_WinAPI_PathIsSameRoot ( $sPath1, $sPath2 )


###Parameters###
@@ParamTable@@
$sPath1
	The first path to be compared.
$sPath2
	The second path to be compared.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success 	True - Both strings have the same root component.
Failure 	False.
@@End@@
