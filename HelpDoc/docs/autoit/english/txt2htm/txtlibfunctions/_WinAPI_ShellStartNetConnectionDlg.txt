###User Defined Function###
_WinAPI_ShellStartNetConnectionDlg

###Description###
Displays a general browsing dialog box for a network resource connection.

###Syntax###
#include <WinAPIDlg.au3>
_WinAPI_ShellStartNetConnectionDlg ( [$sRemote = '' [, $iFlags = 0 [, $hParent = 0]]] )


###Parameters###
@@ParamTable@@
$sRemote
	[optional] The remote network name.
$iFlags
	[optional] The flags that identify the type of resource that the dialog is set to find. This parameter can be
	one or more of the following values.


$hParent
	[optional] Handle of the parent window.
@@End@@

###ReturnValue###
None
