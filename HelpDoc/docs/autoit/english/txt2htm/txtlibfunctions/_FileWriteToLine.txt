###User Defined Function###
_FileWriteToLine

###Description###
Writes text to a specific line in a file.

###Syntax###
#include <File.au3>
_FileWriteToLine ( $sFilePath, $iLine, $sText [, $iOverWrite = 0] )


###Parameters###
@@ParamTable@@
$sFilePath
	The file to write to
$iLine
	The line number to write to
$sText
	The text to write
$iOverWrite
	[optional] If set to 1 will overwrite the old line
	If set to 0 will not overwrite
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	1
Failure:	0 and set @error to non-zero
@error:	1 = File has less lines than $iLine
	2 = File does not exist
	3 = Error when opening file
	4 = $iLine is invalid
	5 = $iOverWrite is invalid
	6 = $sText is invalid
@@End@@


###Remarks###
If _FileWriteToLine is called with $iOverWrite as 1 and $sText as "", it will delete the line.
