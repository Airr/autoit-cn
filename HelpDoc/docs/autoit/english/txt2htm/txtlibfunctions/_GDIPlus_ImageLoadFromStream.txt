###User Defined Function###
_GDIPlus_ImageLoadFromStream

###Description###
Creates an Image object based on a stream

###Syntax###
#include <GDIPlus.au3>
_GDIPlus_ImageLoadFromStream ( $pStream )


###Parameters###
@@ParamTable@@
$pStream
	Pointer to an IStream interface
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	Pointer to a new Image object
Failure:	0 and either:
			@error and @extended are set if DllCall failed
@@End@@


###Remarks###
After you are done with the object, call _GDIPlus_ImageDispose to release the object resources.
