###User Defined Function###
_GUICtrlListView_SetSelectedColumn

###Description###
Sets the index of the selected column

###Syntax###
#include <GuiListView.au3>
_GUICtrlListView_SetSelectedColumn ( $hWnd, $iCol )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
$iCol
	Specifies the column index
@@End@@

###ReturnValue###
None.


###Remarks###
Columns are Zero based index
