###User Defined Function###
_GUICtrlMenu_GetItemText

###Description###
Retrieves the text of the specified menu item

###Syntax###
#include <GuiMenu.au3>
_GUICtrlMenu_GetItemText ( $hMenu, $iItem [, $fByPos = True] )


###Parameters###
@@ParamTable@@
$hMenu
	Handle of the menu
$iItem
	Identifier or position of the menu item
$fByPos
	[optional] Menu identifier flag:
		True - $iItem is a 0-based item position
		False - $iItem is a menu item identifier
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	the menu item text.
Failure:	sets the @error flag to non-zero.
@@End@@


###Remarks###
None.
