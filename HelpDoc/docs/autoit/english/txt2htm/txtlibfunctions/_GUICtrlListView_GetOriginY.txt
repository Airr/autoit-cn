###User Defined Function###
_GUICtrlListView_GetOriginY

###Description###
Retrieves the current vertical view origin for the control

###Syntax###
#include <GuiListView.au3>
_GUICtrlListView_GetOriginY ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
@@End@@

###ReturnValue###
Returns the View Y position


###Remarks###
None.
