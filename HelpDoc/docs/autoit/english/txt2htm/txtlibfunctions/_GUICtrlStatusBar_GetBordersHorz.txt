###User Defined Function###
_GUICtrlStatusBar_GetBordersHorz

###Description###

###Syntax###
#include <GuiStatusBar.au3>
_GUICtrlStatusBar_GetBordersHorz ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	Width of the horizontal border
Failure:	Set @error
@@End@@
