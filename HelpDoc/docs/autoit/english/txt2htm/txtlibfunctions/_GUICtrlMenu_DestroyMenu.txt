###User Defined Function###
_GUICtrlMenu_DestroyMenu

###Description###
Destroys the specified menu and frees any memory that the menu occupies

###Syntax###
#include <GuiMenu.au3>
_GUICtrlMenu_DestroyMenu ( $hMenu )


###Parameters###
@@ParamTable@@
$hMenu
	Menu handle
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True
Failure:	False
@@End@@


###Remarks###
None.
