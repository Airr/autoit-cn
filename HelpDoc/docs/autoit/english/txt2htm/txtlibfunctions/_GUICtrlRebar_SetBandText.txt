###User Defined Function###
_GUICtrlRebar_SetBandText

###Description###
Sets the display text for the band of a rebar control

###Syntax###
#include <GuiReBar.au3>
_GUICtrlRebar_SetBandText ( $hWnd, $iIndex, $sText )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the rebar control
$iIndex
	Zero-based index of the band for which the information will be set
$sText
	New display text for the band
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True
Failure:	False
@@End@@


###Remarks###
None.
