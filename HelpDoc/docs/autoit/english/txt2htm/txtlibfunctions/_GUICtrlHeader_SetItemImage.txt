###User Defined Function###
_GUICtrlHeader_SetItemImage

###Description###
Sets the index of an image within the image list

###Syntax###
#include <GuiHeader.au3>
_GUICtrlHeader_SetItemImage ( $hWnd, $iIndex, $iImage )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$iIndex
	Zero based item index
$iImage
	Zero based image index
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True
Failure:	False
@@End@@
