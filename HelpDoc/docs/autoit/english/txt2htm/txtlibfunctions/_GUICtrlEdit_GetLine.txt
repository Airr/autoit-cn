###User Defined Function###
_GUICtrlEdit_GetLine

###Description###
Retrieves a line of text from an edit control

###Syntax###
#include <GuiEdit.au3>
_GUICtrlEdit_GetLine ( $hWnd, $iLine )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
$iLine
	Zero-based line index to get
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	The line of text
Failure:	Empty string
@@End@@
