###User Defined Function###
_GUICtrlEdit_GetLine

###Description###
Retrieves a line of text from an edit control

###Syntax###
#include <GuiEdit.au3>
_GUICtrlEdit_GetLine ( $hWnd, $iLine )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
$iLine
	0-based line index to get
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	the line of text.
Failure:	an empty string.
@@End@@
