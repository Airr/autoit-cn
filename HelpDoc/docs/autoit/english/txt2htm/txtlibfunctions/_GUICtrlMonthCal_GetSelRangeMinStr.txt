###User Defined Function###
_GUICtrlMonthCal_GetSelRangeMinStr

###Description###
Retrieves the lower date range currently selected by the user in string form

###Syntax###
#include <GuiMonthCal.au3>
_GUICtrlMonthCal_GetSelRangeMinStr ( $hWnd [, $sFormat = "%02d/%02d/%04d"] )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
$sFormat
	[optional] StringFormat string used to format the date
@@End@@

###ReturnValue###
Success: Date in the specified format


###Remarks###
None.
