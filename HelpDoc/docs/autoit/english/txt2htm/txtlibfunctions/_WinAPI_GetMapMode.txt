###User Defined Function###
_WinAPI_GetMapMode

###Description###
Retrieves the current mapping mode.

###Syntax###
#include <WinAPIGdi.au3>
_WinAPI_GetMapMode ( $hDC )


###Parameters###
@@ParamTable@@
$hDC
	Handle to the device context.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success 	The value specifies the mapping mode (MM_*).
Failure 	0.
@@End@@
