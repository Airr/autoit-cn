###User Defined Function###
_GUIScrollBars_ScrollWindow

###Description###
Scrolls the contents of the specified window's client area

###Syntax###
#include <GuiScrollBars.au3>
_GUIScrollBars_ScrollWindow ( $hWnd, $iXAmount, $iYAmount )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the window
$iXAmount
	Specifies the amount, in device units, of horizontal scrolling
$iYAmount
	Specifies the amount, in device units, of vertical scrolling
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True
Failure:	False
@@End@@

None.
