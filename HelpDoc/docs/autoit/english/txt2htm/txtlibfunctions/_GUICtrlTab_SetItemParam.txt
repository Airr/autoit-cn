###User Defined Function###
_GUICtrlTab_SetItemParam

###Description###
Sets the param data of a tab

###Syntax###
#include <GuiTab.au3>
_GUICtrlTab_SetItemParam ( $hWnd, $iIndex, $iParam )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
$iIndex
	0-based item index
$iParam
	Application defined data
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True.
Failure:	False.
@@End@@


###Remarks###
None.
