###User Defined Function###
_GDIPlus_RegionCreate

###Description###
Creates a region that is infinite

###Syntax###
#include <GDIPlus.au3>
_GDIPlus_RegionCreate (  )


###Parameters###
None.

###ReturnValue###
Success: Pointer to a new Region object
Failure: 0


###Remarks###
After you are done with the object, call _GDIPlus_RegionDispose to release the object resources
