###User Defined Function###
_FTP_DirDelete

###Description###
Delete's an Directory on an FTP server

###Syntax###
#include <FTPEx.au3>
_FTP_DirDelete ( $hFTPSession, $sRemote )


###Parameters###
@@ParamTable@@
$hFTPSession
	as returned by <a href="_FTP_Connect.htm">_FTP_Connect()</a>.
$sRemote
	The Directory to deleted.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	1.
Failure:	0.
@@End@@


###Remarks###
None.
