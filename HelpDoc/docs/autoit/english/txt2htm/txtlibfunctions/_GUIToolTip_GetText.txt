###User Defined Function###
_GUIToolTip_GetText

###Description###
Retrieves the text

###Syntax###
#include <GuiToolTip.au3>
_GUIToolTip_GetText ( $hWnd, $hTool, $iID )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$hTool
	Handle to the window that contains the tool
$iID
	Identifier of the tool
@@End@@

###ReturnValue###
Returns the ToolTip text.


###Remarks###
None.
