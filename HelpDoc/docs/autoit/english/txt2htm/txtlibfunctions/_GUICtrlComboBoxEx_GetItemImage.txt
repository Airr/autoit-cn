###User Defined Function###
_GUICtrlComboBoxEx_GetItemImage

###Description###
Retrieves the index of the item's icon

###Syntax###
#include <GuiComboBoxEx.au3>
_GUICtrlComboBoxEx_GetItemImage ( $hWnd, $iIndex )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$iIndex
	Zero based item index
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	Zero based item image index
Failure:	-1
@@End@@


###Remarks###
None.
