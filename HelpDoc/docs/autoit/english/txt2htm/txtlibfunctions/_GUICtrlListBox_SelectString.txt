###User Defined Function###
_GUICtrlListBox_SelectString

###Description###
Searchs for an item that begins with the specified string

###Syntax###
#include <GuiListBox.au3>
_GUICtrlListBox_SelectString ( $hWnd, $sText [, $iIndex = -1] )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
$sText
	String that contains the string for which to search.
$iIndex
	[optional] Specifies the zero based index of the item before the first item to be searched. When the
	search reaches the bottom of the list box, it continues searching from the top of the list box back to the
	item specified by $iIndex. If $iIndex is –1, the entire list box is searched from the beginning.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	The zero based index of the selected item
Failure:	-1
@@End@@


###Remarks###
The list box is scrolled, if necessary, to bring the selected item into view.
Do not use this message with a list box that has the $LBS_MULTIPLESEL or the $LBS_EXTENDEDSEL styles.


###Related###
_GUICtrlListBox_FindString, _GUICtrlListBox_FindInText
