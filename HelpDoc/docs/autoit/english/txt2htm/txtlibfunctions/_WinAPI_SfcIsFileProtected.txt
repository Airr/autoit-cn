###User Defined Function###
_WinAPI_SfcIsFileProtected

###Description###

###Syntax###
#include <WinAPIFiles.au3>
_WinAPI_SfcIsFileProtected ( $sFile )


###Parameters###
@@ParamTable@@
$sFile
	The name of the file to test.
@@End@@

###ReturnValue###
