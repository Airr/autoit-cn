###User Defined Function###
_GDIPlus_RegionGetHRgn

###Description###
Creates a GDI region from a GDI+ Region object

###Syntax###
#include <GDIPlus.au3>
_GDIPlus_RegionGetHRgn ( $hRegion [, $hGraphics = 0] )


###Parameters###
@@ParamTable@@
$hRegion
	Pointer to a Region object
$hGraphics
	[optional] Pointer to a Graphics object that contains the world and page transformations required to
	calculate the device coordinates of the region
@@End@@

###ReturnValue###
Success: Handle to a GDI region (HRGN) object
Failure: 0
