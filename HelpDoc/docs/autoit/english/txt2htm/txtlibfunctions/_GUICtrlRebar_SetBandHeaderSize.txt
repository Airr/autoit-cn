###User Defined Function###
_GUICtrlRebar_SetBandHeaderSize

###Description###
Set the size of the band's header, in pixels

###Syntax###
#include <GuiReBar.au3>
_GUICtrlRebar_SetBandHeaderSize ( $hWnd, $iIndex, $iNewSize )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the rebar control
$iIndex
	0-based index of the band
$iNewSize
	New size of the band's header
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True.
Failure:	False.
@@End@@


###Remarks###
None.
