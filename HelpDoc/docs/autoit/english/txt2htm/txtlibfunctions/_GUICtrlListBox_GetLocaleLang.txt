###User Defined Function###
_GUICtrlListBox_GetLocaleLang

###Description###
Retrieves the current language identifier

###Syntax###
#include <GuiListBox.au3>
_GUICtrlListBox_GetLocaleLang ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
@@End@@

###ReturnValue###
Returns the language identifier of the list box.
