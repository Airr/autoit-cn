###User Defined Function###
_WinAPI_IsChild

###Description###
Tests whether a window is a child window of a specified parent window.

###Syntax###
#include <WinAPISys.au3>
_WinAPI_IsChild ( $hWnd, $hWndParent )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the window to be tested.
$hWndParent
	Handle to the parent window.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success 	True - The window is a child window of the specified parent window.
Failure 	False.
@@End@@
