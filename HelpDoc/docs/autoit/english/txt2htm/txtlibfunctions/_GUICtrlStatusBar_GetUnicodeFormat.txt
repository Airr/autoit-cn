###User Defined Function###
_GUICtrlStatusBar_GetUnicodeFormat

###Description###
Retrieves the Unicode character format flag

###Syntax###
#include <GuiStatusBar.au3>
_GUICtrlStatusBar_GetUnicodeFormat ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
@@End@@

###ReturnValue###
@@ReturnTable@@
True:	the control is using Unicode characters.
False:	the control is using ANSI characters.
@@End@@


###Remarks###
None.
