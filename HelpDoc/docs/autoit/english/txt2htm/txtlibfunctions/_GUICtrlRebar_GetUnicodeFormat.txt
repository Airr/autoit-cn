###User Defined Function###
_GUICtrlRebar_GetUnicodeFormat

###Description###
Retrieves the Unicode character format flag for the control

###Syntax###
#include <GuiReBar.au3>
_GUICtrlRebar_GetUnicodeFormat ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the rebar control
@@End@@

###ReturnValue###
@@ReturnTable@@
True:	the control is using Unicode characters.
False:	the control is using ANSI characters.
@@End@@


###Remarks###
None.
