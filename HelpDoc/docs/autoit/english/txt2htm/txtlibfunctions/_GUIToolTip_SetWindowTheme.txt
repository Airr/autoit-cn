###User Defined Function###
_GUIToolTip_SetWindowTheme

###Description###
Sets the visual style

###Syntax###
#include <GuiToolTip.au3>
_GUIToolTip_SetWindowTheme ( $hWnd, $sStyle )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the ToolTip control (returned by <a href ="_GUIToolTip_Create.htm">_GUIToolTip_Create.</a>)
$sStyle
	String that contains the ToolTip visual style to set
@@End@@

None.
