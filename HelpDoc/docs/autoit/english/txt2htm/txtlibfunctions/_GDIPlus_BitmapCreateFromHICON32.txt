###User Defined Function###
_GDIPlus_BitmapCreateFromHICON32

###Description###
Creates a  Bitmap object based on a 32bit icon

###Syntax###
#include <GDIPlus.au3>
_GDIPlus_BitmapCreateFromHICON32 ( $hIcon )


###Parameters###
@@ParamTable@@
$hIcon
	Handle to an icon
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	a handle to a new Bitmap object.
Failure:	0 and sets the @error flag to non-zero, @extended may contain GPSTATUS error code ($GPID_ERR*).
@@End@@


###Remarks###
When you are done with the Bitmap object, call <a href="_GDIPlus_ImageDispose.htm">_GDIPlus_ImageDispose()</a> to release the resources.


###Related###
_GDIPlus_ImageDispose, _WinAPI_LoadIcon, _WinAPI_LoadImage
