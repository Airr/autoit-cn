###User Defined Function###
_GUICtrlStatusBar_GetTextLength

###Description###
Retrieves the length of a part text

###Syntax###
#include <GuiStatusBar.au3>
_GUICtrlStatusBar_GetTextLength ( $hWnd, $iPart )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$iPart
	0-based part index
@@End@@

###ReturnValue###
Returns the length of part text.


###Remarks###
None.
