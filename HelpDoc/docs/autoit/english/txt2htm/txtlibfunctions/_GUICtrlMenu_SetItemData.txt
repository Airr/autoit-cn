###User Defined Function###
_GUICtrlMenu_SetItemData

###Description###
Sets the application defined value for a menu item

###Syntax###
#include <GuiMenu.au3>
_GUICtrlMenu_SetItemData ( $hMenu, $iItem, $iData [, $fByPos = True] )


###Parameters###
@@ParamTable@@
$hMenu
	Menu handle
$iItem
	Identifier or position of the menu item
$iData
	Application defined value
$fByPos
	[optional] Menu identifier flag:
		True - $iItem is a 0-based item position
		False - $iItem is a menu item identifier
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True.
Failure:	False.
@@End@@


###Remarks###
None.
