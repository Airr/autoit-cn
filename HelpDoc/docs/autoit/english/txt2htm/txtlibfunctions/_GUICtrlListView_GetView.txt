###User Defined Function###
_GUICtrlListView_GetView

###Description###
Retrieves the current view of the control

###Syntax###
#include <GuiListView.au3>
_GUICtrlListView_GetView ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	The current view:
		0 - Details
		1 - Large Icon
		2 - List
		3 - Small Icon
		4 - Tile
Failure:	-1.
@@End@@


###Remarks###
None.
