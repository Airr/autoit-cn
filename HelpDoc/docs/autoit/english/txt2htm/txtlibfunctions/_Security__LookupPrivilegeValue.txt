###User Defined Function###
_Security__LookupPrivilegeValue

###Description###
Retrieves the locally unique identifier (LUID) for a privilege value in form of 64bit integer

###Syntax###
#include <Security.au3>
_Security__LookupPrivilegeValue ( $sSystem, $sName )


###Parameters###
@@ParamTable@@
$sSystem
	Specifies the name of the system on which the privilege name is retrieved. If blank, the
	function attempts to find the privilege name on the local system.
$sName
	Specifies the name of the privilege
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	LUID by which the privilege is known
Failure:	0
@@End@@

None.
