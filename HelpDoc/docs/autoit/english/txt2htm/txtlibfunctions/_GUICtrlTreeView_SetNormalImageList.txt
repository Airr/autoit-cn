###User Defined Function###
_GUICtrlTreeView_SetNormalImageList

###Description###
Sets the normal image list for the control

###Syntax###
#include <GuiTreeView.au3>
_GUICtrlTreeView_SetNormalImageList ( $hWnd, $hImageList )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
$hImageList
	Handle to the image list. If 0, all images are removed
@@End@@

###ReturnValue###
Returns the handle to the previous image list


###Remarks###
None.
