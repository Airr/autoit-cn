###User Defined Function###
_GUICtrlToolbar_GetHotImageList

###Description###
Retrieves the hot button image list

###Syntax###
#include <GuiToolbar.au3>
_GUICtrlToolbar_GetHotImageList ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	Handle to the hot image list
Failure:	0
@@End@@


###Remarks###
A button is considered hot when the cursor is over it
