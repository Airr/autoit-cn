###User Defined Function###
_GUICtrlIpAddress_SetArray

###Description###
Sets the address in the IP address control

###Syntax###
#include <GuiIPAddress.au3>
_GUICtrlIpAddress_SetArray ( $hWnd, $aAddress )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$aAddress
	Array formatted as follows:
	[0] - 1st address field
	[1] - 2nd address field
	[2] - 3rd address field
	[3] - 4th address field
@@End@@

###ReturnValue###
None.


###Remarks###
None.
