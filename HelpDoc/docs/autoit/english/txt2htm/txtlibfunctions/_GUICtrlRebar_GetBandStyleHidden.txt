###User Defined Function###
_GUICtrlRebar_GetBandStyleHidden

###Description###
Determine if flag is set

###Syntax###
#include <GuiReBar.au3>
_GUICtrlRebar_GetBandStyleHidden ( $hWnd, $iIndex )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the rebar control
$iIndex
	Zero-based index of the band
@@End@@

###ReturnValue###
True: Flag is set (The band will not be visible)
False: Flag not set


###Remarks###
None.


###Related###
_GUICtrlRebar_SetBandStyleHidden, _GUICtrlRebar_GetBandStyle
