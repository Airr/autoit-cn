###User Defined Function###
_GUICtrlMenu_IsMenu

###Description###
Determines whether a handle is a menu handle

###Syntax###
#include <GuiMenu.au3>
_GUICtrlMenu_IsMenu ( $hMenu )


###Parameters###
@@ParamTable@@
$hMenu
	Handle to be tested
@@End@@

###ReturnValue###
True: Handle is a menu handle
False: Handle is not a menu handle

None.
