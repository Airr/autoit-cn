###User Defined Function###
_GUICtrlHeader_SetItemWidth

###Description###
Sets the item's width

###Syntax###
#include <GuiHeader.au3>
_GUICtrlHeader_SetItemWidth ( $hWnd, $iIndex, $iWidth )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$iIndex
	0-based item index
$iWidth
	New width for item
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True.
Failure:	False.
@@End@@


###Remarks###
None.
