###User Defined Function###
_GUICtrlStatusBar_GetText

###Description###
Retrieves the text from the specified part

###Syntax###
#include <GuiStatusBar.au3>
_GUICtrlStatusBar_GetText ( $hWnd, $iPart )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$iPart
	Zero based part index
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	Part text
Failure:	set @error
@@End@@


###Remarks###
None.
