###User Defined Function###
_GUICtrlComboBoxEx_GetUnicode

###Description###
Retrieves if control is using Unicode

###Syntax###
#include <GuiComboBoxEx.au3>
_GUICtrlComboBoxEx_GetUnicode ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
@@End@@

###ReturnValue###
@@ReturnTable@@
True:	Using Unicode.
False:	Not using Unicode.
@@End@@
