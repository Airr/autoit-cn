###User Defined Function###
_GUIToolTip_DelTool

###Description###
Removes a tool

###Syntax###
#include <GuiToolTip.au3>
_GUIToolTip_DelTool ( $hWnd, $hTool [, $iID = 0] )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$hTool
	Handle to the window that contains the tool
$iID
	[optional] Identifier of the tool
@@End@@

###ReturnValue###
None.


###Remarks###
None.
