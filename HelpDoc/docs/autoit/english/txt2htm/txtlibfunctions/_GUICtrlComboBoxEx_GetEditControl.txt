###User Defined Function###
_GUICtrlComboBoxEx_GetEditControl

###Description###
Gets the handle to the edit control portion of a ComboBoxEx control

###Syntax###
#include <GuiComboBoxEx.au3>
_GUICtrlComboBoxEx_GetEditControl ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
@@End@@

###ReturnValue###
Returns the handle to the edit control within the ComboBoxEx control if it uses the $CBS_DROPDOWN style.
