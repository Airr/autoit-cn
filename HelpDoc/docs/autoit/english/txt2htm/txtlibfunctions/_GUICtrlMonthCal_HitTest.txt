###User Defined Function###
_GUICtrlMonthCal_HitTest

###Description###
Determines which portion of a month calendar control is at a given point

###Syntax###
#include <GuiMonthCal.au3>
_GUICtrlMonthCal_HitTest ( $hWnd, $iX, $iY )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
$iX
	X position to test
$iY
	Y position to test
@@End@@

###ReturnValue###
Returns a $tagMCHITTESTINFO structure.


###Remarks###
None.
