###User Defined Function###
_WinAPI_SetFont

###Description###
Sets a window font

###Syntax###
#include <WinAPI.au3>
_WinAPI_SetFont ( $hWnd, $hFont [, $fRedraw = True] )


###Parameters###
@@ParamTable@@
$hWnd
	Window handle
$hFont
	Font handle
$fRedraw
	[optional] True to redraw the control
@@End@@

###ReturnValue###
None.


###Remarks###
None.
