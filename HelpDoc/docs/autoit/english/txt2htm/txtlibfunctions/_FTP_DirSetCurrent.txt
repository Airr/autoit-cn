###User Defined Function###
_FTP_DirSetCurrent

###Description###
Set Current Directory on an FTP server.

###Syntax###
#include <FTPEx.au3>
_FTP_DirSetCurrent ( $hFTPSession, $sRemote )


###Parameters###
@@ParamTable@@
$hFTPSession
	as returned by _FTP_Connect().
$sRemote
	The Directory to be set.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	1
Failure:	0  and sets @error
@@End@@


###Remarks###
None.


###Related###
_FTP_Connect
