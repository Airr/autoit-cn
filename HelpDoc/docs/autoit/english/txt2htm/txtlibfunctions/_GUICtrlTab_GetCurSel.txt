###User Defined Function###
_GUICtrlTab_GetCurSel

###Description###
Determines the currently selected tab

###Syntax###
#include <GuiTab.au3>
_GUICtrlTab_GetCurSel ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	The zero based index of the selected tab
Failure:	-1
@@End@@


###Remarks###
None.
