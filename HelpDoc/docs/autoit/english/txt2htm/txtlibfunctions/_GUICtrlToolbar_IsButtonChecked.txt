###User Defined Function###
_GUICtrlToolbar_IsButtonChecked

###Description###
Indicates whether the specified button is checked

###Syntax###
#include <GuiToolbar.au3>
_GUICtrlToolbar_IsButtonChecked ( $hWnd, $iCommandID )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$iCommandID
	Button command ID
@@End@@

###ReturnValue###
@@ReturnTable@@
True:	the button is checked.
False:	the button is not checked.
@@End@@
