###User Defined Function###
_GDIPlus_ImageGetHeight

###Description###
Get the image height

###Syntax###
#include <GDIPlus.au3>
_GDIPlus_ImageGetHeight ( $hImage )


###Parameters###
@@ParamTable@@
$hImage
	Handle to an image object
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	the image height, in pixels.
Failure:	-1 and sets the @error flag to non-zero, @extended may contain GPSTATUS ($GPID_ERR*).
@@End@@


###Remarks###
None.
