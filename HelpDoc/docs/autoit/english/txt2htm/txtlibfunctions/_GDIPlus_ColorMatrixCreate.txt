###User Defined Function###
_GDIPlus_ColorMatrixCreate

###Description###
Creates and initializes an identity color matrix

###Syntax###
#include <GDIPlus.au3>
_GDIPlus_ColorMatrixCreate (  )


###Parameters###
None.

###ReturnValue###
Returns a $tagGDIPCOLORMATRIX structure.


###Remarks###
An identity color matrix is a color matrix that does nothing. No color transformation, shearing, scaling, ...


###Related###
$tagGDIPCOLORMATRIX
