###User Defined Function###
_GUICtrlStatusBar_IsSimple

###Description###
Checks a status bar control to determine if it is in simple mode

###Syntax###
#include <GuiStatusBar.au3>
_GUICtrlStatusBar_IsSimple ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
@@End@@

###ReturnValue###
@@ReturnTable@@
True:	the status bar is in simple mode.
False:	the status bar is not in simple mode.
@@End@@


###Remarks###
None.
