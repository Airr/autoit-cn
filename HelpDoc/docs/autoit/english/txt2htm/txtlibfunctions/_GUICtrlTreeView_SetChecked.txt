###User Defined Function###
_GUICtrlTreeView_SetChecked

###Description###
Sets whether a item has it's checkbox checked or not

###Syntax###
#include <GuiTreeView.au3>
_GUICtrlTreeView_SetChecked ( $hWnd, $hItem [, $fCheck = True] )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
$hItem
	Handle to the item
$fCheck
	[optional] Value to set checked state to:
	True - Checked
	False - Not checked
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True
Failure:	False
@@End@@


###Remarks###
None.
