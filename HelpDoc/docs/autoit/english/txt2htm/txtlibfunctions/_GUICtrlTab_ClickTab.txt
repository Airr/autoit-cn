###User Defined Function###
_GUICtrlTab_ClickTab

###Description###
Clicks a tab

###Syntax###
#include <GuiTab.au3>
_GUICtrlTab_ClickTab ( $hWnd, $iIndex [, $sButton = "left" [, $fMove = False [, $iClicks = 1 [, $iSpeed = 1]]]] )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
$iIndex
	Specifies the 0-based index of the item
$sButton
	[optional] Button to click with
$fMove
	[optional] If True, the mouse will be moved. If False, the mouse does not move.
$iClicks
	[optional] Number of clicks
$iSpeed
	[optional] Mouse movement speed
@@End@@

###ReturnValue###
None.

None.
