###User Defined Function###
_WinAPI_GetActiveWindow

###Description###
Retrieves the window handle to the active window attached to the calling process's message queue.

###Syntax###
#include <WinAPISys.au3>

###Parameters###
None

###ReturnValue###
@@ReturnTable@@

Failure 	0.
@@End@@
