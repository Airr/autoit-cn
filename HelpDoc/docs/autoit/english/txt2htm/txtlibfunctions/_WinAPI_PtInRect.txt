###User Defined Function###
_WinAPI_PtInRect

###Description###
Determines whether the specified point lies within the specified rectangle

###Syntax###
#include <WinAPI.au3>
_WinAPI_PtInRect ( ByRef $tRect, ByRef $tPoint )


###Parameters###
@@ParamTable@@
$tRect
	$tagRECT structure that contains the specified rectangle
$tPoint
	$tagPOINT structure that contains the specified point
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True, point lies within the rectangle
Failure:	False, point does not lie within the rectangle
@@End@@


###Remarks###
None.


###Related###
$tagRECT, $tagPOINT
