###User Defined Function###
_GUICtrlListView_GetItemEx

###Description###
Retrieves some or all of an item's attributes

###Syntax###
#include <GuiListView.au3>
_GUICtrlListView_GetItemEx ( $hWnd, ByRef $tItem )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
$tItem
	$tagLVITEM structure that specifies the information to retrieve
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True.
Failure:	False.
@@End@@


###Remarks###
None.
