###User Defined Function###
_WinAPI_IsWindow

###Description###
Determines whether the specified window handle identifies an existing window

###Syntax###
#include <WinAPI.au3>
_WinAPI_IsWindow ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to be tested
@@End@@

###ReturnValue###
@@ReturnTable@@
True:	window is existing.
False:	window is not existing.
@@End@@
