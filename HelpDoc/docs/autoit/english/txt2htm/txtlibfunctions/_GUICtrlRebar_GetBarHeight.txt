###User Defined Function###
_GUICtrlRebar_GetBarHeight

###Description###
Retrieves the height of the rebar control

###Syntax###
#include <GuiReBar.au3>
_GUICtrlRebar_GetBarHeight ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the rebar control
@@End@@

###ReturnValue###
Returns the Value that represents the height, in pixels, of the control.
