###User Defined Function###
_GUICtrlSlider_GetPageSize

###Description###
Retrieves the number of logical positions the slider moves

###Syntax###
#include <GuiSlider.au3>
_GUICtrlSlider_GetPageSize ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
@@End@@

###ReturnValue###
Success: The page size for the slider


###Remarks###
None.
