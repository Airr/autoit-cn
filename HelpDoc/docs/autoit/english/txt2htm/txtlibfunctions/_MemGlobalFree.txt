###User Defined Function###
_MemGlobalFree

###Description###
Frees the specified global memory object and invalidates its handle

###Syntax###
#include <Memory.au3>
_MemGlobalFree ( $hMemory )


###Parameters###
@@ParamTable@@
$hMemory
	Handle to the global memory object
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	0.
Failure:	$hMemory.
@@End@@


###Remarks###
None.
