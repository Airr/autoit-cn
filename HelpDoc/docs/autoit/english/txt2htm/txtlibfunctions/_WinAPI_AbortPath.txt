###User Defined Function###
_WinAPI_AbortPath

###Description###
Closes and discards any paths in the specified device context

###Syntax###
#include <WinAPIGdi.au3>
_WinAPI_AbortPath ( $hDC )


###Parameters###
@@ParamTable@@
$hDC
	Handle to the device context from which a path will be discarded.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True
Failure:	False
@@End@@
