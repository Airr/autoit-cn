###User Defined Function###
_ColorGetRed

###Description###
Returns the red component of a given color

###Syntax###
#include <Color.au3>
_ColorGetRed ( $iColor )


###Parameters###
@@ParamTable@@
$iColor
	The RGB color to work with (0x00RRGGBB).
@@End@@

###ReturnValue###
Returns the component color in the range 0-255.


###Remarks###
None.
