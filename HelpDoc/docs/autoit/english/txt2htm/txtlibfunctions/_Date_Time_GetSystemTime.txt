###User Defined Function###
_Date_Time_GetSystemTime

###Description###
Retrieves the current system date and time expressed in UTC

###Syntax###
#include <Date.au3>
_Date_Time_GetSystemTime (  )


###Parameters###
None.

###ReturnValue###
Returns a $tagSYSTEMTIME structure with the current system date and time.
