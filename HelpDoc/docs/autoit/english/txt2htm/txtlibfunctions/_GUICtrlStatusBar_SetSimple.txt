###User Defined Function###
_GUICtrlStatusBar_SetSimple

###Description###
Specifies whether a status window displays simple text or displays all window parts

###Syntax###
#include <GuiStatusBar.au3>
_GUICtrlStatusBar_SetSimple ( $hWnd [, $fSimple = True] )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$fSimple
	[optional] Sets the display of the windows
	True - The window displays simple text
	False - The window displays multiple parts
@@End@@

###ReturnValue###
None.


###Remarks###
None.
