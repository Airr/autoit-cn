###User Defined Function###
_GUICtrlEdit_SetRECTEx

###Description###

###Syntax###
#include <GuiEdit.au3>
_GUICtrlEdit_SetRECTEx ( $hWnd, $tRect )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
$tRect
	$tagRECT structure that contains formatting rectangle of an edit control
@@End@@

###ReturnValue###
None.
