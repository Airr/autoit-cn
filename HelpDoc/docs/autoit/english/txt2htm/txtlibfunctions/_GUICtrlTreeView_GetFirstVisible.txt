###User Defined Function###
_GUICtrlTreeView_GetFirstVisible

###Description###
Retrieves the first visible item in the control

###Syntax###
#include <GuiTreeView.au3>
_GUICtrlTreeView_GetFirstVisible ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	the handle of the first visible item.
Failure:	0.
@@End@@
