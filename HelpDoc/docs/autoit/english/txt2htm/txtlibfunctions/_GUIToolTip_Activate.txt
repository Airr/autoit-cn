###User Defined Function###
_GUIToolTip_Activate

###Description###
Activates or deactivates the control

###Syntax###
#include <GuiToolTip.au3>
_GUIToolTip_Activate ( $hWnd [, $fActivate = True] )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$fActivate
	[optional] Activation flag:
	True - Activate ToolTip
	False - Deactivate ToolTip
@@End@@

###ReturnValue###
None.


###Remarks###
None.
