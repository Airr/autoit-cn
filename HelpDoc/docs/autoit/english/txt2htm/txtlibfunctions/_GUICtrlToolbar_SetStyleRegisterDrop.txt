###User Defined Function###
_GUICtrlToolbar_SetStyleRegisterDrop

###Description###
Sets whether the control generates TBN_GETOBJECT notification messages

###Syntax###
#include <GuiToolbar.au3>
_GUICtrlToolbar_SetStyleRegisterDrop ( $hWnd, $fState )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$fState
	True to set, false to unset
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True
Failure:	False
@@End@@

None.
