###User Defined Function###
_GUICtrlHeader_GetImageList

###Description###
Retrieves the handle to the image list

###Syntax###
#include <GuiHeader.au3>
_GUICtrlHeader_GetImageList ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	the handle to the image list set for the header control.
Failure:	0.
@@End@@
