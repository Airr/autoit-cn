###User Defined Function###
_GUICtrlHeader_GetImageList

###Description###

###Syntax###
#include <GuiHeader.au3>
_GUICtrlHeader_GetImageList ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	A handle to the image list set for the header control

@@End@@
