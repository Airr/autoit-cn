###User Defined Function###
_GUICtrlHeader_SetItem

###Description###
Sets information about an item

###Syntax###
#include <GuiHeader.au3>
_GUICtrlHeader_SetItem ( $hWnd, $iIndex, ByRef $tItem )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$iIndex
	0-based item index
$tItem
	DllStructCreate($tagHDITEM) structure
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True.
Failure:	False.
@@End@@
