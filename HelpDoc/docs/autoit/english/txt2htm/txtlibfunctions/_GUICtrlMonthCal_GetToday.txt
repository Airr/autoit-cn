###User Defined Function###
_GUICtrlMonthCal_GetToday

###Description###
Retrieves the date information for the date specified as "today"

###Syntax###
#include <GuiMonthCal.au3>
_GUICtrlMonthCal_GetToday ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
@@End@@

###ReturnValue###
Returns a $tagSYSTEMTIME structure.

None.
