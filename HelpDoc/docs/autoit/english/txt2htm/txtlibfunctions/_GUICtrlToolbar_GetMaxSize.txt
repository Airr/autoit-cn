###User Defined Function###
_GUICtrlToolbar_GetMaxSize

###Description###
Retrieves the total size of all of the visible buttons and separators

###Syntax###
#include <GuiToolbar.au3>
_GUICtrlToolbar_GetMaxSize ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
@@End@@

###ReturnValue###
Returns an array with the following format:
	[0] - Maximum width
	[1] - Maximum height


###Remarks###
None.
