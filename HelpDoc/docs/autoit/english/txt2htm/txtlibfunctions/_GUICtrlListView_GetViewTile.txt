###User Defined Function###
_GUICtrlListView_GetViewTile

###Description###
Determines whether the view mode is in tile mode

###Syntax###
#include <GuiListView.au3>
_GUICtrlListView_GetViewTile ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
@@End@@

###ReturnValue###
@@ReturnTable@@
True:	view is in tile mode.
False:	view is not in tile mode.
@@End@@
