###User Defined Function###
_ArrayInsert

###Description###
Add a new value at the specified position

###Syntax###
#include <Array.au3>
_ArrayInsert ( ByRef $aArray, $iElement [, $vValue = ""] )


###Parameters###
@@ParamTable@@
$aArray
	Array to modify
$iElement
	Position to insert item at
$vValue
	[optional] Value of item to insert
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	the new size of the array.
Failure:	0 and sets the @error flag to non-zero.
@error:	1 - $aArray is not an array
	2 - $aArray is not a 1 dimensional array
	3 - $iElement is outside array bounds
@@End@@

None.
