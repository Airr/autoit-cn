###User Defined Function###
_FTP_DirGetCurrent

###Description###
Get Current Directory on an FTP server.

###Syntax###
#include <FTPEx.au3>
_FTP_DirGetCurrent ( $hFTPSession )


###Parameters###
@@ParamTable@@
$hFTPSession
	as returned by _FTP_Connect().
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	the directory name.
Failure:	0 and sets the @error flag to non-zero.
@@End@@


###Remarks###
None.


###Related###
_FTP_Connect
