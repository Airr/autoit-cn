###User Defined Function###
_GDIPlus_EncodersGetCount

###Description###
Get the number of available image encoders

###Syntax###
#include <GDIPlus.au3>
_GDIPlus_EncodersGetCount (  )


###Parameters###
None.

###ReturnValue###
@@ReturnTable@@
Success:	the number of image encoders.
Failure:	-1 and sets the @error flag to non-zero, @extended may contain GPSTATUS ($GPID_ERR*).
@@End@@
