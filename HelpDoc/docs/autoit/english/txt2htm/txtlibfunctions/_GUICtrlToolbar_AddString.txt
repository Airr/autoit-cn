###User Defined Function###
_GUICtrlToolbar_AddString

###Description###
Adds a new string to the toolbar's string pool

###Syntax###
#include <GuiToolbar.au3>
_GUICtrlToolbar_AddString ( $hWnd, $sString )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$sString
	String to add
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	Zero based index of the new string
Failure:	-1
@@End@@


###Remarks###
None.
