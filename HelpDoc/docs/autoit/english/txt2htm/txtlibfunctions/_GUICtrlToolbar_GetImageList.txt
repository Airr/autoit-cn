###User Defined Function###
_GUICtrlToolbar_GetImageList

###Description###
Retrieves the default state image list

###Syntax###
#include <GuiToolbar.au3>
_GUICtrlToolbar_GetImageList ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	the handle to the image list.
Failure:	0.
@@End@@


###Remarks###
A toolbar control uses this image list to display buttons when they are not hot or disabled.
