###User Defined Function###
_GUICtrlTab_GetItemImage

###Description###
Retrieves the image index for a tab

###Syntax###
#include <GuiTab.au3>
_GUICtrlTab_GetItemImage ( $hWnd, $iIndex )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
$iIndex
	0-based item index
@@End@@

###ReturnValue###
Returns the 0-based image index of the tab.


###Remarks###
None.
