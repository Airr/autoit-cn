###User Defined Function###
_GUIImageList_GetIconHeight

###Description###
Retrieves the height of the images in an image list

###Syntax###
#include <GuiImageList.au3>
_GUIImageList_GetIconHeight ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the imagelist
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	Height, in pixels, of each image
Failure:	0
@@End@@


###Remarks###
None.
