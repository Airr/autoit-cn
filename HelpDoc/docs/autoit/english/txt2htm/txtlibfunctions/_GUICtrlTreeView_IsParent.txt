###User Defined Function###
_GUICtrlTreeView_IsParent

###Description###
Indicates whether one item is the parent of another item

###Syntax###
#include <GuiTreeView.au3>
_GUICtrlTreeView_IsParent ( $hWnd, $hParent, $hItem )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
$hParent
	Handle to parent item
$hItem
	Handle to the item to test
@@End@@

###ReturnValue###
@@ReturnTable@@
True:	$hParent is a parent of $hItem.
False:	$hParent is not a parent of $hItem.
@@End@@
