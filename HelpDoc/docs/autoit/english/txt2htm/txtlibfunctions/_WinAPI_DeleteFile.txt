###User Defined Function###
_WinAPI_DeleteFile

###Description###
Deletes an existing file.

###Syntax###
#include <WinAPIFiles.au3>
_WinAPI_DeleteFile ( $sFile )


###Parameters###
@@ParamTable@@
$sFile
	The name of the file to be deleted.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success 	True.
Failure 	False, call _WinAPI_GetLastError() to get extended error information.
@@End@@
