###User Defined Function###
_GUICtrlListView_GetExtendedListViewStyle

###Description###
Retrieves the extended styles that are currently in use

###Syntax###
#include <GuiListView.au3>
_GUICtrlListView_GetExtendedListViewStyle ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
@@End@@

###ReturnValue###
Returns a DWORD that represents the styles currently in use for a given list-view


###Remarks###
None.
