###User Defined Function###
_GDIPlus_PathClone

###Description###
Clones a path

###Syntax###
#include <GDIPlus.au3>
_GDIPlus_PathClone ( $hPath )


###Parameters###
@@ParamTable@@
$hPath
	Pointer to a GraphicsPath object to be cloned
@@End@@

###ReturnValue###
Success: Pointer to a new cloned GraphicsPath object
Failure: 0


###Remarks###
After you are done with the object, call _GDIPlus_PathDispose to release the object resources
