###User Defined Function###
_GDIPlus_PathClone

###Description###
Clones a path

###Syntax###
#include <GDIPlus.au3>
_GDIPlus_PathClone ( $hPath )


###Parameters###
@@ParamTable@@
$hPath
	Pointer to a GraphicsPath object to be cloned
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	a pointer to a new cloned GraphicsPath object.
Failure:	0 and sets the @error flag to non-zero, @extended may contain GPSTATUS ($GPID_ERR*).
@@End@@


###Remarks###
After you are done with the object, call _GDIPlus_PathDispose() to release the object resources.
