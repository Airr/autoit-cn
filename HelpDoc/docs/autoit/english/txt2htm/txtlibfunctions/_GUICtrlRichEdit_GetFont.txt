###User Defined Function###
_GUICtrlRichEdit_GetFont

###Description###
Gets the font attributes of a selection or, if no selection, at the insertion point

###Syntax###
#include <GuiRichEdit.au3>
_GUICtrlRichEdit_GetFont ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	an array containing values:
	[0] - point size
	0 if sizes are mixed in selection
	[1] - the name of the font
	"" if fonts are mixed in selection
	[2] - the character set
	$ANSI_CHARSET - 0
	$BALTIC_CHARSET - 186
	$CHINESEBIG5_CHARSET - 136
	$EASTEUROPE_CHARSET - 238
	$GB2312_CHARSET - 134
	$GREEK_CHARSET - 161
	$HANGEUL_CHARSET - 129
	$MAC_CHARSET - 77
	$OEM_CHARSET - 255
	$RUSSIAN_CHARSET - 204
	$SHIFTJIS_CHARSET - 128
	$SYMBOL_CHARSET - 2
	$TURKISH_CHARSET - 162
	$VIETNAMESE_CHARSET - 163
Failure:	sets @error
@error:	101 - $hWnd is not a handle
@@End@@


###Remarks###
None.
