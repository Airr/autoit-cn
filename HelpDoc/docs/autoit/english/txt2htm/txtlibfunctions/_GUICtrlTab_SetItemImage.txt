###User Defined Function###
_GUICtrlTab_SetItemImage

###Description###
Sets the image of a tab

###Syntax###
#include <GuiTab.au3>
_GUICtrlTab_SetItemImage ( $hWnd, $iIndex, $iImage )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
$iIndex
	0-based item index
$iImage
	0-based item image
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True.
Failure:	False.
@@End@@

None.
