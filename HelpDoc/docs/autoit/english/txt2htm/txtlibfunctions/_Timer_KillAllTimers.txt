###User Defined Function###
_Timer_KillAllTimers

###Description###
Destroys all the timers

###Syntax###
#include <Timers.au3>
_Timer_KillAllTimers ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the window associated with the timers.
	This value must be the same as the $hWnd value passed to the <a href="_Timer_SetTimer.htm">_Timer_SetTimer()</a> function that created the timer
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True
Failure:	False
@@End@@


###Remarks###
The <a href="_Timer_KillAllTimers.htm">_Timer_KillAllTimers()</a> function does not remove WM_TIMER messages already posted to the message queue


###Related###
_Timer_KillTimer, _Timer_SetTimer
