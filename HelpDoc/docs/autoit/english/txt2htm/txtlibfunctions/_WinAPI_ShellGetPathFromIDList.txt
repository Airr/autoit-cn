###User Defined Function###
_WinAPI_ShellGetPathFromIDList

###Description###
Converts an item identifier list to a file system path

###Syntax###
#include <WinAPIShellEx.au3>
_WinAPI_ShellGetPathFromIDList ( $PIDL )


###Parameters###
@@ParamTable@@
$PIDL
	The address of an item identifier list that specifies a file or directory location relative to the
	root of the namespace (the desktop).
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	The file system path.
Failure:	Empty string.
@@End@@
