###User Defined Function###
_GUICtrlHeader_GetItemImage

###Description###
Retrieves the index of an image within the image list

###Syntax###
#include <GuiHeader.au3>
_GUICtrlHeader_GetItemImage ( $hWnd, $iIndex )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$iIndex
	0-based item index
@@End@@

###ReturnValue###
Returns the 0-based index of the image.


###Remarks###
None.
