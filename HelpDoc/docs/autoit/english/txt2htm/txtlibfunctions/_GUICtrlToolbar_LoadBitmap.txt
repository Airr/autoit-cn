###User Defined Function###
_GUICtrlToolbar_LoadBitmap

###Description###
Adds a bitmap to the image list from a file

###Syntax###
#include <GuiToolbar.au3>
_GUICtrlToolbar_LoadBitmap ( $hWnd, $sFileName )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to toolbar
$sFileName
	Fully qualified path to bitmap file
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	the 0-based index of the new image.
Failure:	-1.
@@End@@


###Remarks###
None.
