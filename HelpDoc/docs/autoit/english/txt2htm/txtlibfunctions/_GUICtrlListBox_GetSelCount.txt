###User Defined Function###
_GUICtrlListBox_GetSelCount

###Description###
Retrieves the total number of selected items

###Syntax###
#include <GuiListBox.au3>
_GUICtrlListBox_GetSelCount ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	the count of selected items.
Failure:	-1 if no items are selected.
@@End@@
