###User Defined Function###
_GDIPlus_EncodersGetCLSID

###Description###
Return the encoder CLSID for a specific image file type

###Syntax###
#include <GDIPlus.au3>
_GDIPlus_EncodersGetCLSID ( $sFileExt )


###Parameters###
@@ParamTable@@
$sFileExt
	File extension to search for (BMP, JPG, TIF, etc.)
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	CLSID of the encoder in string format
Failure:	Empty string
@@End@@


###Remarks###
None.
