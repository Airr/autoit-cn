###Structure Name###
$tagNMLVSCROLL

###Description###
Provides information about a scrolling operation

###Syntax###
Global Const $tagNMLVSCROLL = $tagNMHDR & ";int DX;int DY"


###Fields###
@@ParamTable@@
$tagNMHDR
	Contains information about a notification message
DX
	Specifies in pixels the horizontal position where a scrolling operation should begin or end
DY
	Specifies in pixels the vertical position where a scrolling operation should begin or end
@@End@@

###Remarks###
None.
