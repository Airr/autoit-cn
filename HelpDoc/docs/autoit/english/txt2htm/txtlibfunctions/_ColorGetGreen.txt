###User Defined Function###
_ColorGetGreen

###Description###
Returns the green component of a given color.

###Syntax###
#include <Color.au3>
_ColorGetGreen ( $nColor )


###Parameters###
@@ParamTable@@
$nColor
	The RGB color to work with (0x00RRGGBB).
@@End@@

###ReturnValue###
Returns the component color in the range 0-255.


###Remarks###
None.

_ColorGetBlue, _ColorGetRed, _ColorGetRGB
