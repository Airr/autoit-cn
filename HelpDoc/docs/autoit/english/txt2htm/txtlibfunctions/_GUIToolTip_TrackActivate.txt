###User Defined Function###
_GUIToolTip_TrackActivate

###Description###
Activates or deactivates a tracking ToolTip

###Syntax###
#include <GuiToolTip.au3>
_GUIToolTip_TrackActivate ( $hWnd [, $fActivate = True [, $hTool = 0 [, $iID = 0]]] )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$fActivate
	[optional] True to activate, False to deactivate
$hTool
	[optional] Handle to the window that contains the tool
$iID
	[optional] Application-defined identifier of the tool
@@End@@

###ReturnValue###
None.


###Remarks###
None.
