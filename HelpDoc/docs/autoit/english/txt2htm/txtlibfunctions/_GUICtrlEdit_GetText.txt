###User Defined Function###
_GUICtrlEdit_GetText

###Description###
Get the text from the edit control

###Syntax###
#include <GuiEdit.au3>
_GUICtrlEdit_GetText ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	a string from the edit control.
Failure:	an empty string.
@@End@@
