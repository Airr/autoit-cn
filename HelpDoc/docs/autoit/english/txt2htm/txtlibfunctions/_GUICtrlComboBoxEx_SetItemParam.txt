###User Defined Function###
_GUICtrlComboBoxEx_SetItemParam

###Description###

###Syntax###
#include <GuiComboBoxEx.au3>
_GUICtrlComboBoxEx_SetItemParam ( $hWnd, $iIndex, $iParam )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$iIndex
	Zero based index of the item
$iParam
	Item specific value
@@End@@
