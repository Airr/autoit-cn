###User Defined Function###
_GUICtrlMonthCal_GetTodayStr

###Description###
Retrieves the date information for the date specified as "today" in string format

###Syntax###
#include <GuiMonthCal.au3>
_GUICtrlMonthCal_GetTodayStr ( $hWnd [, $sFormat = "%02d/%02d/%04d"] )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
$sFormat
	[optional] StringFormat string used to format the date
@@End@@

###ReturnValue###
Success: Date in the specified format


###Remarks###
None.
