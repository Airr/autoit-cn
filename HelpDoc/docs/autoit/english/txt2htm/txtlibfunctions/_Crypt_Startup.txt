###User Defined Function###
_Crypt_Startup

###Description###
Initialize the Crypt library

###Syntax###
#include <Crypt.au3>
_Crypt_Startup (  )


###Parameters###
None.

###ReturnValue###
@@ReturnTable@@
Success:	True.
Failure:	False and sets the @error flag to non-zero.
@error:	1 - Failed to open Advapi32.dll
	2 - Failed to acquire crypt context
@@End@@


###Remarks###
Calling this function is optional.
