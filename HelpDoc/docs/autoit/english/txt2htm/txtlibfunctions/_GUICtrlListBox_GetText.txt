###User Defined Function###
_GUICtrlListBox_GetText

###Description###
Returns the item (string) at the specified index

###Syntax###
#include <GuiListBox.au3>
_GUICtrlListBox_GetText ( $hWnd, $iIndex )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
$iIndex
	Specifies the 0-based index of the string to retrieve
@@End@@

###ReturnValue###
Returns the item (string).


###Remarks###
None.
