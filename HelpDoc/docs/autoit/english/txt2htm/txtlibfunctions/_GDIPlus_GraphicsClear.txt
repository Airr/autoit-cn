###User Defined Function###
_GDIPlus_GraphicsClear

###Description###
Clears a Graphics object to a specified color

###Syntax###
#include <GDIPlus.au3>
_GDIPlus_GraphicsClear ( $hGraphics [, $iARGB = 0xFF000000] )


###Parameters###
@@ParamTable@@
$hGraphics
	Handle to a Graphics object
$iARGB
	[optional] Alpha, Red, Green and Blue components of color
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True.
Failure:	False and sets the @error flag to non-zero, @extended may contain GPSTATUS ($GPID_ERR*).
@@End@@

None.
