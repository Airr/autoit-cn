###User Defined Function###
_GDIPlus_GraphicsDispose

###Description###
Release a Graphics object

###Syntax###
#include <GDIPlus.au3>
_GDIPlus_GraphicsDispose ( $hGraphics )


###Parameters###
@@ParamTable@@
$hGraphics
	Handle to a Graphics object
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True.
Failure:	False and sets the @error flag to non-zero, @extended may contain GPSTATUS ($GPID_ERR*).
@@End@@


###Remarks###
None.
