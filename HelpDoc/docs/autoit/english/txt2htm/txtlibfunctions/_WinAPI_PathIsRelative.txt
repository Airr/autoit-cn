###User Defined Function###
_WinAPI_PathIsRelative

###Description###
Searches a path and determines if it is relative.

###Syntax###
#include <WinAPIShPath.au3>
_WinAPI_PathIsRelative ( $sPath )


###Parameters###
@@ParamTable@@
$sPath
	The path to be searched.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success 	True - The path is relative.
Failure 	False.
@@End@@
