###User Defined Function###
_WinAPI_MAKELANGID

###Description###
Construct language id from a primary language id and a sublanguage id

###Syntax###
#include <WinAPI.au3>
_WinAPI_MAKELANGID ( $lgidPrimary, $lgidSub )


###Parameters###
@@ParamTable@@
$lgidPrimary
	Primary Language id
$lgidSub
	Sub-Language id
@@End@@

###ReturnValue###
Returns the Language identifier


###Remarks###
None.
