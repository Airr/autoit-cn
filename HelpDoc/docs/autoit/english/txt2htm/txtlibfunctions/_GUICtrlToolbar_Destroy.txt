###User Defined Function###
_GUICtrlToolbar_Destroy

###Description###
Delete the control

###Syntax###
#include <GuiToolbar.au3>
_GUICtrlToolbar_Destroy ( ByRef $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True, Handle is set to 0
Failure:	False
@@End@@


###Remarks###
Restricted to only be used on Toolbar controls created with _GUICtrlToolbar_Create


###Related###
_GUICtrlToolbar_Create
