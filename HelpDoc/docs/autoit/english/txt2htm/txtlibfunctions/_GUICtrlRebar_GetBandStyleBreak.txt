###User Defined Function###
_GUICtrlRebar_GetBandStyleBreak

###Description###
Determine if flag is set

###Syntax###
#include <GuiReBar.au3>
_GUICtrlRebar_GetBandStyleBreak ( $hWnd, $iIndex )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the rebar control
$iIndex
	Zero-based index of the band
@@End@@

###ReturnValue###
True: Flag is set (The band is on a new line)
False: Flag not set


###Remarks###
None.


###Related###
_GUICtrlRebar_SetBandStyleBreak, _GUICtrlRebar_GetBandStyle
