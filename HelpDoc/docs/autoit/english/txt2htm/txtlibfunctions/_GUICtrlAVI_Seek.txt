###User Defined Function###
_GUICtrlAVI_Seek

###Description###
Directs an AVI control to display a particular frame of an AVI clip

###Syntax###
#include <GuiAVI.au3>
_GUICtrlAVI_Seek ( $hWnd, $iFrame )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
$iFrame
	Zero based index of the frame to display
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True
Failure:	False
@@End@@


###Remarks###
None.
