###User Defined Function###
_GUICtrlComboBox_GetListArray

###Description###
Retrieves all items from the list portion of a ComboBox control

###Syntax###
#include <GuiComboBox.au3>
_GUICtrlComboBox_GetListArray ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
@@End@@

###ReturnValue###
Success: Array in the following format:
	[0] - Number of items
	[1] - Item 1
	[2] - Item 2
	[n] - Item n


###Remarks###
None.
