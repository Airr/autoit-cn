###User Defined Function###
_GUICtrlHeader_GetItemRectEx

###Description###
Retrieves the bounding rectangle for a given item

###Syntax###
#include <GuiHeader.au3>
_GUICtrlHeader_GetItemRectEx ( $hWnd, $iIndex )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$iIndex
	0-based item index
@@End@@

###ReturnValue###
Returns a $tagRECT structure that receives the bounding rectangle.


###Remarks###
None.
