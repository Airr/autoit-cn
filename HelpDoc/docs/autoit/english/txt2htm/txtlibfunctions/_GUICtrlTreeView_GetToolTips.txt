###User Defined Function###
_GUICtrlTreeView_GetToolTips

###Description###
Retrieves the handle to the child ToolTip control

###Syntax###
#include <GuiTreeView.au3>
_GUICtrlTreeView_GetToolTips ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	Handle to the child ToolTip control
Failure:	0 if the control is not using ToolTips
@@End@@


###Remarks###
When created, controls automatically create a ToolTip control. To cause a control not to use ToolTips, create
the control with the $TVS_NOTOOLTIPS style
