###User Defined Function###
_GUICtrlRebar_GetBandCount

###Description###
Retrieves the count of bands currently in the rebar control

###Syntax###
#include <GuiReBar.au3>
_GUICtrlRebar_GetBandCount ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the rebar control
@@End@@

###ReturnValue###
Returns the number of bands assigned to the control
