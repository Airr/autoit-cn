###User Defined Function###
_GUICtrlTreeView_GetChildCount

###Description###
Retrieves the number of children of an parent item

###Syntax###
#include <GuiTreeView.au3>
_GUICtrlTreeView_GetChildCount ( $hWnd, $hItem )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
$hItem
	Handle to item
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	Returns the Number of Children
Failure:	Returns -1 if 0 Children were found.
@@End@@


###Remarks###
None.


###Related###
_GUICtrlTreeView_GetCount, _GUICtrlTreeView_GetSiblingCount
