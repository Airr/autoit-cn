###User Defined Function###
_StringToHex

###Description###
Convert a string to a hex string

###Syntax###
#include <String.au3>
_StringToHex ( $sString )


###Parameters###
@@ParamTable@@
$sString
	String to be converted.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	a hex string.
Failure:	an empty string.
@@End@@
