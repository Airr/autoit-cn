###User Defined Function###
_GUICtrlListBox_GetCaretIndex

###Description###
Return index of item that has the focus rectangle

###Syntax###
#include <GuiListBox.au3>
_GUICtrlListBox_GetCaretIndex ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	Zero based index of the selected item
Failure:	-1 if nothing is selected
@@End@@


###Remarks###
None.
