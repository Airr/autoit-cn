###User Defined Function###
_GUICtrlMonthCal_GetCalendarCount

###Description###
Gets the number of calendars currently displayed in the calendar control

###Syntax###
#include <GuiMonthCal.au3>
_GUICtrlMonthCal_GetCalendarCount ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
@@End@@

###ReturnValue###
Success: Number of calendars currently displayed in the calendar control. The maximum number of allowed calendars is 12


###Remarks###
Minimum OS - Windows Vista.
