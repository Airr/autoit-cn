###User Defined Function###
_GUICtrlMenu_SetItemBmp

###Description###
Sets the bitmap displayed for the item

###Syntax###
#include <GuiMenu.au3>
_GUICtrlMenu_SetItemBmp ( $hMenu, $iItem, $hBmp [, $fByPos = True] )


###Parameters###
@@ParamTable@@
$hMenu
	Handle of the menu
$iItem
	Identifier or position of the menu item
$hBmp
	Handle to the item bitmap
$fByPos
	[optional] Menu identifier flag:
		True - $iItem is a 0-based item position
		False - $iItem is a menu item identifier
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True.
Failure:	False.
@@End@@


###Remarks###
None.
