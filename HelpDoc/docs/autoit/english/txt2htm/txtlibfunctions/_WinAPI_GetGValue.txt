###User Defined Function###
_WinAPI_GetGValue

###Description###
Retrieves an intensity value for the green component of a 32-bit RGB value.

###Syntax###
#include <WinAPIGdi.au3>
_WinAPI_GetGValue ( $iRGB )


###Parameters###
@@ParamTable@@
$iRGB
	The color value, in RGB.
@@End@@

###ReturnValue###
Returns the intensity of the green component of the specified RGB color.
