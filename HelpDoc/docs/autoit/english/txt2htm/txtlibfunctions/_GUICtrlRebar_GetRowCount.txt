###User Defined Function###
_GUICtrlRebar_GetRowCount

###Description###
Retrieves the number of rows of bands in a rebar control

###Syntax###
#include <GuiReBar.au3>
_GUICtrlRebar_GetRowCount ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the rebar control
@@End@@

###ReturnValue###
Returns a value that represents the number of band rows in the control.
