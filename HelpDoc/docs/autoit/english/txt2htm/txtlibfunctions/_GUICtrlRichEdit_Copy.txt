###User Defined Function###
_GUICtrlRichEdit_Copy

###Description###
Copy text to clipboard

###Syntax###
#include <GuiRichEdit.au3>
_GUICtrlRichEdit_Copy ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True
Failure:	False and sets @error
@error:	101 - $hWnd is not a handle
@@End@@
