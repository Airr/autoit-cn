###User Defined Function###
_GUICtrlListView_GetItemPositionY

###Description###
Retrieves the Y position of an item

###Syntax###
#include <GuiListView.au3>
_GUICtrlListView_GetItemPositionY ( $hWnd, $iIndex )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
$iIndex
	0-based index of the item
@@End@@

###ReturnValue###
Return the Y position of item.


###Remarks###
None.
