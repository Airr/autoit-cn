###User Defined Function###
_WinAPI_IsWindowUnicode

###Description###
Determines whether the specified window is a native Unicode window.

###Syntax###
#include <WinAPISys.au3>
_WinAPI_IsWindowUnicode ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the window to test.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success 	True - The window is a native Unicode window.
Failure 	False.
@@End@@
