###User Defined Function###
_MouseTrap

###Description###
Confine the Mouse Cursor to specified coords

###Syntax###
#include <Misc.au3>
_MouseTrap ( [$iLeft = 0 [, $iTop = 0 [, $iRight = 0 [, $iBottom = 0]]]] )


###Parameters###
@@ParamTable@@
$iLeft
	[optional] Left coord
$iTop
	[optional] Top coord
$iRight
	[optional] Right coord
$iBottom
	[optional] Bottom coord
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True.
Failure:	False.
@@End@@


###Remarks###
Use <a href="_MouseTrap.htm">_MouseTrap()</a> with no params to release the Mouse Cursor.
