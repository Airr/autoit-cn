###User Defined Function###
_GDIPlus_Startup

###Description###
Initialize Microsoft Windows GDI+

###Syntax###
#include <GDIPlus.au3>
_GDIPlus_Startup (  )


###Parameters###
None.

###ReturnValue###
@@ReturnTable@@
Success:	True.
Failure:	False and sets the @error flag to non-zero, @extended may contain GPSTATUS ($GPID_ERR*).
@@End@@


###Remarks###
Call _GDIPlus_Startup() before you create any GDI+ objects.
