###User Defined Function###
_WinAPI_ShortToWord

###Description###
Converts a value of type SHORT to a value of type WORD

###Syntax###
#include <WinAPIMisc.au3>
_WinAPI_ShortToWord ( $iValue )


###Parameters###
@@ParamTable@@
$iValue
	The value to be converted.
@@End@@

###ReturnValue###
Returns the converted (WORD) value.
