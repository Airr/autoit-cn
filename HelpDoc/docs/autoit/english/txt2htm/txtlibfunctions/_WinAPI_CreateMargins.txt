###User Defined Function###
_WinAPI_CreateMargins

###Description###
Creates $tagMARGINS structure with specified left, right, top, and bottom retaining borders

###Syntax###
#include <WinAPIMisc.au3>
_WinAPI_CreateMargins ( $iLeftWidth, $iRightWidth, $iTopHeight, $iBottomHeight )


###Parameters###
@@ParamTable@@
$iLeftWidth
	The width of the left border that retains its size.
$iRightWidth
	The width of the right border that retains its size.
$iTopHeight
	The height of the top border that retains its size.
$iBottomHeight
	The height of the bottom border that retains its size.
@@End@@

###ReturnValue###
Returns $tagMARGINS structure that contains the specified retaining borders.


###Remarks###
None.
