###User Defined Function###
_GDIPlus_GraphicsCreateFromHWND

###Description###
Create a Graphics object from a window handle

###Syntax###
#include <GDIPlus.au3>
_GDIPlus_GraphicsCreateFromHWND ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to a window
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	a handle to a Graphics object.
Failure:	0 and sets the @error flag to non-zero, @extended may contain GPSTATUS ($GPID_ERR*).
@@End@@


###Remarks###
When you are done with the Graphics object, call _GDIPlus_GraphicsDispose() to release the resources.
