###User Defined Function###
_GUICtrlTreeView_SetInsertMarkColor

###Description###
Sets the color used to draw the insertion mark

###Syntax###
#include <GuiTreeView.au3>
_GUICtrlTreeView_SetInsertMarkColor ( $hWnd, $iColor )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
$iColor
	Insertion mark color
@@End@@

###ReturnValue###
Returns the Previous insertion mark color.


###Remarks###
None.
