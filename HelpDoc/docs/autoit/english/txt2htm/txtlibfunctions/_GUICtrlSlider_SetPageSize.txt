###User Defined Function###
_GUICtrlSlider_SetPageSize

###Description###
Sets the number of logical positions the slider moves

###Syntax###
#include <GuiSlider.au3>
_GUICtrlSlider_SetPageSize ( $hWnd, $iPageSize )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
$iPageSize
	New page size
@@End@@

###ReturnValue###
Success: The previous page size


###Remarks###
None.
