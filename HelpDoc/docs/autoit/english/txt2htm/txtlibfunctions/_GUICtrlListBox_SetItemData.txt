###User Defined Function###
_GUICtrlListBox_SetItemData

###Description###
Sets the value associated with the specified item

###Syntax###
#include <GuiListBox.au3>
_GUICtrlListBox_SetItemData ( $hWnd, $iIndex, $iValue )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
$iIndex
	Specifies the zero based index of the item
$iValue
	Specifies the value to be associated with the item
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True
Failure:	False
@@End@@


###Remarks###
None.
