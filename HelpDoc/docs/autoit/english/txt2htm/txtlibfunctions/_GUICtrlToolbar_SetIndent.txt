###User Defined Function###
_GUICtrlToolbar_SetIndent

###Description###
Sets the indentation for the first button control

###Syntax###
#include <GuiToolbar.au3>
_GUICtrlToolbar_SetIndent ( $hWnd, $iIndent )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$iIndent
	Indentation in pixels
@@End@@

###ReturnValue###

@@End@@
