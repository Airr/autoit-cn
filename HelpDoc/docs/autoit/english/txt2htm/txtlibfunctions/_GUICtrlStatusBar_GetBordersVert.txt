###User Defined Function###
_GUICtrlStatusBar_GetBordersVert

###Description###
Retrieves the current width of the vertical border

###Syntax###
#include <GuiStatusBar.au3>
_GUICtrlStatusBar_GetBordersVert ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	Width of the vertical border
Failure:	Set @error
@@End@@
