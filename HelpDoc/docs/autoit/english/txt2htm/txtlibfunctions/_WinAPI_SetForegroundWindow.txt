###User Defined Function###
_WinAPI_SetForegroundWindow

###Description###
Puts the specified window into the foreground and activates its.

###Syntax###
#include <WinAPISys.au3>
_WinAPI_SetForegroundWindow ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the window that should be activated and brought to the foreground.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success 	True
Failure 	False
@@End@@
