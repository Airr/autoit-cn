###User Defined Function###
_GUICtrlComboBoxEx_SetItemImage

###Description###
Sets the index of the item's icon in the control's image list

###Syntax###
#include <GuiComboBoxEx.au3>
_GUICtrlComboBoxEx_SetItemImage ( $hWnd, $iIndex, $iImage )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$iIndex
	0-based index of the item
$iImage
	0-based index into the control's image list
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True.
Failure:	False.
@@End@@


###Remarks###
None.
