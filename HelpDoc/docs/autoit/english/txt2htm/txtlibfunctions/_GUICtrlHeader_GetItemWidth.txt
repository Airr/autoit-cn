###User Defined Function###
_GUICtrlHeader_GetItemWidth

###Description###
Retrieves the item's width

###Syntax###
#include <GuiHeader.au3>
_GUICtrlHeader_GetItemWidth ( $hWnd, $iIndex )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$iIndex
	0-based item index
@@End@@

###ReturnValue###
Returns the width of the item.


###Remarks###
None.
