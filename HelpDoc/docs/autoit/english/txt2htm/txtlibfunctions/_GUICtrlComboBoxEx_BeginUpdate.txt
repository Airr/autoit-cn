###User Defined Function###
_GUICtrlComboBoxEx_BeginUpdate

###Description###
Prevents updating of the control until the EndUpdate function is called

###Syntax###
#include <GuiComboBoxEx.au3>
_GUICtrlComboBoxEx_BeginUpdate ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True
Failure:	False
@@End@@


###Remarks###
None.
