###User Defined Function###
_GDIPlus_ImageLoadFromFile

###Description###
Create an image object based on a file

###Syntax###
#include <GDIPlus.au3>
_GDIPlus_ImageLoadFromFile ( $sFileName )


###Parameters###
@@ParamTable@@
$sFileName
	Fully qualified image file name
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	an handle to the new image object.
Failure:	-1 and sets the @error flag to non-zero, @extended may contain GPSTATUS ($GPID_ERR*).
@@End@@


###Remarks###
None.

_GDIPlus_ImageSaveToFile, _GDIPlus_ImageSaveToFileEx
