###User Defined Function###
_GUICtrlRebar_GetBandID

###Description###
Get the value that the control uses to identify this band for custom draw notification messages

###Syntax###
#include <GuiReBar.au3>
_GUICtrlRebar_GetBandID ( $hWnd, $iIndex )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the rebar control
$iIndex
	0-based index of the band
@@End@@

###ReturnValue###
Returns the id of the band.


###Remarks###
None.
