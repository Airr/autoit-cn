###User Defined Function###
_DateToMonth

###Description###
Returns the name of the Month, based on the specified month number.

###Syntax###
#include <Date.au3>
_DateToMonth ( $iMonth [, $ishort = 0] )


###Parameters###
@@ParamTable@@
$iMonth
	Month number (1 = January, 12 = December).
$ishort
	[optional] 0 - Long name of the Month.
	1 - Abbreviated name of the Month.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	the name of the given Month.
Failure:	an empty string and sets the @error flag to non-zero.
@error:	1 - Invalid month number or format.
@@End@@


###Remarks###
This function returns English day names only.
