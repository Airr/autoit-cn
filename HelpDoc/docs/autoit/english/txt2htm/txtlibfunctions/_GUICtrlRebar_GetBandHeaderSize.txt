###User Defined Function###
_GUICtrlRebar_GetBandHeaderSize

###Description###
Retrieves the size of the band's header, in pixels

###Syntax###
#include <GuiReBar.au3>
_GUICtrlRebar_GetBandHeaderSize ( $hWnd, $iIndex )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the rebar control
$iIndex
	0-based index of the band
@@End@@

###ReturnValue###
Returns the size of the band's header.


###Remarks###
None.
