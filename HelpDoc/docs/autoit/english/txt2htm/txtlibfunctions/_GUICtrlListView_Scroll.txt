###User Defined Function###
_GUICtrlListView_Scroll

###Description###
Scrolls the content of a list-view

###Syntax###
#include <GuiListView.au3>
_GUICtrlListView_Scroll ( $hWnd, $iDX, $iDY )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
$iDX
	Value of type int that specifies the amount of horizontal scrolling in pixels.
	If the list-view control is in list-view, this value specifies the number of columns to scroll
$iDY
	Value of type int that specifies the amount of vertical scrolling in pixels
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True
Failure:	False
@@End@@


###Remarks###
When the list-view control is in report view, the control can only be scrolled vertically in whole
line increments. Therefore, the $iDY parameter will be rounded to the nearest number of pixels
that form a whole line increment. For example, if the height of a line is 16 pixels and 8 is passed
for $iDY, the list will be scrolled by 16 pixels (1 line). If 7 is passed for $iDY, the list will be
scrolled 0 pixels (0 lines).
