###User Defined Function###
_WinAPI_LoWord

###Description###
Returns the low word of a longword

###Syntax###
#include <WinAPI.au3>
_WinAPI_LoWord ( $iLong )


###Parameters###
@@ParamTable@@
$iLong
	Longword value
@@End@@

###ReturnValue###
Returns an low order word of the longword value
