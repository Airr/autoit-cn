###User Defined Function###
_GUICtrlHeader_GetItemText

###Description###
Retrieves the item text

###Syntax###
#include <GuiHeader.au3>
_GUICtrlHeader_GetItemText ( $hWnd, $iIndex )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$iIndex
	0-based item index
@@End@@

###ReturnValue###
Returns the item text.

None.
