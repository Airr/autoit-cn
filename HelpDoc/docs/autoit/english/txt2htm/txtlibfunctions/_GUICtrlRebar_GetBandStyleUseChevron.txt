###User Defined Function###
_GUICtrlRebar_GetBandStyleUseChevron

###Description###
Determine if band use chevron flag is set

###Syntax###
#include <GuiReBar.au3>
_GUICtrlRebar_GetBandStyleUseChevron ( $hWnd, $iIndex )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the rebar control
$iIndex
	0-based index of the band
@@End@@

###ReturnValue###
@@ReturnTable@@
True:	the flag is set (Display drop-down button).
False:	the flag not set.
@@End@@


###Remarks###
None.

_GUICtrlRebar_GetBandStyle, _GUICtrlRebar_SetBandStyleUseChevron
