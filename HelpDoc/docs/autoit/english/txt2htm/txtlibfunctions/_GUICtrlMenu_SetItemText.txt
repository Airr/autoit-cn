###User Defined Function###
_GUICtrlMenu_SetItemText

###Description###
Sets the text for a menu item

###Syntax###
#include <GuiMenu.au3>
_GUICtrlMenu_SetItemText ( $hMenu, $iItem, $sText [, $fByPos = True] )


###Parameters###
@@ParamTable@@
$hMenu
	Menu handle
$iItem
	Identifier or position of the menu item
$sText
	Menu item text
$fByPos
	[optional] Menu identifier flag:
		True - $iItem is a 0-based item position
		False - $iItem is a menu item identifier
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True.
Failure:	False.
@@End@@


###Remarks###
None.
