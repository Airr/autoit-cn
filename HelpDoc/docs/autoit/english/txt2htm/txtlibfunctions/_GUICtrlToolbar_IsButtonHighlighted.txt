###User Defined Function###
_GUICtrlToolbar_IsButtonHighlighted

###Description###
Indicates whether the specified button is hilighted

###Syntax###
#include <GuiToolbar.au3>
_GUICtrlToolbar_IsButtonHighlighted ( $hWnd, $iCommandID )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$iCommandID
	Button command ID
@@End@@

###ReturnValue###
@@ReturnTable@@
True:	the button is highlighted.
False:	the button is not highlighted.
@@End@@
