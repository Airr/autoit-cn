###User Defined Function###
_GUICtrlToolbar_IsButtonHighlighted

###Description###
Indicates whether the specified button is hilighted

###Syntax###
#include <GuiToolbar.au3>
_GUICtrlToolbar_IsButtonHighlighted ( $hWnd, $iCommandID )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$iCommandID
	Button command ID
@@End@@

###ReturnValue###
True: Button is hilighted
False: Button is not hilighted


###Remarks###
None.
