###User Defined Function###
_WinAPI_FreeMRUList

###Description###
Frees the handle associated with the most recently used (MRU) list and writes cached data to the registry.

###Syntax###
#include <WinAPIReg.au3>
_WinAPI_FreeMRUList ( $hMRU )


###Parameters###
@@ParamTable@@
$hMRU
	Handle of the MRU list to free.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success 	True
Failure 	False
@@End@@


###Remarks###
None
