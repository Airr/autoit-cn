###User Defined Function###
_GDIPlus_MatrixGetElements

###Description###
Gets the elements of a matrix

###Syntax###
#include <GDIPlus.au3>
_GDIPlus_MatrixGetElements ( $hMatrix )


###Parameters###
@@ParamTable@@
$hMatrix
	Pointer to a Matrix object
@@End@@

###ReturnValue###
Success: Array of the elements of the matrix:

	[0] - Row 1, column 1
	[1] - Row 1, column 2
	[2] - Row 2, column 1
	[3] - Row 2, column 2
	[4] - Row 3, column 1
	[5] - Row 3, column 2
Failure: -1
