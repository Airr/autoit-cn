###User Defined Function###
_GUICtrlMonthCal_GetMinReqWidth

###Description###
Retrieves the minimum width required to display a full month

###Syntax###
#include <GuiMonthCal.au3>
_GUICtrlMonthCal_GetMinReqWidth ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
@@End@@

###ReturnValue###
Returns the minimum width, in pixels.


###Remarks###
None.
