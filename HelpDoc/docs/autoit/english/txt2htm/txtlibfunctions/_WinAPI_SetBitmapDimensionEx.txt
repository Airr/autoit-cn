###User Defined Function###
_WinAPI_SetBitmapDimensionEx

###Description###
Assigns preferred dimensions to a compatible bitmap

###Syntax###
#include <WinAPIGdi.au3>
_WinAPI_SetBitmapDimensionEx ( $hBitmap, $iWidth, $iHeight )


###Parameters###
@@ParamTable@@
$hBitmap
	Handle to the bitmap. This bitmap cannot be a DIB-section bitmap.
$iWidth
	The width, in 0.1-millimeter units, of the bitmap.
$iHeight
	The height, in 0.1-millimeter units, of the bitmap.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True
Failure:	False
@@End@@


###Remarks###
The bitmap cannot be a DIB section, which is a bitmap created by the <a href="_WinAPI_CreateDIB.htm">_WinAPI_CreateDIB()</a> or <a href="_WinAPI_CreateDIBSection.htm">_WinAPI_CreateDIBSection()</a>
functions. If the bitmap is a DIB section, the <a href="_WinAPI_SetBitmapDimensionEx.htm">_WinAPI_SetBitmapDimensionEx()</a> function fails.

Call <a href="_WinAPI_GetExtended.htm">_WinAPI_GetExtended()</a> to retrieve a $tagSIZE structure containing the previous dimensions of the bitmap.


###Related###
_WinAPI_CreateDIB, _WinAPI_CreateDIBSection, _WinAPI_GetExtended
