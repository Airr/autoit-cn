###User Defined Function###
_GUICtrlComboBox_GetDroppedControlRectEx

###Description###
Retrieve the screen coordinates of a combo box in its dropped-down state

###Syntax###
#include <GuiComboBox.au3>
_GUICtrlComboBox_GetDroppedControlRectEx ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
@@End@@

###ReturnValue###
Returns a $tagRECT structure that receives the screen coordinates.


###Remarks###
None.
