###User Defined Function###
_WinAPI_LoByte

###Description###
Returns the low BYTE of a 16-bit (2 bytes) value

###Syntax###
#include <WinAPIMisc.au3>
_WinAPI_LoByte ( $iValue )


###Parameters###
@@ParamTable@@
$iValue
	16-bit value.
@@End@@

###ReturnValue###
Returns low BYTE value.


###Remarks###
None.
