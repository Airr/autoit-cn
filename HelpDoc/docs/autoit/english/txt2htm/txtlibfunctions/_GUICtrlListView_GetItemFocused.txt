###User Defined Function###
_GUICtrlListView_GetItemFocused

###Description###
Determines whether the item is highlighted as a drag-and-drop target

###Syntax###
#include <GuiListView.au3>
_GUICtrlListView_GetItemFocused ( $hWnd, $iIndex )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
$iIndex
	Zero based index of the item
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True
Failure:	False
@@End@@


###Remarks###
None.
