###User Defined Function###
_GUICtrlToolbar_SetButtonText

###Description###
Sets the display text of a button

###Syntax###
#include <GuiToolbar.au3>
_GUICtrlToolbar_SetButtonText ( $hWnd, $iCommandID, $sText )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$iCommandID
	Button command ID
$sText
	Button text
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True.
Failure:	False.
@@End@@
