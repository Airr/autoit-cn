###User Defined Function###
_GUICtrlToolbar_SetMaxTextRows

###Description###
Sets the maximum number of text rows displayed button

###Syntax###
#include <GuiToolbar.au3>
_GUICtrlToolbar_SetMaxTextRows ( $hWnd, $iMaxRows )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$iMaxRows
	Maximum number of rows of text that can be displayed
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True.
Failure:	False.
@@End@@
