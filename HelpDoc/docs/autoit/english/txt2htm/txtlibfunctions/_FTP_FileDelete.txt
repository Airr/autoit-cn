###User Defined Function###
_FTP_FileDelete

###Description###
Delete an file from an FTP server.

###Syntax###
#include <FTPEx.au3>
_FTP_FileDelete ( $hFTPSession, $sRemoteFile )


###Parameters###
@@ParamTable@@
$hFTPSession
	as returned by _FTP_Connect()
$sRemoteFile
	The remote Location for the file.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	1
Failure:	0  and sets @error
@@End@@


###Remarks###
None.


###Related###
_FTP_Connect
