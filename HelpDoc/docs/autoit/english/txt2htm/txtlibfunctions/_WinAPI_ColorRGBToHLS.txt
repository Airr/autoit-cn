###User Defined Function###
_WinAPI_ColorRGBToHLS

###Description###
Converts colors from RGB to hue-luminance-saturation (HLS) format.

###Syntax###
#include <WinAPIGdi.au3>
_WinAPI_ColorRGBToHLS ( $iRGB, ByRef $iHue, ByRef $iLuminance, ByRef $iSaturation )


###Parameters###
@@ParamTable@@
$iRGB
	RGB color.
$iHue
	Returns HLS hue value.
$iLuminance
	Returns HLS luminance value.
$iSaturation
	Returns HLS saturation value.
@@End@@

###ReturnValue###
None
