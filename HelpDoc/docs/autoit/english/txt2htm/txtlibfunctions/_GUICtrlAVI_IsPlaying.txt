###User Defined Function###
_GUICtrlAVI_IsPlaying

###Description###
Checks whether an Audio-Video Interleaved (AVI) clip is playing

###Syntax###
#include <GuiAVI.au3>
_GUICtrlAVI_IsPlaying ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True.
Failure:	False.
@@End@@


###Remarks###
Minimum OS: Windows Vista.
