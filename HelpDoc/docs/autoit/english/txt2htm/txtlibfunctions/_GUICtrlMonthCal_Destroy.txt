###User Defined Function###
_GUICtrlMonthCal_Destroy

###Description###
Delete the MonthCal control

###Syntax###
#include <GuiMonthCal.au3>
_GUICtrlMonthCal_Destroy ( ByRef $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True, Handle is set to 0
Failure:	False
@@End@@


###Remarks###
Restricted to only be used on Month Calendar controls created with _GUICtrlMonthCal_Create

_GUICtrlMonthCal_Create
