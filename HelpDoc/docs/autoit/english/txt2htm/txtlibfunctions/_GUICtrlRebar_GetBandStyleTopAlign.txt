###User Defined Function###
_GUICtrlRebar_GetBandStyleTopAlign

###Description###
Determine if flag is set

###Syntax###
#include <GuiReBar.au3>
_GUICtrlRebar_GetBandStyleTopAlign ( $hWnd, $iIndex )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the rebar control
$iIndex
	Zero-based index of the band
@@End@@

###ReturnValue###
True: Flag is set (Keep band in top row)
False: Flag not set


###Remarks###
None.

_GUICtrlRebar_SetBandStyleTopAlign, _GUICtrlRebar_GetBandStyle
