###User Defined Function###
_GDIPlus_BitmapCreateFromStream

###Description###
Creates a Bitmap object based on an IStream COM interface

###Syntax###
#include <GDIPlus.au3>
_GDIPlus_BitmapCreateFromStream ( $pStream )


###Parameters###
@@ParamTable@@
$pStream
	Pointer to an IStream COM interface
@@End@@

###ReturnValue###
Success: Returns a handle to a new Bitmap object
Failure: 0 and either:

	@error and @extended are set if DllCall failed


###Remarks###
After you are done with the object, call _GDIPlus_ImageDispose to release the object resources


###Related###
_GDIPlus_ImageDispose, _WinAPI_CreateStreamOnHGlobal
