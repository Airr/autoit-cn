###User Defined Function###
_GDIPlus_PathBrushSetLinearBlend

###Description###
Sets the blend shape of a path gradient brush to create a custom blend based on a triangular shape

###Syntax###
#include <GDIPlus.au3>
_GDIPlus_PathBrushSetLinearBlend ( $hPathGradientBrush, $fFocus [, $fScale = 1] )


###Parameters###
@@ParamTable@@
$hPathGradientBrush
	Pointer to a PathGradientBrush object
$fFocus
	Number in the range 0.0 to 1.0 that specifies where the center color will be at its
	highest intensity
$fScale
	[optional] Number in the range 0.0 to 1.0 that specifies the maximum intensity of center color that
	gets blended with the boundary color
@@End@@

###ReturnValue###
Success: True
Failure: False


###Remarks###
None
