###User Defined Function###
_WinAPI_FatalExit

###Description###
Transfers execution control to the debugger

###Syntax###
#include <WinAPIDiag.au3>
_WinAPI_FatalExit ( $iCode )


###Parameters###
@@ParamTable@@
$iCode
	The error code associated with the exit.
@@End@@

###ReturnValue###
None.


###Remarks###
@error can be return if a Windows error occured.
