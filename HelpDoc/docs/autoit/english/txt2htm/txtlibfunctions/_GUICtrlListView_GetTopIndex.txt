###User Defined Function###
_GUICtrlListView_GetTopIndex

###Description###
Retrieves the index of the topmost visible item when in list or report view

###Syntax###
#include <GuiListView.au3>
_GUICtrlListView_GetTopIndex ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
@@End@@

###ReturnValue###
Returns the 0-based index of the item.
	Zero if the list-view control is in icon or small icon view.

None.
