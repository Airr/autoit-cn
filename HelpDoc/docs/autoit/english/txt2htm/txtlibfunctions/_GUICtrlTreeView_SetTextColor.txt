###User Defined Function###
_GUICtrlTreeView_SetTextColor

###Description###

###Syntax###

_GUICtrlTreeView_SetTextColor ( $hWnd, $vRGBColor )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
$vRGBColor
	New Hex text color
@@End@@

###ReturnValue###
Returns the Previous Hex RGB text color.
