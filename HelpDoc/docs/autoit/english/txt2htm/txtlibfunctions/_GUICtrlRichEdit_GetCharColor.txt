###User Defined Function###
_GUICtrlRichEdit_GetCharColor

###Description###
Retrieves the color of the selected text or, if none selected, of the character to the right of the insertion point

###Syntax###
#include <GuiRichEdit.au3>
_GUICtrlRichEdit_GetCharColor ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	a COLORREF value an @extended set to 1 if the color is the same throught the selection.
Failure:	sets the @error flag to non-zero.
@error:	101 - $hWnd is not a handle
@@End@@


###Remarks###
If you need to create an color array use <a href="_ColorGetCOLORREF.htm">_ColorGetCOLORREF()</a> not <a href="_ColorGetRGB.htm">_ColorGetRGB()</a>.
