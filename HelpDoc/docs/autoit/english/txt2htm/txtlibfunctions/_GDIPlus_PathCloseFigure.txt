###User Defined Function###
_GDIPlus_PathCloseFigure

###Description###
Closes the current figure of a path

###Syntax###
#include <GDIPlus.au3>
_GDIPlus_PathCloseFigure ( $hPath )


###Parameters###
@@ParamTable@@
$hPath
	Pointer to a GraphicsPath object
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True.
Failure:	False and sets the @error flag to non-zero, @extended may contain GPSTATUS ($GPID_ERR*).
@@End@@
