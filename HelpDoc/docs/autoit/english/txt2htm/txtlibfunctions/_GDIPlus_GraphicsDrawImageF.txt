###User Defined Function###
_GDIPlus_GraphicsDrawImageF

###Description###
Draw an Image object

###Syntax###
#include <GDIPlus.au3>
_GDIPlus_GraphicsDrawImageF ( $hGraphics, $hImage, $fX, $fY )


###Parameters###
@@ParamTable@@
$hGraphics
	Handle to a Graphics object
$hImage
	Handle to an Image object
$fX
	The X coordinate of the upper left corner of the rendered image
$fY
	The Y coordinate of the upper left corner of the rendered image
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True
Failure:	False and sets the @error flag to non-zero, @extended may contain GPSTATUS ($GPID_ERR*).
@@End@@

None.
