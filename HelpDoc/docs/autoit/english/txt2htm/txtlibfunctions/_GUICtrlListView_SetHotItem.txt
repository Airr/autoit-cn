###User Defined Function###
_GUICtrlListView_SetHotItem

###Description###
Sets the hot item

###Syntax###
#include <GuiListView.au3>
_GUICtrlListView_SetHotItem ( $hWnd, $iIndex )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
$iIndex
	0-based index of the item to be set as the hot item
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	the index of the item that was previously hot.
Failure:	-1.
@@End@@


###Remarks###
None.
