###User Defined Function###
_GUICtrlToolbar_GetUnicodeFormat

###Description###
Retrieves the Unicode character format flag

###Syntax###
#include <GuiToolbar.au3>
_GUICtrlToolbar_GetUnicodeFormat ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
@@End@@

###ReturnValue###
True: Control is using Unicode characters
False: Control is using ANSI characters


###Remarks###
None.
