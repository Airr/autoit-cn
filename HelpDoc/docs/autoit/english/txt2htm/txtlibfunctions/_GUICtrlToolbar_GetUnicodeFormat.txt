###User Defined Function###
_GUICtrlToolbar_GetUnicodeFormat

###Description###
Retrieves the Unicode character format flag

###Syntax###
#include <GuiToolbar.au3>
_GUICtrlToolbar_GetUnicodeFormat ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
@@End@@

###ReturnValue###
@@ReturnTable@@
True:	the control is using Unicode characters.
False:	the control is using ANSI characters.
@@End@@


###Remarks###
None.
