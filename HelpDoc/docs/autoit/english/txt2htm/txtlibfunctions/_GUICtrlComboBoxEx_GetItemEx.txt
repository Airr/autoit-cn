###User Defined Function###
_GUICtrlComboBoxEx_GetItemEx

###Description###
Retrieves some or all of an item's attributes

###Syntax###
#include <GuiComboBoxEx.au3>
_GUICtrlComboBoxEx_GetItemEx ( $hWnd, ByRef $tItem )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$tItem
	$tagCOMBOBOXEXITEM structure that specifies the information to retrieve
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True
Failure:	False
@@End@@


###Remarks###
None.
