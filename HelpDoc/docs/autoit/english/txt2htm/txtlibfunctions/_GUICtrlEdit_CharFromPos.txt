###User Defined Function###
_GUICtrlEdit_CharFromPos

###Description###
Retrieve information about the character closest to a specified point in the client area

###Syntax###
#include <GuiEdit.au3>
_GUICtrlEdit_CharFromPos ( $hWnd, $iX, $iY )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
$iX
	horizontal position
$iY
	vertical position
@@End@@

###ReturnValue###
Success: Array in the following format:

	[0] - Zero-based index of the character nearest the specified point
	[1] - Zero-based index of the line that contains the character

None.
