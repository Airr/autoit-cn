###User Defined Function###
_GUICtrlToolbar_GetButtonInfoEx

###Description###
Retrieves extended information for a button

###Syntax###
#include <GuiToolbar.au3>
_GUICtrlToolbar_GetButtonInfoEx ( $hWnd, $iCommandID )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$iCommandID
	Button command ID
@@End@@

###ReturnValue###
Returns a $tagTBBUTTONINFO structure.


###Remarks###
None.
