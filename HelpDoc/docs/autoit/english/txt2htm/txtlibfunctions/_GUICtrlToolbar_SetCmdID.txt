###User Defined Function###
_GUICtrlToolbar_SetCmdID

###Description###
Sets the command identifier of a toolbar button

###Syntax###
#include <GuiToolbar.au3>
_GUICtrlToolbar_SetCmdID ( $hWnd, $iIndex, $iCommandID )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$iIndex
	Zero based index of the button whose command identifier is to be set
$iCommandID
	Command identifier
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True
Failure:	False
@@End@@
