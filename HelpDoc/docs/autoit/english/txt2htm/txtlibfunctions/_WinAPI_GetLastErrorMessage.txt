###User Defined Function###
_WinAPI_GetLastErrorMessage

###Description###
Returns the calling threads last error message

###Syntax###
#include <WinAPI.au3>
_WinAPI_GetLastErrorMessage (  )


###Parameters###
None.

###ReturnValue###
@@ReturnTable@@
Success:	the last error message.
Failure:	sets the @error flag to non-zero.
@@End@@
