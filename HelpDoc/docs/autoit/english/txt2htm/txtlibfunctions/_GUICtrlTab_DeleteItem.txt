###User Defined Function###
_GUICtrlTab_DeleteItem

###Description###
Deletes a tab

###Syntax###
#include <GuiTab.au3>
_GUICtrlTab_DeleteItem ( $hWnd, $iIndex )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
$iIndex
	Zero based index of the item
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True
Failure:	False
@@End@@


###Remarks###
This does not delete the controls on the tabitems
