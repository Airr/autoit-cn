###User Defined Function###
_GDIPlus_ImageSaveToFile

###Description###
Save an image to file

###Syntax###
#include <GDIPlus.au3>
_GDIPlus_ImageSaveToFile ( $hImage, $sFileName )


###Parameters###
@@ParamTable@@
$hImage
	Handle to an image object
$sFileName
	Fully qualified image file name
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True.
Failure:	False and sets the @error flag to non-zero, @extended may contain GPSTATUS ($GPID_ERR*).
@@End@@


###Remarks###
None.
