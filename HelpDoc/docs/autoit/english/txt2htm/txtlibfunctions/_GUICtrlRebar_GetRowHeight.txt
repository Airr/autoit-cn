###User Defined Function###
_GUICtrlRebar_GetRowHeight

###Description###
Retrieves the height of a specified row in a rebar control

###Syntax###
#include <GuiReBar.au3>
_GUICtrlRebar_GetRowHeight ( $hWnd, $iIndex )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the rebar control
$iIndex
	0-based index of a band. The height of the row that contains the specified band will be retrieved
@@End@@

###ReturnValue###
Returns a value that represents the row height, in pixels.

None.
