###User Defined Function###
_WinAPI_WordToShort

###Description###
Converts a value of type WORD to a value of type SHORT

###Syntax###
#include <WinAPIMisc.au3>
_WinAPI_WordToShort ( $iValue )


###Parameters###
@@ParamTable@@
$iValue
	The value to be converted.
@@End@@

###ReturnValue###
Returns the converted (SHORT) value.
