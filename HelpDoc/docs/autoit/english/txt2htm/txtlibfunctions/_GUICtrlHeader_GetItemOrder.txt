###User Defined Function###
_GUICtrlHeader_GetItemOrder

###Description###
Retrieves the order in which the item appears

###Syntax###
#include <GuiHeader.au3>
_GUICtrlHeader_GetItemOrder ( $hWnd, $iIndex )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$iIndex
	0-based item index
@@End@@

###ReturnValue###
Returns the 0-based item order.


###Remarks###
None.
