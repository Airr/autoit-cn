###User Defined Function###
_WinAPI_IsIconic

###Description###
Determines whether the specified window is minimized (iconic).

###Syntax###
#include <WinAPISys.au3>
_WinAPI_IsIconic ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the window to test.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success 	True - The window is iconic.
Failure 	False.
@@End@@
