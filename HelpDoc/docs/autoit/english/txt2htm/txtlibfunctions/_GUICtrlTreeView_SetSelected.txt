###User Defined Function###
_GUICtrlTreeView_SetSelected

###Description###
Sets whether the item appears in the selected state

###Syntax###
#include <GuiTreeView.au3>
_GUICtrlTreeView_SetSelected ( $hWnd, $hItem [, $fFlag = True] )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
$hItem
	Handle to the item
$fFlag
	[optional] True if item is to be selected, otherwise False
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True.
Failure:	False.
@@End@@


###Remarks###
None.
