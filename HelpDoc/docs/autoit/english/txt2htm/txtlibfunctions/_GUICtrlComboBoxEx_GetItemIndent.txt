###User Defined Function###
_GUICtrlComboBoxEx_GetItemIndent

###Description###
Retrieves the number of image widths the item is indented

###Syntax###
#include <GuiComboBoxEx.au3>
_GUICtrlComboBoxEx_GetItemIndent ( $hWnd, $iIndex )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$iIndex
	0-based item index
@@End@@

###ReturnValue###
Returns the item indention.


###Remarks###
None.
