###User Defined Function###
_GDIPlus_FontGetHeight

###Description###
Gets the line spacing of a font in the current unit of a specified Graphics object

###Syntax###
#include <GDIPlus.au3>
_GDIPlus_FontGetHeight ( $hFont, $hGraphics )


###Parameters###
@@ParamTable@@
$hFont
	Pointer to a Font object
$hGraphics
	Pointer to a Graphics object
@@End@@

###ReturnValue###
Success: Returns the line spacing of this font
Failure: 0


###Remarks###
The line spacing is the vertical distance between the base lines of two consecutive lines of text. Thus, the
+line spacing includes the blank space between lines along with the height of the character itself
