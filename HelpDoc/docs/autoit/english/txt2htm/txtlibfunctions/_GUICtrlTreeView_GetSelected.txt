###User Defined Function###
_GUICtrlTreeView_GetSelected

###Description###
Indicates whether the item appears in the selected state

###Syntax###
#include <GuiTreeView.au3>
_GUICtrlTreeView_GetSelected ( $hWnd, $hItem )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
$hItem
	Handle to the item
@@End@@

###ReturnValue###
@@ReturnTable@@
True:	Item is in the selected state
False:	Item is not in the selected state
@@End@@


###Remarks###
None.
