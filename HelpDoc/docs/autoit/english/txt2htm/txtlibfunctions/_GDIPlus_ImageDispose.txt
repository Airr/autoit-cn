###User Defined Function###
_GDIPlus_ImageDispose

###Description###

###Syntax###
#include <GDIPlus.au3>
_GDIPlus_ImageDispose ( $hImage )


###Parameters###
@@ParamTable@@
$hImage
	Handle to an image object
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True.
