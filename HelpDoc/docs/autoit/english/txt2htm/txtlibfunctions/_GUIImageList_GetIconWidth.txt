###User Defined Function###
_GUIImageList_GetIconWidth

###Description###
Retrieves the width of the images in an image list

###Syntax###
#include <GuiImageList.au3>
_GUIImageList_GetIconWidth ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the imagelist
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	Width, in pixels, of each image
Failure:	0
@@End@@


###Remarks###
None.
