###User Defined Function###
_GUICtrlTab_Destroy

###Description###
Delete the control

###Syntax###
#include <GuiTab.au3>
_GUICtrlTab_Destroy ( ByRef $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True, $hWnd is set to 0.
Failure:	False.
@@End@@


###Remarks###
Restricted to only be used on Tab controls created with _GUICtrlTab_Create().


###Related###
_GUICtrlTab_Create
