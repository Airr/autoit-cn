###User Defined Function###
_HexToString

###Description###
Convert a hex string to a string.

###Syntax###
#include <String.au3>
_HexToString ( $sHex )


###Parameters###
@@ParamTable@@
$sHex
	A hexadecimal string
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	Returns a string
Failure:	Returns "" blank string and sets @error to non-zero
@@End@@
