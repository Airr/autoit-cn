###User Defined Function###
_WinNet_ConnectionDialog

###Description###
Starts a general browsing dialog box for connecting to network resources

###Syntax###
#include <WinNet.au3>
_WinNet_ConnectionDialog ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the owner window for the dialog box
@@End@@

###ReturnValue###
