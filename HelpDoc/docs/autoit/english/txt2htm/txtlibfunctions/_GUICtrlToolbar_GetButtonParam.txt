###User Defined Function###
_GUICtrlToolbar_GetButtonParam

###Description###
Retrieves the button param value

###Syntax###
#include <GuiToolbar.au3>
_GUICtrlToolbar_GetButtonParam ( $hWnd, $iCommandID )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$iCommandID
	Button command ID
@@End@@

###ReturnValue###
Returns the application-defined value.


###Remarks###
None.
