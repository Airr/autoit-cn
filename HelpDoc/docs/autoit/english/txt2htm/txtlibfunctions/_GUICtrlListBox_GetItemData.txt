###User Defined Function###
_GUICtrlListBox_GetItemData

###Description###
Retrieves the application defined value associated with an item

###Syntax###
#include <GuiListBox.au3>
_GUICtrlListBox_GetItemData ( $hWnd, $iIndex )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
$iIndex
	Specifies the zero based index of the item
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	Application defined value
Failure:	-1
@@End@@


###Remarks###
None.
