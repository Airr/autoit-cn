###User Defined Function###
_FTP_FileGetSize

###Description###
Gets filesize of a file on the FTP server.

###Syntax###
#include <FTPEx.au3>
_FTP_FileGetSize ( $hFTPSession, $sFileName )


###Parameters###
@@ParamTable@@
$hFTPSession
	as returned by _FTP_Connect().
$sFileName
	The file name.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	the filesize as an uint64.
Failure:	sets the @error flag to non-zero.
@@End@@


###Remarks###
None.

_FTP_Connect
