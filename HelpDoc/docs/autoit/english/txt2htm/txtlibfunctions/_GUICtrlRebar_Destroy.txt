###User Defined Function###
_GUICtrlRebar_Destroy

###Description###
Delete the control

###Syntax###
#include <GuiReBar.au3>
_GUICtrlRebar_Destroy ( ByRef $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the rebar control
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True, $hWnd is set to 0.
Failure:	False.
@@End@@


###Remarks###
Restricted to only be used on Rebar controls created with <a href="_GUICtrlRebar_Create.htm">_GUICtrlRebar_Create()</a>.
