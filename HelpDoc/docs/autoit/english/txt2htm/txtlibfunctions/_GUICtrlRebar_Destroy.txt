###User Defined Function###
_GUICtrlRebar_Destroy

###Description###
Delete the control

###Syntax###
#include <GuiReBar.au3>
_GUICtrlRebar_Destroy ( ByRef $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the rebar control
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True, Handle is set to 0
Failure:	False
@@End@@


###Remarks###
Restricted to only be used on Rebar controls created with _GUICtrlRebar_Create


###Related###
_GUICtrlRebar_Create
