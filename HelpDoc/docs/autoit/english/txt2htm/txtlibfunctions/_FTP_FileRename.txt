###User Defined Function###
_FTP_FileRename

###Description###
Renames an file on an FTP server.

###Syntax###
#include <FTPEx.au3>
_FTP_FileRename ( $hFTPSession, $sExisting, $sNew )


###Parameters###
@@ParamTable@@
$hFTPSession
	as returned by _FTP_Connect().
$sExisting
	The old file name.
$sNew
	The new file name.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	1
Failure:	0 and sets @error to non-zero
@@End@@


###Remarks###
None.

_FTP_Connect
