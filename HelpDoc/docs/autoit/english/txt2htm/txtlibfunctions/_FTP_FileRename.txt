###User Defined Function###
_FTP_FileRename

###Description###
Renames an file on an FTP server

###Syntax###
#include <FTPEx.au3>
_FTP_FileRename ( $hFTPSession, $sExisting, $sNew )


###Parameters###
@@ParamTable@@
$hFTPSession
	as returned by <a href="_FTP_Connect.htm">_FTP_Connect()</a>.
$sExisting
	The old file name.
$sNew
	The new file name.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	1.
Failure:	0 and sets the @error flag to non-zero.
@@End@@


###Remarks###
None.

_FTP_Connect
