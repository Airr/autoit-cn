###User Defined Function###
_WinAPI_GetWindowWidth

###Description###
Returns the width of the window

###Syntax###
#include <WinAPI.au3>
_WinAPI_GetWindowWidth ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to a window
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	the width of window.
Failure:	sets the @error flag to non-zero.
@@End@@


###Remarks###
None.
