###User Defined Function###
_GUICtrlTreeView_SetBold

###Description###
Sets whether the item is drawn using a bold sytle

###Syntax###
#include <GuiTreeView.au3>
_GUICtrlTreeView_SetBold ( $hWnd, $hItem [, $fFlag = True] )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
$hItem
	Handle to the item
$fFlag
	[optional] True if item is drawn bold, otherwise False
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True
Failure:	False
@@End@@


###Remarks###
None.
