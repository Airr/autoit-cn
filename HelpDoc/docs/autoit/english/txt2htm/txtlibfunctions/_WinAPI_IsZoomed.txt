###User Defined Function###
_WinAPI_IsZoomed

###Description###
Determines whether a window is maximized.

###Syntax###
#include <WinAPISys.au3>
_WinAPI_IsZoomed ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the window to test.
@@End@@

###ReturnValue###
@@ReturnTable@@
True:	the window is zoomed.
False:	the window is not zoomed.
@@End@@
