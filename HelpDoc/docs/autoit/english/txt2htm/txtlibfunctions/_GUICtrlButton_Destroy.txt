###User Defined Function###
_GUICtrlButton_Destroy

###Description###
Delete the Button control

###Syntax###
#include <GuiButton.au3>
_GUICtrlButton_Destroy ( ByRef $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True, $hWnd is set to 0.
Failure:	False.
@@End@@


###Remarks###
Restricted to only be used on Button controls created with _GUICtrlButton_Create().


###Related###
_GUICtrlButton_Create
