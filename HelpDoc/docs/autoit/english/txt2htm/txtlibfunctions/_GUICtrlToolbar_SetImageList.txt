###User Defined Function###
_GUICtrlToolbar_SetImageList

###Description###
Sets the default button image list

###Syntax###
#include <GuiToolbar.au3>
_GUICtrlToolbar_SetImageList ( $hWnd, $hImageList )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$hImageList
	Handle to the image list to set. If this parameter is 0, no images are displayed in the buttons.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	the handle to the image list previously used to display buttons in their default state.
Failure:	0.
@@End@@


###Remarks###
None.
