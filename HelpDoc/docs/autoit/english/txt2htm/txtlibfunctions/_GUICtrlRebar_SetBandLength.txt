###User Defined Function###
_GUICtrlRebar_SetBandLength

###Description###
Set the size length of the band.

###Syntax###
#include <GuiReBar.au3>
_GUICtrlRebar_SetBandLength ( $hWnd, $iIndex, $icx )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the rebar control
$iIndex
	0-based index of the band
$icx
	Length of the band, in pixels
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True.
Failure:	False.
@@End@@


###Remarks###
None.
