###User Defined Function###
_FTP_FindFileNext

###Description###
Find Next File on an FTP server.

###Syntax###
#include <FTPEx.au3>
_FTP_FindFileNext ( $hFTPFind )


###Parameters###
@@ParamTable@@
$hFTPFind
	Handle as returned by _FTP_FindFileFirst().
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	an array:
		[0]  - Number of elements
		[1]  - File Attributes
		[2]  - Creation Time Low
		[3]  - Creation Time Hi
		[4]  - Access Time Low
		[5]  - Access Time Hi
		[6]  - Last Write Low
		[7]  - Last Write Hi
		[8]  - File Size High
		[9]  - File Size Low
		[10] - File Name
		[11] - Altername
Failure:	0  and sets the @error flag to non-zero.
@@End@@


###Remarks###
None.
