###User Defined Function###
_GUICtrlMenu_MapAccelerator

###Description###
Maps a menu accelerator key to it's position in the menu

###Syntax###
#include <GuiMenu.au3>
_GUICtrlMenu_MapAccelerator ( $hMenu, $cAccel )


###Parameters###
@@ParamTable@@
$hMenu
	Handle to menu
$cAccel
	Accelerator key
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	The zero based position of the item in the menu
Failure:	-1
@@End@@
