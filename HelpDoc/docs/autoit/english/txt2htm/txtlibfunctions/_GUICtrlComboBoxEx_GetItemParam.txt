###User Defined Function###
_GUICtrlComboBoxEx_GetItemParam

###Description###
Retrieves the application specific value of the item

###Syntax###
#include <GuiComboBoxEx.au3>
_GUICtrlComboBoxEx_GetItemParam ( $hWnd, $iIndex )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$iIndex
	0-based item index
@@End@@

###ReturnValue###
Returns an application specific value.


###Remarks###
None.
