###User Defined Function###
_GDIPlus_ImageGetGraphicsContext

###Description###
Get the graphics context of the image

###Syntax###
#include <GDIPlus.au3>
_GDIPlus_ImageGetGraphicsContext ( $hImage )


###Parameters###
@@ParamTable@@
$hImage
	Handle to an image object
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	a handle to a Graphics object.
Failure:	-1 and sets the @error flag to non-zero, @extended may contain GPSTATUS ($GPID_ERR*).
@@End@@

None.
