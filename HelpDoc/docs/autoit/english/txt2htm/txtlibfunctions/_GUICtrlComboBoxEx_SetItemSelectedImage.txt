###User Defined Function###
_GUICtrlComboBoxEx_SetItemSelectedImage

###Description###
Sets the index of the item's overlay icon in the control's image list

###Syntax###
#include <GuiComboBoxEx.au3>
_GUICtrlComboBoxEx_SetItemSelectedImage ( $hWnd, $iIndex, $iImage )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$iIndex
	Zero based index of the item
$iImage
	Zero based index into the control's image list
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True
Failure:	False
@@End@@


###Remarks###
None.
