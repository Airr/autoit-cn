###User Defined Function###
_GUICtrlTreeView_GetSiblingCount

###Description###
Retrieves the number of siblings at the level of an item

###Syntax###
#include <GuiTreeView.au3>
_GUICtrlTreeView_GetSiblingCount ( $hWnd, $hItem )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
$hItem
	Handle to item
@@End@@

###ReturnValue###
Returns the Number of siblings


###Remarks###
None.


###Related###
_GUICtrlTreeView_GetCount, _GUICtrlTreeView_GetChildCount
