###User Defined Function###
_GUICtrlToolbar_GetStyleTransparent

###Description###
Indicates that the control is transparent

###Syntax###
#include <GuiToolbar.au3>
_GUICtrlToolbar_GetStyleTransparent ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
@@End@@

###ReturnValue###
True: Style is present
False: Style is not present


###Remarks###
None.
