###User Defined Function###
_GUICtrlHeader_SetItemOrder

###Description###
Sets the order in which the item appears

###Syntax###
#include <GuiHeader.au3>
_GUICtrlHeader_SetItemOrder ( $hWnd, $iIndex, $iOrder )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$iIndex
	Zero based item index
$iOrder
	Zero based item order
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True
Failure:	False
@@End@@


###Remarks###
None.
