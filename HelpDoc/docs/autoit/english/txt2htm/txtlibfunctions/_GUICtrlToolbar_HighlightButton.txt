###User Defined Function###
_GUICtrlToolbar_HighlightButton

###Description###
Sets the highlight state of a given button control

###Syntax###
#include <GuiToolbar.au3>
_GUICtrlToolbar_HighlightButton ( $hWnd, $iCommandID [, $fHighlight = True] )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$iCommandID
	Button command ID
$fHighlight
	[optional] Highlight state:
		True - Button will be highlighted
		False - Button will be unhighlighted
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True.
Failure:	False.
@@End@@
