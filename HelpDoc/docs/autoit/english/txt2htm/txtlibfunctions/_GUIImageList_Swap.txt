###User Defined Function###
_GUIImageList_Swap

###Description###
Swap image between Source and Destination image's index

###Syntax###
#include <GuiImageList.au3>
_GUIImageList_Swap ( $hWnd, $iSource, $iDestination )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the imagelist
$iSource
	The 0-based index of the image to be used as the source of the swap operation
$iDestination
	The 0-based index of the image to be used as the destination of the swap operation
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True.
Failure:	False.
@@End@@

None.
