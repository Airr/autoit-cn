###User Defined Function###
_GUICtrlRebar_GetBandText

###Description###
Retrieves the display text for the band

###Syntax###
#include <GuiReBar.au3>
_GUICtrlRebar_GetBandText ( $hWnd, $iIndex )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the rebar control
$iIndex
	Zero-based index of the band for which the information will be retrieved
@@End@@

###ReturnValue###
Returns the display text for the band


###Remarks###
None.
