###User Defined Function###
_GUICtrlToolbar_SetUnicodeFormat

###Description###
Sets the Unicode character format flag

###Syntax###
#include <GuiToolbar.au3>
_GUICtrlToolbar_SetUnicodeFormat ( $hWnd, $fUnicode = False )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$fUnicode
	Unicode character setting:
	True - Control uses Unicode characters
	False - Control uses ANSI characters
@@End@@

###ReturnValue###
Success: Previous character format flag setting


###Remarks###
None.
