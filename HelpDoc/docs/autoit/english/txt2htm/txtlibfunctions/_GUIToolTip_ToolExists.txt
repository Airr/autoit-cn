###User Defined Function###
_GUIToolTip_ToolExists

###Description###
Determines whether a current tool exists

###Syntax###
#include <GuiToolTip.au3>
_GUIToolTip_ToolExists ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
@@End@@

###ReturnValue###
True: Tool exists
False: Tool does not exist


###Remarks###
None.
