###User Defined Function###
_GUICtrlTab_SetItem

###Description###
Sets information about a tab

###Syntax###
#include <GuiTab.au3>
_GUICtrlTab_SetItem ( $hWnd, $iIndex [, $sText = -1 [, $iState = -1 [, $iImage = -1 [, $iParam = -1]]]] )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
$iIndex
	0-based item index
$sText
	[optional] Item text
$iState
	[optional] Item state
$iImage
	[optional] 0-based item image
$iParam
	[optional] Application defined data
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True.
Failure:	False.
@@End@@

None.
