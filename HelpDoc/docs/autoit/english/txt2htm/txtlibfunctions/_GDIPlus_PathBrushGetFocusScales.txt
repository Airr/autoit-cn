###User Defined Function###
_GDIPlus_PathBrushGetFocusScales

###Description###
Gets the focus scales of a path gradient brush

###Syntax###
#include <GDIPlus.au3>
_GDIPlus_PathBrushGetFocusScales ( $hPathGradientBrush )


###Parameters###
@@ParamTable@@
$hPathGradientBrush
	Pointer to a PathGradientBrush object
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	an array containing the focus scales:
		[0] - X focus scale
		[1] - Y focus scale
Failure:	sets the @error flag to non-zero, @extended may contain GPSTATUS ($GPID_ERR*).
@@End@@


###Remarks###
None.
