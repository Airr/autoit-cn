###User Defined Function###
_GDIPlus_PathBrushGetFocusScales

###Description###
Gets the focus scales of a path gradient brush

###Syntax###
#include <GDIPlus.au3>
_GDIPlus_PathBrushGetFocusScales ( $hPathGradientBrush )


###Parameters###
@@ParamTable@@
$hPathGradientBrush
	Pointer to a PathGradientBrush object
@@End@@

###ReturnValue###
Success: Array containing the focus scales:

	[0] - X focus scale
	[1] - Y focus scale
Failure: -1


###Remarks###
None
