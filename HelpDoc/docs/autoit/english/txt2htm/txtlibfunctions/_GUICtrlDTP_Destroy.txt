###User Defined Function###
_GUICtrlDTP_Destroy

###Description###
Delete the control

###Syntax###
#include <GuiDateTimePicker.au3>
_GUICtrlDTP_Destroy ( ByRef $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True, Handle is set to 0
Failure:	False
@@End@@


###Remarks###
Restricted to only be used on Date Time Picker controls created with _GUICtrlDTP_Create


###Related###
_GUICtrlDTP_Create
