###User Defined Function###
_GUICtrlButton_GetImage

###Description###
Retrieves a handle to the image (icon or bitmap) associated with the button

###Syntax###
#include <GuiButton.au3>
_GUICtrlButton_GetImage ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	Handle to the image
Failure:	0
@@End@@


###Remarks###
None.
