###User Defined Function###
_GUICtrlTab_GetItemText

###Description###
Retrieves the text of a tab

###Syntax###
#include <GuiTab.au3>
_GUICtrlTab_GetItemText ( $hWnd, $iIndex )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
$iIndex
	0-based item index
@@End@@

###ReturnValue###
Returns the tab text.


###Remarks###
None.
