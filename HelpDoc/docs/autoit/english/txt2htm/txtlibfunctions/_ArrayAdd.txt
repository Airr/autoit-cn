###User Defined Function###
_ArrayAdd

###Description###
Adds a specified value at the end of an existing array

###Syntax###
#include <Array.au3>
_ArrayAdd ( ByRef $aArray, $vValue )


###Parameters###
@@ParamTable@@
$aArray
	Array to modify
$vValue
	Value to add
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	the index of last added item.
Failure:	-1 and sets the @error flag to non-zero.
@error:	1 - $aArray is not an array
	2 - $aArray is not a 1 dimensional array
@@End@@

None.
