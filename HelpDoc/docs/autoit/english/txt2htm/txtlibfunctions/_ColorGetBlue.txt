###User Defined Function###
_ColorGetBlue

###Description###
Returns the blue component of a given color

###Syntax###
#include <Color.au3>
_ColorGetBlue ( $iColor )


###Parameters###
@@ParamTable@@
$iColor
	The RGB color to work with (0x00RRGGBB).
@@End@@

###ReturnValue###
Returns the component color in the range 0-255.


###Remarks###
None.
