###User Defined Function###
_GUICtrlToolbar_IsButtonIndeterminate

###Description###
Indicates whether the specified button is indeterminate

###Syntax###
#include <GuiToolbar.au3>
_GUICtrlToolbar_IsButtonIndeterminate ( $hWnd, $iCommandID )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$iCommandID
	Button command ID
@@End@@

###ReturnValue###
@@ReturnTable@@
True:	the button is indeterminate.
False:	the button is not indeterminate.
@@End@@

None.
