###User Defined Function###
_GUICtrlComboBoxEx_GetItemTextLen

###Description###
Gets the length, in characters, of a string in the list of a combo box

###Syntax###
#include <GuiComboBoxEx.au3>
_GUICtrlComboBoxEx_GetItemTextLen ( $hWnd, $iIndex )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$iIndex
	index item
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	the length of the string.
Failure:	-1.
@@End@@
