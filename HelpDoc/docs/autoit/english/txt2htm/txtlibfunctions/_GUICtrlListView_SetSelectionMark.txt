###User Defined Function###
_GUICtrlListView_SetSelectionMark

###Description###
Sets the selection mark in the control

###Syntax###
#include <GuiListView.au3>
_GUICtrlListView_SetSelectionMark ( $hWnd, $iIndex )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
$iIndex
	Zero based index of the new selection mark. If -1, the selection mark is removed.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	The previous selection mark
Failure:	-1 if there is no previous selection mark
@@End@@


###Remarks###
The selection mark is the item index from which a multiple selection starts. This message does not affect the
selection state of the item.
