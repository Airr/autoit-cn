###User Defined Function###
_GUICtrlListView_SetView

###Description###
Sets the view of the control

###Syntax###
#include <GuiListView.au3>
_GUICtrlListView_SetView ( $hWnd, $iView )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
$iView
	View state for the control:
		0 - Details
		1 - Large Icon
		2 - List
		3 - Small Icon
		4 - Tile
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True.
Failure:	False.
@@End@@

None.
