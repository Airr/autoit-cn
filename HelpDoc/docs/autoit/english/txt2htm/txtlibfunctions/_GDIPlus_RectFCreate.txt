###User Defined Function###
_GDIPlus_RectFCreate

###Description###

###Syntax###
#include <GDIPlus.au3>
_GDIPlus_RectFCreate ( [$iX = 0 [, $iY = 0 [, $iWidth = 0 [, $iHeight = 0]]]] )


###Parameters###
@@ParamTable@@
$iX
	[optional] X coordinate of upper left hand corner of rectangle
$iY
	[optional] Y coordinate of upper left hand corner of rectangle
$iWidth
	[optional] Rectangle width
$iHeight
	[optional] Rectangle height
@@End@@

###ReturnValue###
Returns a $tagGDIPRECTF structure.
