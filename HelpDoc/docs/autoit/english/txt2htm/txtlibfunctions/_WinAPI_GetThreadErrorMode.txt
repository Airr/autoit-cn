###User Defined Function###
_WinAPI_GetThreadErrorMode

###Description###
Retrieves the error mode for the calling thread

###Syntax###
#include <WinAPIProc.au3>
_WinAPI_GetThreadErrorMode (  )


###Parameters###
None.

###ReturnValue###
Returns the process error mode ($SEM_*).


###Remarks###
This function requires Windows 7 or later.
