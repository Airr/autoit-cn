###User Defined Function###
_GUICtrlComboBoxEx_GetExtendedStyle

###Description###
Gets the extended styles that are in use for a ComboBoxEx control

###Syntax###
#include <GuiComboBoxEx.au3>
_GUICtrlComboBoxEx_GetExtendedStyle ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
@@End@@

###ReturnValue###
Success: DWORD value that contains the ComboBoxEx control extended styles in use for the control


###Remarks###
None.
