###User Defined Function###
_GUICtrlTab_GetToolTips

###Description###
Retrieves the handle to the ToolTip control associated with the control

###Syntax###
#include <GuiTab.au3>
_GUICtrlTab_GetToolTips ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	Handle to the ToolTip control
Failure:	0
@@End@@


###Remarks###
None.
