###User Defined Function###
_GUICtrlTab_InsertItem

###Description###
Inserts a new tab

###Syntax###
#include <GuiTab.au3>
_GUICtrlTab_InsertItem ( $hWnd, $iIndex, $sText [, $iImage = -1 [, $iParam = 0]] )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
$iIndex
	Index of the new tab
$sText
	Tab text
$iImage
	[optional] Zero based tab image index
$iParam
	[optional] Application defined data
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	The zero based index of the new tab
Failure:	-1
@@End@@
