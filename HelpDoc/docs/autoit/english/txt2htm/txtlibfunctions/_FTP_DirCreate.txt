###User Defined Function###
_FTP_DirCreate

###Description###
Makes an Directory on an FTP server.

###Syntax###
#include <FTPEx.au3>
_FTP_DirCreate ( $hFTPSession, $sRemote )


###Parameters###
@@ParamTable@@
$hFTPSession
	as returned by _FTP_Connect().
$sRemote
	The Directory to Create.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	1.
Failure:	0.
@@End@@


###Remarks###
None.
