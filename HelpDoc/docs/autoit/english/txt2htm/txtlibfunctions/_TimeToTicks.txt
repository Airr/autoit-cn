###User Defined Function###
_TimeToTicks

###Description###
Converts the specified hours, minutes, and seconds to ticks.

###Syntax###
#include <Date.au3>
_TimeToTicks ( [$iHours = @HOUR [, $iMins = @MIN [, $iSecs = @SEC]]] )


###Parameters###
@@ParamTable@@
$iHours
	[optional] The hours.
$iMins
	[optional] The minutes.
$iSecs
	[optional] The seconds.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	Returns the number of ticks.
Failure:	0
@error:	0 - No error.
	1 - The specified hours, minutes, or seconds are not valid.
@@End@@

None.
