###User Defined Function###
_GUICtrlTab_SetCurSel

###Description###
Selects a tab

###Syntax###
#include <GuiTab.au3>
_GUICtrlTab_SetCurSel ( $hWnd, $iIndex )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
$iIndex
	Zero based item index
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	The index of the previously selected tab
Failure:	-1
@@End@@


###Remarks###
A tab control does not send a $TCN_SELCHANGING or $TCN_SELCHANGE notification message when
a tab is selected using this function.
