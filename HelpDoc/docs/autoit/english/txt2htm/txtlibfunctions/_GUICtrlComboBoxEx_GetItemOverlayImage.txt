###User Defined Function###
_GUICtrlComboBoxEx_GetItemOverlayImage

###Description###
Retrieves the index of the item's overlay image icon

###Syntax###
#include <GuiComboBoxEx.au3>
_GUICtrlComboBoxEx_GetItemOverlayImage ( $hWnd, $iIndex )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$iIndex
	0-based item index
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	a 0-based item overlay image index.
Failure:	-1.
@@End@@


###Remarks###
None.
