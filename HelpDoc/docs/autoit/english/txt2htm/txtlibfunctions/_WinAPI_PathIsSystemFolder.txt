###User Defined Function###
_WinAPI_PathIsSystemFolder

###Description###
Determines if an existing folder contains the attributes that make it a system folder.

###Syntax###
#include <WinAPIShPath.au3>
_WinAPI_PathIsSystemFolder ( $sPath )


###Parameters###
@@ParamTable@@
$sPath
	The name of an existing folder to check the system folder attributes.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success 	True - The path is a system folder.
Failure 	False.
@@End@@
