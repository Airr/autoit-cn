###User Defined Function###
_GUICtrlToolbar_SetStyleTransparent

###Description###
Sets whether the control is transparent

###Syntax###
#include <GuiToolbar.au3>
_GUICtrlToolbar_SetStyleTransparent ( $hWnd, $fState )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$fState
	True to set, false to unset
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True
Failure:	False
@@End@@


###Remarks###
None.
