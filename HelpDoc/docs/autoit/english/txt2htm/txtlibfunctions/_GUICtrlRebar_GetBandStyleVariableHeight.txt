###User Defined Function###
_GUICtrlRebar_GetBandStyleVariableHeight

###Description###
Determine if flag is set

###Syntax###
#include <GuiReBar.au3>
_GUICtrlRebar_GetBandStyleVariableHeight ( $hWnd, $iIndex )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the rebar control
$iIndex
	Zero-based index of the band
@@End@@

###ReturnValue###
True: Flag is set (The band can be resized by the rebar control)
False: Flag not set


###Remarks###
None.


###Related###
_GUICtrlRebar_SetBandStyleVariableHeight, _GUICtrlRebar_GetBandStyle
