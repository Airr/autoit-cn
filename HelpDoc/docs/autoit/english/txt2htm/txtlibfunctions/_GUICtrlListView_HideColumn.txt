###User Defined Function###
_GUICtrlListView_HideColumn

###Description###
Hides the column "sets column width to zero"

###Syntax###
#include <GuiListView.au3>
_GUICtrlListView_HideColumn ( $hWnd, $iCol )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
$iCol
	Column to hide
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True.
Failure:	False.
@@End@@
