###User Defined Function###
_GUICtrlToolbar_GetStyleRegisterDrop

###Description###
Indicates that the control generates TBN_GETOBJECT notification messages

###Syntax###
#include <GuiToolbar.au3>
_GUICtrlToolbar_GetStyleRegisterDrop ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
@@End@@

###ReturnValue###
True: Style is present
False: Style is not present


###Remarks###
None.
