###User Defined Function###
_GUICtrlListBox_SetCaretIndex

###Description###
Set the focus rectangle to the item at the specified index in a multiple-selection list box

###Syntax###
#include <GuiListBox.au3>
_GUICtrlListBox_SetCaretIndex ( $hWnd, $iIndex [, $fPartial = False] )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
$iIndex
	Specifies the zero based index of the item
$fPartial
	[optional] If False, the item is scrolled until it is fully visible; if it is True, the item is scrolled
	until it is at least partially visible.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True
Failure:	False
@@End@@


###Remarks###
None.
