###User Defined Function###
_GUICtrlTreeView_IsFirstItem

###Description###
Indicates whether the tree item is very first

###Syntax###
#include <GuiTreeView.au3>
_GUICtrlTreeView_IsFirstItem ( $hWnd, $hItem )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
$hItem
	Handle to the item
@@End@@

###ReturnValue###
@@ReturnTable@@
True:	Item is the first item.
False:	Item is not the first item.
@@End@@

None.
