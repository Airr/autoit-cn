###User Defined Function###
_GUIToolTip_GetTipBkColor

###Description###
Retrieves the background color

###Syntax###
#include <GuiToolTip.au3>
_GUIToolTip_GetTipBkColor ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the ToolTip control (returned by <a href ="_GUIToolTip_Create.htm">_GUIToolTip_Create.</a>)
@@End@@

###ReturnValue###
Returns the COLORREF (BGR) value of the background color of the ToolTip control.


###Remarks###
None.
