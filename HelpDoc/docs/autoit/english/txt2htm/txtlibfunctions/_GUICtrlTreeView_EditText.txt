###User Defined Function###
_GUICtrlTreeView_EditText

###Description###
Begins in-place editing of the specified item's text

###Syntax###
#include <GuiTreeView.au3>
_GUICtrlTreeView_EditText ( $hWnd, $hItem )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
$hItem
	Handle to the item to edit
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	The handle to the edit control
Failure:	0
@@End@@


###Remarks###
None.
