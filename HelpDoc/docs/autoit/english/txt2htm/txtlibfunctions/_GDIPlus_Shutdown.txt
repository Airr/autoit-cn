###User Defined Function###
_GDIPlus_Shutdown

###Description###
Clean up resources used by Microsoft Windows GDI+

###Syntax###
#include <GDIPlus.au3>
_GDIPlus_Shutdown (  )


###Parameters###
None.

###ReturnValue###
@@ReturnTable@@
Success:	True
Failure:	False
@@End@@


###Remarks###
You must dispose of all of your GDI+ objects before you call _GDIPlus_Shutdown
