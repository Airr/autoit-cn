###User Defined Function###
_GUICtrlRebar_GetBKColor

###Description###
Retrieves a rebar control's default background color

###Syntax###
#include <GuiReBar.au3>
_GUICtrlRebar_GetBKColor ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the rebar control
@@End@@

###ReturnValue###
Returns a COLORREF value that represent the current default background color


###Remarks###
None.
