###User Defined Function###
_WinAPI_PathIsFileSpec

###Description###
Searches a path for any path-delimiting characters.

###Syntax###
#include <WinAPIShPath.au3>
_WinAPI_PathIsFileSpec ( $sPath )


###Parameters###
@@ParamTable@@
$sPath
	The path to be searched.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success 	True - There are no path-delimiting characters (":" or "\") within the path.
Failure 	False.
@@End@@
