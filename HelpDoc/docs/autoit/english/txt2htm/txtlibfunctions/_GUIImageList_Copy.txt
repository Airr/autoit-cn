###User Defined Function###
_GUIImageList_Copy

###Description###
Source image is copied to the destination image's index

###Syntax###
#include <GuiImageList.au3>
_GUIImageList_Copy ( $hWnd, $iSource, $iDestination )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the imagelist
$iSource
	The 0-based index of the image to be used as the source of the copy operation
$iDestination
	The 0-based index of the image to be used as the destination of the copy operation
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True.
Failure:	False.
@@End@@


###Remarks###
None.
