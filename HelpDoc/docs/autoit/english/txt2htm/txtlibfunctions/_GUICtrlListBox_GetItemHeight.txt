###User Defined Function###
_GUICtrlListBox_GetItemHeight

###Description###
Retrieves the height of items

###Syntax###
#include <GuiListBox.au3>
_GUICtrlListBox_GetItemHeight ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	The height, in pixels, of each item in the list box
Failure:	0
@@End@@


###Remarks###
None.
