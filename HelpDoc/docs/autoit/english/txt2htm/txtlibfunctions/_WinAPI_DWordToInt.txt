###User Defined Function###
_WinAPI_DWordToInt

###Description###
Converts a value of type DWORD to a value of type INT

###Syntax###
#include <WinAPIMisc.au3>
_WinAPI_DWordToInt ( $iValue )


###Parameters###
@@ParamTable@@
$iValue
	The value to be converted.
@@End@@

###ReturnValue###
Returns the converted (INT) value.
