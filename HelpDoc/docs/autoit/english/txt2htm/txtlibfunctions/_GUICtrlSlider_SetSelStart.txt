###User Defined Function###
_GUICtrlSlider_SetSelStart

###Description###
Sets the starting logical position of the current selection range

###Syntax###
#include <GuiSlider.au3>
_GUICtrlSlider_SetSelStart ( $hWnd, $iMinimum )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
$iMinimum
	Starting logical position for the selection range
@@End@@

###ReturnValue###
None.


###Remarks###
None.
