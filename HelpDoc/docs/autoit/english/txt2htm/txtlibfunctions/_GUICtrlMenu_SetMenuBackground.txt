###User Defined Function###
_GUICtrlMenu_SetMenuBackground

###Description###
Sets the background brush for the menu

###Syntax###
#include <GuiMenu.au3>
_GUICtrlMenu_SetMenuBackground ( $hMenu, $hBrush )


###Parameters###
@@ParamTable@@
$hMenu
	Handle of the menu
$hBrush
	Brush to use for the background
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True

@@End@@
