###User Defined Function###
_GUICtrlTreeView_Delete

###Description###
Removes an item and all its children

###Syntax###
#include <GuiTreeView.au3>
_GUICtrlTreeView_Delete ( $hWnd [, $hItem = 0] )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
$hItem
	[optional] Handle/Control ID of item
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True
Failure:	False
@@End@@


###Remarks###
None.
