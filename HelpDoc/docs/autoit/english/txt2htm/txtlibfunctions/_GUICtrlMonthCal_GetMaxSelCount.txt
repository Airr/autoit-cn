###User Defined Function###
_GUICtrlMonthCal_GetMaxSelCount

###Description###
Retrieves the maximum date range that can be selected in a month calendar control

###Syntax###
#include <GuiMonthCal.au3>
_GUICtrlMonthCal_GetMaxSelCount ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
@@End@@

###ReturnValue###
Success: INT value that represents the total number of days that can be selected for the control


###Remarks###
None.
