###User Defined Function###
_GDIPlus_LineBrushGetColors

###Description###
Gets the starting color and ending color of a linear gradient brush

###Syntax###
#include <GDIPlus.au3>
_GDIPlus_LineBrushGetColors ( $hLineGradientBrush )


###Parameters###
@@ParamTable@@
$hLineGradientBrush
	Pointer to a LinearGradientBrush object
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	Array containing the starting and ending colors of the LinearGradientBrush:
			[0] - Starting color
			[1] - Ending color
Failure:	-1 and either:
			@error and @extended are set if DllCall failed
@@End@@

None.
