###User Defined Function###
_GUICtrlTreeView_GetText

###Description###
Retrieve the item text

###Syntax###
#include <GuiTreeView.au3>
_GUICtrlTreeView_GetText ( $hWnd [, $hItem = 0] )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
$hItem
	[optional] item ID/handle
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	the text from item.
Failure:	an empty string.
@@End@@

None.
