###User Defined Function###
_GUICtrlTreeView_GetParentParam

###Description###
Retrieve the parent control ID/Param of item

###Syntax###
#include <GuiTreeView.au3>
_GUICtrlTreeView_GetParentParam ( $hWnd [, $hItem = 0] )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
$hItem
	[optional] item ID/Param
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	the parent control param.
Failure:	False.
@@End@@
