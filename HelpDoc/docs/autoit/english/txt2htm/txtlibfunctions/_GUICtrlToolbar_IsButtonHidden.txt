###User Defined Function###
_GUICtrlToolbar_IsButtonHidden

###Description###
Indicates whether the specified button is hidden

###Syntax###
#include <GuiToolbar.au3>
_GUICtrlToolbar_IsButtonHidden ( $hWnd, $iCommandID )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$iCommandID
	Button command ID
@@End@@

###ReturnValue###
True: Button is hidden
False: Button is not hidden


###Remarks###
None.
