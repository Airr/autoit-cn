###User Defined Function###
_GUICtrlTab_SetItemText

###Description###
Sets the text of a tab

###Syntax###
#include <GuiTab.au3>
_GUICtrlTab_SetItemText ( $hWnd, $iIndex, $sText )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
$iIndex
	0-based item index
$sText
	Item text
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True.
Failure:	False.
@@End@@

None.
