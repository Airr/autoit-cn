###User Defined Function###
_RunDos

###Description###
Executes a DOS command in a hidden command window

###Syntax###
#include <Process.au3>
_RunDos ( $sCommand )


###Parameters###
@@ParamTable@@
$sCommand
	Command to execute
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	the exit code of the command.
Failure:	0 and sets the @error flag to non-zero.
@@End@@
