###User Defined Function###
_WinAPI_GetRValue

###Description###
Retrieves an intensity value for the red component of a 32-bit RGB value.

###Syntax###
#include <WinAPIGdi.au3>
_WinAPI_GetRValue ( $iRGB )


###Parameters###
@@ParamTable@@
$iRGB
	The color value, in RGB.
@@End@@

###ReturnValue###
Returns the intensity of the red component of the specified RGB color.
