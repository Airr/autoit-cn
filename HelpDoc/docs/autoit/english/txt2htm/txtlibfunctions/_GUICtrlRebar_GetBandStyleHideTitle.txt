###User Defined Function###
_GUICtrlRebar_GetBandStyleHideTitle

###Description###
Determine if band hide title flag is set

###Syntax###
#include <GuiReBar.au3>
_GUICtrlRebar_GetBandStyleHideTitle ( $hWnd, $iIndex )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the rebar control
$iIndex
	0-based index of the band
@@End@@

###ReturnValue###
@@ReturnTable@@
True:	the flag is set (Keep band title hidden).
False:	the flag not set.
@@End@@


###Remarks###
None.

_GUICtrlRebar_GetBandStyle, _GUICtrlRebar_SetBandStyleHideTitle
