###User Defined Function###
_GUICtrlMonthCal_GetMonthRangeMaxStr

###Description###
Retrieves date information that represents the high limit of the controls display in string format

###Syntax###
#include <GuiMonthCal.au3>
_GUICtrlMonthCal_GetMonthRangeMaxStr ( $hWnd [, $sFormat = "%02d/%02d/%04d"] )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
$sFormat
	[optional] StringFormat string used to format the date
@@End@@

###ReturnValue###
Returns a date in the specified format.


###Remarks###
None.
