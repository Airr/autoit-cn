###User Defined Function###
_GUICtrlToolbar_MoveButton

###Description###
Moves a button from one index to another

###Syntax###
#include <GuiToolbar.au3>
_GUICtrlToolbar_MoveButton ( $hWnd, $iOldPos, $iNewPos )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$iOldPos
	0-based index of the button to be moved
$iNewPos
	0-based index where the button will be moved
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True.
Failure:	False.
@@End@@
