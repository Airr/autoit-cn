###User Defined Function###
_GUICtrlMenu_SetItemGrayed

###Description###
Sets the grayed state of a menu item

###Syntax###
#include <GuiMenu.au3>
_GUICtrlMenu_SetItemGrayed ( $hMenu, $iItem [, $fState = True [, $fByPos = True]] )


###Parameters###
@@ParamTable@@
$hMenu
	Menu handle
$iItem
	Identifier or position of the menu item
$fState
	[optional] Item state to set:
	True - State is enabled
	False - State is disabled
$fByPos
	[optional] Menu identifier flag:
	True - $iItem is a zero based item position
	False - $iItem is a menu item identifier
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True
Failure:	False
@@End@@


###Remarks###
None.
