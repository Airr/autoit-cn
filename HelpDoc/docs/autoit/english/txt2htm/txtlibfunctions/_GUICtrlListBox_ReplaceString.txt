###User Defined Function###
_GUICtrlListBox_ReplaceString

###Description###
Replaces the text of an item

###Syntax###
#include <GuiListBox.au3>
_GUICtrlListBox_ReplaceString ( $hWnd, $iIndex, $sText )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
$iIndex
	Specifies the 0-based index of the item
$sText
	String to replace old string
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True.
Failure:	False.
@@End@@
