###User Defined Function###
_GUICtrlSlider_SetLineSize

###Description###
Sets the number of logical positions the slider moves

###Syntax###
#include <GuiSlider.au3>
_GUICtrlSlider_SetLineSize ( $hWnd, $iLineSize )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
$iLineSize
	New line size
@@End@@

###ReturnValue###
Success: The previous line size


###Remarks###
The default setting for the line size is 1
