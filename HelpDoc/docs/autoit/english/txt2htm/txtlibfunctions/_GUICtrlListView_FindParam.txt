###User Defined Function###
_GUICtrlListView_FindParam

###Description###
Searches for an item with the specified lParam

###Syntax###
#include <GuiListView.au3>
_GUICtrlListView_FindParam ( $hWnd, $iParam [, $iStart = -1] )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
$iParam
	Param value to search for
$iStart
	[optional] Zero based index of the item to begin the search with or -1 to start from the beginning. The
	specified item is itself excluded from the search.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	The zero based index of the item
Failure:	-1
@@End@@


###Remarks###
None.
