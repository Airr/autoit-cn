###User Defined Function###
_WinAPI_PathUnmakeSystemFolder

###Description###
Removes the attributes from a folder that make it a system folder.

###Syntax###
#include <WinAPIShPath.au3>
_WinAPI_PathUnmakeSystemFolder ( $sPath )


###Parameters###
@@ParamTable@@
$sPath
	The name of an existing folder that will have the system folder attributes removed.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success 	True
Failure 	False
@@End@@
