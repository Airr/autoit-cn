###User Defined Function###
_GUICtrlTreeView_GetState

###Description###
Retrieve the state of the item

###Syntax###
#include <GuiTreeView.au3>
_GUICtrlTreeView_GetState ( $hWnd [, $hItem = 0] )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
$hItem
	[optional] item ID/handle
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	the state of the item.
Failure:	sets the @error flag to non-zero.
@@End@@


###Remarks###
None.
