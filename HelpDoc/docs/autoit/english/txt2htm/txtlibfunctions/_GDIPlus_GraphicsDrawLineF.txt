###User Defined Function###
_GDIPlus_GraphicsDrawLineF

###Description###
Draw a line

###Syntax###
#include <GDIPlus.au3>
_GDIPlus_GraphicsDrawLineF ( $hGraphics, $fX1, $fY1, $fX2, $fY2 [, $hPen = 0] )


###Parameters###
@@ParamTable@@
$hGraphics
	Handle to a Graphics object
$fX1
	The X coordinate of the starting point of the line
$fY1
	The Y coordinate of the starting point of the line
$fX2
	The X coordinate of the ending point of the line
$fY2
	The Y coordinate of the ending point of the line
$hPen
	[optional] Handle to a pen object that is used to draw the arc. If 0, a solid black pen with a width of 1
@@End@@

###ReturnValue###
Success: True
Failure: False


###Remarks###
None.
