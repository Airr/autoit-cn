###User Defined Function###
_WinAPI_MakeWord

###Description###
Returns a WORD (16-bit) value from two BYTE (8-bit) values.

###Syntax###
#include <WinAPIMisc.au3>
_WinAPI_MakeWord ( $iLo, $iHi )


###Parameters###
@@ParamTable@@
$iLo
	Low byte.
$iHi
	Hi byte.
@@End@@

###ReturnValue###
Returns the WORD value.

None
