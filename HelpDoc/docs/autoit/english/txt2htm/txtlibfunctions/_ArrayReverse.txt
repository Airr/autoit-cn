###User Defined Function###
_ArrayReverse

###Description###
Takes the given array and reverses the order in which the elements appear in the array

###Syntax###
#include <Array.au3>
_ArrayReverse ( ByRef $aArray [, $iStart = 0 [, $iEnd = 0]] )


###Parameters###
@@ParamTable@@
$aArray
	Array to modify
$iStart
	[optional] Index of array to start modifying at
$iEnd
	[optional] Index of array to stop modifying at
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	1.
Failure:	0 and sets the @error flag to non-zero.
@error:	1 - $aArray is not an array
	2 - $iStart is greater than $iEnd
	3 - $aArray is not a 1 dimensional array
	4 - $aArray is empty
@@End@@
