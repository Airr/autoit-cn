###User Defined Function###
_WinAPI_SetLayeredWindowAttributes

###Description###
Sets Layered Window Attributes

###Syntax###
#include <WinAPI.au3>
_WinAPI_SetLayeredWindowAttributes ( $hWnd, $iTransColor [, $hTransGUI = 255 [, $iFlags = 0x03 [, $fColorRef = False]]] )


###Parameters###
@@ParamTable@@
$hWnd
	Handle of GUI to work on
$iTransColor
	Transparent color
$hTransGUI
	[optional] Set Transparancy of GUI
$iFlags
	[optional] Flags.
$fColorRef
	[optional] If True, $iTransColor is a COLORREF( 0x00bbggrr ), else an RGB-Color
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True
Failure:	False, call <a href="_WinAPI_GetLastError.htm">_WinAPI_GetLastError()</a> to get extended error information
@@End@@

None.
