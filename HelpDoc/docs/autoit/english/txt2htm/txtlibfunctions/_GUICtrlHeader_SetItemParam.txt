###User Defined Function###
_GUICtrlHeader_SetItemParam

###Description###
Sets the param value of the item

###Syntax###
#include <GuiHeader.au3>
_GUICtrlHeader_SetItemParam ( $hWnd, $iIndex, $iParam )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$iIndex
	Zero based item index
$iParam
	Item param value
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True
Failure:	False
@@End@@

None.
