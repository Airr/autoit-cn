###User Defined Function###
_WinAPI_SetProcessWindowStation

###Description###
Assigns the specified window station to the calling process.

###Syntax###
#include <WinAPISys.au3>
_WinAPI_SetProcessWindowStation ( $hStation )


###Parameters###
@@ParamTable@@
$hStation
	Handle to the window station. This window station must be associated with the current session.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success 	True
Failure 	False
@@End@@
