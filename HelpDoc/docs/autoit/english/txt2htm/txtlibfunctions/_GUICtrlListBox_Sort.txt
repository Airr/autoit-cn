###User Defined Function###
_GUICtrlListBox_Sort

###Description###
Re-sorts list box if it has the $LBS_SORT style

###Syntax###
#include <GuiListBox.au3>
_GUICtrlListBox_Sort ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Control ID/Handle to the control
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True.
Failure:	False.
@@End@@


###Remarks###
Re-sorts list box if it has the $LBS_SORT style.
Might be useful if you use <a href="_GUICtrlListBox_InsertString.htm">_GUICtrlListBox_InsertString()</a>.
