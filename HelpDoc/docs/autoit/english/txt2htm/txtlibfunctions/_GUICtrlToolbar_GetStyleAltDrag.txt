###User Defined Function###
_GUICtrlToolbar_GetStyleAltDrag

###Description###
Indicates that the control allows buttons to be dragged

###Syntax###
#include <GuiToolbar.au3>
_GUICtrlToolbar_GetStyleAltDrag ( $hWnd )


###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
@@End@@

###ReturnValue###
True: Style is present
False: Style is not present


###Remarks###
None.
