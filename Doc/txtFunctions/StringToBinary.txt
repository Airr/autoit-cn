###Function###
StringToBinary

###Description###
转换字符串为二进制数据. 

###Syntax###
StringToBinary ( 表达式[,标志]) 


###Parameters###
@@ParamTable@@
表达式
	需要转换的字符串表达式.
标志
	[可选参数] 修改数据编码转换格式:
	标志 = 1 (默认), 二进制数据为 ANSI 编码
	标志 = 2, 二进制数据为 UTF16 小编码
	标志 = 3, 二进制数据为 UTF16 大编码
	标志 = 4, 二进制数据为 UTF8 编码
@@End@@


###ReturnValue###
返回一个二进制变量. 


###Remarks###
None.

###Related###
Binary, BinaryToString, IsBinary, String
