###Function###
TCPSend

###Description###
在已经连接的套接字(socket)上面发送数据.

###Syntax###
TCPSend ( 主套接字, 数据 )


###Parameters###
@@ParamTable@@
主套接字
	一个已经连接的套接字标识 (SocketID),返回于 <a href="TCPConnect.htm">TCPConnect</a> 函数.
数据
	要发送给已连接的套接字的二进制数据/字符串.   
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回发送到已经连接的套接字的字节数量.   
失败:	返回 0 并设置 @error 的值(依照 Windows API WSAGetError 返回).
@@End@@


###Remarks###
None.

TCPStartup, TCPConnect, TCPListen, TCPTimeout (Option)
