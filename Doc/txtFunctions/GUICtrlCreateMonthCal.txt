###Function###
GUICtrlCreateMonthCal

###Description###
在GUI上面创建一个月历控件.

###Syntax###
GUICtrlCreateMonthCal ( "文本", 左侧, 顶部 [, 宽度 [, 高度 [, 样式 [, 扩展样式]]]] )


###Parameters###
@@ParamTable@@
文本
	预选日期.(总是为 "yyyy/mm/dd" 格式).
左侧
	控件左侧的位置.若此值为 -1 则根据 <a href="AutoItSetOption.htm#GUICoordMode">GUICoordMode</a> 的设置来计算左侧位置.
顶部
	控件上方的位置.若此值为 -1 则根据 <a href="AutoItSetOption.htm#GUICoordMode">GUICoordMode</a> 的设置来计算上方位置.
宽度
	[可选参数] 控件的宽度(默认值(default)为上一个控件的宽度).
高度
	[可选参数] 控件的高度(默认值(default)为上一个控件的高度).
样式
	[可选参数] 指定控件的样式.请查看附录中关于 <a href="../appendix/GUIStyles.htm#MonthCal">GUI 控件样式</a> 的说明.
	
	默认(default)(-1): 无.
	强制性样式: $WS_TABSTOP
扩展样式
	[可选参数] 指定控件的扩展样式.请查看附录的 <a href="../appendix/GUIStyles.htm#Extended">扩展样式表</a>.
	默认值(default) (-1) : WS_EX_CLIENTEDGE
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回新控件的控件标识符(控件ID).
失败:	返回值为 0.
@@End@@


###Remarks###
若要获得控件的各种值,请参考 <a href="GUICtrlRead.htm">GUICtrlRead</a>.

默认大小为: $GUI_DOCKSIZE.


###Related###
GUICoordMode (Option), GUIGetMsg, GUICtrlRead
