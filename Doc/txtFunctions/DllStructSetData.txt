###Function###
DllStructSetData

###Description###
在 struct 中设置数据元素.

###Syntax###
DllStructSetData ( Struct, Element, value [, index ] )


###Parameters###
@@ParamTable@@
Struct
	被 DllStructCreate 送回的 struct.
Element
	你想要存取的 struct 的一个元素, 起始于 1 或 DllStructCreate 所定义的元素名.
value
	放到 struct 内的新值.
index
	如果元素是一个数组，你需要设定一个索引，否则它会设定索引为 1. 第一个元素是 1.
@@End@@

###ReturnValue###
@@ReturnTable@@
成功: 	值, 从 struct 读取.
失败:	0.
@Error:	0 = 没有错误. 
	1 = 从 DllStructCreate 送回的 Struct not 不正确. 
	2 = 元素值超出范围. 
	3 = 索引在 struct 的外面. 
	4 = 元素数据种类未知 
	5 = 索引 &lt;= 0.
@@End@@


###Remarks###
当元素是字符[n]时, byte[n] 或 ubyte[n] 数据可以是字串，否则一定是个数字.

###Related###
DllCall, DllStructCreate, DllStructGetData
