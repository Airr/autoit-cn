###Function###
StringToASCIIArray

###Description###
对包含ASCII[代]码的数组的每个字符转换一个字串.

###Syntax###
StringToASCIIArray( "字符串", [ 开始 [, 结束 [, 编码 ] ] ] )


###Parameters###
@@ParamTable@@
"字符串"
	转换数组 ASCII[代]码的字串.
开始
	[可选] 以 0 为基础的启动处理位置 (默认: 0).
结束
	[可选] 以 0 为基础结束处理的位置 (默认: StringLen("字符串")).
编码
	[可选] 被返回的数组将包含指定字符集的值:
		0 - UTF-16 (默认)
		1 - ANSI
		2 - UTF-8
@@End@@


###ReturnValue###
成功 - 每个元素在对应位置是统一码字符的数组[代]码.
失效 - 返回一个空串.

###Remarks###
字串可能包含嵌入的 Chr(0).  These will appear in the returned array as well as any data past them.  当字串转换结束时，唯一的处理是停机或用户指定的结束.
