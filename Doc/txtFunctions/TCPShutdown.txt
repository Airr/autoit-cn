###Function###
TCPShutdown, UDPShutdown

###Description###
停止 TCP/UDP 服务. 


###Syntax###
TCPShutdown ( )
UDPShutdown ( )


###Parameters###
None.


###ReturnValue###
@@ReturnTable@@
成功:	返回1.
失败:	返回 0 并设置 @error 的值(依照 Windows API WSACleanup 返回).    
@@End@@


###Remarks###
每当成功完成了一个TCPStartup()调用,必须调用一个TCPShutdown()来关闭初始化的网络.
UDPShutdown() 是 TCPShutdown()的一个别名.

###Related###
TCPStartup, TCPListen, TCPCloseSocket, UDPCloseSocket
