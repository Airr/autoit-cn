###Function###
TrayCreateItem

###Description###
在系统托盘上面创建一个菜单项目控件.

###Syntax###
TrayCreateItem ( 文本 [, 菜单ID [, 菜单索引 [, 菜单可选项目]]] )


###Parameters###
@@ParamTable@@
文本
	菜单项目显示的文本.
菜单ID
	[可选参数] 定义此子菜单要附加到的目标菜单ID.若此值为 -1 则使用第一级菜单(默认)
菜单索引
	[可选参数] 允许定义要创建的菜单项的(次序)编号.编号从0开始.若此值为 -1 则添加到最后一个编号之后(默认).
菜单可选项目
	[可选参数] 0 (默认) = 创建一个标准菜单项目, 1 = 创建一个可选菜单项目.
@@End@@


###ReturnValue###
@@ReturnTable@@
成功:	返回创建的菜单项目的控件ID (controlID).
失败:	返回 0.
@@End@@


###Remarks###
如果 '文本' 为空字符串 ( "" ) 就添加一个分割线.

默认状态,正常选中的菜单项目 (不是单选项目)将会在您单击后自动取消选中!
要关闭这个选项,请设置 <a href="../functions/AutoItSetOption.htm#TrayMenuMode">TrayMenuMode</a> 里面的值为 2. 

单选项目会自动集合在一个组内,如果包含分割线( separator ),将不会显示为单选项目.
默认状态,正常选中的项目会选中,其它项目会取消选中状态!
要关闭这个选项,请设置 <a href="../functions/AutoItSetOption.htm#TrayMenuMode">TrayMenuMode</a> 里面的值为 8.
