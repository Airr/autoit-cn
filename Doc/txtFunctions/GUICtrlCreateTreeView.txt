###Function###
GUICtrlCreateTreeView

###Description###
在GUI上创建一个 TreeView 控件.

###Syntax###
GUICtrlCreateTreeView ( 左侧, 上方 [, 宽度 [, 高度 [, 样式 [, 扩展样式]]]] )


###Parameters###
@@ParamTable@@
左侧  
	控件左侧的位置.若此值为 -1 则根据 <a href="AutoItSetOption.htm#GUICoordMode">GUICoordMode</a> 的设置来计算左侧位置.   
上方  
	控件上方的位置.若此值为 -1 则根据 <a href="AutoItSetOption.htm#GUICoordMode">GUICoordMode</a> 的设置来计算上方位置.   
宽度  	
	[可选参数] 控件的宽度(默认值为上一个控件的宽度).   
高度  
	[可选参数] 控件的高度(默认值为上一个控件的高度).   
样式  
	[可选参数] 指定控件的样式.请查看附录中关于  <a href="../appendix/GUIStyles.htm#TreeView">GUI 控件样式</a>  的说明.
	
	默认值(-1)：$TVS_HASBUTTONS, $TVS_HASLINES, $TVS_LINESATROOT, $TVS_DISABLEDRAGDROP, $TVS_SHOWSELALWAYS 
	强制性风格： $WS_TABSTOP
扩展样式  
	[可选参数] 指定控件的扩展样式.请查看附录的 <a href="../appendix/GUIStyles.htm#Extended">扩展样式表</a>.
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回控件标识符(控件ID).   
失败:	返回值为0.
@@End@@


###Remarks###
若要设置或修改控件的各种信息请查看 <a href="..\guiref\GUICtrlSet.htm">GUICtrlSet...</a>.

要在默认样式的基础上添加一个新的样式可使用 <b>BitOr</b>($GUI_SS_DEFAULT_TREEVIEW, 新样式,...).

To use the values specified above you must #include <TreeViewConstants.au3> in your script.
