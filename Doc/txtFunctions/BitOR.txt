###Function###
BitOR

###Description###
BitOR 按位"或"运算.

###Syntax###
BitOR ( 值1, 值2 [, 值 n] ) 


###Parameters###
@@ParamTable@@
值1
	第一个数.
值2
	第二个数.
值n
	[可选参数] 第N个数,最高可计算255个值.
@@End@@

###ReturnValue###
返回值 返回所有参数进行按位或运算后的结果.
位操作把所有数值当作32位整数来进行运算.


###Remarks###
记住数值可使用十六进制符号(0x).
BitOR returns 0 in each bit position where all input arguments have a 0 in the corresponding 
position and 1 wherever there is at least one 1-bit.


###Related###
BitAND, BitNOT, BitShift, BitXOR, Hex, BitRotate
