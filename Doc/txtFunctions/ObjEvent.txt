###Function###
ObjEvent

###Description###
包含一个已经得到的Object(对象)的事件句柄.

###Syntax###
ObjEvent ( $ObjectVar, "functionprefix" [, "接口名称"] )
ObjEvent ( "AutoIt.Error" [, "函数名称"] )

###Parameters###
@@ParamTable@@
$ObjectVar
	一个包含要接收事件对象的变量
"functionprefix"
	定义接收事件函数句柄的前缀. 
	前缀后拼接对象方法名.
"接口名称"
	事件接口的可选名称. 
	Note: It must be a supported as outgoing for the
	Object AND it must be of type DISPATCH.
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 object (对象) 或者 函数名称.
失败:	返回 空字符串 "" 并设置 @error 为 1.
@@End@@


###Remarks###
第一行语句接收来自给定对象的事件. 
为了要接收一个特定的事件, 使用与事件名字一起附加的给定前缀产生一个 AutoIt 函数名.

第二行语句为 COM 差错处理. 如果任何的 COM 差错发生，给定的函数将被调用. 
如果第二个参数被省略, 它将会返回目前差错处理程式函数的名字, 如果存在.

参考 <a href="../intro/ComRef.htm">Obj/COM 参考</a> 相关说明.
