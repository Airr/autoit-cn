###Function###
AdlibUnRegister

###Description###
反注册 adlib 函数.

###Syntax###
AdlibUnRegister( [ "函数" ] )


###Parameters###
@@ParamTable@@
函数
	[可选参数:] 需要取消注册的 adlib 函数名称.参考备注得到更多信息.
@@End@@

###ReturnValue###
@@ReturnTable@@
The number of adlib functions currently registered.
@@End@@


###Remarks###
If the function name is not specified then the last registered function will be unregistered.

AdlibRegister
