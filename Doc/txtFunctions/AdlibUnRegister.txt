###Function###
AdlibUnRegister

###Description###
Unregisters an adlib function.

###Syntax###
AdlibUnRegister( "function" )


###Parameters###
@@ParamTable@@
function
	The name of the adlib function to be Unregistered.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	Returns 1.
Failure:	Returns 0.
@@End@@


###Remarks###
None.

###Related###
AdlibRegister, AdLibDisable
