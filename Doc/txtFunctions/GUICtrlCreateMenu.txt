###Function###
GUICtrlCreateMenu

###Description###
在GUI上创建一个菜单控件.

###Syntax###
GUICtrlCreateMenu ( "子菜单文本" [, menuID [, menuentry]] ) 


###Parameters###
@@ParamTable@@
子菜单文本
	子菜单文本.
menuID
	[可选参数] 定义此菜单项要附加到的目标菜单ID.若此值为 -1 则使用第一级菜单.   
menuentry
	[可选参数] 允许定义要创建的菜单项的(次序)编号.编号从0开始.
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回控件标识符(控件ID).
失败:	返回值为0.  
@@End@@


###Remarks###
To set or change information in the control see <a href="GUICtrlUpdate Managenent.htm">GUICtrlUpdate...</a>.
