###Function###
GUISetStyle

###Description###
改变窗口样式.

###Syntax###
GUISetStyle ( 样式 [,扩展样式 [, 窗口句柄]] )


###Parameters###
@@ParamTable@@
样式
	windows默认样式请看 <a href="../appendix/GUIStyles.htm">GUI Control Styles Appendix</a>.
	
	-1为不改变.
扩展样式
	[可选参数] 默认扩展样式请看 <a href="GUICreate.htm#Extended Style Table">Extended Style Table</a> below.  默认值为-1.
	-1为不改变.
窗口句柄
	[可选参数] 由GUICreate返回的窗口句柄 (默认为上一窗口句柄).
@@End@@


###ReturnValue###
@@ReturnTable@@
成功:	返回 1.
失败:	返回 0.
@@End@@


###Remarks###
No checking is done on style value, neither non interaction with already defined control. It is the designer responsability to take care of this compatibility.


###Related###
GUICreate, GUIGetStyle
