###Function###
BitRotate

###Description###
执行位移操作, with rotation. 

###Syntax###
BitRotate ( 值, 位移[, 大小] ) 


###Parameters###
@@ParamTable@@
值
	要被操作的值.
位移
	左移转数 (负值右移).  如不指定, 默认 1.
大小
	[可选] A string that determines the rotation size, 默认 (16 bits). 见下文.
@@End@@
<br>
Size parameter :
@@ParamTable@@
"B"
	rotate bits within the low-order byte (8 bits).
"W"
	rotate bits within the low-order word (16 bits).
"D"
	rotate bits within the entire double-word (32 bits).
@@End@@

###ReturnValue###
Returns the value rotated by the required number of bits.
Bit operations are performed as 32-bit integers.


###Remarks###
Remember hex notation can be used for numbers.


###Related###
BitShift, BitAND, BitNOT, BitOR, BitXOR, Hex
