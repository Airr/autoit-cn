###Function###
UDPCloseSocket

###Description###
关闭一个 UDP 套接字(socket).


###Syntax###
UDPCloseSocket ( 套接字数组 )


###Parameters###
@@ParamTable@@
套接字数组
	一个由 <a href="UDPBind.htm">UDPBind</a> 或者 <a href="UDPOpen.htm">UDPOpen</a> 函数返回的套接字/数组.    
@@End@@


###ReturnValue###
@@ReturnTable@@
成功:	返回1.
失败:	返回0 并设置 @error 的值(依照 Windows API WSAGetError 返回).    
@@End@@


###Remarks###
None.

###Related###
UDPBind, UDPOpen, TCPShutdown, TCPStartup
