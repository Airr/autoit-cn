###Function###
GUICtrlRegisterListViewSort

###Description###
为 listview 控件注册一个用户定义排序函数.

###Syntax###
GUICtrlRegisterListViewSort ( controlID, "function" )

###Parameters###
@@ParamTable@@
controlID
	listview 控件ID.
function
	用户定义排序函数名称.
@@End@@


###ReturnValue###
@@ReturnTable@@
成功:	1
失败:	0
@@End@@


###Remarks###
!!! 要使用户定义函数能工作，必须定义它 最多4个参数 ，否则函数将不能被调用 !!!
例如:
Func MySortFunction($nListViewID, $LParam1, $LParam2, $nColumn)
...
EndFunc

或

Func MySortFunction($nListViewID, $LParam1, $LParam2)
...
EndFunc


当用户函数被调用时，这 4个参数有下列值:
@@StandardTable1@@
<b>位置</b>	<b>参数</b>	<b>意义</b>
1	controlID	排序函数使用的 listview 控件ID.
2	lParam1		第一个项目的 lParam 值.(默认为项目控件ID).
3	lParam2		第二个项目的 lParam 值.(默认为项目控件ID).
4	column		点击排序的列.(第一列为 0).
@@End@@

下列是改变排序调用行为的返值:
@@StandardTable1@@
<b>返回值</b>	<b>意义</b>
-1	第一个项目应该在第二个之前.（好像是废话哦！）
0	没有改变.
1	第一个项目应该跟随第二个.（又是废话哦！）
@@End@@

参阅 selfcreated GUI 中 listview 列排序的例子.
