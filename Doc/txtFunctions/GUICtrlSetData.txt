###Function###
GUICtrlSetData

###Description###
修改指定控件的数据.

###Syntax###
GUICtrlSetData ( 控件ID, 数据 [, 默认值] )


###Parameters###
@@ParamTable@@
控件ID
	控件标识符(控件ID),可由<a href="../guiref/GUICtrlCreate.htm">GUICtrlCreate...</a>函数的返回值获得.
数据
	对于 Combo (列表框), List (列表), ListView (列表查看), ListViewItem (列表查看项目):  由 <a href="AutoItSetOption.htm#GUIDataSeparatorChar">Opt("GUIDataSeparatorChar",...) </a>分开的项目文本,默认使用"|"分开.
	对于 Progress (进度条): 百分比 
	对于 Slider (滑动条): 值
	对于 Group (群组), Label (标签), Button (按钮), Checkbox (检查框), Radio (单选框), Combo (组合框), List (列表), Input (输入框), Edit (编辑框), TabItem (标签项目): 更新文本
	对于 Date (日期): 日期或时间,具体取决于控件样式
	对于 TreeViewItem (树形列表查看项目): 更新文本
	对于 Dummy : 值.
默认
	[可选参数]
	对于 Combo (组合框), List (列表): 默认选中的项目.
	对于 Edit (编辑框), Input (输入框): 若此参数有定义而且不是 "" 则参数“数据”所含字符串将被插入到指针所在位置后(并非覆盖原有内容).
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回1.
失败:	返回0.
	返回值为 -1,说明给定的数据无效.
@@End@@


###Remarks###
对于 Combo (组合框), List (列表)控件 :
如果 "数据" 符合一个现有项目，它将当做默认被设定.
如果 "数据" 以 GUIDataSeparatorChar 开始或者是一个空的串 ""，早先的列表将被破坏.

对于ListView (列表查看), ListViewItem (列表查看项目) 控件 :
仅仅更新一个特定列,其余忽略. 例如： "||update" 将更新第三列.
如果 "更新" 是空值，列/子项将被抹掉. 举例来说， "|" 将会抹掉第二列/子项, "" 将会抹掉第一.


###Related###
GUICtrlCreate..., GUICtrlSet..., GUICtrlRead, GUIDataSeparatorChar (Option)
