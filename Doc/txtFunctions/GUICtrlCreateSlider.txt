###Function###
GUICtrlCreateSlider

###Description###
在GUI上创建一个滑动条(Slider)控件.

###Syntax###
GUICtrlCreateSlider ( 左侧, 上方 [, 宽度 [, 高度 [, 样式 [, 扩展样式]]]] ) 


###Parameters###
@@ParamTable@@
左侧  
	控件左侧的位置.若此值为 -1 则根据 GUICoordMode 的设置来计算左侧位置.   
上方  
	控件上方的位置.若此值为 -1 则根据 GUICoordMode 的设置来计算上方位置.   
宽度  
	[可选参数] 控件的宽度(默认值为上一个控件的宽度).   
高度  
	[可选参数] 控件的高度(默认值为上一个控件的高度).   
样式  
	[可选参数] 指定控件的样式.请查看附录中关于 <a href="../appendix/GUIStyles.htm#Slider">GUI 控件样式</a>  的说明.
	
	默认值(-1)：$TBS_AUTOTICKS 
扩展样式  
	[可选参数] 指定控件的扩展样式.请查看附录的 <a href="../appendix/GUIStyles.htm#Extended">扩展样式表</a>.
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回控件标识符(控件ID).   
失败:	返回值为0.  
@@End@@


###Remarks###
若要获得控件的各种值请查看 <a href="GUICtrlRead.htm">GUICtrlRead</a>.
若要设置或修改控件的各种信息请查看 <a href="../guiref/GUICtrlSet.htm">GUICtrlSet...</a>.

若要更新滑块的位置请使用 <a href="GUICtrlSetData.htm">GUICtrlSetData</a>.
若要设置最小值和最大值请使用 <a href="GUICtrlSetLimit.htm">GUICtrlSetLimit</a>.

要在默认样式的基础上添加一个新的样式可使用 <b>BitOr</b>($GUI_SS_DEFAULT_SLIDER, 新样式,...).

要使用上方指定的值,您必须包含 #include <SliderConstants.au3> 在你的脚本中.

默认大小为: $GUI_DOCKAUTO 


###Related###
GUICoordMode (Option), GUICtrlSetData, GUICtrlSetLimit, GUICtrlSet..., GUIGetMsg
