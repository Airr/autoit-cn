###Function###
Asc

###Description###
返回某字符的 ASCII 码数值.

###Syntax###
Asc ( "字符" ) 


###Parameters###
@@ParamTable@@
字符
	要获得ASCII码的字符,若传递到此参数的是一个字符串则仅返回该字符串首字符的ASCII码.
@@End@@

###ReturnValue###
返回值 返回指定字符的 ASCII 码值. 



###Remarks###
附录 带有一个完整的 ASCII 列表.
Asc("0") == 48, Asc("9") == 57, Asc("A") == 65, Asc("Z") == 90, Asc("a") == 97, Asc("z") == 122, 等等.


###Related###
AscW, Chr, ChrW
