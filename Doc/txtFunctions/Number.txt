###Function###
Number

###Description###
返回某表达式的数值结果. 

###Syntax###
Number ( 表达式 ) 


###Parameters###
@@ParamTable@@
表达式
	要转换成数字的一个 表达式. 
@@End@@

###ReturnValue###
返回一个数字.
 

###Remarks###
以字母开头的字符串将被视为数值零.以数字开头的字符串将返回被(首个)非数字字符截断后的结果.


###Related###
Int, String
