###Function###
GUICtrlCreateCheckbox

###Description###
在GUI上创建一个复选框(Checkbox)控件. 

###Syntax###
GUICtrlCreateCheckbox ( "文本", 左侧, 上方 [, 宽度 [, 高度 [, 样式 [, 扩展样式]]]] ) 


###Parameters###
@@ParamTable@@
文本
	控件的文本.
左侧
	控件左侧的位置。若此值为 -1 则根据 <a href="AutoItSetOption.htm#GUICoordMode">GUICoordMode</a> 的设置来计算左侧位置。
上方
	控件上方的位置。若此值为 -1 则根据 <a href="AutoItSetOption.htm#GUICoordMode">GUICoordMode</a> 的设置来计算上方位置。
宽度
	[可选参数] 控件的宽度(默认值为上一个控件的宽度).
高度
	[可选参数] 控件的高度(默认值为上一个控件的高度).
样式
	[可选参数] 指定控件的样式.请查看附录中关于 <a href="../appendix/GUIStyles.htm#Checkbox">GUI 控件样式</a> 的说明.
	
	默认值(-1): $BS_AUTOCHECKBOX. 
	强制性样式:$WS_TABSTOP, $BS_AUTOCHECKBOX,(如果没有定义它的样式)
扩展样式
	[可选参数] 指定控件的扩展样式.请查看附录的 扩展样式表 .
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回控件标识符(控件ID).
失败:	返回值为0.
@@End@@


###Remarks###
若要获得控件的各种值请查看 <a href="GUICtrlRead.htm">GUICtrlRead</a>.
To set or change information in the control see <a href="GUICtrlUpdate Managenent.htm">GUICtrlUpdate...</a>.

使用 $BS_ICON 或 $BS_BITMAP 样式即可让复选框显示一个图标或图片.使用 <a href="GUICtrlSetImage.htm">GUICtrlSetImage</a> 来指定图片文件.

要在默认样式的基础上添加一个新的样式可使用 <b>BitOr</b>($GUI_SS_DEFAULT_CHECKBOX, 新样式,...).

要使用上方指定的值,您必须包含 #include <ButtonConstants.au3> 在你的脚本中.

默认大小为: $GUI_DOCKHEIGHT.
