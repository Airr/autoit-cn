###Function###
TCPCloseSocket

###Description###
关闭一个 TCP 套接字(socket). 


###Syntax###
TCPCloseSocket ( 套接字 ) 


###Parameters###
@@ParamTable@@
套接字
	套接字标识(SocketID) ,通常是 <a href="TCPListen.htm">TCPListen</a> 或 <a href="TCPAccept.htm">TCPAccept</a> 函数的返回值.    
@@End@@


###ReturnValue###
@@ReturnTable@@
成功:	返回 1.   
失败:	返回 0 并设置 @error 的值(依照 Windows API WSAGetError 返回).    
@@End@@


###Remarks###
None.

###Related###
TCPStartup, TCPListen, TCPAccept
