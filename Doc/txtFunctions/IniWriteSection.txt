###Function###
IniWriteSection

###Description###
将数据写入到标准INI文件的一个字段.

###Syntax###
IniWriteSection ( "文件名", "字段", "数据" [, 索引] )


###Parameters###
@@ParamTable@@
文件名
	.ini 文件的路径.
字段
	.ini 文件中的字段名称.
数据
	需要写入的数据. 数据请按照标准的INI文件格式写入 键 = 值 这样成对的写入,如果有多行,请使用 @LF 换行.
索引
	If an array is passed as data, this specifies the index to start writing from.  By default, this is 1 so that the return value of IniReadSection() can be used immediately.  For manually created arrays, this value may need to be different depending on how the array was created.  This parameter is ignored if a string is passed as data.
@@End@@


###ReturnValue###
@@ReturnTable@@
成功:	返回 1.
失败:	返回 0. T如果 @error 被设置为1,那么您的数据格式是错误的.
@@End@@


###Remarks###
一个标准的INI文件的结构如下:
<i>[段名]
关键字=数值</i>
若目标文件并不存在,则程序将自动创建该文件,关键字和段将被添加到后面部分而且并不按任何规则排列.
如果字段是存在的,数据将会覆盖.
