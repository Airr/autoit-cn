###Function###
PluginOpen

###Description###
Open a plugin file.

###Syntax###
PluginOpen ( filename )


###Parameters###
@@ParamTable@@
filename
	Filename of the plugin file to open.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	Returns a dll "handle" to be used to close the plugin.
Failure:	Returns 0.
@@End@@


###Remarks###
None.

###Related###
PluginClose
