###Function###
GUICtrlCreateEdit

###Description###
在GUI上创建一个编辑框(Edit)控件. 

###Syntax###
GUICtrlCreateEdit ( "文本", 左侧, 上方 [, 宽度 [, 高度 [, 样式 [, 扩展样式]]]] ) 


###Parameters###
@@ParamTable@@
文本
	控件的文本.
左侧
	控件左侧的位置.若此值为 -1 则根据 <a href="AutoItSetOption.htm#GUICoordMode">GUICoordMode</a> 的设置来计算左侧位置.
上方
	控件上方的位置.若此值为 -1 则根据 <a href="AutoItSetOption.htm#GUICoordMode">GUICoordMode</a> 的设置来计算上方位置.
宽度
	[可选参数] 控件的宽度(默认值为上一个控件的宽度).
高度
	[可选参数] 控件的高度(默认值为上一个控件的高度).
样式
	[可选参数] 指定控件的样式.请查看附录中关于 <a href="../appendix/GUIStyles.htm#Edit">GUI 控件样式</a> 的说明.
	
	默认值(-1): $ES_WANTRETURN, $WS_VSCROLL, $WS_HSCROLL, $ES_AUTOVSCROLL, $ES_AUTOHSCROLL
	强制性样式:$ES_MULTILINE, $WS_TABSTOP (如果 不是$ES_READONLY(只读)) 
扩展样式
	[可选参数] 指定控件的扩展样式.请查看附录的 <a href="../appendix/GUIStyles.htm#Extended">扩展样式表</a> .
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回控件标识符(控件ID).
失败:	返回值为0.
@@End@@


###Remarks###
若要获得控件的各种值请查看 <a href="GUICtrlRead.htm">GUICtrlRead</a>.
若要设置或修改控件的各种信息请查看 <a href="../guiref/GUICtrlSet.htm">GUICtrlSet...</a>.

要在默认样式的基础上添加一个新的样式可使用 <b>BitOr</b>($GUI_SS_DEFAULT_EDIT, 新样式,...).

如果您想拖放一个文件到这个控件,请添加 <a href="../appendix/GUIStyles.htm#Extended">WS_EX_ACCEPTFILES 扩展样式</a> 到 <a href="GUICreate.htm">GUICreate()</a> 并设置状态为 $GUI_DROPACCEPTED.
拖动多个文件会被分割开. 

要使用上方指定的值,您必须包含 #include <EditConstants.au3> 在你的脚本中.

默认大小为: $GUI_DOCKAUTO .坐标随机. 

创建一个 RichEdit 控件来结合本控件,将不能再包含一个基本控件. 
您可以使用 GuiCtrlCreateObj. 请参考第二个例子,如果您想包含RichEdit 控件.
