###Function###
ProcessClose

###Description###
结束某个进程.

###Syntax###
ProcessClose ( "进程") 


###Parameters###
@@ParamTable@@
进程
	要终止的进程的名称或 PID(进程标识符).
@@End@@


###ReturnValue###
@@ReturnTable@@
成功:	返回 1.
失败:	返回 0 并设置 @error 到:</td>
	1 = 打开进程失败</td>
	2 = AdjustTokenPrivileges 失败</td>
	3 = 终止进程失败</td>
@extended 按照 WinAPI GetLastError() 设置
@@End@@



###Remarks###
进程名是指可执行文件的名称(无需给出完整路径),例如:"notepad.exe" 或 "winword.exe"

若有多个进程的名称相同则 PID 最高的进程将被结束(并不是以最近启动的进程为准).

PID 是标识进程的唯一的数值.PID 可通过 ProcessExists 或 Run 命令的返回值获得.

每隔 250 毫秒左右进程将被检测一次.
