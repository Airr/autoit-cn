###Function###
TCPNameToIP

###Description###
转化一个因特网名称(域名)到一个 IP 地址. 

###Syntax###
TCPNameToIP ( 域名 ) 


###Parameters###
@@ParamTable@@
域名
	一个域名的字符串.
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回一个包含 IP 地址的字符串.
失败:	返回空字符串 "" 并设置 @error 的值(依照 Windows API WSAGetError 返回).    
@@End@@


###Remarks###
TCPStartup 必须在前面调用(进行网络初始化).


###Related###
TCPStartup
