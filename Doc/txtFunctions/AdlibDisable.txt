###Function###
AdlibDisable

###Description###
禁用 Adlib 功能.

###Syntax###
AdlibDisable ( [ "function"])


###Parameters###
@@ParamTable@@
function
	The name of the adlib function to be disabled
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	Returns 1.
Failure:	Returns 0.
@@End@@


###Remarks###
If no function is supplied then the last AdlibRegister function will be disabled. Allows compatibility with previous single adlib functions

###Related###
AdlibRegister, AdLibUnRegister
