###Function###
Int

###Description###
返回某个表达式的整数(整数部分)值. 

###Syntax###
Int ( 表达式 )


###Parameters###
@@ParamTable@@
表达式
	要转换为整数的表达式.
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回整数.
失败:	返回 0 并设置 @error 为1(如果表达式不是一个整数,浮点或者字符串)  
@@End@@


###Remarks###
小数部分将被约去,因此 Int(1.999999) 的返回值将是 1 
Int(0/0) 的返回值将是 -9.22337203685478e+018.令人奇怪?


###Related###
Number, String, Round, HWnd
