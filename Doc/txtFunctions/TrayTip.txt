###Function###
TrayTip

###Description###
在托盘图标上显示一个气球提示 (仅支持2000/XP) 

###Syntax###
TrayTip ( "标题", "文本", 超时时间 [, 选项] ) 


###Parameters###
@@ParamTable@@
标题
	在气球提示窗口上方显示的标题文字,粗体(最高可达63个字符).   
文本
	气球提示的消息正文(最高可达255个字符).
超时时间
	气球提示显示的时间长度(粗略值).Windows 允许设置的时间范围是 10-30 秒,但并不准时.
选项
	[可选参数] 请查看下面的注意部分.  0=没有图标(默认), 
	1=消息, 2=警告, 3=错误    
@@End@@

###ReturnValue###
None.


###Remarks###
TrayTip 函数只可用于 Windows 2000/XP 或更高版本.

如果托盘图标消失则气球提示也会消失.因此,如果设置了 AutoItSetOption("TrayIconHide", 1) 或者用户在注册表中把气球提示功能禁止了,那么气球提示将不会显示! 

Windows XP 在显示气球提示时会播放一个声音(系统通知),如果要禁止播放声音则可在 <i>选项<i> 参数中增加数值 16 或者设置标题为空.

如果要清除已有的气球提示,只要再次调用此函数并设置 文本 参数为空字符串(标题任意)即可. 

###Related###
MsgBox, ToolTip, TrayIconHide (Option)
