###Function###
_WinAPI_RegQueryInfoKey

###Description###
Retrieves information about the specified registry key.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_RegQueryInfoKey ( $hKey )

###Parameters###
@@ParamTable@@
$hKey
	Handle to an open registry key. The key must have been opened with the $KEY_QUERY_VALUE access right.
	This handle is returned by the _WinAPI_RegCreateKey() or _WinAPI_RegOpenKey() function. It can also
	be one of the following predefined keys.
	
	$HKEY_CLASSES_ROOT
	$HKEY_CURRENT_CONFIG
	$HKEY_CURRENT_USER
	$HKEY_LOCAL_MACHINE
	$HKEY_PERFORMANCE_DATA
	$HKEY_USERS
	
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	The array containing the following parameters:
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;[0]	The number of subkeys that are contained by the specified key.
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;[1]	The size of the key's subkey with the longest name, in characters, not including the terminating null character.
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;[2]	The number of values that are associated with the key.
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;[3]	The size of the key's longest value name, in characters. The size does not include the terminating null character.
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;[4]	The size of the longest data component among the key's values, in bytes.
Failure	0 and sets the @error flag to non-zero, @extended flag may contain the system error code.
@@End@@
