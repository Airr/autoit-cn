###User Defined Function###
_WinAPI_GetStockObject

###Description###
检索预定义的常用笔, 画笔, 字体, 或调色板的句柄

###Syntax###
#Include <WinAPI.au3>
_WinAPI_GetStockObject($iObject)

###Parameters###
@@ParamTable@@
$iObject
	指定常规对象的类型.参数可以是任何以下值之一:
		$BLACK_BRUSH - 黑色画笔
		$DKGRAY_BRUSH - 暗灰色画笔
		$GRAY_BRUSH - 灰色画笔
		$HOLLOW_BRUSH - 空心画笔 (相当于 NULL_BRUSH)
		$LTGRAY_BRUSH - 浅灰色画笔
		$NULL_BRUSH - 空画笔 (相当于 HOLLOW_BRUSH)
		$WHITE_BRUSH - 白色画笔
		$BLACK_PEN - 黑色笔
		$NULL_PEN - 空笔
		$WHITE_PEN - 白色笔
		$ANSI_FIXED_FONT - Windows 固定间距(等宽)系统字体
		$ANSI_VAR_FONT - Windows 可变间距(比例空间)的系统字体
		$DEVICE_DEFAULT_FONT - 设备相关的字体
		$DEFAULT_GUI_FONT - 默认用户界离开象的字体
		$OEM_FIXED_FONT - OEM 固定间距(等宽)字体
		$SYSTEM_FONT - 系统字体
		$SYSTEM_FIXED_FONT - 固定间距(等宽)系统字体,在早于 Windows 3.0版本系统中使用
		$DEFAULT_PALETTE - 默认调色板. 这个调色板包括系统调色板中的静态颜色.
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回请求的逻辑对象
失败:	返回 0
@@End@@
