###Function###
_WinAPI_ActivateKeyboardLayout

###Description###
Sets the input locale identifier for the calling thread or the current process.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_ActivateKeyboardLayout ( $hLocale [, $iFlag] )

###Parameters###
@@ParamTable@@
$hLocale
	The input locale identifier to be activated. This parameter must be either the handle to a keyboard
	layout or one of the following values.
	
	$HKL_NEXT
	$HKL_PREV
	
$iFlag
	The flag that specifies how the input locale identifier is to be activated. This parameter can be
	one of the following values.
	
	$KLF_REORDER
	$KLF_RESET
	$KLF_SETFORPROCESS
	$KLF_SHIFTLOCK
	
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	The previous input locale identifier.
Failure	0 and sets the @error flag to non-zero.
@@End@@
