###User Defined Function###
_WinAPI_WideCharToMultiByte

###Description###
转换一个 Unicode 字符串到一个多字节字符串.

###Syntax###
#Include <WinAPI.au3>
_WinAPI_WideCharToMultiByte($pUnicode [, $iCodePage = 0 [, $iFlags = 0 [, $bRetString = True]]])

###Parameters###
@@ParamTable@@
$pUnicode
	一个包含 Unicode 文本的数据结构数组.
$iCodePage
	[可选参数] 用于转换的代码页:
		0 - 当前系统 Windows ANSI 代码页
		1 - 当前系统 OEM 代码页
		2 - 当前系统 Macintosh 代码页
		3 - 当前线程 Windows ANSI 代码页
		42 - Symbol 代码页
		65000 - UTF-7
		65001 - UTF-8
$iFlags
	[optional] Flags indicating the conversion type, see MSDN (default 0)
$bRetString
	[optional] Flags that indicate whether to return a String or a DllStruct  (default True : String)
@@End@@

###ReturnValue###
Returns a string or a DllStruct containing a multibyte string

###Remarks###
None.
