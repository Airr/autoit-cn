###User Defined Function###
_WinAPI_GetObject

###Description###
检索指定图形对象的信息

###Syntax###
#Include <WinAPI.au3>
_WinAPI_GetObject($hObject, $iSize, $pObject)

###Parameters###
@@ParamTable@@
$hObject
	标识逻辑笔,画笔,字体,位图,区域或调色板
$iSize
	指定写入到缓冲区的字节数
$pObject
	接收信息的缓冲区指针.
		可以为每个类型对象指定接收信息的缓冲区类型:
		HBITMAP - BITMAP 或 DIBSECTION
		HPALETTE - 逻辑调色板条目计数
		HPEN - EXTLOGPEN 或 LOGPEN
		HBRUSH - LOGBRUSH
		HFONT - LOGFONT
	[可选参数] 如 $pObject 为 0, 函数返回值是所需的字节数,
		字节存储的信息将写入到指定的图形对象缓冲区.
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	如 $pObject 是一个有效的指针, 返回值是缓冲区中的字节数.
	如果函数成功, 与 $pObject 为 0, 则返回值是需要的字节数,此字节用来保存将存储到缓冲区的信息.
失败:	返回 0
@@End@@
