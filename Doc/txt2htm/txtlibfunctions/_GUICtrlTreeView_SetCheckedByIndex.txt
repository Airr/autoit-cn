###User Defined Function###
_GUICtrlTreeView_SetCheckedByIndex

###Description###
设置指定索引的项目复选框选中

###Syntax###
#Include <GuiTreeView.au3>
_GUICtrlTreeView_SetCheckedByIndex($hWnd, $hItem, $iIndex [, $fCheck = True])

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
$hItem
	项目句柄
$iIndex
	指定 $hItem 项目列表中的 0 基索引
$fCheck
	[可选参数] 选中状态值:
		 True - 选中
		False - 不选中
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 True
失败:	返回 False
@@End@@
