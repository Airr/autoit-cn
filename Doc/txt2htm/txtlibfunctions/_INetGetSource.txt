###User Defined Function###
_INetGetSource

###Description###
获取 URL 源且不写临时文件.

###Syntax###
#include <INet.au3>
_INetGetSource ( $s_URL )


###Parameters###
@@ParamTable@@
$s_URL
	网站的网址. 例如: 'www.autoitscript.com'
$bString
	[可选参数] 如果为 True,则返回数据的字符串格式,否则为二进制格式.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success - The read string and sets @extended to the number of bytes returned.
Failure - An empty string and and sets @error to non-zero.
@@End@@


###Remarks###
None.
