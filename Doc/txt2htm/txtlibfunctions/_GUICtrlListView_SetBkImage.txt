###User Defined Function###
_GUICtrlListView_SetBkImage

###Description###
设置控件背景图像

###Syntax###
#Include <GuiListView.au3>
_GUICtrlListView_SetBkImage($hWnd[, $sURL = ""[, $iStyle = 0[, $iXOffset = 0[, $iYOffset = 0]]]])

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
$sURL
	[可选参数] 背景图像的 URL 地址. 如果空白,表示控件没有任何背景
$iStyle
	[可选参数] 确定背景图像的样式:
		0 - 标准
		1 - 平铺
$iXOffset
	[可选参数] 图像占控件客户区百分比的水平偏移量.
		仅在 $iStyle 为 0 时有效.
$iYOffset
	[可选参数] 图像占控件客户区百分比的垂直偏移量.
		仅在 $iStyle 为 0 时有效.
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 True
失败:	返回 False
@@End@@


###Remarks###
 当应用程序终止时调用 CoUninitialize.
 函数只适用于 _GUICtrlListView_Create 创建的或外部的 ListViews 控件
