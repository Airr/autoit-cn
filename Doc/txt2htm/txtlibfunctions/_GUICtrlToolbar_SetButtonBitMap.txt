###User Defined Function###
_GUICtrlToolbar_SetButtonBitMap

###Description###
设置按钮相关的位图索引

###Syntax###
#Include <GuiToolBar.au3>
_GUICtrlToolbar_SetButtonBitMap($hWnd, $iCommandID, $iIndex)

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
$iCommandID
	按钮命令 ID
$iIndex
	图像在图像列表中的 0 基索引
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 True
失败:	返回 False
@@End@@

None.
