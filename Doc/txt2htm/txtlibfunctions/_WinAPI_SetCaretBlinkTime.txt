###User Defined Function###
_WinAPI_SetCaretBlinkTime

###Description###
设置插入符得闪烁时间.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_SetCaretBlinkTime ( $iDuration )

###Parameters###
@@ParamTable@@
$iDuration
	新闪烁时间的毫秒值.如果此参数为 -1, 插入符号不闪烁.
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回先前闪烁时间的毫秒值.
失败:	返回 0,并设置@error标志为非 0 值.
@@End@@

###Remarks###
用户可以使用控制面板设置闪烁时间,应用程序应尊重用户的选择设置.
 函数仅使用允许用户设置闪烁时间的应用程序,例如一个控制面板小程序.
