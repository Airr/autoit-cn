###User Defined Function###
_GUICtrlToolbar_EnableButton

###Description###
启用或禁用指定按钮

###Syntax###
#Include <GuiToolBar.au3>
_GUICtrlToolbar_EnableButton($hWnd, $iCommandID[, $fEnable = True])

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
$iCommandID
	按钮命令 ID
$fEnable
	[可选参数] 可用标志:
		True - 启用
		False - 禁用
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	True
失败:	False
@@End@@

###Remarks###
None.
