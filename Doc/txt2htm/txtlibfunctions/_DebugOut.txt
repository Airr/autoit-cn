###User Defined Function###
_DebugOut

###Description###
输出调试信息到 _DebugSetup() 设置的调试会话框

###Syntax###
#include <Debug.au3>
_DebugOut ( $sOutput [, $bActivate] )


###Parameters###
@@ParamTable@@
$sOutput
	要输出到调试会话的字符串或其他可打印值.
$bActivate
	[可选参数] 忽略,保持向后兼容性.
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 1, @Error 与 @Extended 和以前一样调用 _DebugOut().
失败:	返回 0 , 设置 @Error :.
@Error:	0 = 没有错误.
	1 = $sOutput 是一个不兼容的类型.
	3 = _DebugSetup() 没有正常运行.
@@End@@

###Remarks###
 每次调用 _DebugOut(), 输出都将占用各自的行.

 _DebugSetup() 必须在第一次调用 _DebugOut() 之前, 否则将返回 @Error = 3..
