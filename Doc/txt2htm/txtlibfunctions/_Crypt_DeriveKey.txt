###User Defined Function###
_Crypt_DeriveKey

###Description###
用算法和密码创建密钥

###Syntax###
#Include <Crypt.au3>
_Crypt_DeriveKey($vPassword, $iALG_ID [, $iHash_ALG_ID = $CALG_MD5 ] )

###Parameters###
@@ParamTable@@
$vPassword
	要使用的密码
$iALG_ID
	加密算法的 ID
$iHash_ALG_ID
	[可选参数] hash 算法的 ID 与密码 译注:关于 Hash 见下文注意部分
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回密钥句柄
	设置 @error 为 0
失败:	返回 -1 并设置 @error:
	1 - 无法创建 hash 对象
	2 - 没有 hash 密码
	3 - 无法生成密钥
@@End@@

###Remarks###
密钥需要用 _Crypt_DestroyKey 销毁.
AES 算法不适用于 Windows 2000.
