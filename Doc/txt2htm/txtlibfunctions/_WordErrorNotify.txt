###User Defined Function###
_WordErrorNotify

###Description###
指定是否 Word.au3 自动通知警告和错误 (在控制台)

###Syntax###
#include <Word.au3> 
_WordErrorNotify ( [$f_notify = -1] )

###Parameters###
@@ParamTable@@
$f_notify
	[可选参数] 指定通知或关闭
	-1 = (默认) 返回当前设置
	True = 打开
	False = 关闭
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	If $f_notify = -1, 返回当前设置的通知，否则返回 1
失败:	返回 0,
@@End@@


###Remarks###
 Word.au3 默认写诊断信息，警告和错误的控制台. 当使用 SciTe 编辑器来运行你的脚本,这些信息可以很容易地看到.

 此函数允许你打开或关闭此功能.
