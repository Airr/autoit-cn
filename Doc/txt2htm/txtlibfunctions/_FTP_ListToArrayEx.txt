###User Defined Function###
_FTP_ListToArrayEx
###Description###
获取当前远程目录的文件/目录的名称, 大小, 属性和时间.
###Syntax###
#Include <FTPEx.au3>
_FTP_ListToArrayEx($l_FTPSession [, $Return_Type = 0 [, $l_Flags = 0 [, $b_Fmt = 1 [, $l_Context]]]])
###Parameters###
@@ParamTable@@
$l_FTPSession
	_FTP_Connect() 返回值.
$Return_type
	[可选参数] 0 = 获取文件及目录, 1 = 仅获取目录, 2 = 仅获取文件.
$l_Flags
	[可选参数] 参见 _FTP_FindFileFirst().
$b_Fmt
	[可选参数] 日期时间类型 : 1 = yyyy/mm/dd, 0 = mm/dd/yyyy.
$l_Context
	[可选参数] 一个变量, 它包含应用程序定义的值, 关联应用程序数据的搜索.
	用于应用程序已调用 _FTP_SetStatusCallback() 时,设置一个状态回调函数.
@@End@@
###ReturnValue###
@@ReturnTable@@
成功:	返回一个二维数组, 见备注.
失败:	返回 Array[0][0] = 0.
@@End@@
###Remarks###
 Array[0][0] = 找到的条目数
 Array[x][0] 文件名
 Array[x][1] 文件大小
 Array[x][2] 文件属性
 Array[x][3] 文件修改日期时间
 Array[x][4] 文件创建日期时间
 Array[x][5] 文件访问日期时间
###Related###
_FTP_Connect, _FTP_SetStatusCallback
