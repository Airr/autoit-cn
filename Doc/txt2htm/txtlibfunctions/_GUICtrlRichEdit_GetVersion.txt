###用户自定义函数###
_GUICtrlRichEdit_GetVersion

###描述###
获取富文本编辑控件的版本

###语法###
#Include <GuiRichEdit.au3>
_GUICtrlRichEdit_GetVersion()

###参数###
@@ParamTable@@
无
@@End@@

###返回值###
@@ReturnTable@@
无
@@End@@

###备注###
包含在Windows版本中的富文本编辑控件的版本:
3.0 - Windows 2000, XP
3.1 - Windows Vista
4.0 - Windows XP SP1
4.1 - Windows Vista/Windows 7
