###User Defined Function###
_GUICtrlRichEdit_GetVersion

###Description###
获取富文本编辑控件的版本

###Syntax###

_GUICtrlRichEdit_GetVersion()

@@End@@

###Remarks###
包含在Windows版本中的富文本编辑控件的版本:
3.0 - Windows 2000, XP
3.1 - Windows Vista
4.0 - Windows XP SP1
4.1 - Windows Vista/Windows 7
