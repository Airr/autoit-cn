###User Defined Function###
_GUICtrlRichEdit_GetVersion

###Description###
检索 RichEdit 的版本

###Syntax###
#Include <GuiRichEdit.au3>
_GUICtrlRichEdit_GetVersion()

###Parameters###
None.


###ReturnValue###
None.

###Remarks###
 包括在 Windows 版本中的 RichEdit 版本
 3.0 - Windows 2000, XP
 3.1 - Windows Vista
 4.0 - Windows XP SP1
 4.1 - Windows Vista
