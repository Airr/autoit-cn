###Function###
_WinAPI_AddMRUString

###Description###
Adds a string to the top of the most recently used (MRU) list.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_AddMRUString ( $hMRU, $sStr )

###Parameters###
@@ParamTable@@
$hMRU
	Handle of the MRU list.
$sStr
	The string be added.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	1.
Failure	0 and sets the @error flag to non-zero.
@@End@@

###Remarks###
None
