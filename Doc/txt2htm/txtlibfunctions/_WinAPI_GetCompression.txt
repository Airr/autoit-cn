###User Defined Function###
_WinAPI_GetCompression

###Description###
Retrieves the current compression state of a file or directory.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_GetCompression ( $sPath )

###Parameters###
@@ParamTable@@
$sPath
	Path to file or directory to retrieve compression state.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	The current compression state.
Failure	0 and sets the @error flag to non-zero.
@@End@@

###Remarks###
None
