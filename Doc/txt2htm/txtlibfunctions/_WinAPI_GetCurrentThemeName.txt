###User Defined Function###
_WinAPI_GetCurrentThemeName

###Description###
Retrieves the name of the current visual styles, color scheme name, and size name.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_GetCurrentThemeName ( )

###Parameters###
@@ParamTable@@
None.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	The array that contains the following information.
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;[0]	The theme path and file name.
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;[1]	The color scheme name.
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;[2]	The size name.
Failure	0 and sets the @error flag to non-zero, @extended flag may contain the system error code.
@@End@@
