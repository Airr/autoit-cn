###User Defined Function###
_WinAPI_PathCompactPath

###Description###
Truncates a file path to fit within a given pixel width by replacing path components with ellipses.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_PathCompactPath ( $hWnd, $sPath [, $iWidth] )

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the window used for font metrics.
$sPath
	The path to be modified.
$iWidth
	[可选参数] The width, in pixels, in which the string must fit. If this parameter is (-1), width will be equal to
	the width of the window.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	The modified path.
Failure	The original $sPath parameter and sets the @error flag to non-zero.
@@End@@

###Remarks###
This function will not compact the path beyond the base file name preceded by ellipses.
