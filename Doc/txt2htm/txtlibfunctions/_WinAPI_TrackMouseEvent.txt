###User Defined Function###
_WinAPI_TrackMouseEvent

###Description###
Posts messages when the mouse pointer leaves a window or hovers over a window for a specified amount of time.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_TrackMouseEvent ( $hWnd, $iFlags [, $iTime] )

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the window to track.
$iFlags
	The services requested. This parameter can be a combination of the following values.
	
	$TME_CANCEL
	$TME_HOVER
	$TME_LEAVE
	$TME_NONCLIENT
	$TME_QUERY
	
$iTime
	[可选参数] The hover time-out (if $TME_HOVER was specified in $Flags), in milliseconds. Can be (-1), which
	means to use the system default hover time-out.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	1.
Failure	0 and sets the @error flag to non-zero.
@@End@@
