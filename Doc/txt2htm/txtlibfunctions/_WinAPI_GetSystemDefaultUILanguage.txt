###User Defined Function###
_WinAPI_GetSystemDefaultUILanguage

###Description###
Retrieves the language identifier for the system default UI language of the operating system.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_GetSystemDefaultUILanguage ( )

###Parameters###
@@ParamTable@@
None.
@@End@@

###ReturnValue###

失败:	返回 0 并设置 @error 标志为非 0 值.
