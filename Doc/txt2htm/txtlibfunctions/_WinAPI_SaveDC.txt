###User Defined Function###
_WinAPI_SaveDC

###Description###
Saves the current state of the specified device context (DC) by copying data describing selected objects and
;                  graphic modes to a context stack.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_SaveDC ( $hDC )

###Parameters###
@@ParamTable@@
$hDC
	Handle to the DC whose state is to be saved.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	The value identifies the saved state.
Failure	0 and sets the @error flag to non-zero.
@@End@@

###Remarks###
None
