###User Defined Function###
_WinAPI_EnumProcessWindows

###Description###
Enumerates the windows that belong to the specified process.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_EnumProcessWindows ( [$PID [, $fVisible]] )

###Parameters###
@@ParamTable@@
$PID
	[可选参数] The PID of the process. Default (0) is the current process.
$fVisible
	[可选参数] Specifies whether enumerates the invisible window, valid values:
		TRUE     - Enumerate only visible windows. (Default)
		FALSE    - Enumerate all windows.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	The 2D array of the handles to the windows and classes for the specified process.
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;[0][0]	Number of rows in array (n)
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;[0][1]	Unused
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;[1][0]	Window handle
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;[1][1]	Window class name
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;[n][0]	Window handle
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;[n][1]	Window class name
Failure	0 and sets the @error flag to non-zero.
@@End@@
