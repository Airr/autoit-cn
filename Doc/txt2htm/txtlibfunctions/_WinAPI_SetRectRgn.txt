###Function###
_WinAPI_SetRectRgn

###Description###
Converts a region into a rectangular region with the specified coordinates.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_SetRectRgn ( $hRgn, $tRECT )

###Parameters###
@@ParamTable@@
$hRgn
	Handle to the region.
$tRECT
	$tagRECT structure that contains the coordinates of the rectangular region in logical units.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	1.
Failure	0 and sets the @error flag to non-zero.
@@End@@

###Remarks###
None
