###Function###
_WinAPI_PrintWindow

###Description###
Copies a visual window into the specified device context.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_PrintWindow ( $hWnd, $hDC [, $fClient] )

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the window that will be copied.
$hDC
	Handle to the device context.
$fClient
	Specifies whether copies only the client area of the window, valid values:
		TRUE    - Only the client area of the window is copied to device context.
		FALSE   - The entire window is copied. (Default)
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	1.
Failure	0 and sets the @error flag to non-zero.
@@End@@

###Remarks###
None
