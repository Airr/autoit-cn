###User Defined Function###
_GUICtrlListBox_SetCaretIndex

###Description###
设置多选择列表框指定索引项目的焦点矩形

###Syntax###
#Include <GuiListBox.au3>
_GUICtrlListBox_SetCaretIndex($hWnd, $iIndex[, $fPartial = False])

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
$iIndex
	项目的 0 基索引
$fPartial
	[可选参数] 如果为 False,项目将滚动至完全可见;
		如果 True, 项目滚动至部分可见.
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 True
失败:	返回 False
@@End@@

###Remarks###
None.
