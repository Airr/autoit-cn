###User Defined Function###
_GUICtrlToolbar_CheckButton

###Description###
选中或取消选中某个按钮

###Syntax###
#Include <GuiToolBar.au3>
_GUICtrlToolbar_CheckButton($hWnd, $iCommandID[, $fCheck = True])

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
$iCommandID
	按钮命令 ID
$fCheck
	[可选参数] 选择状态:
		True - 按钮选中
		False - 按钮未选中
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 True
失败:	返回 False
@@End@@

###Remarks###
 当按钮被选中时,它会显示在按下状态
