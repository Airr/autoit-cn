###Function###
_WinAPI_GetPosFromRect

###Description###
Interprets the coordinates of the rectangle as offset and position coordinates.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_GetPosFromRect ( $tRECT )

###Parameters###
@@ParamTable@@
$tRECT
	$tagRECT structure that contains the logical coordinates of the rectangle.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	The array containing the following parameters:
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;[0]	The x-coordinate of the upper-left corner of the rectangle.
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;[1]	The y-coordinate of the upper-left corner of the rectangle.
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;[2]	The width of the rectangle.
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;[3]	The height of the rectangle.
Failure	0 and sets the @error flag to non-zero.
@@End@@

###Remarks###
This function is inverse to the _WinAPI_CreateRectEx().
