###User Defined Function###
_IE_Introduction

###Description###
在新浏览器窗口中显示一个关于 IE.au3 的介绍信息

###Syntax###
#include <IE.au3> 
_IE_Introduction ( [$s_module = "basic"] )

###Parameters###
@@ParamTable@@
$s_module
	[可选参数] 指定要运行的模块
	basic = (默认) 基本介绍.
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回一个指向 InternetExplorer.Application 对象的对象变量
失败:	返回 0 并设置 @ERROR
@Error:	0 ($_IEStatus_Success) = 没有错误
	5 ($_IEStatus_InvalidValue) = 无效值
@Extended:	包含无效参数的数量.
@@End@@


###Remarks###
AutoIt 帮助文件目前不允许常规 UDFs 文档包含一系列函数,而是把重点放在一个单一的函数上.本函数目的是代替常规文档.它会不断的改进.
