###User Defined Function###
_GUICtrlMenu_SetItemData

###Description###
设置应用程序定义的菜单项值

###Syntax###
#Include <GuiMenu.au3>
_GUICtrlMenu_SetItemData($hMenu, $iItem, $iData[, $fByPos = True])

###Parameters###
@@ParamTable@@
$hMenu
	菜单句柄
$iItem
	菜单项的标识符或位置,由 $fByPos 参数决定.
$iData
	应用程序定义的值
$fByPos
	[可选参数] 菜单识别标志:
		True - $iItem 为项目的 0 基位置
		False - $iItem 为菜单项的标识符
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 True
失败:	返回 False
@@End@@

###Remarks###
None.
