###User Defined Function###
_GUICtrlEdit_HideBalloonTip

###Description###
Hides any balloon tip associated with an edit control

###Syntax###
#Include <GuiEdit.au3>
_GUICtrlEdit_HideBalloonTip($hWnd)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True
Failure:	False
@@End@@

###Remarks###
Minimum OS - Windows XP.
