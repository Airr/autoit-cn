###User Defined Function###
_WordAttach

###Description###
获取Microsoft Word的首个实例并按选择的搜索方式搜索匹配字符串

###Syntax###
#include <Word.au3> 
_WordAttach ( $s_string [, $s_mode = "FilePath"] )

###Parameters###
@@ParamTable@@
$s_string
	要搜索的字符串
$s_mode
	[可选参数] 指定搜索方式
	FileName = 打开文件的名称
	FilePath = (默认) 打开文件的完整路径
	HWND = 窗口的句柄
	Text = 来自文档正文的文字
	Title = 窗口的标题
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回指向 Word.Application 对象的对象变量
失败:	返回 0 并设置 @ERROR
@Error:	$_WordStatus_Success = 无错误
	$_WordStatus_GeneralError = 一般错误
	$_WordStatus_InvalidValue = 无效值
	$_WordStatus_NoMatch = 不匹配
@Extended:	参数数量无效
@@End@@
