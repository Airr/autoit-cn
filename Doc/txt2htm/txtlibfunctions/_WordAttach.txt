###User Defined Function###
_WordAttach

###Description###
获取Microsoft Word的首个实例并按选择的搜索方式搜索匹配字符串

###Syntax###
#Include <Word.au3>
_WordAttach ( $s_string [,$s_mode = "FilePath"] )

###Parameters###
@@ParamTable@@
$s_string 
   搜索的字符串
$s_mode 
   可选: 指定搜索方式
    FileName = 打开文件的名称
    FilePath = (默认)
 
      打开文件的完整路径
    HWND = 窗口的句柄
    Text = 来自文档正文的文字
    Title =
 
  窗口的标题
@@End@@

###ReturnValue###
@@ReturnTable@@
成功: 返回指向Word.Application对象的对象变量
失败: 返回0并设置@ERROR
@Error: 0 ($_WordStatus_Success) = 无错误
     1 ($_WordStatus_GeneralError) = 一般错误
     3 ($_WordStatus_InvalidDataType) = 无效数据类型
     5 ($_WordStatus_InvalidValue) = 无效值
     7 ($_WordStatus_NoMatch) = 不匹配
@Extended: 包含无效参数数量
@@End@@
