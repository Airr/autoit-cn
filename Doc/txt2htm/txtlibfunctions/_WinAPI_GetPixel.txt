###User Defined Function###
_WinAPI_GetPixel

###Description###
检索指定坐标的像素颜色值.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_GetPixel ( $hDC, $iX, $iY )

###Parameters###
@@ParamTable@@
$hDC
	设备环境句柄.
$iX
	像素的 x 坐标逻辑单位.
$iY
	像素的 y 坐标逻辑单位.
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回像素颜色的 RGB 值.
失败:	返回 -1,设置@error:标志为非 0 值.
@@End@@

None
