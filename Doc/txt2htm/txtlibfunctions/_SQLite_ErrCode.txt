###用户自定义函数###
_SQLite_ErrCode

###描述###
返回最近的错误号(数字)

###语法###
#Include <SQLite.au3>
_SQLite_ErrCode ( [ $hDB ] )

###参数###
@@ParamTable@@
 $hDB  [可选] 打开的数据库, 默认为最近打开的.
@@End@@

###返回值###
@@ReturnTable@@
成功: 返回可与$SQLITE_*常量比较的值.
失败: 返回0.
@error: 1 - 调用SQLite API 'sqlite3_errcode'错误
@@End@@
