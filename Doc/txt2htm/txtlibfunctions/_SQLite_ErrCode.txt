###User Defined Function###
_SQLite_ErrCode

###Description###
返回最后一个错误代码 (数字)

###Syntax###
#include <SQLite.au3>
_SQLite_ErrCode ( [ $hDB ] )


###Parameters###
@@ParamTable@@
$hDB
	[可选参数] 打开的数据库,默认为最后打开的数据库
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回值可能违反 $SQLITE_* 常量
失败:	返回 0
@错误:	1 - 错误调用 SQLite API 'sqlite3_errcode'
	2 - 调用被安全模式阻止
@@End@@

None.
