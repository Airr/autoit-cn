###User Defined Function###
_SQLite_ErrCode

###Description###
Returns last error code (numeric)

###Syntax###
#include <SQLite.au3>
_SQLite_ErrCode ( [ $hDB ] )


###Parameters###
@@ParamTable@@
$hDB
	[optional] An open database, default is the last opened database
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	Return value can be compared against $SQLITE_* constants
Failure:	Returns 0
@error:	1 - error calling SQLite API 'sqlite3_errcode'
	2 - Call Prevented by SafeMode
@@End@@

None.
