###User Defined Function###
_WinAPI_GetTickCount64

###Description###
检索系统已启动以来运行的毫秒数.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_GetTickCount64 ( )

###Parameters###
@@ParamTable@@
None.
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回系统启动以来运行的毫秒数.
失败:	返回 0,并设置@error标志为非 0 值.
@@End@@

###Remarks###
本函数需要 Windows Vista 或更高版本系统.
