###User Defined Function###
_WinAPI_IsDoorOpen

###Description###
检查光驱托盘的开/关状态.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_IsDoorOpen ( $sDrive )

###Parameters###
@@ParamTable@@
$sDrive
	CD 盘驱动器号, 如 D:, E:, 等等.
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 1 - CD (DVD) 托盘打开.
	返回 0 - 未打开.
失败:	返回 0,设置@error:标志为非 0 值.
@@End@@
