###User Defined Function###
_GUICtrlHeader_SetImageList

###Description###
Assigns an image list

###Syntax###
#Include <GuiHeader.au3>
_GUICtrlHeader_SetImageList($hWnd, $hImage)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to control
$hImage
	Handle to an image list
@@End@@

###ReturnValue###
Success: The handle to the image list previously associated with the control
Failure: 0

###Remarks###
None.
