###User Defined Function###
_SQLite_Close

###Description###
关闭 SQLite 数据库.

###Syntax###
#include <SQLite.au3>
_SQLite_Close ( [ $hDB ] )


###Parameters###
@@ParamTable@@
$hDB
	[可选参数] 数据库句柄.
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 $SQLITE_OK.
失败:	返回值可能违反 $SQLITE_* 常量.
@错误:	-1 - SQLite 报告错误 (检查返回值)
	1 - 错误调用 SQLite API 'sqlite3_close'
	2 - 调用被安全模式阻止
@@End@@

###Remarks###
如果没有数据库句柄, 最后打开的数据库将被关闭.
