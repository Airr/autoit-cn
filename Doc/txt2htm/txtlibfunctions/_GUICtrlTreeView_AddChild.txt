###User Defined Function###
_GUICtrlTreeView_AddChild

###Description###
添加子项目

###Syntax###
#Include <GuiTreeView.au3>
_GUICtrlTreeView_AddChild($hWnd, $hParent, $sText [, $iImage = -1 [, $iSelImage = -1]])

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
$hParent
	父项目
$sText
	新项目文本
$iImage
	[可选参数] 项目图标在图像列表中的 0 基索引
$iSelImage
	[可选参数] 子项目图标在图像列表中的 0 基索引
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回新项目句柄
失败:	返回 0
@@End@@

###Remarks###
 添加项目作为 $hParent 的子项添加到 $hParent 子项列表的末尾.
 如果控件有排序功能,则新项目插入到控件子项目排序的正确位置.
