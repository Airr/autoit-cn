###User Defined Function###
_GUICtrlTreeView_SetChildren

###Description###
设置项目的子项标志

###Syntax###
#Include <GuiTreeView.au3>
_GUICtrlTreeView_SetChildren($hWnd, $hItem[, $fFlag = True])

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
$hItem
	项目句柄
$fFlag
	[可选参数] 设置标志:
		True - 项目有子项
		False - 清除项目的子项标志
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	True
失败:	False
@@End@@

###Remarks###
None.
