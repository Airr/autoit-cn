###User Defined Function###
_WinAPI_PathGetCharType

###Description###
Determines the type of character in relation to a path.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_PathGetCharType ( $sChar )

###Parameters###
@@ParamTable@@
$sChar
	The character for which to determine the type.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	Returns one or more of the following values that define the type of character.
Failure	(-1) and sets the @error flag to non-zero.
@@End@@

None
