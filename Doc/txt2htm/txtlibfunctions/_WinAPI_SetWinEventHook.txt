###User Defined Function###
_WinAPI_SetWinEventHook

###Description###
Sets an event hook function for a range of events.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_SetWinEventHook( $iEventMin, $iEventMax, $pWinEventProc [, $iProcessID [, $iThreadID [, $iFlags]]] )

###Parameters###
@@ParamTable@@
$iEventMin
	The lowest event value in the range of events ($EVENT_*) that are handled by the hook function.
$iEventMax
	The highest event value in the range of events ($EVENT_*) that are handled by the hook function.
$pEventProc
	The address of an application-defined hook function that the system calls in response to
	events generated by an accessible object.
$iProcessID
	[可选参数] The ID of the process from which the hook function receives events. If this parameter is 0,
	the hook function is associated with all existing processes on the current desktop.
$iThreadID
	[可选参数] The ID of the thread from which the hook function receives events. If this parameter is 0,
	the hook function is associated with all existing threads on the current desktop.
$iFlags
	[可选参数] The flags that specify the location of the hook function and of the events to be skipped.
	The following flags are valid:
	
	$WINEVENT_INCONTEXT
	$WINEVENT_OUTOFCONTEXT
	$WINEVENT_SKIPOWNPROCESS
	$WINEVENT_SKIPOWNTHREAD
	
	The following single flags, or flag combinations are valid:
	
	$WINEVENT_INCONTEXT
	$WINEVENT_OUTOFCONTEXT
	$WINEVENT_INCONTEXT | $WINEVENT_SKIPOWNPROCESS
	$WINEVENT_INCONTEXT | $WINEVENT_SKIPOWNTHREAD
	$WINEVENT_OUTOFCONTEXT | $WINEVENT_SKIPOWNPROCESS
	$WINEVENT_OUTOFCONTEXT | $WINEVENT_SKIPOWNTHREAD
	
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	A value that identifies this event hook instance.
Failure	0 and sets the @error flag to non-zero.
@@End@@

###Remarks###
Clients can call _WinAPI_SetWinEventHook() multiple times if they want to register additional hook functions
or listen for additional events.
