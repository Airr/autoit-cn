###User Defined Function###
_GUICtrlComboBoxEx_SetCurSel

###Description###
在组合控件列表框中选择字符串

###Syntax###
#Include <GuiComboBoxEx.au3>
_GUICtrlComboBoxEx_SetCurSel($hWnd[, $iIndex = -1])

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
$iIndex
	[可选参数] 指定字符串的 0 基索引
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 True
失败:	返回 False
@@End@@

###Remarks###
 如果 $iIndex 为 –1, 列表中的任何当前选项将被删除且编辑框被清空.
 如果 $iIndex 大于列表中的项目数量,
 或 $iIndex 为 –1, 则返回值为 -1,选中项将被清除.
