###User Defined Function###
_GUICtrlRichEdit_SetModified

###Description###
设置或清除修改标志

###Syntax###
#Include <GuiRichEdit.au3>
_GUICtrlRichEdit_SetModified($hWnd, $fState = True)

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
$fState
	指定标志的修改新值:
		True - 文本已被修改（默认）
		False - 没有修改.
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 True
失败:	返回 False设置@error
@error:	101 - $hWnd 参数值不是句柄
	102 - $fState 必须是 True 或 False
@@End@@

###Remarks###
 控件创建时,系统自动设置修改标志为 False.
 如果文本被用户或编程修改时,系统设置修改标志为 True.
 调用 _GUICtrlRichEdit_IsModified 检索标志的当前状态.
