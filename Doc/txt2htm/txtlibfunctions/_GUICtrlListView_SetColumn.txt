###User Defined Function###
_GUICtrlListView_SetColumn

###Description###
设置列的属性

###Syntax###
#Include <GuiListView.au3>
_GUICtrlListView_SetColumn($hWnd, $iIndex, $sText [, $iWidth = -1 [, $iAlign = -1 [, $iImage = -1 [, $fOnRight = False]]]])

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
$iIndex
	列的 0 基索引
$sText
	列标题文本
$iWidth
	[可选参数] 列像素宽度
$iAlign
	[可选参数] 列标题和列项目文本对齐方式:
		0 - 文本左对齐
		1 - 文本右对齐
		2 - 文本居中
$iImage
	[可选参数] 图像列表中的图像 0 基索引
$fOnRight
	[可选参数] 如为 True, 则列图像显示在文字右侧
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 True
失败:	返回 False
@@End@@

None.
