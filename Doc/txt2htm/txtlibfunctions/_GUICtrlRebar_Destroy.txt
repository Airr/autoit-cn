###User Defined Function###
_GUICtrlRebar_Destroy

###Description###
Delete the control

###Syntax###
#Include <GuiRebar.au3>
_GUICtrlRebar_Destroy(ByRef $hWnd)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	True, Handle is set to 0
失败:	False
@@End@@

###Remarks###
Restricted to only be used on control created with _GUICtrlRebar_Create
