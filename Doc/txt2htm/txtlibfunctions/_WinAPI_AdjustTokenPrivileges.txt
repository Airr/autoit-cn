###User Defined Function###
_WinAPI_AdjustTokenPrivileges

###Description###
启用或禁用指定的访问令牌权限.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_AdjustTokenPrivileges ( $hToken, $aPrivileges, $iState )

###Parameters###
@@ParamTable@@
$hToken
	访问令牌的句柄.
		 该句柄必须有 $TOKEN_ADJUST_PRIVILEGES 与 $TOKEN_QUERY 访问权标.
$aPrivileges
	特权变量. 如果此参数为 (-1), 函数将禁用特许权标,
		 并忽略 $iState 参数. 此参数可以是以下类型之一:
		 单特许常量 ($SE_*).
		 $SE_* 常量的一维数组.
		 $SE_* 常量及其属性(见 $iState)的二维数组.
		 [0][0] - 特许
		 [0][1] - 属性
		 [n][0] - 特许
		 [n][1] - 属性
$iState
	特权属性. 如果 $aPrivileges 参数为一维数组, $iState 将应用于整个数组.
		 如果 $aPrivileges 参数为 (-1) 或二维数组, 函数将忽略此参数,使用该数组中指定的属性.
		 参数可以是以下值之一:
		 0 - 禁用特许权限.
		 1 - 启用特许权限.
		 2 - 特许权限由默认启用.
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	如果 $aPrivileges 是单 $SE_* 常量, 则返回先前的特许属性 (0 或 1),
	否则总是返回 1. 要确定函数是否调整了指定的全部特权,
	当函数成功时,检查 @extended 标志返回的下列值:
	0 - 函数已调整所有指定的特权.
	1 - $aPrivileges 参数没有指定一或多个特权.
失败:	返回 0,设置@error:标志为非 0 值.
@@End@@

###Remarks###
函数不能添加新的权限访问令牌.它只能启用或禁用该令牌的现有权限.
