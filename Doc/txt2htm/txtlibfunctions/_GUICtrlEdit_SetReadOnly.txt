###User Defined Function###
_GUICtrlEdit_SetReadOnly

###Description###
设置或删除只读样式 ($ES_READONLY)

###Syntax###
#Include <GuiEdit.au3>
_GUICtrlEdit_SetReadOnly($hWnd, $fReadOnly)

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
$fReadOnly
	可选值:
		True - 设置 $ES_READONLY 样式
		False - 删除 $ES_READONLY 样式
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 True
失败:	返回 False
@@End@@

None.
