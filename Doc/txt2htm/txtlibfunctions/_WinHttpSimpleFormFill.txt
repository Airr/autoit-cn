###User Defined Function###
_WinHttpSimpleFormFill

###Description###
填写表单

###Syntax###
#Include <WinHttp.au3>
_WinHttpSimpleFormFill(ByRef $hInternet [, $sActionPage = Default [, $sFormId = Default [, $sFieldId1 = Default [, $sData1 = Default [, (...)]]]]])

###Parameters###
@@ParamTable@@
$hInternet
	_WinHttpConnect()返回的句柄. 在表要求重定向的情况下该句柄将关闭并由新的需要的句柄替换.
$sActionPage
	[可选参数] 表单所在页的路径(默认: "" - 空字符串. 表示服务器上的'默认'页).
$sFormId
	[可选参数] 表单ID. 可为名称或0基索引
$sFieldId1
	[可选参数] 输入框ID. 
$sData1
	[可选参数] 设置到相应字段的数据
(...)
	[可选参数] 其他字段/数据对. 不超过40
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回由被提交表单所在服务器返回的HTML源文件
失败:	返回空字符串设置@error
	1 - 页上无表单
	2 - 无效表单
	3 - 页上没有指定属性的表单
	4 - 连接问题
@@End@@

###Remarks###
$sFormId指定表单的Id如.getElementById(FormId). 其他的可使用"index:FormIndex"按0基索引的数字标识表单(如一些页面的三个表单则第一个索引为0, 第二个索引为1, 第三个索引为2). 使用"name:FormName"标识表单由其名称如.getElementsByName(FormName). 表单名为颜色标记. 在此情况下填充首个有名称的表单. 不用于$sFieldId情况. 如果使用"name:InputName"选项则以指定$sData数据填充的全部该名称的字段. 该函数一次可填充40个字段. 未设置字段保留默认.
