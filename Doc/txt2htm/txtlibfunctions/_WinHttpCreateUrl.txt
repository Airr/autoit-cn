###User Defined Function###
_WinHttpCreateUrl

###Description###
Creates a URL from array of components such as the host name and path.

###Syntax###
#Include <WinHttp.au3>
_WinHttpCreateUrl($aURLArray)

###Parameters###
@@ParamTable@@
$aURLArray
	Array of URL data.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	Returns created URL
Failure	Returns empty string and sets @error:
	1 - Invalid input.
	2 - Initial DllCall failed
	3 - Main DllCall failed
@@End@@

###Remarks###
Input is one dimensional 8 elements in size array:
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;- first element [0] is scheme name
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;- second element [1] is internet protocol scheme
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;- third element [2] is host name
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;- fourth element [3] is port number
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;- fifth element [4] is user name
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;- sixth element [5] is password
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;- seventh element [6] is URL path
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;- eighth element [7] is extra information
