###User Defined Function###
_WinAPI_LZNTDecompress

###Description###
Decompresses an input data.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_LZNTDecompress ( $tInput, ByRef $tOutput, $iBufferSize )

###Parameters###
@@ParamTable@@
$tInput
	"byte[n]" or any other structure that contains the data to be decompressed.
$tOutput
	"byte[n]" structure that is created by this function, and contain the decompressed data.
$iBufferSize
	The size of the buffer which are used for decompressing, in bytes. In most cases, the buffer size
	should be several times (not less x2) larger than the size of the input data.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	The size of the decompressed data, in bytes.
Failure	0 and sets the @error flag to non-zero, @extended flag may contain the NTSTATUS code.
@@End@@

###Remarks###
The input and output buffers must be different, otherwise, the function fails.
