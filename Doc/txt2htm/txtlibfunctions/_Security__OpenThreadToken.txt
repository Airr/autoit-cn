###User Defined Function###
_Security__OpenThreadToken

###Description###
打开与线程相关访问令牌

###Syntax###
#Include <Security.au3>
_Security__OpenThreadToken($iAccess[, $hThread = 0 [, $fOpenAsSelf = False]])

###Parameters###
@@ParamTable@@
$iAccess
	访问掩码,指定访问令牌要求的访问类型
		请求的访问类型协调令牌的自由访问控制列表（DACL）,
		以确定允许或拒绝访问
$hThread
	[可选参数] 打开访问令牌的线程句柄
$fOpenAsSelf
	[可选参数] 指定如何检查线程调用 OpenThreadToken 函数的访问安全环境
		如果此参数为 False,由调用线程检查访问的安全环境.
		如果线程模拟客户端,此安全环境可以是客户端进程.
		如果参数为 True,由调用进程的线程检查访问的安全环境.
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回新开访问令牌的句柄
失败:	返回 0
@@End@@
