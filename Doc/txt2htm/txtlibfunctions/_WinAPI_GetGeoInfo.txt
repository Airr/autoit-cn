###User Defined Function###
_WinAPI_GetGeoInfo

###Description###
Retrieves information about a specified geographical location.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_GetGeoInfo ( $GEOID, $iType [, $iLanguage] )

###Parameters###
@@ParamTable@@
$GEOID
	The identifier for the geographical location (GEOID) for which to get information.
$iType
	The type of information to retrieve. It can be one of the following values.
	
	$GEO_NATION
	$GEO_LATITUDE
	$GEO_LONGITUDE
	$GEO_ISO2
	$GEO_ISO3
	$GEO_LCID
	$GEO_FRIENDLYNAME
	$GEO_OFFICIALNAME
	$GEO_TIMEZONES
	$GEO_OFFICIALLANGUAGES
	
	Windows Vista or later
	
	$GEO_RFC1766
	
	Windows 8 or later
	
	$GEO_ISO_UN_NUMBER
	$GEO_PARENT
	
$iLanguage
	[可选参数] The language identifier, used with the geographical location.
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 1.
失败:	返回 0 并设置 @error 标志为非 0 值.
@@End@@
