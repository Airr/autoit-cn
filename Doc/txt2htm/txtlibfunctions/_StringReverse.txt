###User Defined Function###
_StringReverse

###Description###
Reverses the contents of the specified string.

###Syntax###
#Include <String.au3>
_StringReverse($s_String)

###Parameters###
@@ParamTable@@
$s_String
	String to reverse
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	Returns reversed string
Failure:	Returns an empty string and sets @error
@Error:	0 = No error
	1 = One of the parameters is invalid
	2 = Dll error
@@End@@
