###User Defined Function###
_GUICtrlMonthCal_GetMonthRangeMinStr

###Description###
Retrieves date information that represents the low limit of the controls display in string format

###Syntax###
#Include <GuiMonthCal.au3>
_GUICtrlMonthCal_GetMonthRangeMinStr($hWnd[, $sFormat = "%02d/%02d/%04d"])

###Parameters###
@@ParamTable@@
$hWnd
	Handle to control
$sFormat
	[optional] StringFormat string used to format the date
@@End@@

###ReturnValue###
Success: Date in the specified format

###Remarks###
None.
