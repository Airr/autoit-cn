###User Defined Function###
_IE_Example

###Description###
显示一个已预载入文档的新浏览器窗口,用于IE.au3中的例子或者您自己的测试.

###Syntax###
#include <IE.au3> 
_IE_Example ( [$s_module = "basic"] )

###Parameters###
@@ParamTable@@
$s_module
	[可选参数] 指定要运行的哪个模块
	basic = (默认) 带有文本,链接和图片的简单 HTML 页面
	form = 带有多个表单(form)元素的简单 HTML 页面
	frameset = 带有框架(frames)的简单 HTML 页面
	iframe = 带有框架(frames)的简单 HTML 页面
	table = 带有表(tables)的简单 HTML 页面
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回一个指向 InternetExplorer.Application 对象的对象变量
失败:	返回 0 并设置 @ERROR
@Error:	0 ($_IEStatus_Success) = 没有错误
	5 ($_IEStatus_InvalidValue) = 无效值
@Extended:	包含无效参数数量
@@End@@


###Remarks###
这个函数显示一个预载入HTML和HTML元素的浏览器窗口,用于整个帮助文件和 IE.au3 例子中. 同时也推荐您在 IE.au3 函数中使用它们.
