    ###User Defined Function###
    _WinAPI_GetCursorInfo

###Description###
    获取全局鼠标键的信息

###Syntax###
    #Include
    <WinAPI.au3>
    _WinAPI_GetCursorInfo()
@@End@@

###ReturnValue###
@@ReturnTable@@
成功: 如下格式数组:
&nbsp ;  $aCursor[0] - 成功为真, 否则为假
&nbsp ;  $aCursor[1] - 箭头处于显示状态为真, 否则为假
&nbsp ;  $aCursor[2] - 鼠标箭头句柄
&nbsp ;  $aCursor[3] - 箭头X坐标
&nbsp ;  $aCursor[4] - 箭头Y坐标
@@End@@
