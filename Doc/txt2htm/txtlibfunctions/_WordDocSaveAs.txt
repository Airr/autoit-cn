###User Defined Function###
_WordDocSaveAs

###Description###
_WordDocSaveAs
用新名字或新格式保存文档

###Syntax###
#Include <Word.au3>
_WordDocSaveAs ( ByRef $o_object [, $s_FilePath = "" [, $i_Format = 0 [, $b_ReadOnlyRecommended = 0 [, $b_AddToRecentFiles = 0 [, $b_LockComments = 0 [, $s_Password = "" [, $s_WritePassword = ""]]]]]]] )

###Parameters###
@@ParamTable@@
$o_object 
   Word.Application的文档对象变量
$s_FilePath 
   可选: 保存文档的完整路径. (见备注)
    "" = (默认) 如果文档从没被保存过, 可使用默认名称(例如,
 
      Document1.doc)
$i_Format 
   可选: 文档保存的格式.
    0 = (默认) Microsoft Word格式
    1 = Microsoft Word模板格式
    2 = Microsoft Windows文本格式
    3 = Microsoft Windows保留断行符的文本格式
    4 = Microsoft DOS文本格式
    5 = Microsoft DOS保留断行的文本格式
    6 = 富文本格式 (RTF)
    7 = Unicode 文本格式或编码的文本格式
    8 = 标准HTML格式
    9 = 网络档案格式
    10 = 过滤的HTML格式
    11 = 可扩展标记语言(XML)格式
$b_ReadOnlyRecommended 
   可选: 指定Microsoft Word是否假定只读无论文件是否已打开.
    0 = (默认) 不假定
    1 =
 
    假定只读
$b_AddToRecentFiles 
   可选: 指定是否将文件名添加到文件菜单底部的最近使用的文件中.
    0 = (默认)不添加
    1 = 添加
$b_LockComments 
   可选: 指定是否锁定批注
    0 = (默认) 不锁定
    1 = 锁定
$s_Password 
   可选: 打开文件时使用的密码 (见备注)
$s_WritePassword 
   可选: 保存文件时使用的密码 (见备注)
@@End@@

###ReturnValue###
@@ReturnTable@@
 成功:  返回1
 失败:  返回0并设置 @ERROR
 @Error:  0 ($_WordStatus_Success) = 无错误
  1 ($_WordStatus_GeneralError) = 一般错误
  3 ($_WordStatus_InvalidDataType) = 无效数据类型
  4 ($_WordStatus_InvalidObjectType) = 无效对象类型
 @Extended:  参数数量无效
@@End@@

###Remarks###
如果指定文件名的文件已存在 , 文件将在没有提示的条件下被覆盖.
如果文件无法保存脚本将暂停至注册WordErrorHandler为止.
尽量避免在程序中使用强密码. 如果进程需要密码 , 可从用户处获取 , 将密码存为变量 , 然后在代码中使用变量
