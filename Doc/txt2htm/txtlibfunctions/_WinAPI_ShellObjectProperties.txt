###User Defined Function###
_WinAPI_ShellObjectProperties

###Description###
Invokes the Properties context menu command on a Shell object.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_ShellObjectProperties ( $sPath [, $iType [, $sProperty [, $hParent]]] )

###Parameters###
@@ParamTable@@
$sPath
	The object name.
$iType
	[可选参数] The value that specifies the type of object.
	
	$SHOP_PRINTERNAME
	$SHOP_FILEPATH
	$SHOP_VOLUMEGUID
	
$sProperty
	[可选参数] The name of the property sheet page to be opened initially.
$hParent
	[可选参数] Handle of the parent window of the dialog box.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	1.
Failure	0 and sets the @error flag to non-zero.
@@End@@

None
