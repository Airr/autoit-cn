###Structure Name###
$tagHDITEM

###Description###
包含标头控件中的项目信息

###Syntax###
Global Const $tagHDITEM = "int Mask;int XY;ptr Text;hwnd hBMP;int TextMax;int Fmt;int Param;int Image;int Order;int Type;ptr pFilter;int State"

###Fields###
@@ParamTable@@
Mask
	说明哪些其它结构成员包含有效数据,或必须填写
XY
	项目的宽度或高度
Text
	项目地址的字符串
hBMP
	项目位图句柄
TextMax
	项目字符串长度
Fmt
	指定项目格式
Param
	应用程序定义的项目数据
Image
	图像在图像列表中的 0 基索引
Order
	项目在标头控件中从左至右出现
Type
	由 pFilter 指定的过滤器类型
pFilter
	应用程序定义的数据项目地址
State
	项目状态
@@End@@

###Remarks###
None.
