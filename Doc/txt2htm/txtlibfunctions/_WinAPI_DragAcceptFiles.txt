###User Defined Function###
_WinAPI_DragAcceptFiles

###Description###
Registers whether a window accepts dropped files.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_DragAcceptFiles ( $hWnd [, $fAccept] )

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the window that is registering whether it will accept dropped files.
$fAccept
	[可选参数] Specifies whether a window accepts dropped files, valid values:
		TRUE    - Accept dropped files. (Default)
		FALSE   - Discontinue accepting dropped files.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	1.
Failure	0 and sets the @error flag to non-zero.
@@End@@
