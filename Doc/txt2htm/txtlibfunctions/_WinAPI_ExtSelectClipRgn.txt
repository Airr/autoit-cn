###Function###
_WinAPI_ExtSelectClipRgn

###Description###
Combines the specified region with the current clipping region.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_ExtSelectClipRgn ( $hDC, $hRgn [, $iMode] )

###Parameters###
@@ParamTable@@
$hDC
	Handle to the device context.
$hRgn
	Handle to the region to be selected. This handle can only be 0 when the $RGN_COPY mode is specified.
$iMode
	The operation to be performed. It must be one of the following values.
	
	$RGN_AND
	$RGN_COPY
	$RGN_DIFF
	$RGN_OR
	$RGN_XOR
	
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	The value that specifies the new clipping region's complexity&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;it can be one of the following values.
Failure	0 and sets the @error flag to non-zero.
@@End@@
