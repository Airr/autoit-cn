###User Defined Function###
_WinAPI_IsBadCodePtr

###Description###
Determines whether the calling process has read access to the memory at the specified address.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_IsBadCodePtr ( $pAddress )

###Parameters###
@@ParamTable@@
$pAddress
	A pointer to a memory address.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	1 - The calling process does not have read access to the specified memory.
0	Otherwise.
Failure	0 and sets the @error flag to non-zero.
@@End@@

###Remarks###
None
