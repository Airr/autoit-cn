###结构常量###
$tagGDIPENCODERPARAM

###Description###
$tagGDIPENCODERPARAM 结构

###Syntax###
Global Const $tagGDIPENCODERPARAM =  "byte GUID[16];dword Count;dword Type;ptr Values"

###成员###
@@ParamTable@@
GUID
   参数类别标识符(GDI_EPG常量)
Count
   由Value成员指定的数组中的值的数量
Type
   参数数据类型标识符(GDI_EPT常量)
Values
   数组的值的指针. 每个值均具有由Type成员指定的类别.
@@end@@
