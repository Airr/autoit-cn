###User Defined Function###
_GUICtrlMenu_InsertMenuItem

###Description###
插入一个新菜单项到指定位置

###Syntax###
#Include <GuiMenu.au3>
_GUICtrlMenu_InsertMenuItem($hMenu, $iIndex, $sText[, $iCmdID = 0[, $hSubMenu = 0]])

###Parameters###
@@ParamTable@@
$hMenu
	菜单句柄
$iIndex
	插入的菜单项 0 基位置
$sText
	菜单项的文本,如果空白,将被插入一个分隔符.
$iCmdID
	[可选参数] 分配给该项目的命令 ID
$hSubMenu
	[可选参数] 与菜单项关联的子菜单句柄
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 True
失败:	返回 False
@@End@@

###Remarks###
None.
