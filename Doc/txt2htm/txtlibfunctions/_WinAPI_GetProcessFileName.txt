###Function###
_WinAPI_GetProcessFileName

###Description###
Retrieves the fully-qualified path of the executable file for the specified process.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_GetProcessFileName ( [$PID] )

###Parameters###
@@ParamTable@@
$PID
	The PID of the process. Default (0) is the current process.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	The fully-qualified path to the file.
Failure	Empty string and sets the @error flag to non-zero.
@@End@@

###Remarks###
None
