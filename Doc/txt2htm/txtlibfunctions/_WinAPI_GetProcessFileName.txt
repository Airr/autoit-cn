###User Defined Function###
_WinAPI_GetProcessFileName

###Description###
检索指定进程可执行文件的完整路径.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_GetProcessFileName ( [$PID] )

###Parameters###
@@ParamTable@@
$PID
	[可选参数] 进程 PID. 默认 0, 为当前进程.
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回文件的完整路径.
失败:	返回空串,设置@error:标志为非 0 值.
@@End@@

None
