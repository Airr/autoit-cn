###User Defined Function###
_GUICtrlEdit_EndUpdate

###Description###
Enables screen repainting that was turned off with the BeginUpdate function

###Syntax###
#Include <GuiEdit.au3>
_GUICtrlEdit_EndUpdate($hWnd)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
@@End@@

###ReturnValue###
成功: True
失败: False

###Remarks###
None.
