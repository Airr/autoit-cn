###用户自定义函数###
_IEGetObjByName

###描述###
返回指定名称(name)或者id的对象变量

###语法###
#Include <IE.au3>
_IEGetObjByName ( ByRef $o_object, $s_Id [, $i_index = 0] )

###参数###
@@ParamTable@@
 对象变量  InternetExplorer.Application , 窗口或者框架(frame)对象的对象变量
 ID  定义你想要匹配的对象的名称或者ID
 索引  [可选]: 如果名称或者ID出现不止一次 , 定义从0开始的具体对象.
0 (默认)或者正整数返回对应索引号的对象
-1 返回指定对象的集合
@@End@@

###返回值###
@@ReturnTable@@
成功: 返回指向指定对象的对象变量 , @EXTENDED = 指定对象数量
失败: 返回0并设置@ERROR为:
    0 ($_IEStatus_Success) = 无错误
    3 ($_IEStatus_InvalidDataType) = 无效数据类型
    7 ($_IEStatus_NoMatch) = 不匹配
    @Extended: 包含无效参数数量
@@End@@

###备注###
这个函数将返回指定名称或者ID的对象的对象引用. DOM组件可以有名称或者ID属性或者二者都有. 一个特殊的ID可以被分配给一个单独的元素而一个特殊的名称(name)可以被分配给多个元素. 在IE里面名称和ID是被共享的.
