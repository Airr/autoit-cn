###User Defined Function###
_GUICtrlListView_SetItemSelected

###Description###
设置项目是否被选中

###Syntax###
#Include <GuiListView.au3>
_GUICtrlListView_SetItemSelected($hWnd, $iIndex[, $fSelected = True[, $fFocused = False]])

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
$iIndex
	项目的 0 基索引, -1, 则设置所有项目的选定状态
$fSelected
	[可选参数] 如果为 True, 项目被选中,否则不是.
$fFocused
	[可选参数] 如果 True,项目具有焦,否则不是.
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 True
失败:	返回 False
@@End@@
