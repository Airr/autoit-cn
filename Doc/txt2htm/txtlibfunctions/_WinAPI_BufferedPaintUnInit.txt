###User Defined Function###
_WinAPI_BufferedPaintUnInit

###Description###
Closes down buffered painting for the current thread.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_BufferedPaintUnInit ( )

###Parameters###
@@ParamTable@@
None.
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 1.
失败:	返回 0 并设置 @error 标志为非 0 值, @extended 标志可能包含一个系统错误代码.
@@End@@

###Remarks###
本函数需要 Windows Vista 或更高版本系统.
