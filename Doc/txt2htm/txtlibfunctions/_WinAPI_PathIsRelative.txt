###Function###
_WinAPI_PathIsRelative

###Description###
Searches a path and determines if it is relative.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_PathIsRelative ( $sPath )

###Parameters###
@@ParamTable@@
$sPath
	The path to be searched.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	1 - The path is relative.
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;0	Otherwise.
Failure	0 and sets the @error flag to non-zero.
@@End@@
