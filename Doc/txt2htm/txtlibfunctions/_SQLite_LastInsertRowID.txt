###用户自定义函数###
_SQLite_LastInsertRowID

###描述###
返回最近插入到数据库的ROWID

###语法###
#Include <SQLite.au3>
_SQLite_LastInsertRowID ( [ $hDB ] )

###参数###
@@ParamTable@@
 $hDB  [可选] 打开的数据库, -1为最近打开的数据库
@@End@@

###返回值###
@@ReturnTable@@
成功: 返回ROWID
失败: 返回0
@error: 1 - 调用SQLite API 'sqlite3_last_insert_rowid'错误
   2 - 安全模式阻止调用
@@End@@
