###User Defined Function###
_SQLite_LastInsertRowID

###Description###
返回数据库中最新插入的 ROWID

###Syntax###
#include <SQLite.au3>
_SQLite_LastInsertRowID ( [ $hDB ] )


###Parameters###
@@ParamTable@@
$hDB
	[可选参数] 打开的数据库,如为 -1, 则使用最后打开的数据库
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 ROWID
失败:	返回 0@错误:
	1 - 错误调用 SQLite API 'sqlite3_last_insert_rowid'
	2 - 调用被安全模式阻止
@@End@@
