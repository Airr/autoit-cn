###User Defined Function###
_SQLite_LastInsertRowID

###Description###
返回最近插入到数据库的ROWID

###Syntax###
#Include <SQLite.au3>
_SQLite_LastInsertRowID ( [ $hDB ] )

###Parameters###
@@ParamTable@@
 $hDB  [可选] 打开的数据库, -1为最近打开的数据库
@@End@@

###ReturnValue###
@@ReturnTable@@
成功: 返回ROWID
失败: 返回0
@error: 1 - 调用SQLite API 'sqlite3_last_insert_rowid'错误
   2 - 安全模式阻止调用
@@End@@
