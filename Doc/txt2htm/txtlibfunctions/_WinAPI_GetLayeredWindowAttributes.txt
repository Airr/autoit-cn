###User Defined Function###
_WinAPI_GetLayeredWindowAttributes

###Description###
获取分层窗口属性

###Syntax###
#Include <WinAPI.au3>
_WinAPI_GetLayeredWindowAttributes($hWnd, ByRef $i_transcolor, ByRef $Transparency [, $asColorRef = False])

###Parameters###
@@ParamTable@@
$hwnd
	工作 GUI 的句柄
$i_transcolor
	返回透明的颜色 ( 双字 0x00bbggrr 或字符串 "0xRRGGBB")
$Transparency
	返回透明的 GUI
$asColorRef
	[可选参数] 如为 True, $i_transcolor 将是 COLORREF( 0x00bbggrr ), 否则为 RGB 颜色
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回应用的 LWA_ALPHA 与 LWA_COLORKEY (使用 BitAnd)
失败:	返回 0
@error:	
	1 到 3 - DllCall 错误
	4 - 函数不成功@extended:
	包含 _WinAPI_GetLastError()
@@End@@

###Remarks###
 最低操作系统 - Windows XP.
