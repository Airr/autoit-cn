###User Defined Function###
_SoundSeek

###Description###
寻道指定位置的声音 

###Syntax###
#include <Sound.au3>
_SoundSeek (ByRef $aSnd_id, $iHour, $iMin, $iSec )

###Parameters###
@@ParamTable@@
$aSnd_id
	由 _SoundOpen() 返回的声音数组 ID
$iHour
	寻道到小时
$iMin
	寻道到分钟
$iSec
	寻道到秒钟
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 1
失败:	返回 0,设置@error:	
@error:	1 = 失败
	3 = 无效的声音标识. 使用 _SoundOpen()返回的数组.
@@End@@

###Remarks###
 使用 _SoundSeek 的声音,必须使用 _SoundPlay 恢复播放.
 此函数强制使用 _SoundOpen 返回的 ID 数组, 无论编码类型,确保文件都从正确位置播放.
 该函数的时间校正因子 VBR 可能被改变时, ID 数组应该更新.
