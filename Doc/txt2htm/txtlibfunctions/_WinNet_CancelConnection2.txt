###User Defined Function###
_WinNet_CancelConnection2

###Description###
取消现有的网络连接

###Syntax###
#Include <WinNet.au3>
_WinNet_CancelConnection2($sName [, $fUpdate = True [, $fForce = True]])

###Parameters###
@@ParamTable@@
$sName
	指定重定向本地设备的名称,或要断开的远程网络资源的名称．
		当此参数指定一个重定向的本地设备,函数取消指定的设备重定向;
		如果参数指定远程网络资源,只有在没有设备连接到远程网络时才被取消.
$fUpdate
	[可选参数] 如果为True,用户配置文件被更新,
		信息连接已经不再是一个持续连接.
$fForce
	[可选参数] 如果有打开的文件或连接工作,指定是否应该发生断线．
		如果此参数为　False,且存在打开的文件或工作,该函数失败.
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	True
失败:	False
@@End@@
