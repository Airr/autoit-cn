###User Defined Function###
_GUICtrlMenu_InsertMenuItemEx

###Description###
插入一个新菜单项到指定位置(Ex)

###Syntax###
#Include <GuiMenu.au3>
_GUICtrlMenu_InsertMenuItemEx($hMenu, $iIndex, ByRef $tMenu[, $fByPos = True])

###Parameters###
@@ParamTable@@
$hMenu
	菜单句柄
$iIndex
	项目位置,函数将在此位置之前插入新项目
$tMenu
	$tagMENUITEMINFO 结构
$fByPos
	[可选参数] 菜单识别标志:
		True - 限定 $iIndex 参数指定项目的 0 基位置
		False - 限定 $iIndex 参数指定项目的标识符
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 True
失败:	返回 False
@@End@@

###Remarks###
None.
