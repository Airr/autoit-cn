###User Defined Function###
_WinAPI_UrlUnescape

###Description###
Converts escape sequences back into ordinary characters.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_UrlUnescape ( $sUrl [, $iFlags] )

###Parameters###
@@ParamTable@@
$sUrl
	The URL.
$iFlags
	[可选参数] The flags that control which characters in the URL should be unescaped (an escape sequence has the form "%xy").
	It can be a combination of the following values.
	
	$URL_DONT_UNESCAPE_EXTRA_INFO
	
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	The unescaped URL.
Failure	Empty string and sets the @error flag to non-zero, @extended flag may contain the system error code.
@@End@@
