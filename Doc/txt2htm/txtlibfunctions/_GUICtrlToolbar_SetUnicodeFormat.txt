###User Defined Function###
_GUICtrlToolbar_SetUnicodeFormat

###Description###
设置 Unicode 字符格式标志

###Syntax###
#Include <GuiToolBar.au3>
_GUICtrlToolbar_SetUnicodeFormat($hWnd, $fUnicode = False)

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
$fUnicode
	设置 Unicode 字符:
		True - 控件使用 Unicode 字符
		False - 控件使用 ANSI 字符
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回先前设置的字符格式
@@End@@

###Remarks###
None.
