###User Defined Function###
_WinHttpWriteData

###Description###
写入请求数据到一个HTTP服务器.

###Syntax###
#Include <WinHttp.au3>
_WinHttpWriteData($hRequest, $vData [, $iMode = Default ])

###Parameters###
@@ParamTable@@
$hRequest
	由 _WinHttpSendRequest() 返回的有效句柄.
$vData
	要写入的数据
$iMode
	[可选参数] 整数方式定义的写入模式. 默认为 0 - 写入一个 ANSI 字符串.
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回1设置@error 0
	@extended 接收被写入的字节数
失败:	返回0设置@error 且当其为1时表示DllCall失败
@@End@@

###Remarks###
$data变量是要写入的字符串或二进制数据
$iMode可为下列值:
0 - 写入ANSI字符串
1 - 写入二进制字符串
