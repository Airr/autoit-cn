###Function###
_WinAPI_BeginDeferWindowPos

###Description###
Allocates memory for a multiple-window-position structure.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_BeginDeferWindowPos ( [$iAmount] )

###Parameters###
@@ParamTable@@
$iAmount
	The initial number of windows for which to store position information.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	The value identifies the multiple-window-position structure.
Failure	0 and sets the @error flag to non-zero.
@@End@@

None
