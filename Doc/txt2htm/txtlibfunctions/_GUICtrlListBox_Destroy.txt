###User Defined Function###
_GUICtrlListBox_Destroy

###Description###
Delete the control

###Syntax###
#Include <GuiListBox.au3>
_GUICtrlListBox_Destroy(ByRef $hWnd)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
@@End@@

###ReturnValue###
成功: True, 句柄设置为 0
失败: False

###Remarks###
Restricted to only be used on Listbox created with _GUICtrlListBox_Create
