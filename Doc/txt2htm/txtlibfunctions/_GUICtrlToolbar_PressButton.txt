###User Defined Function###
_GUICtrlToolbar_PressButton

###Description###
按压或释放指定的按钮

###Syntax###
#Include <GuiToolBar.au3>
_GUICtrlToolbar_PressButton($hWnd, $iCommandID[, $fPress = True])

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
$iCommandID
	按钮命令 ID
$fPress
	[可选参数] 按下状态
		True - 按钮将被设置为按下状态
		False - 按钮将被设置为未按下状态
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 True
失败:	返回 False
@@End@@

###Remarks###
函数不触发下拉式按钮单击事件.
如果你想确保该按钮单击事件被激发,应该使用 _GUICtrlToolbar_ClickButton 函数,不涉及按钮样式
