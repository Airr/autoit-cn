###User Defined Function###
_GUICtrlListView_SetGroupInfo

###Description###
设置组信息

###Syntax###
#Include <GuiListView.au3>
_GUICtrlListView_SetGroupInfo($hWnd, $iGroupID, $sHeader[, $iAlign = 0[, $iState = $LVGS_NORMAL]])

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
$iGroupID
	组的 ID
$sHeader
	标题文本
$iAlign
	[可选参数] 组标题文本对齐:
		0 - 左
		1 - 居中
		2 - 右
$iState
	[可选参数] 支持 Vista 及以上系统, 可以有以下值之一:
		$LVGS_NORMAL - 展开组,显示组名称及组中全部项目.
		$LVGS_COLLAPSED - 折叠组.
		$LVGS_HIDDEN - 隐藏组.
		$LVGS_NOHEADER - 不显示组标题.
		$LVGS_COLLAPSIBLE - 组可以折叠.
		$LVGS_FOCUSED - 组拥有键盘焦点.
		$LVGS_SELECTED - 组被选中.
		$LVGS_SUBSETED - 仅显示部分项目.
		$LVGS_SUBSETLINKFOCUSED - 组子集链接有键盘焦点
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 True
失败:	返回 False
@@End@@


###Remarks###
 最低操作系统 - Windows XP.
