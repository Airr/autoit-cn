###User Defined Function###
_GUICtrlTreeView_Add

###Description###
添加主项目

###Syntax###
#Include <GuiTreeView.au3>
_GUICtrlTreeView_Add($hWnd, $hSibling, $sText[, $iImage = -1[, $iSelImage = -1]])

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
$hSibling
	同级项目
$sText
	新项目文本
$iImage
	[可选参数] 项目图标在图像列表中的 0 基索引
$iSelImage
	[可选参数] 子项目图标在图像列表中的 0 基索引
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回新项目句柄
失败:	返回 0
@@End@@

###Remarks###
 新项目将作为 $hSibling 的最后一个项目.
 如果控件有排序功能,则新项目插入到控件排序的正确位置
