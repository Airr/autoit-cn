###User Defined Function###
_GUICtrlRichEdit_SetParaShading

###Description###
设置(首个)被选段落或(无选定时的)当前段落的阴影样式

###Syntax###
#Include <GuiRichEdit.au3>
_GUICtrlRichEdit_SetParaShading($hWnd [, $iWeight = Default [, $sStyle = Default [, $sForeColor = Default [, $sBackColor = Default]]]])

###Parameters###
@@ParamTable@@
 $hWnd  控件句柄
 $iWeight  [可选]: 前景色的百分比, 剩余的为背景色
 $sStyle  [可选]: 阴影样式 - 包含一个下列的字符串:
non - 无
dhz - 水平黑
dvt - 垂直黑
ddd - 向下对角线黑
dud - 向上对角线黑
dgr - 网格黑
dtr - 网格黑
lhz - 水平亮
lvt - 垂直亮
ldd - 向下对角线亮
lud - 向上对角线亮
lgr - 网格亮
ltr - 网格亮
 $sForeColor  [可选]: 前景色 - 下列之一:
"blk" - 黑(初始值)
"blu" - 蓝
"cyn" - 青
"grn" - 绿
"mag" - 品红
"red" - 红
"yel" - 黄
"whi" - 白
"dbl" - 蓝黑
"dgn" - 墨绿
"dmg" - 深紫红
"drd" - 暗红
"dyl" - 暗黄
"dgy" - 深灰
"lgy" - 浅灰
 $sBackColor  [可选]: 与$sForeColor相同
@@End@@

###ReturnValue###
@@ReturnTable@@
 成功:  真
 失败:  假并设置@error
 @error:  101 - $hWnd不是句柄
  1021 - $iWeight不是正数
  1022 - $iWeight的值无效
  103 - $sStyle的值无效
  104 - $sForeColor的值无效
  105 - $sBackColor的值无效
  700 - 操作失败
@@End@@

###Remarks###
当选定文本时, 默认是第一个带有选定文本的段落的值.
如果没有选定时, 默认是当前段.
在富文本编辑控件中不显示阴影, 但此处创建的阴影不会在文字中显示
