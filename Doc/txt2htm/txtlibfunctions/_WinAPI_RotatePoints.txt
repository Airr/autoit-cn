###User Defined Function###
_WinAPI_RotatePoints

###Description###
Rotates a points from the array by the specified angle.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_RotatePoints ( ByRef $aPoint, $iXC, $iYC, $iAngle [, $iStart [, $iEnd]] )

###Parameters###
@@ParamTable@@
$aPoint
	The 2D array ([x1, y1, ...], [x2, y2, ...], ... [xN, yN, ...]). Every first two elements from this
	array specifies a point to be rotate. Other array elements (if any) do not change.
$iXC
	X-coordinates of the point on which there is a rotation, in logical units.
$iYC
	Y-coordinates of the point on which there is a rotation, in logical units.
$iAngle
	The angle to rotate, in degree.
$iStart
	[可选参数] The index of array to start rotating at.
$iEnd
	[可选参数] The index of array to stop rotating at.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	1.
Failure	0 and sets the @error flag to non-zero.
@@End@@
