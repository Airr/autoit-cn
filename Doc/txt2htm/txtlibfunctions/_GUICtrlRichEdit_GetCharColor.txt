###User Defined Function###
_GUICtrlRichEdit_GetCharColor

###Description###
Retrieves the color of the selected text or, if none selected, of the character to the right of the insertion point

###Syntax###
#Include <GuiRichEdit.au3>
_GUICtrlRichEdit_GetCharColor($hWnd)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to control
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	COLORREF value an @extended set to 1 if the color is the same throught the selection
Failure:	sets @error
@error:	101 - $hWnd is not a handle
@@End@@

###Remarks###
None.
