###User Defined Function###
_WinAPI_GetMapMode

###Description###
Retrieves the current mapping mode.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_GetMapMode ( $hDC )

###Parameters###
@@ParamTable@@
$hDC
	Handle to the device context.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	The value specifies the mapping mode (MM_*).
失败:	返回 0 并设置 @error 标志为非 0 值.
@@End@@

###Remarks###
None
