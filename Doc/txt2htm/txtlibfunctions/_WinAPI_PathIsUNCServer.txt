###User Defined Function###
_WinAPI_PathIsUNCServer

###Description###
Determines if a string is a valid Universal Naming Convention (UNC) for a server path only.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_PathIsUNCServer ( $sPath )

###Parameters###
@@ParamTable@@
$sPath
	The path to validate.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	1 - The path is a valid UNC path for a server only (no share name).
0	Otherwise.
Failure	0 and sets the @error flag to non-zero.
@@End@@

None
