###User Defined Function###
_GUICtrlMenu_CheckMenuItem

###Description###
设置菜单项复选标记 (√) 的选中或清除选中状态

###Syntax###
#Include <GuiMenu.au3>
_GUICtrlMenu_CheckMenuItem($hMenu, $iItem[, $fCheck = True[, $fByPos = True]])

###Parameters###
@@ParamTable@@
$hMenu
	菜单句柄
$iItem
	菜单项目的标识符或位置,由 $fByPos 参数指定.
$fCheck
	[可选参数] True, 则设置复选标记, False,则删除
$fByPos
	[可选参数] 菜单识别标志:
		True - $iItem 为项目的 0 基位置
		False - $iItem 为菜单项的标识符
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回菜单项的以前状态 ($MF_CHECKED 或 $MF_UNCHECKED)
失败:	返回 -1
@@End@@

###Remarks###
 菜单栏项目不能有复选标记.
 $iItem 参数标识用于打开子菜单或命令的项目.
 对于打开子菜单的项目, $Item 参数必须指定项目的位置; 对于命令, $Item 参数可以指定该项目的位置或标识符.
