###User Defined Function###
_WinAPI_BufferedPaintClear

###Description###
Clears a specified rectangle in the buffer to ARGB = {0,0,0,0}.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_BufferedPaintClear ( $hBP [, $tRECT] )

###Parameters###
@@ParamTable@@
$hBP
	Handle of the buffered paint context.
$tRECT
	[可选参数] $tagRECT structure that specifies the rectangle to clear. If this parameter is 0, the entire buffer is used.
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 1.
失败:	返回 0 并设置 @error 标志为非 0 值, @extended 标志可能包含一个系统错误代码.
@@End@@

###Remarks###
本函数需要 Windows Vista 或更高版本系统.
