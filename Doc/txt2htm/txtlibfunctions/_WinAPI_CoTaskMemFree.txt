###User Defined Function###
_WinAPI_CoTaskMemFree

###Description###
Frees a block of task memory.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_CoTaskMemFree ( $hMemory )

###Parameters###
@@ParamTable@@
$hMemory
	The pointer to the memory block to be freed.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	1.
Failure	0 and sets the @error flag to non-zero.
@@End@@
