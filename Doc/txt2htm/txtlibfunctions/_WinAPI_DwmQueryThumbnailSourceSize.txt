###User Defined Function###
_WinAPI_DwmQueryThumbnailSourceSize

###Description###
返回桌面窗口管理器(DWM)源缩略图的大小.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_DwmUnregisterThumbnail ( $hThumbnail )

###Parameters###
@@ParamTable@@
$hThumbnail
	检索源窗口大小的缩略图句柄.
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回包含源缩略图大小的 $tagSIZE 结构.
失败:	返回 0,设置 @error 标志为非 0 值, @extended 标志可能包含一个系统错误代码.
@@End@@

###Remarks###
本函数需要 Windows Vista 或更高版本系统.
