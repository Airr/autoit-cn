###User Defined Function###
_WinAPI_GetMousePosX

###Description###
返回当前鼠标的 X 位置

###Syntax###
#Include <WinAPI.au3>
_WinAPI_GetMousePosX([$fToClient = False [, $hWnd = 0]])

###Parameters###
@@ParamTable@@
$fToClient
	[可选参数] 如为 True, 坐标将被转换为客户坐标
$hWnd
	[可选参数] 如 $fToClient 为 True, 则指定窗口句柄
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回鼠标的 X 位置
失败:	设置@error	
@@End@@

###Remarks###
 获取鼠标位置时,函数会考虑到目前 MouseCoordMode 的设置.
 它也将传递转换屏幕坐标参数到客户端.
