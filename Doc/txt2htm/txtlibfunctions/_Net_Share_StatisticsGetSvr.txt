###用户自定义函数###
_Net_Share_StatisticsGetSvr

###描述###
获取服务器的操作统计

###语法###
#Include <NetShare.au3>
_Net_Share_StatisticsGetSvr([$sServer = ""])

###参数###
@@ParamTable@@
 $sServer  用以指定函数将操作的远程服务器的DNS或NetBIOS名称的字符串. 如果参数为空则使用本地计算机.
@@End@@

###返回值###
@@ReturnTable@@
成功: 如下格式数组:
    [ 0] - 统计开始的时间. 该值存储为从00:00:00, January 1, 1970, GMT开始所经过秒数
    [ 1] - 服务器上一个文件的打开次数
    [ 2] - 服务器设备的打开次数
    [ 3] - 延迟的服务器打印工作的数量
    [ 4] - 启动服务器会话的次数
    [ 5] - 服务器会话被断开的次数
    [ 6] - 服务器会话因错误失败的次数
    [ 7] - 服务器密码出错的次数
    [ 8] - 服务器获取许可错误的次数
    [ 9] - 服务器系统错误的次数
    [10] - 服务器发送到网络的字节数
    [11] - 服务器从网络收到的字节数
    [12] - 服务器的平均回应的毫秒时间
    [13] - 服务器请求所必需的缓冲区但定位失败的次数
    [14] - 服务器请求所需的大缓冲区但定位失败的次数
@@End@@

###备注###
不需要特殊组成员获取服务器统计信息. 仅本地管理组或服务器操作组可在远程服务器上执行该函数.
