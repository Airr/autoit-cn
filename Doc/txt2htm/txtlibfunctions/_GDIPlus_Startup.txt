###User Defined Function###
_GDIPlus_Startup

###Description###
初始化微软 Windows GDI+

###Syntax###
#Include <GDIPlus.au3>
_GDIPlus_Startup()

###Parameters###
None.


###ReturnValue###
@@ReturnTable@@
成功:	返回 True
失败:	返回 False
@@End@@

###Remarks###
在您创建任何 GDI+ 对象前,先调用 _GDIPlus_Startup.
应用程序运行在 Windows 2000 操作系统中时,GDI+ 需要一个独立的可再发行包才能支持.
