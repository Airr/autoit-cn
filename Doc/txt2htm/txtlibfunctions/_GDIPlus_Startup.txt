###User Defined Function###
_GDIPlus_Startup

###Description###
初始化Microsoft Windows GDI+

###Syntax###
#Include <GDIPlus.au3>
_GDIPlus_Startup()

###Parameters###
@@ParamTable@@
    无
@@End@@

###ReturnValue###
@@ReturnTable@@
成功: 真
失败: 假
@@End@@

###Remarks###
在创建GDI+对象前调用_GDIPlus_Startup. GDI+对于运行于Microsoft Windows NT 4.0 SP6, Windows 2000, Windows 98, 及Windows Me上的应用程序需要再次分发.
