###Function###
_WinAPI_GetDefaultPrinter

###Description###
Retrieves the printer name of the default printer for the current user on the local computer.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_GetDefaultPrinter ( )

###Parameters###
@@ParamTable@@
None.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	String containing the default printer name.
Failure	Empty string and sets the @error flag to:
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;1	Function fails or not found
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;2	No default printer
@@End@@
