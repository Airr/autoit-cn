###User Defined Function###
_WinAPI_GetTempFileName

###Description###
创建临时文件的名称.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_GetTempFileName ( $sPath [, $sPrefix] )

###Parameters###
@@ParamTable@@
$sPath
	文件名的目录路径. 通常应用程序为当前目录指定一个句点(.).
$sPrefix
	[可选参数] 前缀字符串.函数最多使用前 3 个字符串作为文件名的前缀字符
		 file name.
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回临时文件的名称.
失败:	返回空字符串,设置 @error 标志为非 0 值.
@@End@@

###Remarks###
本函数创建的临时文件名称不会自动删除.
