###User Defined Function###
_GUICtrlListView_RedrawItems

###Description###
强制刷新控件范围内的项目

###Syntax###
#Include <GuiListView.au3>
_GUICtrlListView_RedrawItems($hWnd, $iFirst, $iLast)

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
$iFirst
	刷新范围的第一项目 0 基索引
$iLast
	刷新范围的最后项目 0 基索引
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 True
失败:	返回 False
@@End@@

###Remarks###
 指定的项目,在控件接收 $WM_PAINT 消息前,实际上并没有刷新.
