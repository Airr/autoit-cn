###User Defined Function###
_WinAPI_GetErrorMode

###Description###
获取当前进程的错误模式

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_GetErrorMode ( )

###Parameters###
@@ParamTable@@
None.
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回进程的错误模式($SEM_*).
失败:	返回 0,并设置@error标志为非 0 值.
@@End@@

###Remarks###
This function requires Windows Vista or later.
