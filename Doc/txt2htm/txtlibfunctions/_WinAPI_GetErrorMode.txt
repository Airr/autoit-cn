###User Defined Function###
_WinAPI_GetErrorMode

###Description###
Retrieves the error mode for the current process.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_GetErrorMode ( )

###Parameters###
@@ParamTable@@
None.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	The process error mode ($SEM_*).
Failure	0 and sets the @error flag to non-zero.
@@End@@
