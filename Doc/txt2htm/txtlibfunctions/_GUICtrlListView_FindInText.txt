###User Defined Function###
_GUICtrlListView_FindInText

###Description###
搜索包含指定文本的项目

###Syntax###
#Include <GuiListView.au3>
_GUICtrlListView_FindInText($hWnd, $sText[, $iStart = -1[, $fWrapOK = True[, $fReverse = False]]])

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
$sText
	匹配文本
$iStart
	[可选参数] 搜索开始的 0 基索引,或 -1, 从头开始.
		指定索引项自身不包括在搜索范围内.
$fWrapOK
	[可选参数] 如为 True, 则没有找到匹配项时,从第一项继续搜索
$fReverse
	[可选参数] 如为 True, 则搜索将从 $iStart - 1 开始到 0. 如果 $fWrapOK = True 搜索将继续
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回项目的 0 基索引
失败:	返回 -1
@@End@@

###Remarks###
 搜索不区分大小写. 
不像 _GUICtrlListView_FindText,这个函数将搜索所有的文字,包括子项.
