###Function###
_WinAPI_RectVisible

###Description###
Determines whether any part of the specified rectangle lies within the clipping region.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_RectVisible ( $hDC, $tRECT )

###Parameters###
@@ParamTable@@
$hDC
	Handle to the device context.
$tRECT
	$tagRECT structure that contains the logical coordinates of the specified rectangle.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	1, 2 - The rectangle lies within the clipping region.
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;0	Otherwise.
Failure	0 and sets the @error flag to non-zero.
@@End@@
