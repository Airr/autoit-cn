###User Defined Function###
_DebugReport

###Description###
写调试会话

###Syntax###
#include <Debug.au3>
_DebugReport ( $sData [,$bLastError = False [,$bExit = False]])


###Parameters###
@@ParamTable@@
$sData
	须予报告的数据
$bLastError
	[可选参数] 设置为 True, 表示如有 GetLastErrorMessage() 数据必须追加到报告中
$bExit
	[可选参数] 设置为 True, 表示脚本必须终止
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 1.
失败:	返回 0.
@@End@@

###Remarks###
 如果没有 _DebugSetup(), 已发布的函数总是返回.
 @error 为调用方保留.
 如果 $bLastError = True, @extended 可以设置为 Windows API 的 GetlastError(). 否则保留.
