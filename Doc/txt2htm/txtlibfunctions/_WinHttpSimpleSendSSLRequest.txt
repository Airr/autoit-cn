###User Defined Function###
_WinHttpSimpleSendSSLRequest

###Description###
使用SSL请求发送一个简单的表单,但不读取数据

###Syntax###
#Include <WinHttp.au3>
_WinHttpSimpleSendSSLRequest($hConnect [, $sType [, $sPath [, $sReferrer = Default [, $sData = Default [, $sHeader = Default ]]]]])

###Parameters###
@@ParamTable@@
$hConnect
	_WinHttpConnect 函数返回的句柄
$sType
	[可选参数] GET 或 POST (默认: GET)
$sPath
	[可选参数] 请求路径 (默认: "" - 空字符串;表示服务器上的默认页)
$sReferrer
	[可选参数] 引用页面 (默认: $WINHTTP_NO_REFERER)
$sData
	[可选参数] POST数据 (默认: $WINHTTP_NO_REQUEST_DATA)
$sHeader
	[可选参数] 附加头 (默认: $WINHTTP_NO_ADDITIONAL_HEADERS)
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 _WinHttpReceiveResponse 函数之后的请求句柄.
失败:	返回 0 并设置@error
	1 - 无法打开请求
	2 - 无法发送请求
	3 - 无法接收响应
@@End@@
