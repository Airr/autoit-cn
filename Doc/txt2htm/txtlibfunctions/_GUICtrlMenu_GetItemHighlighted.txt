###User Defined Function###
_GUICtrlMenu_GetItemHighlighted

###Description###
检索菜单项的高亮状态

###Syntax###
#Include <GuiMenu.au3>
_GUICtrlMenu_GetItemHighlighted($hMenu, $iItem[, $fByPos = True])

###Parameters###
@@ParamTable@@
$hMenu
	菜单句柄
$iItem
	菜单项的标识符或位置,由 $fByPos 参数决定.
$fByPos
	[可选参数] 菜单识别标志:
		True - $iItem 为项目的 0 基位置
		False - $iItem 为菜单项的标识符
@@End@@

###ReturnValue###
@@ReturnTable@@
True:	项目高亮
False:	项目非高亮
@@End@@

###Remarks###
None.
