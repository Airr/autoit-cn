###User Defined Function###
_WinHttpSetOption

###Description###
Sets an Internet option.

###Syntax###
#Include <WinHttp.au3>
_WinHttpSetOption($hInternet, $iOption, $vSetting [, $iSize = Default ])

###Parameters###
@@ParamTable@@
$hInternet
	Handle on which to set data.
$iOption
	Integer value that contains the Internet option to set.
$vSetting
	Value of setting
$iSize
	[可选参数] Size of $vSetting, required if $vSetting is pointer to memory block
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	Returns 1.
Failure	Returns 0 and sets @error:
	1 - Invalid Internet option
	2 - Size required
	3 - Datatype of value does not fit to option
	4 - DllCall failed
@@End@@
