###Function###
_WinAPI_IsThemePartDefined

###Description###
Retrieves whether a visual style has defined parameters for the specified part.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_IsThemePartDefined ( $hTheme, $iPartId )

###Parameters###
@@ParamTable@@
$hTheme
	Handle to a window's specified theme data.
$iPartId
	The part.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	1 - The theme has defined parameters for the specified $iPartId.
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;0	Otherwise.
Failure	0 and sets the @error flag to non-zero.
@@End@@
