###User Defined Function###
_GUICtrlListView_AddSubItem

###Description###
向控件添加子项目

###Syntax###
#include <GuiListView.au3>
_GUICtrlListView_AddSubItem($hWnd, $iIndex, $sText, $iSubItem [, $iImage = -1])

###Parameters###
@@ParamTable@@
$hWnd
	控件的控件ID/句柄
$iIndex
	主项的 0 基索引
$sText
	子项目文本
$iSubItem
	子项的索引
$iImage
	[可选参数] 控件图标在图像列表中的 0 基索引
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 True
失败:	返回 False
@@End@@

None.
