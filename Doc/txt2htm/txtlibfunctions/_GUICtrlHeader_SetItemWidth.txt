###User Defined Function###
_GUICtrlHeader_SetItemWidth

###Description###

###Syntax###
#Include <GuiHeader.au3>
_GUICtrlHeader_SetItemWidth($hWnd, $iIndex, $iWidth)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to control
$iIndex
	Zero based item index
$iWidth
	New width for item
@@End@@

###ReturnValue###
成功: True
失败: False
