###Structure Name###
$tagNMHDR

###Description###
包含一个关于通知消息的信息

###Syntax###
Global Const $tagNMHDR = "struct; hwnd hWndFrom;uint_ptr IDFrom;INT Code; endstruct"

###Fields###
@@ParamTable@@
hWndFrom
	进行发送消息的目标控件所在的窗口的窗口句柄.
IDFrom
	进行发送消息的目标控件的控件标识.
Code
	通知代码,这个成员可以是一个公共通知代码(<a href="http://msdn.microsoft.com/en-us/library/bb775497(v=VS.85).aspx#notifications" target="_blank">常规控件通知参考</a>),或者控件本身的特殊通知代码. 
@@End@@

###Remarks###
None.
