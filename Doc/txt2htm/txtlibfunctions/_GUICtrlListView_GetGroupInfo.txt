###User Defined Function###
_GUICtrlListView_GetGroupInfo

###Description###
检索组信息,使用组 ID.

###Syntax###
#Include <GuiListView.au3>
_GUICtrlListView_GetGroupInfo($hWnd, $iGroupID)

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
$iGroupID
	指定检索信息的组 ID
@@End@@

###ReturnValue###
@@ReturnTable@@
	返回如下格式数组:
	[0] - 组标题文字
	[1] - 标题对齐方式:
	0 - 左对齐
	1 - 居中对齐
	2 - 右对齐
@@End@@

###Remarks###
 最低操作系统 - Windows XP.
