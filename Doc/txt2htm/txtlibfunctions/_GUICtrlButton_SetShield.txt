###User Defined Function###
_GUICtrlButton_SetShield

###Description###
在指定按钮或链接上显示盾牌图标

###Syntax###
#include <GuiButton.au3>
_GUICtrlButton_SetShield($hWnd [, $fRequired = True])

###Parameters###
@@ParamTable@@
$hWnd
	控件的控件ID/句柄
$fRequired
	[可选参数] 使用 True 绘制一个盾牌图标,使用 False 不绘制.
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 True
失败:	返回 False
@@End@@

###Remarks###
最低操作系统: Windows Vista.
