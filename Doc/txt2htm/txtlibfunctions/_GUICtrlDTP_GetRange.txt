###User Defined Function###
_GUICtrlDTP_GetRange

###Description###
Retrieves the current minimum and maximum allowable system times

###Syntax###
#Include <GuiDateTimePicker.au3>
_GUICtrlDTP_GetRange($hWnd)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
@@End@@

###ReturnValue###
Success: Returns an array with the following format:

	[ 0] - True if Min data is valid, otherwise False
	[ 1] - Min Year
	[ 2] - Min Month
	[ 3] - Min Day
	[ 4] - Min Hour
	[ 5] - Min Minute
	[ 6] - Min Second
	[ 7] - True if Max data is valid, otherwise False
	[ 8] - Max Year
	[ 9] - Max Month
	[10] - Max Day
	[11] - Max Hour
	[12] - Max Minute
	[13] - Max Second

###Remarks###
None.
