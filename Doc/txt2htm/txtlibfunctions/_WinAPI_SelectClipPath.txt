###Function###
_WinAPI_SelectClipPath

###Description###
Selects the current path as a clipping region, combining the new region with any existing clipping region.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_SelectClipPath ( $hDC [, $iMode] )

###Parameters###
@@ParamTable@@
$hDC
	Handle to the device context of the path.
$iMode
	The way to use the path. This parameter can be one of the following values.
	
	$RGN_AND
	$RGN_COPY
	$RGN_DIFF
	$RGN_OR
	$RGN_XOR
	
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	1.
Failure	0 and sets the @error flag to non-zero.
@@End@@

None
