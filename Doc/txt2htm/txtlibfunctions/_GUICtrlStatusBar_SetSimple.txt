###User Defined Function###
_GUICtrlStatusBar_SetSimple

###Description###
指定是否显示简单文本状态窗口, 或显示所有窗口部件

###Syntax###
#Include <GuiStatusBar.au3>
_GUICtrlStatusBar_SetSimple($hWnd, $fSimple = True)

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
$fSimple
	设置 Windows 显示
		 True - 窗口显示简单文本
		False - 窗口显示多个部件
@@End@@

###ReturnValue###
None.

###Remarks###
None.
