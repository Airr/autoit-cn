###User Defined Function###
_WinHttpSetCredentials

###Description###
Passes the required authorization credentials to the server.

###Syntax###
#Include <WinHttp.au3>
_WinHttpSetCredentials($hRequest, $iAuthTargets, $iAuthScheme, $sUserName, $sPassword)

###Parameters###
@@ParamTable@@
$hRequest
	Valid handle returned by _WinHttpOpenRequest().
$iAuthTargets
	Authentication target.
$iAuthScheme
	Authentication scheme.
$sUserName
	Valid user name.
$sPassword
	Valid password.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	Returns 1.
Failure	Returns 0 and sets @error:
	1 - DllCall failed
@@End@@

None.
