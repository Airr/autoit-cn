###User Defined Function###
_GUICtrlHeader_SetItemText

###Description###
设置标题项目文本

###Syntax###
#Include <GuiHeader.au3>
_GUICtrlHeader_SetItemText($hWnd, $iIndex, $sText)

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
$iIndex
	基于0开始的项目索引.
$sText
	新项目文本.参考备注.
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 True
失败:	返回 False
@@End@@

###Remarks###
If a notification callback is needed, you have to specify $sText = -1 (LPSTR_TEXTCALLBACK).
