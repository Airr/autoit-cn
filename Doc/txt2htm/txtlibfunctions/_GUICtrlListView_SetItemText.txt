###User Defined Function###
_GUICtrlListView_SetItemText

###Description###
更改项目或子项文本

###Syntax###
#Include <GuiListView.au3>
_GUICtrlListView_SetItemText($hWnd, $iIndex, $sText[, $iSubItem = 0])

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
$iIndex
	项目的 0 基索引
$sText
	子项目文本
$iSubItem
	[可选参数] 子项的索引, 或 0, 仅设置项目.
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 True
失败:	返回 False
@@End@@

###Remarks###
 如果 $iSubItem = -1,则设置行
