###User Defined Function###
_GUICtrlTreeView_SetFocused

###Description###
设置项目的焦点

###Syntax###
#Include <GuiTreeView.au3>
_GUICtrlTreeView_SetFocused($hWnd, $hItem[, $fFlag = True])

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
$hItem
	项目句柄
$fFlag
	[可选参数] 设置标志:
		True - 设置为焦点
		False - 没有焦点
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 True
失败:	返回 False
@@End@@

###Remarks###
None.
