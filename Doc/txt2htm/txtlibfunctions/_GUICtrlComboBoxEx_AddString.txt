###User Defined Function###
_GUICtrlComboBoxEx_AddString

###Description###
添加字符串

###Syntax###
#Include <GuiComboBoxEx.au3>
_GUICtrlComboBoxEx_AddString($hWnd, $sText[, $iImage = -1[, $iSelecteImage = -1[, $iOverlayImage = -1[, $iIndent = -1[, $iParam = -1]]]]])

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
		要插入为列表框最后一个项目,则设置 $iIndex 参数为 -1.
$sText
	项目文本. 如设置为 -1, 则该值由 $CBEN_GetDISPINFO 通知消息设置.
$iImage
	[可选参数] 控件项目图标在图像列表中的 0 基索引. 
$iSelecteImage
	[可选参数] 控件项目选中状态时, 图标在图像列表中的 0 基索引. 
$iOverlayImage
	[可选参数] 控件项目覆盖状态时, 图标在图像列表中的 0 基索引. 
$iIndent
	[可选参数] 项目缩进的图像宽度数. 一个缩进量等于一个图像宽度..
$iParam
	[可选参数] 项目的具体值
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回新项目索引
失败:	返回 -1
@@End@@

###Remarks###
None.
