###Structure Name###
$tagMENUITEMINFO

###Description###
包含有关菜单项信息

###Syntax###
Global Const $tagMENUITEMINFO = "int Size;int Mask;int Type;int State;int ID;int SubMenu;int BmpChecked;int BmpUnchecked;" & _
		"int ItemData;ptr TypeData;int CCH;int BmpItem"

###Fields###
@@ParamTable@@
Size
	结构大小，以字节为单位
Mask
	获取或设置的成员. 可为一或多个下列值:
		$MIIM_BITMAP - 检索或设置 BmpItem 成员
		$MIIM_CHECKMARKS - 检索或设置 BmpChecked 与 BmpUnchecked 成员
		$MIIM_DATA - 检索或设置 ItemData 成员
		$MIIM_FTYPE - 检索或设置 Type 成员
		$MIIM_ID - 检索或设置 ID 成员
		$MIIM_STATE - 检索或设置  State 成员
		$MIIM_STRING - 检索或设置  TypeData 成员
		$MIIM_SUBMENU - 检索或设置  SubMenu 成员
		$MIIM_TYPE - 检索或设置  Type 与 TypeData 成员
Type
	菜单项类型. 可为一或多个下列值:
		$MFT_BITMAP - 显示使用位图的菜单项
		$MFT_MENUBARBREAK - 将菜单项置于新行或新列上
		$MFT_MENUBREAK - 将菜单项置于新行或新列上
		$MFT_OWNERDRAW - 指定由菜单宿主绘制菜单项
		$MFT_RADIOCHECK - 显示使用单选按钮标记的选定菜单项
		$MFT_RIGHTJUSTIFY - 右对齐菜单项及子项
		$MFT_RIGHTORDER - 指定级联菜单从右到左
		$MFT_SEPARATOR - 指定菜单项为分隔符
State
	菜单项状态. 该成员可为一或多个下列值:
		$MFS_CHECKED - 选中
		$MFS_DEFAULT - 默认
		$MFS_DISABLED - 禁用
		$MFS_ENABLED - 启用
		$MFS_GRAYED - 禁用
		$MFS_HILITE - 高亮显示
ID
	应用程序定义的标识菜单项的16位值
SubMenu
	与菜单项相关的下拉菜单或子菜单的句柄
BmpChecked
	项目选定时显示的位图句柄
BmpUnchecked
	项目未被选定时显示的位图句柄
ItemData
	应用程序定义的与菜单项相关的值
TypeData
	菜单项的内容
CCH
	菜单项目文本的长度
BmpItem
	显示的位图句柄
@@End@@

###Remarks###
None.
