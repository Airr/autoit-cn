###结构常量###
$tagMENUITEMINFO

###Description###
包含菜单项信息的结构

###Syntax###
Global Const $tagMENUITEMINFO =  "int Size;int Mask;int Type;int State;int ID;int SubMenu;int BmpChecked;int BmpUnchecked;int ItemData;ptr TypeData"
  "int CCH;int BmpItem"

###成员###
@@ParamTable@@
Size
   结构的字节大小
Mask
   获取或设置的成员. 该成员可为一个或多个下列值:
    $MIIM_BITMAP - 获取或设置BmpItem成员
    $MIIM_CHECKMARKS - 获取或设置BmpChecked及BmpUnchecked成员
    $MIIM_DATA - 获取或设置ItemData成员
    $MIIM_FTYPE - 获取或设置Type成员
    $MIIM_ID - 获取或设置ID成员
    $MIIM_STATE - 获取或设置State成员
    $MIIM_STRING - 获取或设置TypeData成员
    $MIIM_SUBMENU - 获取或设置SubMenu成员
    $MIIM_TYPE - 获取或设置Type及TypeData成员
Type
   菜单项类型. 该成员可为一个或多个下列值:
    $MFT_BITMAP - 显示使用位图的菜单项
    $MFT_MENUBARBREAK - 将菜单项置于新行或新列上
    $MFT_MENUBREAK - 将菜单项置于新行或新列上
    $MFT_OWNERDRAW - 指定由菜单宿主绘制菜单项
    $MFT_RADIOCHECK - 显示使用单选按钮标记的选定菜单项
    $MFT_RIGHTJUSTIFY - 菜单项及子项右对齐
    $MFT_RIGHTORDER - 指定菜单从右至左层叠
    $MFT_SEPARATOR - 指定菜单项为分割符
State
   菜单项状态. 该成员可为一个或多个下列值:
    $MFS_CHECKED - 复选菜单项
    $MFS_DEFAULT - 指定默认菜单项
    $MFS_DISABLED - 禁用菜单项并灰色显示以便无法选定
    $MFS_ENABLED - 启用菜单项以便选取
    $MFS_GRAYED - 禁用菜单项并灰色显示以便无法选定
    $MFS_HILITE - 高亮菜单项
ID
   应用程序定义的标识菜单项的16位值
SubMenu
   与菜单项相关的下拉菜单或子菜单的句柄
BmpChecked
   项目选定时显示的位图的句柄
BmpUnchecked
   项目未被选定时显示的位图的句柄
ItemData
   应用程序定义的与菜单项相关的值
TypeData
   菜单项的内容
CCH
   菜单项目文本的长度
BmpItem
   被显示的位图的句柄
@@end@@
