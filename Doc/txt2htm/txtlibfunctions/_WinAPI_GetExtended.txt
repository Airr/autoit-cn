###User Defined Function###
_WinAPI_GetExtended

###Description###
Retrieves the last extended function return value.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_GetExtended ( )

###Parameters###
@@ParamTable@@
None.
@@End@@

###ReturnValue###

###Remarks###
不是所有函数都有扩展返回值. 比如, _WinAPI_CeateDIB()将返回DIB位值的指针.
_WinAPI_GetExtended()必须在调用某函数后立即返回.
