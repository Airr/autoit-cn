###User Defined Function###
_GUICtrlToolbar_SetDisabledImageList

###Description###
Sets the disabled image list

###Syntax###
#Include <GuiToolBar.au3>
_GUICtrlToolbar_SetDisabledImageList($hWnd, $hImageList)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$hImageList
	Handle to the image list that will be set
@@End@@

###ReturnValue###
Success: The handle to the image list previously used to display disabled buttons
Failure: 0

###Remarks###
None.
