###User Defined Function###
_WinAPI_GetFileVersionInfo

###Description###
Retrieves version information for the specified file.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_GetFileVersionInfo ( $sFile )

###Parameters###
@@ParamTable@@
$sFile
	The name of the file.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	The "byte[n]" structure (buffer) that contains the file-version information.
Failure	0 and sets the @error flag to non-zero.
@@End@@

###Remarks###
None
