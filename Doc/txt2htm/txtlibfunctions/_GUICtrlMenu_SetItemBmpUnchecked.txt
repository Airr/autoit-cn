###User Defined Function###
_GUICtrlMenu_SetItemBmpUnchecked

###Description###
设置显示项目未选中状态的位图

###Syntax###
#Include <GuiMenu.au3>
_GUICtrlMenu_SetItemBmpUnchecked($hMenu, $iItem, $hBmp[, $fByPos = True])

###Parameters###
@@ParamTable@@
$hMenu
	菜单句柄
$iItem
	菜单项的标识符或位置,由 $fByPos 参数决定.
$hBmp
	位图句柄
$fByPos
	[可选参数] 菜单识别标志:
		True - $iItem 为项目的 0 基位置
		False - $iItem 为菜单项的标识符
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 True
失败:	返回 False
@@End@@

###Remarks###
None.
