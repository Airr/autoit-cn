###User Defined Function###
_WinAPI_GetShellWindow

###Description###
Retrieves a handle to the Shell's desktop window.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_GetShellWindow ( )

###Parameters###
@@ParamTable@@
None.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	The handle of the Shell's desktop window.
失败:	返回 0 并设置 @error 标志为非 0 值.
@@End@@
