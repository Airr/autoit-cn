###User Defined Function###
_GUICtrlTreeView_SetUnicodeFormat

###Description###
设置 Unicode 字符格式标志

###Syntax###
#include <GuiTreeView.au3>
_GUICtrlTreeView_SetUnicodeFormat($hWnd [, $iFormat = True])

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
$iFormat
	[可选参数] 指定字符集标志.
		True - 控件使用 Unicode 字符
		False - 控件使用 ANSI 字符.
@@End@@

###ReturnValue###
返回以前的字符格式设置标志

###Remarks###
None.
