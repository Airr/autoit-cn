###User Defined Function###
_GUICtrlRichEdit_GetCharBkColor

###Description###
检索所选文本的背景颜色,如果没有选定文本,则检索插入点右边的字符

###Syntax###
#Include <GuiRichEdit.au3>
_GUICtrlRichEdit_GetCharBkColor($hWnd)

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	如果选定文本的颜色是一致的,则返回 COLORREF 值的 @extended 设置为 1
失败:	设置 @error
@error:	101 - $hWnd 参数值不是句柄
@@End@@

###Remarks###
If you need to create an color array use _ColorGetCOLORREF() not _ColorGetRGB().
