###User Defined Function###
_GDIPlus_MatrixRotate

###Description###
旋转矩阵

###Syntax###
#Include <GDIPlus.au3>
_GDIPlus_MatrixRotate($hMatrix, $fAngle[, $bAppend = False])

###Parameters###
@@ParamTable@@
$hMatrix
	返回矩阵对象句柄
$fAngle
	旋转角度. 正值指定顺时针旋转.
$bAppend
	[可选参数] 指定旋转方向:
		True - 指定向左旋转矩形
		False - 指定向右旋转矩形
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 True
失败:	返回 False
@@End@@
