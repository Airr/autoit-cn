###User Defined Function###
_GUICtrlRichEdit_ReplaceText

###Description###
替换所选的文本

###Syntax###
#Include <GuiRichEdit.au3>
_GUICtrlRichEdit_ReplaceText($hWnd, $sText[, $fCanUndo = True])

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
$sText
	替换的文本
$fCanUndo
	[可选参数] 确定操作是否可以撤消. True (默认) 或 False
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 True
失败:	返回 False设置@error
@error:	101 - $hWnd 参数值不是句柄
	103 - $fCanUndo 必须是 True 或 False
	-1 - 没有选中的文本
@@End@@

None.
