###Function###
_WinAPI_EnumResourceNames

###Description###
Enumerates resources of a specified type within a binary module.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_EnumResourceNames ( $sModule, $sType )

###Parameters###
@@ParamTable@@
$sModule
	Handle to a module to search.
$sType
	The type of the resource. This parameter can be string or integer type.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	The array of the names for the specified resource type.
Failure	0 and sets the @error flag to non-zero.
@@End@@

None
