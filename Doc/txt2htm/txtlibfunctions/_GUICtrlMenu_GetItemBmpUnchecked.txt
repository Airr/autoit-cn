###User Defined Function###
_GUICtrlMenu_GetItemBmpUnchecked

###Description###
检索显示项目未选中状态的位图

###Syntax###
#Include <GuiMenu.au3>
_GUICtrlMenu_GetItemBmpUnchecked($hMenu, $iItem[, $fByPos = True])

###Parameters###
@@ParamTable@@
$hMenu
	菜单句柄
$iItem
	菜单项的标识符或位置,由 $fByPos 参数决定.
$fByPos
	[可选参数] 菜单识别标志:
		True - $iItem 为项目的 0 基位置
		False - $iItem 为菜单项的标识符
@@End@@

###ReturnValue###
@@ReturnTable@@
	返回显示项目未选中状态的位图句柄
@@End@@

###Remarks###
None.
