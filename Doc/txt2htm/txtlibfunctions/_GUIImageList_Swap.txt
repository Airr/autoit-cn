###User Defined Function###
_GUIImageList_Swap

###Description###

(译注:原解释语句与 _GUIImageList_Copy 函数完全一样,此处臆测翻译)

###Syntax###
#Include <GuiImageList.au3>
_GUIImageList_Swap($hWnd, $iSource, $iDestination)

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
$iSource
	交换操作的源图像 0 基索引
$iDestination
	交换操作的目标图像 0 基索引
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 True
失败:	返回 False
@@End@@
