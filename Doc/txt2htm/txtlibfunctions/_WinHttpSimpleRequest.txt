###用户自定义函数###
_WinHttpSimpleRequest

###描述###
发送请求到简单表单中

###语法###
#Include <WinHTTP.au3>
_WinHttpSimpleRequest( $hConnect, $sType, $sPath [, $sReferrer = Default [, $sData = Default [, $sHeader = Default [, $fGetHeaders = Default [, $iMode = Default ]]]]] )

###参数###
@@ParamTable@@
$hConnect 
   _WinHttpConnect()返回的句柄.
$sType 
   [可选] GET(默认)或POST.
$sPath 
   [可选] 请求路径(默认为空字符串表示服务器上的默认页).
$sReferrer 
   [可选] 引用(默认为$WINHTTP_NO_REFERER)
$sData 
   [可选] POST数据(默认为$WINHTTP_NO_REQUEST_DATA)
$sHeader 
   [可选] 额外头部(默认为$WINHTTP_NO_ADDITIONAL_HEADERS).
$fGetHeaders 
   [可选] 返回应答的头部(默认为False).
$iMode 
   [可选] 结果的读取模式(默认为0).
@@End@@

###返回值###
@@ReturnTable@@
成功: $fGetHeaders为False是返回应答数据(默认)
    $fGetHeaders为True是返回数组
    [0] - 应答头部
    [1] - 应答数据
失败: 返回0并设置@error为:
    1 - 无法打开请求
    2 - 无法发送请求
    3 - 无法接收应答
    4 - 无效$iMode
@@End@@
