###User Defined Function###
_Security__SetPrivilege

###Description###
启用或禁用本地令牌特权

###Syntax###
#include <Security.au3>
_Security__SetPrivilege($hToken, $sPrivilege, $fEnable)

###Parameters###
@@ParamTable@@
$hToken
	令牌句柄
$sPrivilege
	特权名称
$fEnable
	特权设置:
		True - 启用特权
		False - 禁用特权
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 True
失败:	返回 False
@@End@@

None.
