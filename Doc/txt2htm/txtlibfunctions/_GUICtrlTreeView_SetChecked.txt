###User Defined Function###
_GUICtrlTreeView_SetChecked

###Description###
设置项目复选框选中

###Syntax###
#include <GuiTreeView.au3>
_GUICtrlTreeView_SetChecked($hWnd, $hItem [, $fCheck = True])

###Parameters###
@@ParamTable@@
$hWnd
	控件ID/句柄
$hItem
	项目句柄
$fCheck
	[可选参数] 选中状态值:
		True - 选中
		False - 不选中
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	True
失败:	False
@@End@@

###Remarks###
None.
