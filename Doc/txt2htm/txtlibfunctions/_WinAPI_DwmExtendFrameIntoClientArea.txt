###User Defined Function###
_WinAPI_DwmExtendFrameIntoClientArea

###Description###
扩展客户区背后的窗口框架.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_DwmExtendFrameIntoClientArea ( $hWnd [, $tMARGINS] )

###Parameters###
@@ParamTable@@
$hWnd
	框架范围扩展到客户端的窗口句柄.
$tMARGINS
	[可选参数] $tagMARGINS 结构,描述延伸到客户端区域的框架边界.
		 压缩边界用于创建玻璃效果,其中客户区域为一个没有窗口边框的固体表面.
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 1.
失败:	返回 0,设置@error:标志为非 0 值, @extended 标志可能包含一个系统错误代码.
@@End@@

###Remarks###
每当桌面窗口管理器（DWM）切换时,必须调用这个函数.
 WM_DWMCOMPOSITIONCHANGED 消息通知切换的变化信息.
 本函数需要 Windows Vista 或更高版本系统.
This function requires Windows Vista or later.
