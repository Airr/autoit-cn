###User Defined Function###
_WinAPI_ShellGetPathFromIDList

###Description###
转换项目标识符列表到文件系统路径.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_ShellGetPathFromIDList ( $PIDL )

###Parameters###
@@ParamTable@@
$PIDL
	项目标识符列表的地址,指定文件或目录位置相对于命名空间（桌面）的根地址.	
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回文件系统路径
失败:	返回空串,设置@error:标志为非 0 值.
@@End@@

None
