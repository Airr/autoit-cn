###User Defined Function###
_WinAPI_PathIsLFNFileSpec

###Description###
Determines whether a file name is in long format.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_PathIsLFNFileSpec ( $sPath )

###Parameters###
@@ParamTable@@
$sPath
	The file name to be tested.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	1 - The file exceeds the number of characters allowed by the 8.3 format.
0	Otherwise.
Failure	0 and sets the @error flag to non-zero.
@@End@@

None
