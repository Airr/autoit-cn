###User Defined Function###
_SQLite_TotalChanges

###Description###
返回所有从连接开始到现在发生改变的数量(包括通过触发器和外部键)

###Syntax###
#include <SQLite.au3>
_SQLite_TotalChanges ( [ $hDB ] )


###Parameters###
@@ParamTable@@
$hDB
	[可选参数] 打开的数据库,默认为最后打开的数据库
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回更改数量
失败:	返回 0.
@error:	1 - 错误调用 SQLite API 'sqlite3_total_changes'
	2 - 调用被安全模式阻止
@@End@@
