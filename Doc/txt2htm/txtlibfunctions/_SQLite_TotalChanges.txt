###User Defined Function###
_SQLite_TotalChanges

###Description###
返回所有发生改变的数字(包含触发器)

###Syntax###
#include <SQLite.au3>
_SQLite_TotalChanges ( [ $hDB ] )


###Parameters###
@@ParamTable@@
$hDB
	[可选参数] 打开的数据库,默认为最后打开的数据库
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回更改的数字
失败:	返回 0.@错误:
	1 - 错误调用 SQLite API 'sqlite3_changes'
	2 - 调用被安全模式阻止
@@End@@
