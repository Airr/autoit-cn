###User Defined Function###
_MemGlobalUnlock

###Description###
Decrements the lock count associated with a memory object that was allocated with GMEM_MOVEABLE

###Syntax###
#Include <Memory.au3>
_MemGlobalUnlock($hMem)

###Parameters###
@@ParamTable@@
$hMem
	Handle to the global memory object
@@End@@

###ReturnValue###
None.

###Remarks###
None.
