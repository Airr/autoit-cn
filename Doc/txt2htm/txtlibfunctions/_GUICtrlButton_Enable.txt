###User Defined Function###
_GUICtrlButton_Enable

###Description###
应用或者禁用鼠标和键盘对指定按钮的输入

###Syntax###
#Include <GuiButton.au3>
_GUICtrlButton_Enable( $hWnd[, $fEnable = True] )

###Parameters###
@@ParamTable@@
$hWnd 
   控件句柄
$fEnable 
   指定按钮是否启用或禁用:
    True – 启用
    False – 禁用
@@End@@

###ReturnValue###
@@ReturnTable@@
成功: True
失败: False
@@End@@
