###User Defined Function###
_WinHttpSetTimeouts

###Description###
设置处理HTTP的超时

###Syntax###
#Include <WinHttp.au3>
_WinHttpSetTimeouts($hInternet [, $iResolveTimeout = Default [, $iConnectTimeout = Default [, $iSendTimeout = Default [, $iReceiveTimeout = Default ]]]])

###Parameters###
@@ParamTable@@
$hInternet
	_WinHttpOpen()或_WinHttpOpenRequest()返回的句柄
$iResolveTimeout
	[可选参数] 指定用于名称解析的整数毫秒超时值
$iConnectTimeout
	[可选参数] 指定用于服务器连接请求的整数毫秒超时值
$iSendTimeout
	[可选参数] 指定用于发送请求的整数毫秒超时值
$iReceiveTimeout
	[可选参数] 指定用于接收对请求的应答的整数毫秒超时值
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回1设置@error 0
失败:	返回0设置@error 且当其为1时表示DllCall失败
@@End@@

###Remarks###
初始值:
$iResolveTimeout = 0
$iConnectTimeout = 60000
$iSendTimeout = 30000
$iReceiveTimeout = 30000
