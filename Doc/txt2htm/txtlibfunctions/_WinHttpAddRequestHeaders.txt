###User Defined Function###
_WinHttpAddRequestHeaders

###Description###
添加一个或者多个 HTTP 请求头到 HTTP 请求句柄.

###Syntax###
#Include <WinHttp.au3>
_WinHttpAddRequestHeaders ($hRequest, $sHeaders [, $iModifiers = Default ])

###Parameters###
@@ParamTable@@
$hRequest
	由 _WinHttpOpenRequest 函数返回的句柄.
$sHeader
	[可选参数] 要附加到请求的报头.
$iModifiers
	[可选参数] 包含用于更改该函数过程的标志. 默认为 $WINHTTP_ADDREQ_FLAG_ADD_IF_NEW.
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 1
失败:	返回 0 并设置 @error:
	1 - DllCall 失败
@@End@@

###Remarks###
一次添加多个请求时, 必须使用将每个请求以@CRLF分开每个$hRequest, 响应头 $sHeaders 和 $iModifiers.
