###User Defined Function###
_GUICtrlMenu_GetItemDisabled

###Description###
检索菜单项的禁用状态

###Syntax###
#Include <GuiMenu.au3>
_GUICtrlMenu_GetItemDisabled($hMenu, $iItem[, $fByPos = True])

###Parameters###
@@ParamTable@@
$hMenu
	菜单句柄
$iItem
	菜单项的标识符或位置,由 $fByPos 参数决定.
$fByPos
	[可选参数] 菜单识别标志:
		True - $iItem 为项目的 0 基位置
		False - $iItem 为菜单项的标识符
@@End@@

###ReturnValue###
@@ReturnTable@@
True:	项目被禁用
False:	项目未禁用
@@End@@

###Remarks###
None.
