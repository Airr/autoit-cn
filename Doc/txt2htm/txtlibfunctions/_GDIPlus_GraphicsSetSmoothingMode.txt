###User Defined Function###
_GDIPlus_GraphicsSetSmoothingMode

###Description###
设置图形对象绘制质量

###Syntax###
#Include <GDIPlus.au3>
_GDIPlus_GraphicsSetSmoothingMode($hGraphics, $iSmooth)

###Parameters###
@@ParamTable@@
$hGraphics
	图形对象的句柄
$iSmooth
	平滑模式:
		0 - 不使用平滑
		1 - 使用 8 X 4 矩形过滤器
		2 - 使用 8 X 8 矩形过滤器
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 True
失败:	返回 False
@@End@@

###Remarks###
None.
