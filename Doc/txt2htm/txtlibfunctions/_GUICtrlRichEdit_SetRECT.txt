###User Defined Function###
_GUICtrlRichEdit_SetRECT

###Description###
设置控件的格式化矩形 
译注:原描述文字与 _GUICtrlRichEdit_SetModified 函数完全一样,
参照 _GUICtrlRichEdit_GetRECT 函数汉化此函数的描述文字.

###Syntax###
#Include <GuiRichEdit.au3>
_GUICtrlRichEdit_SetRECT($hWnd [,$iLeft = Default [, $iTop = Default [, $iRight = Default [, $iBottom = Default [, $bRedraw = True]]]]]])

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
$iLeft
	[可选参数] 对话框左侧位置
$iTop
	[可选参数] 对话框顶部位置
$iRight
	[可选参数] 话框右侧位置
$iBottom
	[可选参数] 对话框底部位置
$bRedraw
	[可选参数] True, 则强制刷新
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 True
失败:	返回 False设置@error
@error:	101 - $hWnd 参数值不是句柄
	1021 - $iLeft 参数值不是正数
	1022 - $iTop 参数值不是正数
	1023 - $iRight 参数值不是正数
	1024 - $iBottom 参数值不是正数
	1025 - $iLeft >= $iRight
	1026 - $iTop >= $iBottom
@@End@@

###Remarks###
 格式化矩形是指绘制文本的区域,其中一部分可能不可见.
 参数的默认值 = 不改变先前值.
 如果只定义 $hWnd, 则被重置为创建时的状态.
