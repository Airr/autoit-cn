###User Defined Function###
_WinAPI_EndUpdateResource

###Description###
更新或删除资源模块内的修改.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_EndUpdateResource ( $hUpdate [, $fDiscard] )

###Parameters###
@@ParamTable@@
$hUpdate
	_WinAPI_BeginUpdateResource() 与 _WinAPI_UpdateResource() 函数返回的模块句柄.
$fDiscard
	[可选参数] 指定资源更新是否写入文件,有效值:
		TRUE     - 删除更新.
		FALSE    - 写入更新,资源更新将生效. (默认)
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 1.
失败:	返回 0,设置 the @error 为非 0 值.
@@End@@
