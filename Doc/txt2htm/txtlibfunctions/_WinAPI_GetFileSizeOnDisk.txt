###User Defined Function###
_WinAPI_GetFileSizeOnDisk

###Description###
Retrieves the file allocation size on disk.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_GetFileSizeOnDisk ( $sFile )

###Parameters###
@@ParamTable@@
$sFile
	The name of the file to retrieve allocation size.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	The allocation size, in bytes. This value is a multiple of the sector or cluster size of the nderlying
Failure	0 and sets the @error flag to non-zero.
@@End@@
