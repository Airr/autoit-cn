###User Defined Function###
_WinNet_RestoreConnection

###Description###
还原网络资源的连接

###Syntax###
#Include <WinNet.au3>
_WinNet_RestoreConnection([$sDevice = "" [, $hWnd = 0 [, $fUseUI = True]]])

###Parameters###
@@ParamTable@@
$sDevice
	[可选参数] 连接到本地的驱动器名称，例如 "Z:". 如果留空，
		函数重新连接所有存储在当前用户注册表的持久驱动器
$hWnd
	[可选参数] 函数用来显示用户界面的父窗口句柄，网络连接时，提示用户输入名称和密码．
		如果为　0，则没有所有者窗口．
$fUseUI
	[可选参数] 如果为 True，提示调用程序显示用户名/密码
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	True
失败:	False
@@End@@
