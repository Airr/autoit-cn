###Function###
_WinAPI_GetParentProcess

###Description###
Retrieves the PID of the parent process for the specified process.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_GetParentProcess ( [$PID] )

###Parameters###
@@ParamTable@@
$PID
	The PID of the process. Default (0) is the current process.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	The PID of the parent process.
Failure	0 and sets the @error flag to non-zero.
@@End@@
