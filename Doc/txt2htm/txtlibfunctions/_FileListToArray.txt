###User Defined Function###
_FileListToArray

###Description###
列出指定路径下的文件和/或文件夹 (类似使用 Dir /B命令)

###Syntax###
#Include <File.au3>
_FileListToArray($sPath [, $sFilter = "*" [, $iFlag = 0]])

###Parameters###
@@ParamTable@@
$sPath
	要生成文件列表的路径.
$sFilter
	[可选参数] the filter to use, default is *. Search the Autoit3 helpfile for the word "WildCards" For details.
$iFlag
	[可选参数] specifies whether to return files folders or both
		$iFlag=0(Default) Return both files and folders
		$iFlag=1 Return files only
		$iFlag=2 Return Folders only
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回一个数组, 参考备注
失败:	0
@Error:	1 = 路径未找到或者无效
	2 = 无效 $sFilter
	3 = 无效 $iFlag
	4 = 未找到任何文件
@@End@@

###Remarks###
The array returned is one-dimensional and is made up as follows:
$array[0] = Number of Files\Folders returned
$array[1] = 1st File\Folder
$array[2] = 2nd File\Folder
$array[3] = 3rd File\Folder
$array[n] = nth File\Folder
