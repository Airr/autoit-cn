###User Defined Function###
_WinNet_GetProviderName

###Description###
Obtains the provider name for a specific type of network

###Syntax###
#Include <WinNet.au3>
_WinNet_GetProviderName($iType)

###Parameters###
@@ParamTable@@
$iType
	Network type that is unique to the network
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	Network provider name
Failure:	Set @error
@@End@@
