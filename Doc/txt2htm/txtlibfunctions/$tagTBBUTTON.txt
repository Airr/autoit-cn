###结构常量###
$tagTBBUTTON

###描述###
包含工具栏上的按钮信息的结构

###语法###
Global Const $tagTBBUTTON =  "int Bitmap;int Command;byte State;byte Style;short Reserved;int Param;int String"

###成员###
@@ParamTable@@
Bitmap
   0基索引的按钮图像. 设置该成员为$I_IMAGECALLBACK, 且工具栏将发送$TBN_GETDISPINFO通知以在需要时获取图像索引. 设置该成员为$I_IMAGENONE表示按钮无图像. 按钮表面将只有文本且不包括任何留给图像的空间. 如果按钮是一个分隔符, 位图确定分隔符的像素宽度.
Command
   与按钮相关的命令标识符. 该标识符用于按钮被选定时的$WM_COMMAND消息.
State
   按钮状态标记
Style
   按钮样式标记
Reserved
   保留
Param
   应用程序定义的值
String
   按钮字符串的0基索引, 或包含按钮文本的字符串的指针
@@end@@
