###Structure Name###
$tagTBBUTTON

###Description###
包含工具栏按钮的信息

###Syntax###
Global Const $tagTBBUTTON = "int Bitmap;int Command;byte State;byte Style;short Reserved;int Param;int String"

###Fields###
@@ParamTable@@
Bitmap
	按钮图像的 0 基索引.设置此成员维 $I_IMAGECALLBACK,
	工具栏将发送 $TBN_GETDISPINFO 通知检索图像索引时是必要的.
	设置此成员为 $I_IMAGENONE 表明按钮没有图像.
	按钮布局将不包括任何位图空间,只有文字.如果按钮是一个分隔符,则位图决定分隔符的像素宽度.
	pixels.
Command
	与按钮关联的命令标识符.当按钮被选择时,此标识符用在 $WM_COMMAND 消息中.
State
	按钮状态标志
Style
	按钮样式标志
Reserved
	保留
Param
	应用程序定义的值
String
	按钮字符串的 0 基索引,或包含按钮文字指针的字符串
@@End@@

###Remarks###
None.
