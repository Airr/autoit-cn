###User Defined Function###
_SoundStatus

###Description###
返回声音的状态

###Syntax###
#include <Sound.au3>
_SoundStatus ( $aSnd_id )


###Parameters###
@@ParamTable@@
$aSnd_id
	由 _SoundOpen() 返回的声音数组 ID 或文件名称
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回声音状态
失败:	返回 0,设置@error:	
@error:	3 = 无效的声音标识. 使用 _SoundOpen()返回的数组或有效文件名.
@@End@@

###Remarks###
 从 MSDN:
 所有设备可以返回 :
 "not ready", "paused", "playing", 与 "stopped" 值.
 某些设备可以返回更多
 "open", "parked", "recording", 与 "seeking" 值.
