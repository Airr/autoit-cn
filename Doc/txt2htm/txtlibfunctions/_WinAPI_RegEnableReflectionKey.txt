###User Defined Function###
_WinAPI_RegEnableReflectionKey

###Description###
还原禁止的注册表指定键的映射

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_RegEnableReflectionKey( $hKey )

###Parameters###
@@ParamTable@@
$hKey
   打开的键的句柄. 该句柄由_WinAPI_RegCreateKey()或_WinAPI_RegOpenKey()函数返回; 无法指定远程计算机上的键. 如果键不在映射列表上, 函数成功但不产生任何影响.
@@End@@

###ReturnValue###
@@ReturnTable@@
成功: 返回1
失败: 返回0并设置@error非0, @extended标记可包含非0系统错误代码.
@@End@@

###Remarks###
在WOW64系统上, 32位应用程序查看从64位应用程序查看器分离出的注册表树. 注册表映射在两查看器间复制指定注册表键及值.
最低系统要求: Windows Vista, Windows XP Professional x64 Edition
