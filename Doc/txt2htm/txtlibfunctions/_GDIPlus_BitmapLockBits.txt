###User Defined Function###
_GDIPlus_BitmapLockBits

###Description###
锁定位图用于读取和写入的部分

###Syntax###
#Include <GDIPlus.au3>
_GDIPlus_BitmapLockBits($hBitmap, $iLeft, $iTop, $iRight, $iBottom[, $iFlags = 1[, $iFormat = 0x00022009]])

###Parameters###
@@ParamTable@@
$hBitmap 
   位图对象句柄
$iLeft 
   要锁定的矩形的左上角X坐标
$iTop 
   要锁定的矩形的左上角Y坐标
$iRight 
   要锁定的矩形的右下角X坐标
$iBottom 
   要锁定的矩形的右下角Y坐标
$iFlags 
   指定位图的锁定部分是否可用于读取或写入及调用方是否已分配有缓冲的设置标记. 可为下列组合:
    $GDIP_ILMREAD - 用于读取的图像部分锁定
    $GDIP_ILMWRITE - 用于写入的图像部分锁定
    $GDIP_ILMUSERINPUTBUF - 缓冲已分配给用户
$iFormat 
   指定临时缓冲区中像素数据的格式. 可为下列之一:
    $GDIP_PXF01INDEXED = 每像素1位, 索引的
    $GDIP_PXF04INDEXED = 每像素4位, 索引的
    $GDIP_PXF08INDEXED = 每像素8位, 索引的
    $GDIP_PXF16GRAYSCALE = 每像素16位, 灰度
    $GDIP_PXF16RGB555 = 每像素16位; 每RGB组分5位
    $GDIP_PXF16RGB565 = 每像素16位; 红色5位, 绿色6位, 蓝色5位
    $GDIP_PXF16ARGB1555 = 每像素16位; alpha通道1位和每RGB组分5位
    $GDIP_PXF24RGB = 每像素24位; 每RGB组分8位
    $GDIP_PXF32RGB = 每像素32位; 每RGB组分8位. 无alpha组分.
    $GDIP_PXF32ARGB = 每像素32位; 每RGB和alpha组分8位
    $GDIP_PXF32PARGB = 每像素32位; 每RGB和alpha组分8位, 预放大
@@End@@

###ReturnValue###
@@ReturnTable@@
成功: $tagGDIPBITMAPDATA结构
失败: 设置@error
@@End@@

###Remarks###
处理完锁定部分时调用_GDIPlus_BitmapUnlockBits释放该部分
