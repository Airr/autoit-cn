###User Defined Function###
_WinAPI_StringLenW

###Description###
Calculates the size of wide string

###Syntax###
#include <WinAPI.au3>
_WinAPI_StringLenW($vString)

###Parameters###
@@ParamTable@@
$vString
	String buffer to process
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	String length in characters
Failure:	0
@@End@@

###Remarks###
Returned length does not include terminating null character.
