###Structure Name###
$tagNMHDDISPINFO

###Description###
包含用于控制$HDN_GETDISPINFO通知消息的信息的结构

###Syntax###
Global Const $tagNMHDDISPINFO =  "hwnd WndFrom;int IDFrom;int Code;int Item;int Mask;ptr Text;int TextMax;int Imageint lParam"

###Fields###
@@ParamTable@@
WndFrom
   发送消息的控件的窗体句柄
IDFrom
   发送消息的控件的标识
Code
   通知代码
Item
   标题控件中的0基索引项
Mask
   指定须由控件宿主填充的结构的成员数位标记集. 可为下列值的组合:
    $HDI_TEXT - Text字段须被填充
    $HDI_IMAGE - Image字段须被填充
    $HDI_LPARAM - lParam字段须被填充
    $HDI_DI_SETITEM - 表示控件将保存不再询问的项目信息的返回值.
Text
   标题栏项目将显示的文本的无终止符字符串的指针
TextMax
   文本指向的缓冲区的大小
Image
   图像列表中的图像的0基索引. 指定图像将和项目一起显示, 但不替换项目的位图. 如果iImage设置为$I_IMAGECALLBACK, 控件通过使用$HDN_GETDISPINFO通知消息为该项请求图像信息.
lParam
   与项目相关的应用程序定义的值
@@end@@
