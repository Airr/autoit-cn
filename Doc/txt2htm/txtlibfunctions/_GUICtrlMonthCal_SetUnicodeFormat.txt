###User Defined Function###
_GUICtrlMonthCal_SetUnicodeFormat

###Description###
设置控件的 Unicode 字符格式

###Syntax###
#Include <GuiMonthCal.au3>
_GUICtrlMonthCal_SetUnicodeFormat($hWnd[, $fUnicode = False])

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
$fUnicode
	[可选参数] Unicode 格式标志:
		True - 控件使用 Unicode 字符
		False - 控件使用 ANSI 字符
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	The previous Unicode 格式标志
@@End@@

###Remarks###
None.
