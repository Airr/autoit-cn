###User Defined Function###
_GUICtrlMenu_SetItemID

###Description###
Sets the menu item ID

###Syntax###
#Include <GuiMenu.au3>
_GUICtrlMenu_SetItemID($hMenu, $iItem, $iID[, $fByPos = True])

###Parameters###
@@ParamTable@@
$hMenu
	Menu handle
$iItem
	Identifier or position of the menu item
$iID
	Menu item ID
$fByPos
	[optional] Menu identifier flag:
		 True - $iItem is a zero based item position
		False - $iItem is a menu item identifier
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True
Failure:	False
@@End@@

###Remarks###
None.
