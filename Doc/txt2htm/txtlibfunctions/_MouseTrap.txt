###User Defined Function###
_MouseTrap

###Description###
限制鼠标移动到指定的座标.

###Syntax###
#Include <Misc.au3>
_MouseTrap([$iLeft = 0 [, $iTop = 0 [, $iRight = 0 [, $iBottom = 0]]]])

###Parameters###
@@ParamTable@@
$iLeft
	[可选参数] 左座标
$iTop
	[可选参数] 顶坐标
$iRight
	[可选参数] 右座标
$iBottom
	[可选参数] 底座标
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 True
失败:	返回 False
@@End@@

###Remarks###
 使用 _MouseTrap() 时没有释放鼠标的参数
