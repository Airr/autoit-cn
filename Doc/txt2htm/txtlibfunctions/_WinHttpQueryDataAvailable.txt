###User Defined Function###
_WinHttpQueryDataAvailable

###Description###
Returns the availability to be read with _WinHttpReadData().

###Syntax###
#Include <WinHttp.au3>
_WinHttpQueryDataAvailable($hRequest)

###Parameters###
@@ParamTable@@
$hRequest
	handle returned by _WinHttpOpenRequest().
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	Returns 1 if data is available.
available.	@extended receives the number of available bytes.
Failure	Returns 0 and sets @error:
	1 - DllCall failed
@@End@@

###Remarks###
_WinHttpReceiveResponse must have been called for this handle and completed before _WinHttpQueryDataAvailable is called.
