###User Defined Function###
_WinAPI_GetLocaleInfo

###Description###
Retrieves information about a locale specified by identifier.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_GetLocaleInfo ( $LCID, $iType )

###Parameters###
@@ParamTable@@
$LCID
	Locale identifier for which to retrieve information or one of the following predefined values.
	
	$LOCALE_CUSTOM_DEFAULT
	$LOCALE_CUSTOM_UI_DEFAULT
	$LOCALE_CUSTOM_UNSPECIFIED
	
	$LOCALE_INVARIANT
	$LOCALE_SYSTEM_DEFAULT
	$LOCALE_USER_DEFAULT
	
$iType
	The one of the locale information constants ($LOCALE_*) to retrieve.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	String containing the requested information.
Failure	Empty string and sets the @error flag to non-zero.
@@End@@
