###User Defined Function###
_WinAPI_ValidateRgn

###Description###
Removes a region from the current update region of the specified window.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_ValidateRect ( $hWnd [, $hRgn] )

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the window whose update region is to be modified.
$hRgn
	[可选参数] Handle to a region that defines the area to be removed from the update region. If this parameter is 0,
	the entire client area is removed.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	1.
Failure	0 and sets the @error flag to non-zero.
@@End@@

None
