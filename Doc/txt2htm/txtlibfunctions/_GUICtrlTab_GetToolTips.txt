###User Defined Function###
_GUICtrlTab_GetToolTips

###Description###
Retrieves the handle to the ToolTip control associated with the control

###Syntax###
#Include <GuiTab.au3>
_GUICtrlTab_GetToolTips($hWnd)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
@@End@@

###ReturnValue###
Success: Handle to the ToolTip control
Failure: 0

###Remarks###
None.
