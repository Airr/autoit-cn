###User Defined Function###
_IEFormElementRadioSelect

###Description###
返回给定表单元素的值

###Syntax###
#Include <IE.au3>
_IEFormElementRadioSelect ( ByRef $o_object, $s_string , $s_name [, $f_select = 1 [, $s_mode = "byValue" [, $f_fireEvent = 1]]] )

###Parameters###
@@ParamTable@@
 $o_object  InternetExplorer.Application " 窗口或者框架(frame)对象的对象变量
 $s_string  匹配元素的值--基于 $s_mode
 $s_name  单选按钮(Radio)组的名称或者ID
 $f_select  可选: 定义单选按钮(Radio)元素是否需要被选择或者不选择
-1 = 返回选择的状态
0 =
      不选中单选按钮(Radio)元素
1 = (默认) 选中单选按钮(Radio)元素
 $s_mode  可选: 定义搜索模式
byValue = (默认) 想要选择单选按钮(Radio)项目的值
byIndex =
      想要选择单选按钮(Radio)项目的索引值(基于 0)
 $f_fireEvent  可选:定义在改变值时是否触发OnChange和OnClick的事件
0 =
      在设置值后不触发OnChange或者OnClick事件
1 = (默认)
  在设置值后触发OnChange或者OnClick事件
@@End@@

###ReturnValue###
@@ReturnTable@@
成功: 如果参数$f_select = -1, 返回当前选择的状态, 否则返回 1
失败: 返回0并设置@ERROR为:
    0 ($_IEStatus_Success) = 无错误
    3 ($_IEStatus_InvalidDataType) = 无效数据类型
    4 ($_IEStatus_InvalidObjectType) = 无效对象类型
    5 ($_IEStatus_InvalidValue) = 无效值
    7 ($_IEStatus_NoMatch) = 不匹配
    @Extended: 包含无效参数数量
@@End@@

###Remarks###
$f_fireEvent参数只有当表单元素发生了和它相关的onChange事件后才有效.
函数中的$s_Name是个强制参数.
在具有相同名称的Radio组里面执行单选按钮(Radio).同一时间只有一个被选择,选中一个之后其他都是未选择.
