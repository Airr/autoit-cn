###User Defined Function###
_WinAPI_IsValidLocale

###Description###
Determines if the specified locale is installed or supported on the operating system.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_IsValidLocale ( $LCID [, $iFlag] )

###Parameters###
@@ParamTable@@
$LCID
	Locale identifier of the locale to validate.
$iFlag
	[可选参数] Flag specifying the validity test to apply to the locale identifier. This parameter can have one
	of the following values.
	
	$LCID_INSTALLED
	$LCID_SUPPORTED
	
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	1 - The locale identifier passes the specified validity test.
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;0	Otherwise.
Failure	0 and sets the @error flag to non-zero.
@@End@@
