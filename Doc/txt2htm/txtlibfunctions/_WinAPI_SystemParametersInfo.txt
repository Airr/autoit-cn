###User Defined Function###
_WinAPI_SystemParametersInfo

###Description###
检索或设置全系统(system-wide)参数的值

###Syntax###
#Include <WinAPI.au3>
_WinAPI_SystemParametersInfo($iAction [, $iParam = 0 [, $vParam = 0 [, $iWinIni = 0]]])

###Parameters###
@@ParamTable@@
$iAction
	检索或设置的全系统(system-wide)参数
$iParam
	[可选参数] 参数的用法和格式取决于参数的查询或设置
$vParam
	[可选参数] 参数的用法和格式取决于参数的查询或设置
$iWinIni
	[可选参数] 如果系统参数被设置, 指定用户是否需要更新配置文件.
		如果是, 则 $WM_SETTINGCHANGE 消息是否要传播
		如果不更新用户配置文件, 参数可以为 0 , 也可以是一或多个下列值:
		$SPIF_UPDATEINIFILE - 新设置写入到用户配置
		$SPIF_SENDCHANGE - 传播 $WM_SETTINGCHANGE 消息后更新用户配置文件
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 True
失败:	返回 False
@@End@@

###Remarks###
 函数总是调用 Unicode 版本,$vParam 必须匹配使用的字符串.
