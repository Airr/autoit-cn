###Function###
_WinAPI_GetBitmapDimension

###Description###
Retrieves a dimension for the specified bitmap.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_GetBitmapDimension ( $hBitmap )

###Parameters###
@@ParamTable@@
$hBitmap
	Handle to the bitmap to retrieve dimension.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	$tagSIZE structure that contains the bitmap dimension.
Failure	0 and sets the @error flag to non-zero.
@@End@@

None
