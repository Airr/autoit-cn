###User Defined Function###
_GUICtrlListView_SetItemCut

###Description###
设置是否将项目标记为剪切/粘贴操作

###Syntax###
#Include <GuiListView.au3>
_GUICtrlListView_SetItemCut($hWnd, $iIndex[, $fEnabled = True])

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
$iIndex
	项目的 0 基索引
$fEnabled
	[可选参数] 如为 True 则设置为此状态, 否则,不设置.
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 True
失败:	返回 False
@@End@@

###Remarks###
None.
