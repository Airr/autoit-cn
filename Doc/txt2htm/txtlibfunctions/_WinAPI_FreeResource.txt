###User Defined Function###
_WinAPI_FreeResource

###Description###
递减(减少一)加载资源的引用计数.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_FreeResource ( $hData )

###Parameters###
@@ParamTable@@
$hData
	由 _WinAPI_LoadResource() 创建的资源句柄.
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 1.
失败:	返回 0,并设置@error标志为非 0 值.
@@End@@

###Remarks###
每次应用程序为资源调用 _WinAPI_LoadResource() 函数, 资源引用计数递增(增加 1),
 这个函数已过时, 只是为了支持 16 位 Microsoft Windows 的向后兼容性.
 对于32位 Windows 应用程序, 使用 _WinAPI_LoadResource() 函数, 没有必要释放资源.
