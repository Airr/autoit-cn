###User Defined Function###
_GUICtrlMenu_GetItemID

###Description###
检索菜单项的 ID

###Syntax###
#Include <GuiMenu.au3>
_GUICtrlMenu_GetItemID($hMenu, $iItem[, $fByPos = True])

###Parameters###
@@ParamTable@@
$hMenu
	菜单句柄
$iItem
	菜单项的标识符或位置,由 $fByPos 参数决定.
$fByPos
	[可选参数] 菜单识别标志:
		True - $iItem 为项目的 0 基位置
		False - $iItem 为菜单项的标识符
@@End@@

###ReturnValue###
@@ReturnTable@@
	返回菜单项 ID(译注:根据示例脚本翻译,与原文意义完全不同.留原文如下)Returns the Application defined value associated with the menu item
@@End@@
