###User Defined Function###
_WinAPI_PathYetAnotherMakeUniqueName

###Description###

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_PathYetAnotherMakeUniqueName ( $sPath )

###Parameters###
@@ParamTable@@

	The file name that the unique name will be based on.
@@End@@
