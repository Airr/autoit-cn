###User Defined Function###
_WinHttpCheckPlatform

###Description###
确定当前平台是否支持当前版本的Microsoft Windows HTTP Services (WinHTTP)

###Syntax###
#Include <WinHttp.au3>
_WinHttpCheckPlatform()

###Parameters###
None.

###ReturnValue###
@@ReturnTable@@
成功:	当前平台支持时返回1
	当前平台不支持时返回0
	设置@error 0
失败:	返回设置@error 1, 表示DllCall失败
@@End@@

None.
