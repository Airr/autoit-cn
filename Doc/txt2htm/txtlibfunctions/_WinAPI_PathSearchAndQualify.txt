###User Defined Function###
_WinAPI_PathSearchAndQualify

###Description###
格式化路径为完整有效的路径.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_PathSearchAndQualify ( $sPath [, $fExists] )

###Parameters###
@@ParamTable@@
$sPath
	The path to be formated.
$fExists
	[可选参数] 指定路径是否存在,有效的值:
		1 - 路径必须存在,否则,函数将失败.
		0 - 路径可以不存在. (默认)
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回格式化路径.
失败:	返回空字符串,设置 @error 标志为非 0 值.
@@End@@
