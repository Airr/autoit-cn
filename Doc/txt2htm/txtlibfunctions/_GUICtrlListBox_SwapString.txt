###User Defined Function###
_GUICtrlListBox_SwapString

###Description###
互换两个指定索引的项目文本

###Syntax###
#Include <GuiListBox.au3>
_GUICtrlListBox_SwapString($hWnd, $iIndexA, $iIndexB)

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
$iIndexA
	互换项目A的 0 基索引
$iIndexB
	互换项目B的 0 基索引
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 True
失败:	返回 False
@@End@@
