###User Defined Function###
_WinHttpOpenRequest

###Description###
创建HTTP请求句柄

###Syntax###
#Include <WinHttp.au3>
_WinHttpOpenRequest($hConnect [, $sVerb = Default [, $sObjectName = Default [, $sVersion = Default [, $sReferrer = Default [, $sAcceptTypes = Default [, $iFlags = Default ]]]]]])

###Parameters###
@@ParamTable@@
$hConnect
	由_WinHttpConnect()返回的HTTP会话的句柄
$sVerb
	[可选参数] 包含在请求中使用的HTTP动词的字符串. 默认为"GET"
$sObjectName
	[可选参数] 包含指定HTTP动词的目标资源名称的字符串
$sVersion
	[可选参数] 包含HTTP版本的字符串. 默认为"HTTP/1.1"
$sReferrer
	[可选参数] 指定从其中获取所需$sObjectName的URL的文档的URL字符串. 默认为$WINHTTP_NO_REFERER.
$sAcceptTypes
	[可选参数] 指定客户端接受的指定媒体类型的字符串. 默认为$WINHTTP_DEFAULT_ACCEPT_TYPES.
$iFlags
	[可选参数] 包含网络标记值的整数. 默认为$WINHTTP_FLAG_ESCAPE_DISABLE.
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回有效的会话句柄设置@error 0
失败:	返回0设置@error 且其为1时表示DllCall失败
@@End@@
