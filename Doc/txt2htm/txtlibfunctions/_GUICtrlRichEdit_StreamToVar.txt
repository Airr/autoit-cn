###User Defined Function###
_GUICtrlRichEdit_StreamToVar

###Description###
控件内容写入到变量

###Syntax###
#Include <GuiRichEdit.au3>
_GUICtrlRichEdit_StreamToVar($hWnd [, $fRtf = True [, $fIncludeCOM=True [, $iOpts=0 [, $iCodePage = 0]]]])

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
$fRtf
	[可选参数] True - 写入富文本格式(RTF) (默认)
		False - 只写文字
$fIncludeCOM
	[可选参数] True (default):
		如果写 RTF 格式,包括任何 COM 对象（消耗空间）
		如果只写文字,写 COM 对象的文本表达
		False: 写空格代替 COM 对象
$iOpts
	[可选参数] 其他选项:
		$SFF_PLAINTRTF - 只写入对所有语言普遍适用的关键字
		$SF_UNICODE - 写入 Unicode
$iCodePage
	[可选参数] 生成 UTF-8 和文字使用的代码页
		默认 : 没有
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 RTF 或文本
失败:	返回 "" ,设置 @error
@error:	101 - $hWnd 参数值不是句柄
	1041 - $SFF_PLAINRTF 是无效的文本文件
	1042 - $opts: 是无效的选项
	1043 - $SF_UNICODE 仅对文本文件有效
	700 - 内部错误
@@End@@

###Remarks###
 如果文本被选中,只写选择的,否则全部文本写入到控件

 如果 $sFileSpec 指定扩展名为 .rtf, 则写入 RTF, 否则写纯文本

 调用 _GUICtrlRichEdit_IsModified() 确定文本是否发生了变化
