###User Defined Function###
_WinAPI_LoadResource

###Description###
加载指定资源到全局内存.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_LoadResource ( $hInstance, $hResource )

###Parameters###
@@ParamTable@@
$hInstance
	包含资源的可执行文件模块句柄.如果此参数为 0,
		 系统加载创建当前进程的模块资源.
$hResource
	资源句柄. 此句柄由 _WinAPI_FindResource() 或 _WinAPI_FindResourceEx() 函数返回.	
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回相关数据资源的句柄.
失败:	返回 0,设置@error:标志为非 0 值.
@@End@@

###Remarks###
调用 _WinAPI_LockResource() 函数获取资源数据指针.
