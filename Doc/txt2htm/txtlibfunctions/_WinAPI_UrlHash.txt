###User Defined Function###
_WinAPI_UrlHash

###Description###
Hashes a URL string.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_UrlHash ( $sUrl [, $iLength] )

###Parameters###
@@ParamTable@@
$sUrl
	The URL.
$iLength
	[可选参数] The length of the hash data, in bytes. It should be no larger than 256, otherwise, the function fails.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	The hash data in binary form.
Failure	0 and sets the @error flag to non-zero, @extended flag may contain the system error code.
@@End@@
