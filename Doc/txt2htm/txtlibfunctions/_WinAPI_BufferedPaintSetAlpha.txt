###User Defined Function###
_WinAPI_BufferedPaintSetAlpha

###Description###
Sets the alpha to a specified value in a given rectangle.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_BufferedPaintSetAlpha ( $hBP [, $iAlpha [, $tRECT]] )

###Parameters###
@@ParamTable@@
$hBP
	Handle of the buffered paint context.
$iAlpha
	[可选参数] The alpha value to set. The alpha value can range from zero (fully transparent) to 255 (fully opaque).
$tRECT
	[可选参数] $tagRECT structure that specifies the rectangle in which to set the alpha. If this parameter is 0,
	the entire buffer is used.
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 1.
失败:	返回 0 并设置 @error 标志为非 0 值, @extended 标志可能包含一个系统错误代码.
@@End@@

###Remarks###
本函数需要 Windows Vista 或更高版本系统.
