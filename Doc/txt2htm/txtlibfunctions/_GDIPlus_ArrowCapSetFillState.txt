###User Defined Function###
_GDIPlus_ArrowCapSetFillState

###Description###
设置箭头帽填充状态

###Syntax###
#Include <GDIPlus.au3>
_GDIPlus_ArrowCapSetFillState($hArrowCap, $bFilled = True)

###Parameters###
@@ParamTable@@
$hArrowCap
	箭头帽对象句柄
$bFilled
	填充状态:
		True - 以填充状态
		False - 非填充
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 True
失败:	返回 False
@@End@@
