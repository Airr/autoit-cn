###User Defined Function###
_GUICtrlListView_EnableGroupView

###Description###
启用或禁用控件项目分组

###Syntax###
#Include <GuiListView.au3>
_GUICtrlListView_EnableGroupView($hWnd[, $fEnable = True])

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
$fEnable
	[可选参数]指定项目是否分组
		True - 启用分组视图
		False - 禁用分组视图
@@End@@

###ReturnValue###
@@ReturnTable@@
	返回 0:项目分组显示已经启用或禁用返回 1:控件状态已成功更改
@@End@@

###Remarks###
 最低操作系统 - Windows XP.
 本函数不支持 $LVS_OWNERDATA 样式.
