###User Defined Function###
_GUICtrlToolbar_HighlightButton

###Description###
设置按钮高亮状态

###Syntax###
#Include <GuiToolBar.au3>
_GUICtrlToolbar_HighlightButton($hWnd, $iCommandID[, $fHighlight = True])

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
$iCommandID
	按钮命令 ID
$fHighlight
	[可选参数] 高亮状态:
		True - 按钮高亮
		False - 按钮非高亮
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	True
失败:	False
@@End@@
