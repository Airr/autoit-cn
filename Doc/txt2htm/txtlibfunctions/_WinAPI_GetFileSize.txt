###User Defined Function###
_WinAPI_GetFileSize

###Description###
Retrieves the size of the specified file.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_GetFileSize ( $hFile )

###Parameters###
@@ParamTable@@
$hFile
	A handle to the file. The handle must have been created with either the $GENERIC_READ or $GENERIC_WRITE
	access right or equivalent.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	The file size, in bytes.
Failure	0 and sets the @error flag to non-zero.
@@End@@

None
