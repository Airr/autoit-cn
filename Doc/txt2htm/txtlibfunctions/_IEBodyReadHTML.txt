###User Defined Function###
_IEBodyReadHTML

###Description###
返回内有<body>标记的HTML文档.

###Syntax###
#Include <IE.au3>
_IEBodyReadHTML ( ByRef$o_object )

###Parameters###
@@ParamTable@@
 $o_object  一个 InternetExplorer.Application, Window 或框架对象的对象变量
@@End@@

###ReturnValue###
@@ReturnTable@@
 成功:  返回文档<body>内包含的HTML代码
 失败:  返回0 并且设置 @ERROR
 @Error:  0 ($_IEStatus_Success) = 无错误
  3 ($_IEStatus_InvalidDataType) = 无效数据类型
 @Extended:  包含无效参数数量
@@End@@

###Remarks###
这个函数使用文档主体(body)的 .innerHTML属性. .innerHTML
可用于几乎所有使用_IEPropertyGet.的DOM元素
