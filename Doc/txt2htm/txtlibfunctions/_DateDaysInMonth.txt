###User Defined Function###
_DateDaysInMonth

###Description###
返回基于指定年的指定月的月中的天数

###Syntax###
#Include <Date.au3>
_DateDaysInMonth($iYear, $iMonthNum)

###Parameters###
@@ParamTable@@
$iYear
	4位数的年格式.
$iMonthNum
	月份序数(1 = January, 12 = December).
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回指定月份的天数.
失败:	返回 0 并设置 set @error
@error:	0 - 没有错误.
	1 - 无效月份序数或年
@@End@@
