###Function###
_WinAPI_GetCursor

###Description###
Retrieves a handle to the current cursor.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_GetCursor ( )

###Parameters###
@@ParamTable@@
None.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	Handle to the current cursor. If there is no cursor, the return value is 0.
Failure	0 and sets the @error flag to non-zero.
@@End@@
