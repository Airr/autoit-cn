###User Defined Function###
_GUICtrlStatusBar_SetText

###Description###
设置状态窗口指定部件的文本

###Syntax###
#Include <GuiStatusBar.au3>
_GUICtrlStatusBar_SetText($hWnd, $sText = "", $iPart = 0, $iUFlag = 0)

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
$sText
	部件的显示文本
$iPart
	部件的保留文本
$iUFlag
	绘制操作类型.类型可以是以下值:
		0 - 绘制文本的边框低于窗口平面
		$SBT_NOBORDERS - 绘制无边框文本
		$SBT_OWNERDRAW - 由父窗口绘制文本
		$SBT_POPOUT - 绘制文本的边框高于窗口平面
		$SBT_RTLREADING - 文本反向显示到父窗口
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 True
失败:	返回 False
@@End@@

###Remarks###
 为简单状态条设置 $iPart 为 $SB_SIMPLEID
