###User Defined Function###
_IEFormGetCollection

###Description###
返回文档内描述表单的对象变量合集或一个带索引的单一表单

###Syntax###
#Include <IE.au3>
_IEFormGetCollection ( ByRef $o_object [, $i_index = -1] )

###Parameters###
@@ParamTable@@
 $o_object  InternetExplorer.Application , 窗口或者框架(frame)对象的对象变量
 $i_index  可选: 指定返回一个集合还是索引对象
0 或者正整数返回一个索引对象
-1 = (默认)返回一个集合
@@End@@

###ReturnValue###
@@ReturnTable@@
成功: 返回指向表单对象的对象变量 , @EXTENDED = 表单数量
失败: 返回0并设置@ERROR为:
    0 ($_IEStatus_Success) = 无错误
    3 ($_IEStatus_InvalidDataType) = 无效数据类型
    5 ($_IEStatus_InvalidValue) = 无效
    7 ($_IEStatus_NoMatch) = 不匹配
    @Extended: 包含无效参数数量
@@End@@
