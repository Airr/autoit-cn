###User Defined Function###
_GUICtrlTreeView_GetChildren

###Description###
Indicates whether the item children flag is set

###Syntax###
#Include <GuiTreeView.au3>
_GUICtrlTreeView_GetChildren($hWnd, $hItem)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$hItem
	Handle to the item
@@End@@

###ReturnValue###
@@ReturnTable@@
True:	Item children flag is set
False:	Item children flag is not set
@@End@@

None.
