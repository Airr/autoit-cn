###User Defined Function###
_ClipBoard_GetViewer

###Description###
Retrieves the handle to the first window in the clipboard viewer chain

###Syntax###
#Include <Clipboard.au3>
_ClipBoard_GetViewer()

###Parameters###
None.


###ReturnValue###
@@ReturnTable@@
Success:	The handle to the first window in the clipboard viewer chain
Failure:	Zero if there is no clipboard viewer
@@End@@


###Remarks###
None.
