###User Defined Function###
_ClipBoard_GetOwner

###Description###
返回剪切板中数据的所有者窗口句柄

###Syntax###
#Include <Clipboard.au3>
_ClipBoard_GetOwner()

###Parameters###
None.


###ReturnValue###
@@ReturnTable@@
Success:	The handle to the window that owns the clipboard
Failure:	0
@@End@@


###Remarks###
The clipboard can still contain data even if the clipboard is not currently owned. In general, the clipboard
owner is the window that last placed data in clipboard. The _ClipBoard_Empty function assigns clipboard ownership.

_ClipBoard_Empty, _ClipBoard_GetOpenWindow
