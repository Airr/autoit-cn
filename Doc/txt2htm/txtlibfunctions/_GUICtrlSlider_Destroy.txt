###User Defined Function###
_GUICtrlSlider_Destroy

###Description###
Delete the control

###Syntax###
#Include <GuiSlider.au3>
_GUICtrlSlider_Destroy(ByRef $hWnd)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
@@End@@

###ReturnValue###
成功: True, 句柄设置为 0
失败: False

###Remarks###
Restricted to only be used on slider created with _GUICtrlSlider_Create
