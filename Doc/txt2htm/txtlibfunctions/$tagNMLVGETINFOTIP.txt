###Structure Name###
$tagNMLVGETINFOTIP

###Description###
包含和接收列表视图项目显示工具提示所需的信息

###Syntax###
Global Const $tagNMLVGETINFOTIP = $tagNMHDR & ";int Flags;ptr Text;int TextMax;int Item;int SubItem;int lParam"

###Fields###
@@ParamTable@@
$tagNMHDR
	包含通知消息的信息
Flags
	0 或 $LVGIT_UNFOLDED
Text
	字符串缓冲区,它接收任何额外的文本信息.
	如果 Flags 为 0, 该会员将包含现有项目的文本,
	这种情况下，你可以附加任何其他文本到这个字符串的末尾.
TextMax
	指出文本缓冲区的字符大小
	你可以不需设置这个缓冲区任何特定的大小,
	而是由 $INFOTIPSIZE 的值来设置.
Item
	本结构指向的项目的 0 基索引.
SubItem
	结构指向的子项目的 1 基索引.
	如果该成员为 0, 结构指向项目而非子项.
	此成员尚未使用时,将永远是 0 
lParam
	应用程序定义的项目有关值.
	此成员尚未使用时,将永远是 0.
@@End@@

###Remarks###
None.
