###User Defined Function###
_GUICtrlListView_SetItemDropHilited

###Description###
设置是否将项目高亮显示为拖放目标

###Syntax###
#Include <GuiListView.au3>
_GUICtrlListView_SetItemDropHilited($hWnd, $iIndex[, $fEnabled = True])

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
$iIndex
	项目的 0 基索引
$fEnabled
	[可选参数] 如为 True 则设置为此状态, 否则,不设置.
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 True
失败:	返回 False
@@End@@

###Remarks###
None.
