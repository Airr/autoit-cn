###Function###
_WinAPI_FatalExit

###Description###
Transfers execution control to the debugger.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_FatalExit ( $iCode )

###Parameters###
@@ParamTable@@
$iCode
	The error code associated with the exit.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	1.
Failure	0 and sets the @error flag to non-zero.
@@End@@

None
