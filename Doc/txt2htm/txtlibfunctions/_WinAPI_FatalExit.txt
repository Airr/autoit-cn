###User Defined Function###
_WinAPI_FatalExit

###Description###
Transfers execution control to the debugger.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_FatalExit ( $iCode )

###Parameters###
@@ParamTable@@
$iCode
	The error code associated with the exit.
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 1.
失败:	返回 0,设置 the @error 为非 0 值.
@@End@@

None
