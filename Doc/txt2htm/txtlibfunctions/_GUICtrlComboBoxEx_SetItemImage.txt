###User Defined Function###
_GUICtrlComboBoxEx_SetItemImage

###Description###
设置项目图标在图像列表中的索引.

###Syntax###
#Include <GuiComboBoxEx.au3>
_GUICtrlComboBoxEx_SetItemImage($hWnd, $iIndex, $iImage)

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
$iIndex
	项目的 0 基索引
$iImage
	图标在图像列表的 0 基索引
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 True
失败:	返回 False
@@End@@
