###User Defined Function###
_GUICtrlComboBoxEx_SetUnicode

###Description###
设置控件使用 Unicode

###Syntax###
#Include <GuiComboBoxEx.au3>
_GUICtrlComboBoxEx_SetUnicode($hWnd[, $fUnicode = True])

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
$fUnicode
	[可选参数] 可以是下列值之一:
		True - 打开 Unicode
		False - 关闭 Unicode
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 True
失败:	返回 False
@@End@@

###Remarks###
None.
