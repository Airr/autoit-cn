###User Defined Function###
_GUICtrlComboBoxEx_SetUnicode

###Description###
设置控件是否使用Unicode

###Syntax###
#include <GuiComboBoxEx.au3>
_GUICtrlComboBoxEx_SetUnicode($hWnd [, $fUnicode = True])

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
$fUnicode
	[可选参数] 可以是下列值之一:
		True - 启用 Unicode
		False - 禁用 Unicode
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 True
失败:	返回 False
@@End@@

###Remarks###
None.
