###User Defined Function###
_DebugReportEx

###Description###
格式化写入调试会话框的信息

###Syntax###
#include <Debug.au3>
_DebugReportEx($sData [,$bLastError = False [,$bExit = False]])


###Parameters###
@@ParamTable@@
$sData
	须予报告的数据 (例如 "dllname|functionname")
$bLastError
	[可选参数] 设置为 True, 表示如有 GetLastErrorMessage() 数据必须追加到报告中
$bExit
	[可选参数] 设置为 True, 表示脚本必须终止
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 1.
失败:	返回 0.
@@End@@

###Remarks###
根据 @error 错误信息将被格式化如下:
0 - 报告 "DLL 返回错误函数".
1 - 报告 "无法打开 dll"
3 - 报告 "无法找到 DLL 函数".

 如果 $sData 不包含 "|" 或 @error 不是上面所指定的报告内容, 信息将不会被格式化.
 如果没有 _DebugSetup(), 已发布的函数总是返回.
 @error 为调用方保留.
 如果 $bLastError = True, @extended 可以设置为 Windows API 的 GetlastError(). 否则保留.
