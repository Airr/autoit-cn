###User Defined Function###
_DebugReportEx

###Description###
将格式化信息写入调试会话

###Syntax###
#Include <Debug.au3>
_DebugReportEx( $sData [,$bLastError = False [,$bExit = False]] )

###Parameters###
@@ParamTable@@
$sData 
   将被报告的数据
$bLastError 
   [可选] 为True时在报告的数据后追加GetLastErrorMessage()
$bExit 
   [可选] 为True时中断脚本
@@End@@

###ReturnValue###
@@ReturnTable@@
成功: 返回1
失败: 返回0
@@End@@

###Remarks###
消息将根据@error值格式化为:
0 - 报告Bad return from FUNCTION in DLL(dll中的函数返回失败)
1 - 报告unable to open a dll(无法打开dll)
3 - 报告unable to find FUNCTION in DLL(无法在dll中找到函数)
如果$sData不包含"|"或@error未如上指定则不会被格式化
如果未建立_DebugSetup()函数总是返回.
保留调用方的@error.
如果$bLastError为True则@extended可设置为Windows API GetlastError(). 否则将保留.
