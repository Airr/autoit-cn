###User Defined Function###
_GUICtrlStatusBar_SetUnicodeFormat

###Description###
设置 Unicode 字符格式标志

###Syntax###
#Include <GuiStatusBar.au3>
_GUICtrlStatusBar_SetUnicodeFormat($hWnd[, $fUnicode = True])

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
$fUnicode
	[可选参数] Unicode 字符格式标志:
		True - 控件使用 Unicode 字符
		False - 控件使用 ANSI 字符
@@End@@

###ReturnValue###
@@ReturnTable@@
	返回以前设置的字符格式标志
@@End@@

###Remarks###
None.
