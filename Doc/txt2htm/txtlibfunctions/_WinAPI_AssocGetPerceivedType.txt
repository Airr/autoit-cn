###Function###
_WinAPI_AssocGetPerceivedType

###Description###
Retrieves a file's perceived type based on its extension.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_AssocGetPerceivedType ( $sExt )

###Parameters###
@@ParamTable@@
$sExt
	The file's extension. This should include the leading period, for example ".txt".
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	The array containing the following parameters:
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;[0]	The perceived type ($PERCEIVED_TYPE_*).
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;[1]	The source of the perceived type information ($PERCEIVEDFLAG_*).
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;[2]	The perceived type string, for instance "text" or "video".
Failure	0 and sets the @error flag to non-zero, @extended flag may contain the system error code.
@@End@@

None
