###Function###
_WinAPI_ShellExtractAssociatedIcon

###Description###
Returns a handle to the icon that associated with the specified file's.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_ShellExtractAssociatedIcon ( $sFile [, $fSmall] )

###Parameters###
@@ParamTable@@
$sFile
	The full path and file name of the file that contains the icon, or its extension, such as ".txt".
$fSmall
	Specifies whether extracts a small icon, valid values:
		TRUE   - Extracts a small icon.
		FALSE  - Extracts a large icon. (Default)
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	The icon handle.
Failure	0 and sets the @error flag to non-zero.
@@End@@
