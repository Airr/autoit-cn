###User Defined Function###
_GUICtrlListView_GetStringWidth

###Description###
Determines the width of a specified string

###Syntax###
#Include <GuiListView.au3>
_GUICtrlListView_GetStringWidth($hWnd, $sString)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$sString
	String for which the width will be calculated
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	The string width
Failure:	0
@@End@@

###Remarks###
This function returns the exact width, in pixels, of the specified string. If you use the returned string
width as the column width in the SetColumnWidth function the string will be truncated. To retrieve the column
width that can contain the string without truncating it, you must add padding to the returned string width.
