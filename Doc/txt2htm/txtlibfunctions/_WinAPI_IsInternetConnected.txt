###User Defined Function###
_WinAPI_IsInternetConnected

###Description###
确定当前用户是否连接到互联网.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_IsInternetConnected ( )

###Parameters###
None.

###ReturnValue###
@@ReturnTable@@
成功:	返回 1 - 用户已连接互联网.
	返回 0 - 未连接.
失败:	返回 0,设置@error:非 0 值, @extended 可能包含系统错误代码.
@@End@@

###Remarks###
此函数需要 Windows Vista 或更高版本.
