###User Defined Function###
_GUICtrlListView_MapIndexToID

###Description###
映射项目索引到 ID

###Syntax###
#Include <GuiListView.au3>
_GUICtrlListView_MapIndexToID($hWnd, $iIndex)

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
$iIndex
	项目的 0 基索引
@@End@@

###ReturnValue###
@@ReturnTable@@
	返回项目的 ID
@@End@@

###Remarks###
 最低操作系统 - Windows XP.
 控件内部由索引跟踪项目,这可能存在的问题,
 因为在控件的生命周期内,索引是可以改变的.
 ListView 控件可以用项目创建时的 ID 标记项目,
 你可以使用这个 ID,以保证在该控件生命周期内项目的唯一性.
