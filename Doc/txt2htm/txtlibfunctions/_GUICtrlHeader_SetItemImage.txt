###User Defined Function###
_GUICtrlHeader_SetItemImage

###Description###
设置图像列表内图像的索引

###Syntax###
#include <GuiHeader.au3>
_GUICtrlHeader_SetItemImage($hWnd, $iIndex, $iImage)

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
$iIndex
	项目的 0 基索引
$iImage
	图像的 0 基索引
@@End@@

###ReturnValue###
成功:	返回 True
失败:	返回 False

###Remarks###
None.
