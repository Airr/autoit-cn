###User Defined Function###
_SQLite_Display2DResult

###Description###
返回或打印一个二维数组到控制台

###Syntax###
#include <SQLite.au3>
_SQLite_Display2DResult ( $aResult [, $iCellWidth = 0 [, $bReturn = 0 ]] )


###Parameters###
@@ParamTable@@
$aResult
	要操作的数组
$iCellWidth
	[可选参数] 指定数据字段的大小
$bReturn
	[可选参数] 如果为 True, 返回格式化字符串,但不显示
		如果为 False, 格式化字符串发送到 stdout[标准输出(设备)文件]
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	无返回或返回格式化字符串
失败:	@error = 1 - $aResult 没有数组或有错误尺寸
@@End@@
