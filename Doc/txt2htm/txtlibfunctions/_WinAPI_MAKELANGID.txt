###User Defined Function###
_WinAPI_MAKELANGID

###Description###
从主语言 ID 和子语言标识构建语言 ID

###Syntax###
#Include <WinAPI.au3>
_WinAPI_MAKELANGID($lgidPrimary, $lgidSub)

###Parameters###
@@ParamTable@@
$lgidPrimary
	主语言 ID
$lgidSub
	子语言 ID
@@End@@

###ReturnValue###
返回语言标识符

###Remarks###
None.
