###User Defined Function###
_GUICtrlButton_SetFocus

###Description###
将键盘焦点设定到指定的按钮

###Syntax###
#Include <GuiButton.au3>
_GUICtrlButton_SetFocus( $hWnd[, $fFocus = True] )

###Parameters###
@@ParamTable@@
 $hWnd  控件句柄
 $fFocus  参数可能是下列数值之一:
True - 将键盘焦点设定到按钮
False - 移除按钮的键盘焦点
@@End@@

###ReturnValue###
@@ReturnTable@@
成功: True
失败: False
@@End@@
