###User Defined Function###
_GUICtrlButton_SetFocus

###Description###
设置键盘焦点到指定的按钮

###Syntax###
#include <GuiButton.au3>
_GUICtrlButton_SetFocus($hWnd [, $fFocus = True])

###Parameters###
@@ParamTable@@
$hWnd
	控件的控件ID/句柄
$fFocus
	[可选参数] 参数可以是下列值之一:
		True - 将键盘焦点设定到按钮
		False - 移除按钮的键盘焦点
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 True
失败:	返回 False
@@End@@
