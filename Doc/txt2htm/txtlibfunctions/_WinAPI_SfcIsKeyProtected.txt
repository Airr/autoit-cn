###User Defined Function###
_WinAPI_SfcIsKeyProtected

###Description###
确定指定注册表是否处于保护

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_SfcIsKeyProtected ( $hKey [, $sSubKey [, $iFlag]] )

###Parameters###
@@ParamTable@@
$hKey
	注册表根键句柄或下列预定义值:
	
	$HKEY_CLASSES_ROOT
	$HKEY_CURRENT_USER
	$HKEY_LOCAL_MACHINE
	$HKEY_USERS
	
$sSubKey
	[可选参数] 子键
$iFlag
	[可选参数] 指定运行于64位Windows中的可选注册表视图的标记. x86平台忽略.
	可为下列值.
	
	$KEY_WOW64_32KEY
	$KEY_WOW64_64KEY
	
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回1 表示文件处于保护
0	其它
失败:	返回0并设置@error非0
@@End@@

###Remarks###
最低系统要求: Windows Vista
