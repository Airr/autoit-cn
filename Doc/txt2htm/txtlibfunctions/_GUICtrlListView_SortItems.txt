###User Defined Function###
_GUICtrlListView_SortItems

###Description###
Starts the sort call back, also sets the Arrow in the Header

###Syntax###
#Include <GuiListView.au3>
_GUICtrlListView_SortItems($hWnd, $iCol)

###Parameters###
@@ParamTable@@
$hWnd
	Handle of the control
$iCol
	Column clicked
@@End@@

###ReturnValue###
None.

###Remarks###
For use only in conjunction with _GUICtrlListView_RegisterSortCallBack

A down-arrow/up-arrow is drawn on column selected for Windows XP and above if the option was set
when calling _GUICtrlListView_RegisterSortCallBack
