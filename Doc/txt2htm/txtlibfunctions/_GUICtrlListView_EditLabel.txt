###User Defined Function###
_GUICtrlListView_EditLabel

###Description###
就地编辑指定项目文本

###Syntax###
#Include<GuiListView.au3>
_GUICtrlListView_EditLabel($hWnd, $iIndex)

###Parameters###
@@ParamTable@@
 $hWnd  控件句柄
 $iIndex  要编辑的项目的索引(基于 0 )
@@End@@

###ReturnValue###
@@ReturnTable@@
成功: 用于编辑项目的所在控件的句柄
失败: 0
@@End@@

###Remarks###
当用户完成或者取消编辑时, 编辑控件被破坏且句柄不再有效.
可以使用子类编辑控件, 但最好不要破坏它
