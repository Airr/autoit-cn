###User Defined Function###
_GUICtrlMenu_SetItemSubMenu

###Description###
设置菜单项关联的下拉菜单或子菜单

###Syntax###
#Include <GuiMenu.au3>
_GUICtrlMenu_SetItemSubMenu($hMenu, $iItem, $hSubMenu[, $fByPos = True])

###Parameters###
@@ParamTable@@
$hMenu
	菜单句柄
$iItem
	菜单项的标识符或位置,由 $fByPos 参数决定.
$hSubMenu
	下拉菜单或子菜单句柄
$fByPos
	[可选参数] 菜单识别标志:
		True - $iItem 为项目的 0 基位置
		False - $iItem 为菜单项的标识符
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 True
失败:	返回 False
@@End@@

###Remarks###
None.
