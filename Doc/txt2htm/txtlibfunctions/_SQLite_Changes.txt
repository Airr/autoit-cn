###User Defined Function###
_SQLite_Changes

###Description###
返回最后执行操作对数据库影响的行数

###Syntax###
#include <SQLite.au3>
_SQLite_Changes ( [ $hDB ] )


###Parameters###
@@ParamTable@@
$hDB
	[可选参数] 打开的数据库,默认为最后打开的数据库
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回影响的行数
失败:	返回 0
@error	1 - 错误调用 SQLite API 'sqlite3_changes'
	2 - 调用被安全模式阻止
@@End@@

###Remarks###
Changes due to action of triggers or foreign keys are not part of this count.  See _SQLite_TotalChanges()
