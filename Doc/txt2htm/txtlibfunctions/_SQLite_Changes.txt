###User Defined Function###
_SQLite_Changes

###Description###
返回最近发生改变的数量(不包含触发点)

###Syntax###
#Include <SQLite.au3>
_SQLite_Changes ( [ $hDB ] )

###Parameters###
@@ParamTable@@
 $hDB  [可选] 打开的数据库, 默认为最近打开的数据库
@@End@@

###ReturnValue###
@@ReturnTable@@
成功: 返回改变的数量
失败: 返回0
@error: 1 - 调用SQLite API 'sqlite3_changes'出错
     2 - 安全模式阻止调用
@@End@@
