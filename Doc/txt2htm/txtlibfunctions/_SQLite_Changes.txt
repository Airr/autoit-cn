###User Defined Function###
_SQLite_Changes

###Description###
返回最后执行的更改数字(不包括触发器)

###Syntax###
#include <SQLite.au3>
_SQLite_Changes ( [ $hDB ] )


###Parameters###
@@ParamTable@@
$hDB
	[可选参数] 打开的数据库,默认为最后打开的数据库
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回更改的数字
失败:	返回 0
@error:	1 - 错误调用 SQLite API 'sqlite3_changes'
	2 - 调用被安全模式阻止
@@End@@
