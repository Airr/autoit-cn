###用户自定义函数###
_SQLite_Changes

###描述###
返回最近发生改变的数量(不包含触发点)

###语法###
#Include <SQLite.au3>
_SQLite_Changes ( [ $hDB ] )

###参数###
@@ParamTable@@
 $hDB  [可选] 打开的数据库, 默认为最近打开的数据库
@@End@@

###返回值###
@@ReturnTable@@
成功: 返回改变的数量
失败: 返回0
@error: 1 - 调用SQLite API 'sqlite3_changes'出错
     2 - 安全模式阻止调用
@@End@@
