###User Defined Function###
_WinAPI_MultiByteToWideChar

###Description###
映射字符串到宽字符(Unicode)字符串

###Syntax###
#Include <WinAPI.au3>
_WinAPI_MultiByteToWideChar($sText [, $iCodePage = 0 [, $iFlags = 0 [, $bRetString = False]]])

###Parameters###
@@ParamTable@@
$sText
	文本或 Dll 结构包含的多字节文本
$iCodePage
	[可选参数] 执行转换的代码页:
		0 - ANSI 代码页
		1 - OEM 代码页
		2 - Macintosh 代码页
		3 - 当前线程的 Windows ANSI 代码页
		42 - Symbol 代码页
		65000 - UTF-7
		65001 - UTF-8
$iFlags
	[可选参数] 是否转化为预构成或合成的宽字符:
		$MB_PRECOMPOSED - 预构成字符
		$MB_COMPOSITE - 合成宽字符
		$MB_USEGLYPHCHARS - 标志符号字符代替控制字符
$bRetString
	[可选参数] 指定返回字符串或 DLL 结构(默认 false:结构)
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回包含 Unicode 字符串的字符串或 DLL 结构字符
失败:	设置@error	
@@End@@
