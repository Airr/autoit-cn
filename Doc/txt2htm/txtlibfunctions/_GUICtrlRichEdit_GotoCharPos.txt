###User Defined Function###
_GUICtrlRichEdit_GotoCharPos

###Description###
移动插入点到字符区间位置

###Syntax###
#include <GuiRichEdit.au3>
_GUICtrlRichEdit_GotoCharPos($hWnd, $iCharPos)

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
$iCharPos
	字符区间位置
		特殊值: -1 - 文本结束
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 True
失败:	返回 False 并设置@error
@error:	101 - $hWnd 参数值不是句柄
	102 - $iCharPos 既不是正数或零, 也不是 -1
@@End@@

###Remarks###
控件文本的第一字符位置为 0

取消文本选择(如果有)
