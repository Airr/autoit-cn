###User Defined Function###
_WinAPI_LoadCursor

###Description###
Loads the specified cursor resource from the executable (.EXE) file.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_LoadCursor ( $hInstance, $sName )

###Parameters###
@@ParamTable@@
$hInstance
	Handle to an instance of the module whose executable file contains the cursor to be loaded.
$sName
	The name of the cursor resource or resource identifier to be loaded. To use one of the predefined
	cursors, the application must set the $hInstance parameter to 0 and the $sName parameter to one
	of the following values.
	
	$IDC_APPSTARTING
	$IDC_HAND
	$IDC_ARROW
	$IDC_CROSS
	$IDC_IBEAM
	$IDC_ICON
	$IDC_NO
	$IDC_SIZE
	$IDC_SIZEALL
	$IDC_SIZENESW
	$IDC_SIZENS
	$IDC_SIZENWSE
	$IDC_SIZEWE
	$IDC_UPARROW
	$IDC_WAIT
	
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	Handle to the newly loaded cursor.
Failure	0 and sets the @error flag to non-zero.
@@End@@

###Remarks###
This function loads the cursor resource only if it has not been loadedotherwise, it retrieves the handle to
the existing resource.
