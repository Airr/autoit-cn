###User Defined Function###
_GUICtrlHeader_SetUnicodeFormat

###Description###
设置控件的 Unicode 字符格式标志

###Syntax###
#Include <GuiHeader.au3>
_GUICtrlHeader_SetUnicodeFormat($hWnd, $fUnicode)

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
$fUnicode
	Unicode 标志:
		True - 使用 Unicode 字符
		False - 使用 ANSI 字符
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回先前的 Unicode 格式标志
@@End@@

###Remarks###
None.
