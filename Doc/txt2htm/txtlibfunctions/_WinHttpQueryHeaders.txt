###用户自定义函数###
_WinHttpQueryHeaders

###描述###
获取与HTTP请求相关的头部信息

###语法###
#Include <WinHTTP.au3>
_WinHttpQueryHeaders( $hRequest, [, $iInfoLevel = Default [, $sName = Default [, $iIndex = Default ]]] )

###参数###
@@ParamTable@@
$hRequest 
   _WinHttpOpenRequest()返回的句柄
$iInfoLevel 
   [可选] 指定属性和修正标记的组合. 默认为$WINHTTP_QUERY_RAW_HEADERS_CRLF.
$sName 
   [可选] 包含头部名称的字符串. 默认为$WINHTTP_HEADER_NAME_BY_INDEX.
$iIndex 
   [可选] 枚举多个同名头部的索引.
@@End@@

###返回值###
@@ReturnTable@@
成功: 返回包含Http头部的字符串并设置@extended为下一头部的索引
失败: 返回0并设置@error为:
    1 - DllCall失败
@@End@@
