###User Defined Function###
_WinHttpQueryHeaders

###Description###
Retrieves header information associated with an HTTP request.

###Syntax###
#Include <WinHttp.au3>
_WinHttpQueryHeaders($hRequest [, $iInfoLevel = Default [, $sName = Default [, $iIndex = Default ]]])

###Parameters###
@@ParamTable@@
$hRequest
	Handle returned by _WinHttpOpenRequest().
$iInfoLevel
	[可选参数] A combination of attribute and modifier flags. Default is $WINHTTP_QUERY_RAW_HEADERS_CRLF.
$sName
	[可选参数] Header name string. Default is $WINHTTP_HEADER_NAME_BY_INDEX.
$iIndex
	[可选参数] Index used to enumerate multiple headers with the same name
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	Returns string that contains header.
Failure	Returns empty string and sets @error:
	1 - DllCall failed
@@End@@
