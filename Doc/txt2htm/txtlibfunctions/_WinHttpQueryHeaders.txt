###User Defined Function###
_WinHttpQueryHeaders

###Description###
获取与HTTP请求相关的头部信息.

###Syntax###
#Include <WinHttp.au3>
_WinHttpQueryHeaders($hRequest [, $iInfoLevel = Default [, $sName = Default [, $iIndex = Default ]]])

###Parameters###
@@ParamTable@@
$hRequest
	_WinHttpOpenRequest() 函数返回的句柄.
$iInfoLevel
	[可选参数] 一个包含属性和修正标志的组合. 默认为 $WINHTTP_QUERY_RAW_HEADERS_CRLF.
$sName
	[可选参数] 头名称字符串. 默认为 $WINHTTP_HEADER_NAME_BY_INDEX.
$iIndex
	[可选参数] 枚举多个同名头的索引.
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回包含头的字符串.
失败:	返回空字符串并设置 @error:
	1 - DllCall 失败
@@End@@
