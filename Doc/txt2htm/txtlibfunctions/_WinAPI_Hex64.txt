###User Defined Function###
_WinAPI_Hex64

###Description###
Converts an integer value into a string of its hexadecimal representation.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_Hex64 ( $iValue [, $iLenght] )

###Parameters###
@@ParamTable@@
$iValue
	The expression to convert.
$iLenght
	[可选参数] The number of characters to be returned (up to 16) for integer. Characters are truncated from the
	left-hand side if length is too small.
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 1.
Failure	Empty string and sets the @error flag to non-zero.
@@End@@
