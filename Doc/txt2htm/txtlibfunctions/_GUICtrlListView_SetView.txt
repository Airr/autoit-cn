###User Defined Function###
_GUICtrlListView_SetView

###Description###
Sets the view of the control

###Syntax###
#Include <GuiListView.au3>
_GUICtrlListView_SetView($hWnd, $iView)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$iView
	View state for the control:
		0 - Details
		1 - Large Icon
		2 - List
		3 - Small Icon
		4 - Tile
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	True
失败:	False
@@End@@

###Remarks###
Minimum OS - Windows XP.
