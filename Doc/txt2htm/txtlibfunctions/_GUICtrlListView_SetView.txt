###User Defined Function###
_GUICtrlListView_SetView

###Description###
设置控件的视图

###Syntax###
#Include <GuiListView.au3>
_GUICtrlListView_SetView($hWnd, $iView)

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
$iView
	视图状态:
		0 - 报表视图
		1 - 大图标
		2 - 列表视图
		3 - 小图标
		4 - 平铺显示
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 True
失败:	返回 False
@@End@@

###Remarks###
 最低操作系统 - Windows XP.
