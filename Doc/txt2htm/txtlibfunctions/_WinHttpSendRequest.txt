###User Defined Function###
_WinHttpSendRequest

###Description###
发送指定请求到HTTP服务器.

###Syntax###
#Include <WinHttp.au3>
_WinHttpSendRequest($hRequest [, $sHeaders = Default [, $sOptional = Default [, $iTotalLength = Default [, $iContext = Default ]]]])

###Parameters###
@@ParamTable@@
$hRequest
	_WinHttpOpenRequest() 函数返回的句柄.
$sHeaders
	[可选参数] 附加到请求的附加头. 默认为 $WINHTTP_NO_ADDITIONAL_HEADERS.
$sOptional
	[可选参数] 发送请求头后需要发送的可选数据. 默认为 $WINHTTP_NO_REQUEST_DATA.
$iTotalLength
	[可选参数] 要发送的整个可选数据的长度,按字节计算. 默认为 0.
$iContext
	[可选参数] 要传递给请求句柄的应用程序定义的值,一般为一个回调函数. 默认为 0.
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 1.
失败:	返回 0并设置 @error:
	1 - DllCall 失败
@@End@@

###Remarks###
指定的可选数据($sOptional)将使用$iTotalLength定义的大小接收数据,而不是默认值.
