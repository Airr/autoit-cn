###User Defined Function###
_WinHttpSendRequest

###Description###
向HTTP服务器发送指定请求

###Syntax###
#Include <WinHttp.au3>
_WinHttpSendRequest($hRequest [, $sHeaders = Default [, $sOptional = Default [, $iTotalLength = Default [, $iContext = Default ]]]])

###Parameters###
@@ParamTable@@
$hRequest
	_WinHttpOpenRequest() 返回的句柄
$sHeaders
	[可选参数] 包含附加到请求的额外头部的字符串. 默认为 $WINHTTP_NO_ADDITIONAL_HEADERS.
$sOptional
	[可选参数] 包含任意可选的在请求的头部之后立即发送的数据的字符串. 默认为 $WINHTTP_NO_REQUEST_DATA.
$iTotalLength
	[可选参数] 包含发送的全部可选数据字节长度的未签名长整值. 默认为 0.
$iContext
	[可选参数] 表示写入模式的整数. 默认为0(写入ANSI字符串)
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回1
失败:	返回0并设置@error
	1 - 表示DllCall失败
@@End@@

###Remarks###
指定可选数据($sOptional)将使$iTotalLength接收默认大小的数据
