###User Defined Function###
_GUICtrlRichEdit_IsModified

###Description###
检索 RichEdit 控件的修改标志状态

###Syntax###
#Include <GuiRichEdit.au3>
_GUICtrlRichEdit_IsModified($hWnd)

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 True 或 False
失败:	设置@error
@error:	101 - $hWnd 参数值不是句柄
@@End@@

###Remarks###
 控件创建时,系统自动设置修改标志为 False.
 如果文本被用户或编程修改时,系统设置修改标志为 True
 调用 _GUICtrlRichEdit_SetModified 设置或清除该标志
