###User Defined Function###
_MemGlobalSize

###Description###
Retrieves the current size of the specified global memory object

###Syntax###
#Include <Memory.au3>
_MemGlobalSize($hMem)

###Parameters###
@@ParamTable@@
$hMem
	Handle to the global memory object
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	size of the specified global memory object, in bytes
Failure:	0
@@End@@

###Remarks###
None.
