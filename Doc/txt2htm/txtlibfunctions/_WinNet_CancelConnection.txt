###User Defined Function###
_WinNet_CancelConnection

###Description###
取消现有的网络连接

###Syntax###
#Include <WinNet.au3>
_WinNet_CancelConnection($sName [, $fForce = True])

###Parameters###
@@ParamTable@@
$sName
	指定重定向本地设备的名称,或要断开的远程网络资源的名称．
		当此参数指定一个重定向的本地设备,函数取消指定的设备重定向;
		如果参数指定远程网络资源,只有在没有设备连接到远程网络时才被取消.
$fForce
	[可选参数] 如果有打开的文件或连接工作,指定是否应该发生断线．
		如果此参数为　False,且存在打开的文件或工作,该函数失败.
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	True
失败:	False
@@End@@

###Remarks###
 这个函数仅提供与 Windows 16位版本兼容．
 其他基于 Windows 的应用程序应调用 WNet_CancelConnection2 函数．
