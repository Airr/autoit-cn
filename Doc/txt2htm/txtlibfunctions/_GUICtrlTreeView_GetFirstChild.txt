###User Defined Function###
_GUICtrlTreeView_GetFirstChild

###Description###
Retrieves the first child item of the specified item

###Syntax###
#Include <GuiTreeView.au3>
_GUICtrlTreeView_GetFirstChild($hWnd, $hItem)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$hItem
	Handle to the item
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	The handle of the first child item
Failure:	0
@@End@@

###Remarks###
None.
