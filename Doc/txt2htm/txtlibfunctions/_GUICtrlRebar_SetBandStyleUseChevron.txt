###User Defined Function###
_GUICtrlRebar_SetBandStyleUseChevron

###Description###
设置是否显示下拉按钮

###Syntax###
#Include <GuiRebar.au3>
_GUICtrlRebar_SetBandStyleUseChevron($hWnd, $iIndex[, $fEnabled = True])

###Parameters###
@@ParamTable@@
$hWnd
	伸缩条控件句柄
$iIndex
	带区的 0 基索引
$fEnabled
	[可选参数] 设为 True, 则是,设为 False 则不是
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 True
失败:	返回 False
@@End@@

###Remarks###
Version 5.80. 当带区小于 cxIdeal 时显示一个 (∧) 上尖号或下尖号 (∨)

_GUICtrlRebar_GetBandStyleUseChevron, _GUICtrlRebar_SetBandStyle
