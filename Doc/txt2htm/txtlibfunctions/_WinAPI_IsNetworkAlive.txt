###User Defined Function###
_WinAPI_IsNetworkAlive

###Description###
确定本地系统的网络连接状态,并确定网络连接的类型.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_IsNetworkAlive ( )

###Parameters###
None.

###ReturnValue###
@@ReturnTable@@
成功:	如果本地系统连接到网络, 则返回网络连接类型($NETWORK_ALIVE_*).
	返回 0 - 没有连接.
失败:	返回 0,并设置@error标志为非 0 值.
@@End@@

###Remarks###
检查 @error 标志之前, 总是先检查此函数的返回值.
如果 @error 不为 0, 函数失败,下面的值不适用(译注:原文未说明"下面的值"是什么值)
此函数仅适用于 TCP/IP 连接
