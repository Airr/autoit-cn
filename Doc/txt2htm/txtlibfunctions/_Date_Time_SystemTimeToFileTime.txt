###User Defined Function###
_Date_Time_SystemTimeToFileTime

###Description###
转换一个系统时间到文件时间格式

###Syntax###
#Include <Date.au3>
_Date_Time_SystemTimeToFileTime($pSystemTime)

###Parameters###
@@ParamTable@@
$pSystemTime
	Pointer to a $tagSYSTEMTIME structure to be converted.
@@End@@

###ReturnValue###
Returns $tagFILETIME array that contains the converted file time

###Remarks###
None.

###Related###
$tagFILETIME, $tagSYSTEMTIME
