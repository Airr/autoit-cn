###User Defined Function###
_SQLite_SQLiteExe

###Description###
在 SQLite3.exe 中执行命令

###Syntax###
#include <SQLite.au3>
_SQLite_SQLiteExe ( $sDatabaseFile, $sInput, ByRef $sOutput [, $sSQLiteExeFilename = "SQLite3.exe" ] )


###Parameters###
@@ParamTable@@
$sDatabaseFile
	数据库文件名
$sInput
	SQLite3.exe 命令
$sOutput
	SQLite3.exe 原输出
$sSQLiteExeFilename
	[可选参数] SQLite3.exe 路径
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 $SQLITE_OK
失败:	返回值可能违反 $SQLITE_* 常量
@error:	1 - 不能创建新的数据库
	2 - SQLite3.exe 未找到
	3 - SQL 错误 / 未完成 SQL
	4 - 无法打开输入文件
@@End@@

###Remarks###
欲了解更多有关 SQLite3.exe 命令信息,运行 SQLite.exe, 输入" .Help "并按下回车键.

SQLite3.exe 可在 SQLite 网站查找. 在 AutoIt3 \Extras\ SQLite 目录中有一个帮助文档.
