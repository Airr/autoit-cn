###User Defined Function###
_WinAPI_ComputeCrc32

###Description###
Calculates the CRC32 checksum of a block of memory.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_ComputeCrc32 ( $pMemory, $iLenght )

###Parameters###
@@ParamTable@@
$pMemory
	A pointer to the memory to calculate the checksum.
$iLenght
	The number of bytes to be calculated.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	The CRC32 checksum (DWORD).
Failure	0 and sets the @error flag to non-zero.
@@End@@

###Remarks###
None
