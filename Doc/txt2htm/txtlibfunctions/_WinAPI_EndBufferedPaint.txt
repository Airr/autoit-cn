###User Defined Function###
_WinAPI_EndBufferedPaint

###Description###
Completes a buffered paint operation and frees the associated buffered paint handle.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_EndBufferedPaint ( $hBP [, $fUpdate] )

###Parameters###
@@ParamTable@@
$hBP
	Handle of the buffered paint context.
$fUpdate
	[可选参数] Specifies whether copy the buffer to the target DC, valid values:
		TRUE    - The target DC will be updated from the buffer. (Default)
		FALSE   - The target DC remains unchanged.
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 1.
失败:	返回 0 并设置 @error 标志为非 0 值, @extended 标志可能包含一个系统错误代码.
@@End@@

###Remarks###
本函数需要 Windows Vista 或更高版本系统.
