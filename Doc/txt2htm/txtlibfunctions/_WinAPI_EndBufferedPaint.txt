###User Defined Function###
_WinAPI_EndBufferedPaint

###Description###
完成缓存绘制操作并释放句柄

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_EndBufferedPaint( $hBP [, $fUpdate] )

###Parameters###
@@ParamTable@@
$hBP
   缓存绘制场景句柄
$fUpdate
   指定是否将缓存复制到目标场景. 可用值:
    TRUE - 缓存将更新目标场景(默认)
    FALSE - 不更新目标场景
@@End@@

###ReturnValue###
@@ReturnTable@@
成功: 返回1
失败: 返回0并设置@error非0
@@End@@

###Remarks###
最低系统要求: Windows Vista
