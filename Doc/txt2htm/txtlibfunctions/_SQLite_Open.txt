###User Defined Function###
_SQLite_Open

###Description###
打开或创建 SQLite 数据库.

###Syntax###
#include <SQLite.au3>
_SQLite_Open ( [ $sDatabase_Filename = ":memory:" [,$iAccessMode [,$iEncoding ]]] )


###Parameters###
@@ParamTable@@
$sDatabase_Filename
	[可选参数] 数据库文件名,默认打开内存中的数据库.
$iAccessMode
	[可选参数] 访问模式标志. 默认为 $SQLITE_OPEN_READWRITE + $SQLITE_OPEN_CREATE 
$iEncoding
	[可选参数] 编码模式标志,只用于创建时间.默认为 $SQLITE_ENCODING_UTF8
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	Returns 数据库句柄.
失败:	返回 0.@错误:
	-1 - SQLite 报告一个错误 (检查 @extended 的值)@extended:
	值可能违反 $SQLITE_* 常数
@@End@@

###Remarks###
 没有必要存储数据库句柄.
 函数使用的句柄是上次使用默认打开的句柄.
 要创建一个数据库中UFT16编码,使用 $iEncoding=$SQLITE_ENCODING_UTF16.
