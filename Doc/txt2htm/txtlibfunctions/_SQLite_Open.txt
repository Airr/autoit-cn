###用户自定义函数###
_SQLite_Open

###描述###
打开/创建SQLite数据库

###语法###
#Include <SQLite.au3>
_SQLite_Open ( [ $sDatabase_Filename = ":memory:" [,$iAccessMode [,$iEncoding ]]] )

###参数###
@@ParamTable@@
 $sDatabase_Filename  [可选] 数据库名, 默认打开内存数据库.
 $iAccessMode  [可选] 操作模式标记. 默认为$SQLITE_OPEN_READWRITE + $SQLITE_OPEN_CREATE
 $iEncoding  [可选] 编码模式标记, 仅创建时使用. 默认为$SQLITE_ENCODING_UTF8
@@End@@

###返回值###
@@ReturnTable@@
成功: 返回数据库句柄.
失败: 返回0
@extended: 可与$SQLITE_* 常量比较的值
@error: 1 - 调用SQLite API 'sqlite3_open_V2'错误
    -1 - SQLite报错(检查@extended的值)
@@End@@

###备注###
数据库句柄不用保存.
使用句柄的函数默认使用最近打开的.
使用UFT16编码创建数据库只能以$iEncoding=$SQLITE_ENCODING_UTF16创建.
