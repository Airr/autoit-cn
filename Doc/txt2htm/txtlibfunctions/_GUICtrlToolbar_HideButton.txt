###User Defined Function###
_GUICtrlToolbar_HideButton

###Description###
隐藏或显示指定的按钮

###Syntax###
#Include <GuiToolBar.au3>
_GUICtrlToolbar_HideButton($hWnd, $iCommandID[, $fHide = True])

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
$iCommandID
	按钮命令 ID
$fHide
	[可选参数] 状态标志:
		True - 按钮隐藏
		False - 按钮可见
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	True
失败:	False
@@End@@

###Remarks###
None.
