###User Defined Function###
_SQLite_Query

###Description###
准备 SQLite 查询

###Syntax###
#include <SQLite.au3>
_SQLite_Query ( $hDB, $sSQL, ByRef $hQuery )


###Parameters###
@@ParamTable@@
$hDB
	打开的数据库,如为 -1, 则使用最后打开的数据库
$sSQL
	要执行的语句
$hQuery
	传递出的查询句柄
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 $SQLITE_OK
失败:	返回值可能违反 $SQLITE_* 常量@错误:
	-1 - SQLite 报告错误 (检查返回值)
	1 - 错误调用 SQLite API 'sqlite3_prepare16_v2'
	2 - 调用被安全模式阻止
@@End@@
