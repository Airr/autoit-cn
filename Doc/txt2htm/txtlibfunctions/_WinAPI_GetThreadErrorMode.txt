###User Defined Function###
_WinAPI_GetThreadErrorMode

###Description###
Retrieves the error mode for the calling thread.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_GetThreadErrorMode ( )

###Parameters###
@@ParamTable@@
None.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	The process error mode ($SEM_*).
失败:	返回 0 并设置 @error 标志为非 0 值.
@@End@@

###Remarks###
本函数需要 Windows 7 或以上版本系统.
