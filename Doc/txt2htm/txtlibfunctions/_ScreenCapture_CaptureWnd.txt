###User Defined Function###
_ScreenCapture_CaptureWnd

###Description###
Captures a screen shot of a specified window

###Syntax###
#Include <ScreenCapture.au3>
_ScreenCapture_CaptureWnd($sFileName, $hWnd [, $iLeft = 0 [, $iTop = 0 [, $iRight = -1 [, $iBottom = -1 [, $fCursor = True]]]]])

###Parameters###
@@ParamTable@@
$sFileName
	Full path and extension of the image file
$hWnd
	Handle to the window to be captured
$iLeft
	[optional] X coordinate of the upper left corner of the client rectangle
$iTop
	[optional] Y coordinate of the upper left corner of the client rectangle
$iRight
	[optional] X coordinate of the lower right corner of the rectangle
$iBottom
	[optional] Y coordinate of the lower right corner of the rectangle
$fCursor
	[optional] If True the cursor will be captured with the image
@@End@@

###ReturnValue###
None.

###Remarks###
If FileName is not blank this function will capture the screen and save it to file. If FileName is blank, this
function will capture the screen and return a HBITMAP handle to the bitmap image. In this case, after you are
finished with the bitmap you must call _WinAPI_DeleteObject to delete the bitmap handle. All coordinates are in
client coordinate mode.

Requires GDI+: GDI+ requires a redistributable for applications that
run on the Microsoft Windows NT 4.0 SP6, Windows 2000, Windows 98, and Windows Me operating systems.
