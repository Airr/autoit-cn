###User Defined Function###
_GUICtrlButton_Destroy

###Description###
删除按钮控件

###Syntax###
#Include <GuiButton.au3>
_GUICtrlButton_Destroy(ByRef $hWnd)

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
@@End@@

###ReturnValue###
成功: True, 句柄设置为 0
失败: False

###Remarks###
Restricted to only be used on Edit created with _GUICtrlButton_Create
