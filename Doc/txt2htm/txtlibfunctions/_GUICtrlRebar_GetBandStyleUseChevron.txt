###User Defined Function###
_GUICtrlRebar_GetBandStyleUseChevron

###Description###
Determine if flag is set

###Syntax###
#Include <GuiRebar.au3>
_GUICtrlRebar_GetBandStyleUseChevron($hWnd, $iIndex)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to rebar control
$iIndex
	Zero-based index of the band
@@End@@

###ReturnValue###
True: Flag is set (Display drop-down button)
False: Flag not set

###Remarks###
None.
