###User Defined Function###
_GUICtrlTreeView_SetDropTarget

###Description###
设置项目为拖放目标

###Syntax###
#include <GuiTreeView.au3>
_GUICtrlTreeView_SetDropTarget($hWnd, $hItem [, $fFlag = True])

###Parameters###
@@ParamTable@@
$hWnd
	控件ID/句柄
$hItem
	项目句柄
$fFlag
	[可选参数] 设置标志:
		 True - 项目为拖放目标
		False - 项目不是拖放目标
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	True
失败:	False
@@End@@

###Remarks###
None.
