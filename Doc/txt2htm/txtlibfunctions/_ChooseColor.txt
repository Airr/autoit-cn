###User Defined Function###
_ChooseColor

###Description###
创建一个颜色对话框使用户可以选择颜色

###Syntax###
#Include <Misc.au3>
_ChooseColor([$iReturnType = 0 [, $iColorRef = 0 [, $iRefType = 0 [, $hWndOwnder = 0]]]])

###Parameters###
@@ParamTable@@
$iReturnType
	[可选参数] 定义返回类型, 有效值:
		0 - COLORREF RGB颜色
		1 - BGR 16位值
		2 - RGB 16位值
$iColorRef
	[可选参数] 默认选择的颜色
$iRefType
	[可选参数] 传递给 $iColorRef 的类型, 有效值为:
		0 - COLORREF RGB颜色
		1 - BGR 16位值
		2 - RGB 16位值
$hWndOwnder
	[可选参数] 本对话框的父窗口句柄.
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回颜色值
失败:	返回 -1
@@End@@
