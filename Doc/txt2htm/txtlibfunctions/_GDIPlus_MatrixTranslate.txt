###User Defined Function###
_GDIPlus_MatrixTranslate

###Description###
移动矩阵

###Syntax###
#Include <GDIPlus.au3>
_GDIPlus_MatrixTranslate($hMatrix, $fOffsetX, $fOffsetY[, $bAppend = False])

###Parameters###
@@ParamTable@@
$hMatrix
	返回矩阵对象句柄
$fOffsetX
	沿 x 轴添加像素的数量
$fOffsetY
	沿 Y 轴添加像素的数量
$bAppend
	[可选参数] 指定移动方向:
		True - 向左边平移矩阵
		False - 向右边平移矩阵
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 True
失败:	返回 False
@@End@@
