###User Defined Function###
_IETableGetCollection

###Description###
返回用以代表文档中所有表或按索引的单一表的对象变量集合.

###Syntax###
#include <IE.au3> 
_IETableGetCollection ( ByRef $o_object [, $i_index = -1] )

###Parameters###
@@ParamTable@@
$o_object
	InternetExplorer.Application的窗体或框架对象变量
$i_index
	[可选参数]: 指定返回一个集合或索引的实例
	0或正整数 返回索引实例
	-1 = (默认)返回集合
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回文件中所有表的对象集合, @EXTENDED = 表数量
失败:	返回 0 并且设置 @ERROR
@Error:	0 ($_IEStatus_Success) = 无错误
	3 ($_IEStatus_InvalidDataType) = 无效数据类型
	5 ($_IEStatus_InvalidValue) = 无效值
	7 ($_IEStatus_NoMatch) = 无匹配
@Extended:	包含无效参数数量
@@End@@

None.
