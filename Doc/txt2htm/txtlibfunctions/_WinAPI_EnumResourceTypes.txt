###User Defined Function###
_WinAPI_EnumResourceTypes

###Description###
Enumerates resource types within a binary module.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_EnumResourceTypes ( $sModule )

###Parameters###
@@ParamTable@@
$sModule
	Handle to a module to search.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	The array of the types of the resources for the specified module.
Failure	0 and sets the @error flag to non-zero.
@@End@@
