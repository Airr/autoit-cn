###用户自定义函数###
_WinHttpSimpleSendRequest

###描述###
以简单形式发送请求且不记录数据

###语法###
#Include <WinHTTP.au3>
_WinHttpSimpleSendRequest( $hConnect, $sType, $sPath [, $sReferrer = Default [, $sData = Default [, $sHeader = Default ]]] )

###参数###
@@ParamTable@@
$hConnect 
   _WinHttpConnect()返回的句柄.
$sType 
   [可选] GET(默认)或POST.
$sPath 
   [可选] 请求路径(默认为空字符串表示服务器上的默认页).
$sReferrer 
   [可选] 引用(默认为$WINHTTP_NO_REFERER)
$sData 
   [可选] POST数据(默认为$WINHTTP_NO_REQUEST_DATA)
$sHeader 
   [可选] 额外头部(默认为$WINHTTP_NO_ADDITIONAL_HEADERS).
@@End@@

###返回值###
@@ReturnTable@@
成功: 返回以前定义的状态回调函数的指针或当在没有定义状态回调函数时返回空值, 并设置@error为0.
失败: 返回0并设置@error且当其为1时表示DllCall失败.
@@End@@
