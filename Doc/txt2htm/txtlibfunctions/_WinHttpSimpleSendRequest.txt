###User Defined Function###
_WinHttpSimpleSendRequest

###Description###
以简单形式发送请求且不记录数据

###Syntax###
#Include <WinHTTP.au3>
_WinHttpSetStatusCallback( $hConnect, $sType, $sPath [, $sReferrer = Default [, $sData = Default [, $sHeader = Default ]]] )

###Parameters###
@@ParamTable@@
$hConnect
	_WinHttpConnect()返回的句柄.
$sType
	[可选参数] GET(默认)或POST.
$sPath
	[可选参数] 请求路径(默认为空字符串表示服务器上的默认页).
$sReferrer
	[可选参数] 引用(默认为$WINHTTP_NO_REFERER)
$sData
	[可选参数] POST数据(默认为$WINHTTP_NO_REQUEST_DATA)
$sHeader
	[可选参数] 额外头部(默认为$WINHTTP_NO_ADDITIONAL_HEADERS).
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回以前定义的状态回调函数的指针或当在没有定义状态回调函数时返回空值, 设置@error 0.
失败:	返回0
@error:	且当其为1时表示DllCall失败.
@@End@@
