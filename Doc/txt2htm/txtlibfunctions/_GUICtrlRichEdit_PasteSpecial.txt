###User Defined Function###
_GUICtrlRichEdit_PasteSpecial

###Description###
从剪贴板粘贴 RTF 或 RTF 格式的对象(专用)

###Syntax###
#Include <GuiRichEdit.au3>
_GUICtrlRichEdit_PasteSpecial($hWnd, $fAndObjects = True)

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
$fAndObjects
	粘贴对象以及 RTF - True (默认) 或 False
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 True
失败:	返回 False设置@error
@error:	101 - $hWnd 参数值不是句柄
	102 - $fAndObjects 既不是 True, 也不是 False
@@End@@

###Remarks###
None.
