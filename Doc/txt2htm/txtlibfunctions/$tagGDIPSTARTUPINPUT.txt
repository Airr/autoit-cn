###Structure Name###
$tagGDIPSTARTUPINPUT

###Description###
$tagGDIPSTARTUPINPUT 结构

###Syntax###
Global Const $tagGDIPSTARTUPINPUT = "int Version;ptr Callback;int NoThread;int NoCodecs"

###Fields###
@@ParamTable@@
Version
	指定 Microsoft Windows GDI+ 版本.
Callback
	在调试中，为断言和警告调用 GDI + 的回调函数指针
NoThread
	指定是否限制 GDI+ 后台线程，参数值为布尔值
	如果设置为 True, GdiplusStartup 返回一个指向钩子函数和脱钩函数的指针,
	你必须适当地调用这些函数，以取代后台线程.
	如果不想对钩子函数和脱钩函数进行回应, 设置该成员为  False.
NoCodecs
	指定 GDI+ 是否抑制外部图像编解码器，参数值为布尔值.
	GDI+ 的1.0版本不支持外部图像编解码器,所以这个参数被忽略.
@@End@@

###Remarks###
None.
