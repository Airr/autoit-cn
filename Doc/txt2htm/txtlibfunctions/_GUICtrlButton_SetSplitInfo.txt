###User Defined Function###
_GUICtrlButton_SetSplitInfo

###Description###
获取分割按钮控件的信息

###Syntax###
#include <GuiButton.au3>
_GUICtrlButton_SetSplitInfo($hWnd [, $himlGlyph = -1 [, $iSplitStyle = $BCSS_ALIGNLEFT [, $iWidth = 0 [, $iHeight = 0]]]])

###Parameters###
@@ParamTable@@
$hWnd
	控件的控件ID/句柄
$himlGlyph
	[可选参数] 图像列表句柄
$iSplitStyle
	[可选参数] 拆分按钮样式. 可以是下列标志之一或多个标志:.
		$BCSS_ALIGNLEFT - 水平左对齐图像或拆分符号
		$BCSS_IMAGE - 用 ico 图标作为拆分符号
		$BCSS_NOSPLIT - 没有拆分符号
		$BCSS_STRETCH - 伸展拆分符号, 但需保持宽高比
$iWidth
	[可选参数] 拆分符号宽度
$iHeight
	[可选参数] 拆分符号高度
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 True
失败:	返回 False
@@End@@

###Remarks###
此函数仅适用于具有 $BS_SPLITBUTTON 与 $BS_DEFSPLITBUTTON 样式的按钮.

最低操作系统: Windows Vista.
