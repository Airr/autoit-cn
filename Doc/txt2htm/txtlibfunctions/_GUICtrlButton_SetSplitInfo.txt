###User Defined Function###
_GUICtrlButton_SetSplitInfo

###Description###
获取分割按钮控件的信息

###Syntax###
#Include <GuiButton.au3>
_GUICtrlButton_SetSplitInfo($hWnd[, $himlGlyph = -1[, $iSplitStyle = $BCSS_ALIGNLEFT[, $iWidth = 0[, $iHeight = 0]]]])

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
$himlGlyph
	[可选参数] 图像列表句柄
$iSplitStyle
	[可选参数] 分割按钮样式. 可以是下列标志之一或多项.
		  $BCSS_ALIGNLEFT - Align the image or glyph horizontally with the left margin
		  $BCSS_IMAGE - Draw an icon image as the glyph
		  $BCSS_NOSPLIT - 不分割
		  $BCSS_STRETCH - Stretch glyph, but try to retain aspect ratio
$iWidth
	[可选参数] Width of the glyph
$iHeight
	[可选参数] Height of the glyph
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	True
失败:	False
@@End@@

###Remarks###
这个函数只能作用于 $BS_SPLITBUTTON and $BS_DEFSPLITBUTTON 样式的按钮

最低操作系统: Windows Vista
