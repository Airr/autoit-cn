###User Defined Function###
_GUIToolTip_AddTool

###Description###
注册工具提示控件

###Syntax###
#Include <GuiToolTip.au3>
_GUIToolTip_AddTool($hTool, $hWnd, $sText[, $iID = 0[, $iLeft = 0[, $iTop = 0[, $iRight = 0[, $iBottom = 0[, $iFlags = 8[, $iParam = 0]]]]]]])

###Parameters###
@@ParamTable@@
$hTool
	工具提示句柄
$hWnd
	工具提示控件的窗口句柄
$sText
	工具提示控件文本
$iID
	[可选参数] 工具提示控件的标识符或窗口句柄
$iLeft
	[可选参数] 提示框矩形左上角 X 坐标
$iTop
	[可选参数] 提示框矩形左上角 Y 坐标
$iRight
	[可选参数] 提示框矩形右下角 X 坐标
$iBottom
	[可选参数] 提示框矩形右下角 Y 坐标
$iFlags
	[可选参数] 工具提示显示标志:
		1 - 指定 $iID 参数值是窗口句柄, 而不是工具提示控件的 ID
		2 - $iID 指定的提示控件在窗口下方居中
		4 - 指定文本反方向显示
		8 - 指定提示控件是窗口的子类
		16 - 提示工具位置对应有提示信息的控件旁边
		32 - TTM_TRACKPOSITION 提供同一位置的窗口坐标
		64 - 控件向父窗口发送鼠标信息
		128 - 指定解析链接的提示文本
$iParam
	[可选参数] 应用程序定义的工具关联值
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	True
失败:	False
@@End@@

None.
