###User Defined Function###
_GUICtrlMenu_RemoveMenu

###Description###
从指定菜单删除菜单项或分离子菜单

###Syntax###
#Include <GuiMenu.au3>
_GUICtrlMenu_RemoveMenu($hMenu, $iItem[, $fByPos = True])

###Parameters###
@@ParamTable@@
$hMenu
	进行更改的菜单句柄
$iItem
	菜单项的标识符或位置,由 $fByPos 参数决定.
$fByPos
	[可选参数] 菜单识别标志:
		True - $iItem 为项目的 0 基位置
		False - $iItem 为菜单项的标识符
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 True
失败:	返回 False
@@End@@

###Remarks###
 如果菜单项打开一个下拉菜单或子菜单,本函数不破坏菜单或句柄,允许菜单被重用.
 在函数调用前,应该用 _GUICtrlMenu_GetItemSubMenu 函数检索下拉式菜单或子菜单句柄.
