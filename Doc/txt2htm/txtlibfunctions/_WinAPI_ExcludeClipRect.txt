###Function###
_WinAPI_ExcludeClipRect

###Description###
Creates a new clipping region that consists of the existing clipping region minus the specified rectangle.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_ExcludeClipRect ( $hDC, $tRECT )

###Parameters###
@@ParamTable@@
$hDC
	Handle to the device context.
$tRECT
	$tagRECT structure that contains the logical coordinates of the specified rectangle.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	The value that specifies the new clipping region's complexity&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;it can be one of the following values.
Failure	0 and sets the @error flag to non-zero.
@@End@@
