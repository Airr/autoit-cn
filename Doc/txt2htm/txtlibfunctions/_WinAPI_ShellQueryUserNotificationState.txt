###User Defined Function###
_WinAPI_ShellQueryUserNotificationState

###Description###
检查当前用户的计算机状态.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_ShellQueryUserNotificationState ( )

###Parameters###
@@ParamTable@@
None.
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回目前计算机的状态 (QUNS_*).
失败:	返回 0,设置@error:	
标志为非 0 值, @extended 标志可能包含一个系统错误代码.
@@End@@

###Remarks###
本函数需要 Windows Vista 或更高版本系统.
