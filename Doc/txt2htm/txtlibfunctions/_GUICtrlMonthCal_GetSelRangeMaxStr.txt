###User Defined Function###
_GUICtrlMonthCal_GetSelRangeMaxStr

###Description###
Retrieves the upper date range currently selected by the user in string form

###Syntax###
#Include <GuiMonthCal.au3>
_GUICtrlMonthCal_GetSelRangeMaxStr($hWnd[, $sFormat = "%02d/%02d/%04d"])

###Parameters###
@@ParamTable@@
$hWnd
	Handle to control
$sFormat
	[optional] StringFormat string used to format the date
@@End@@

###ReturnValue###
Success: Date in the specified format

###Remarks###
None.
