###User Defined Function###
_WinAPI_CreateDirectory

###Description###
Creates a new directory.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_CreateDirectory ( $sDir [, $tSecurity] )

###Parameters###
@@ParamTable@@
$sDir
	The path of the directory to be created.
$tSecurity
	[可选参数] $tagSECURITY_ATTRIBUTES structure that specifies a security descriptor for the new directory.
	If this parameter is 0, the directory gets a default security descriptor. The ACL in the default
	security descriptor for a directory are inherited from its parent directory.
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 1.
失败:	返回 0 并设置 @error 标志为非 0 值.
@@End@@
