###Function###
_WinAPI_GetProcAddress

###Description###
Retrieves the address of a function from the specified dynamic-link library (DLL).

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_GetProcAddress ( $hModule, $sProc )

###Parameters###
@@ParamTable@@
$hModule
	Handle to the DLL module that contains the function.
$sProc
	The function name.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	The address of the function.
Failure	0 and sets the @error flag to non-zero.
@@End@@

###Remarks###
None
