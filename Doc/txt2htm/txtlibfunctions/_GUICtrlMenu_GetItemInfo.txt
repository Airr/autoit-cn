###User Defined Function###
_GUICtrlMenu_GetItemInfo

###Description###
检索菜单项的信息

###Syntax###
#Include <GuiMenu.au3>
_GUICtrlMenu_GetItemInfo($hMenu, $iItem[, $fByPos = True])

###Parameters###
@@ParamTable@@
$hMenu
	菜单句柄
$iItem
	菜单项的标识符或位置,由 $fByPos 参数决定.
$fByPos
	[可选参数] 菜单识别标志:
		True - $iItem 为项目的 0 基位置
		False - $iItem 为菜单项的标识符
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 $tagMENUITEMINFO 结构
失败:	设置@error:	
@@End@@

###Remarks###
None.
