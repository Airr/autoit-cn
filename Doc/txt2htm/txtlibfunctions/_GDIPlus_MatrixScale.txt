###User Defined Function###
_GDIPlus_MatrixScale

###Description###
缩放矩阵

###Syntax###
#Include <GDIPlus.au3>
_GDIPlus_MatrixScale($hMatrix, $fScaleX, $fScaleY[, $bOrder = False])

###Parameters###
@@ParamTable@@
$hMatrix
	返回矩阵对象句柄
$fScaleX
	X 轴缩放比例
$fScaleY
	Y 轴缩放比例
$bOrder
	[可选参数] 指定缩放方向:
		True - 缩放矩阵在左边
		False - 缩放矩阵在右边
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 True
失败:	返回 False
@@End@@
