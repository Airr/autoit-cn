###User Defined Function###
_GUICtrlMonthCal_GetMinReqRectArray

###Description###
检索显示完整月历所需的最低尺寸

###Syntax###
#Include <GuiMonthCal.au3>
_GUICtrlMonthCal_GetMinReqRectArray($hWnd)

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回如下格式的数组:
	[0] - 元素数量 
	[1] - 月历左上角 X 坐标 
	[2] - 月历左上角 Y 坐标 
	[3] - 月历右下角 X 坐标,如[1]为 0 , 则此值可看作月历的宽度 
	[4] - 月历右下角 Y 坐标,如[2]为 0 , 则此值可看作月历的高度
失败:	返回 FalseSuccess:True, Handle is set to 0 (	成功:	返回 True, 句柄设置为 0 ) Failure:False (
失败:	返回 False )
@@End@@

###Remarks###
 月日历控件所需的窗口大小取决于当前选定的字体,
 控件样式, 运行的系统和区域设置.
