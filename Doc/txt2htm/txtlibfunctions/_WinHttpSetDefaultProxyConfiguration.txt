###User Defined Function###
_WinHttpSetDefaultProxyConfiguration

###Description###
Sets the default WinHttp proxy configuration.

###Syntax###
#Include <WinHttp.au3>
_WinHttpSetDefaultProxyConfiguration($iAccessType [, $sProxy = "" [, $sProxyBypass = ""])

###Parameters###
@@ParamTable@@
$iAccessType
	Integer. Access type.
$sProxy
	[可选参数] String. Proxy server list.
$sProxyBypass
	[可选参数] String. Proxy bypass list.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	Returns 1.
Failure	Returns 0 and sets @error:
	1 - DllCall failed
@@End@@
