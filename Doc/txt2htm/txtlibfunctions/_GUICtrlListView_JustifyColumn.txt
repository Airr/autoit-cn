###User Defined Function###
_GUICtrlListView_JustifyColumn

###Description###
设置控件列对齐

###Syntax###
#Include <GuiListView.au3>
_GUICtrlListView_JustifyColumn($hWnd, $iIndex[, $iAlign = -1])

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
$iIndex
	列的 0 基索引
$iAlign
	[可选参数] 列标题和列项目文本对齐方式:
		0 - 文本左对齐
		1 - 文本右对齐
		2 - 文本居中
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 True
失败:	返回 False
@@End@@

###Remarks###
None.
