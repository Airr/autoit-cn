###Structure Name###
$tagNMLVDISPINFO

###Description###
包含 $LVN_GETDISPINFO 或 $LVN_SETDISPINFO 通知消息

###Syntax###
Global Const $tagNMLVDISPINFO = $tagNMHDR & ";int Mask;int Item;int SubItem;int State;int StateMask;" & _
		"ptr Text;int TextMax;int Image;int Param;int Indent;int GroupID;int Columns;ptr pColumns"

###Fields###
@@ParamTable@@
$tagNMHDR
	包含通知消息的信息
Mask
	Set of flags that specify which members of this structure contain data to be set or which members
	are being requested. 该成员可设置为一或多个下列值:
		$LVIF_COLUMNS - Columns 成员有效
		$LVIF_DI_SETITEM - 操作系统保存请求的列表箱信息
		$LVIF_GROUPID - GroupID 成员有效
		$LVIF_IMAGE - Image 成员有效
		$LVIF_INDENT - Indent 成员有效
		$LVIF_NORECOMPUTE - 控件将不会产生 LVN_GETDISPINFO 文本信息检索
		$LVIF_PARAM - Param 成员有效
		$LVIF_STATE - State 成员有效
		$LVIF_TEXT - Text 成员有效
Item
	项目的 0 基索引
SubItem
	子项目的 1 基索引
State
	项目的状态, 状态图像, 及覆盖图像
StateMask
	指定将被获取或修改 state 成员的值
Text
	项目文本字符串
TextMax
	缓冲区中包括结束符字符串文本的字节数量
Image
	项目图标在图像列表中的索引
Param
	项目的具体值
Indent
	项目数缩进的图像宽度数
GroupID
	项目平铺视图组的标识符
Columns
	显示项目的平铺视图中列的数量
pColumns
	列索引数组的指针
@@End@@

###Remarks###
None.
