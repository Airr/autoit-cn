###User Defined Function###
_WinAPI_DwmIsCompositionEnabled

###Description###
确定是否启用桌面窗口管理器(DWM)的组成.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_DwmIsCompositionEnabled ( )

###Parameters###
@@ParamTable@@
None.
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 1 - 启用 DWM 的组成
	0 - 不启用.
失败:	返回 0,设置 @error 标志为非 0 值, @extended 标志可能包含一个系统错误代码.
@@End@@

###Remarks###
本函数需要 Windows Vista 或更高版本系统.
