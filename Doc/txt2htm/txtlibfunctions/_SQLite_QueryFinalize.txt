###User Defined Function###
_SQLite_QueryFinalize

###Description###
结束一次基于_SQLite_Query()的查询. 查询中断.

###Syntax###
#Include <SQLite.au3>
_SQLite_QueryFinalize ( $hQuery )

###Parameters###
@@ParamTable@@
 $hQuery  由SQLite_Query()生成的查询句柄
@@End@@

###ReturnValue###
@@ReturnTable@@
成功: 返回$SQLITE_OK
失败: 返回可与$SQLITE_* 常量比较的值
@error: -1 - SQLite报错(检查返回值)
      1 - 调用SQLite API 'sqlite3_finalize'错误
      2 - 安全模式阻止调用
@@End@@

###Remarks###
_SQLite_QueryFinalize函数被调用以删除准备好的以前通过调用_SQLite_Query获得的SQL语句. 如果语句成功执行, 或者根本未被执行的话, 那么返回SQLITE_OK. 如果语句执行失败则返回错误代码.
所有准备好的语句必须在调用_SQLite_Close之前完成否则_SQLite_Close将因为SQLITE_BUSY返回码失败.
