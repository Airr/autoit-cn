###User Defined Function###
_GUICtrlListView_RemoveGroup

###Description###
Removes a group from the control

###Syntax###
#Include <GuiListView.au3>
_GUICtrlListView_RemoveGroup($hWnd, $iGroupID)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$iGroupID
	ID that specifies the group to remove
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	The index of the group
Failure:	-1
@@End@@

###Remarks###
Minimum OS - Windows XP.
