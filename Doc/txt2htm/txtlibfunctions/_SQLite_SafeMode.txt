###User Defined Function###
_SQLite_SafeMode

###Description###
禁用或启用安全模式

###Syntax###
#include <SQLite.au3>
_SQLite_SafeMode ( $fSafeModeState )


###Parameters###
@@ParamTable@@
$fSafeModeState
	True 或 False 用于启用或禁用安全模式
@@End@@

###ReturnValue###
@@ReturnTable@@
	返回 $SQLITE_OK
@@End@@

###Remarks###
 安全模式防止无效查询或数据库句柄造成的 SQLite 崩溃.
 默认为启用.
