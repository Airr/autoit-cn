###User Defined Function###
_ExcelReadCell

###Description###
从一个指定Excel对象的活动工作表读取(单元格)信息.

###Syntax###
#Include <Excel.au3>
_ExcelReadCell($oExcel, $sRangeOrRow [, $iColumn = 1])

###Parameters###
@@ParamTable@@
$oExcel
	Excel object opened by a preceding call to _ExcelBookOpen() or _ExcelBookNew()
$sRangeOrRow
	Either an A1 range, or an integer row number to read from if using R1C1
$iColumn
	[optional] The column to read from if using R1C1 (default = 1)
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回指定单元格中的数据
失败:	返回 0, 并设置 @error:
@error:	0 _ 无错误
	1 - 指定的对象不存在
	2 - 参数不正确
@extended:	0 - 行超出有效范围
	1 - 列超出有效范围
@@End@@

###Remarks###
调用函数每次只读取一个单元格
如果指定范围含有多个单元格, 只返回左上角单元格的内容
