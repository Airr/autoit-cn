###User Defined Function###
_ScreenCapture_SaveImage

###Description###
保存一个图像到文件

###Syntax###
#Include <ScreenCapture.au3>
_ScreenCapture_SaveImage($sFileName, $hBitmap [, $fFreeBmp = True])

###Parameters###
@@ParamTable@@
$sFileName
	保存位图的文件扩展名和完整路径
$hBitmap
	HBITMAP(位图)句柄
$fFreeBmp
	[可选参数] 如为 True, 成功保存侯, hBitmap 句柄将被释放
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	True
失败:	False
@@End@@

###Remarks###
 函数保存位图文件,将其转换为指定文件扩展名的图像格式.
 对于 Windows XP, 有效的扩展为 BMP, GIF, JPEG, PNG 与 TIF.
 需要 GDI+: GDI+ 需要一个可再发行的应用程序
 运行于微软 Windows NT 4.0 SP6, Windows 2000, Windows 98, 与 Windows Me 操作系统.
