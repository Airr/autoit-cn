###User Defined Function###
_Min

###Description###
返回两数中较小的值

###Syntax###
#Include <Math.au3>
_Min ( $nNum1, $nNum2 )

###Parameters###
@@ParamTable@@
 $nNum1  第一个数.
 $nNum2  第二个数.
参数 成功: 返回较小的一个数.
失败: 返回0并设置@error为:
   0 = 无错误.
   1 = $nNum1不是数字
   2 = $nNum2不是数字
@@End@@
