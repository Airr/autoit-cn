###User Defined Function###
_GUICtrlHeader_SetItemFormat

###Description###
设置项目格式

###Syntax###
#Include <GuiHeader.au3>
_GUICtrlHeader_SetItemFormat($hWnd, $iIndex, $iFormat)

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
$iIndex
	项目的 0 基索引
$iFormat
	结合下面的格式标识符:
		$HDF_CENTER - 项目内容居中对齐
		$HDF_LEFT - 项目内容左对齐
		$HDF_RIGHT - 项目内容右对齐
		$HDF_BITMAP - 项目显示位图
		$HDF_BITMAP_ON_RIGHT - 位图显示在文字右侧
		$HDF_OWNERDRAW - 标头控件的所有者提请项目
		$HDF_STRING - 项目显示为字符串
		$HDF_IMAGE - 显示来自图像列表的图像
		$HDF_RTLREADING - 从父窗口中反方向读取文本.
		$HDF_SORTDOWN - 在项目上绘制向下箭头( Windows XP 或更高版本)
		$HDF_SORTUP - 在项目上绘制向上箭头( Windows XP 或更高版本)
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 True
失败:	返回 False
@@End@@

###Remarks###
None.
