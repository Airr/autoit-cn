###User Defined Function###
_WinAPI_EqualRgn

###Description###
检查两个指定区域是否相同.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_EqualRgn ( $hRgn1, $hRgn2 )

###Parameters###
@@ParamTable@@
$hRgn1
	区域句柄.
$hRgn2
	区域句柄.
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 1 - 两个区域相同.
	返回 0 - 不相同.
失败:	返回 0,并设置@error标志为非 0 值.
@@End@@
