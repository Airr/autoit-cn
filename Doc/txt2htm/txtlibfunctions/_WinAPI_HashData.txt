###User Defined Function###
_WinAPI_HashData

###Description###
Hashes a memory block.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_HashData ( $pMemory, $iSize [, $iLength] )

###Parameters###
@@ParamTable@@
$pMemory
	A pointer to a memory block containing data to hash.
$iSize
	The size of the memory block, in bytes.
$iLength
	[可选参数] The length of the hash data, in bytes. It should be no larger than 256, otherwise, the function fails.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	The hash data in binary form.
Failure	0 and sets the @error flag to non-zero, @extended flag may contain the system error code.
@@End@@
