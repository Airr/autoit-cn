###User Defined Function###
_IEErrorNotify

###Description###
设定ie.au3是否自动报告提醒和错误(到控制台).

###Syntax###
#include <IE.au3> 
_IEErrorNotify ( [$f_notify = -1] )

###Parameters###
@@ParamTable@@
$f_notify
	[可选参数]: 指定是否开启通知或关闭
	-1 = (默认)返回当前设置
	真 = 开启
	假 = 关闭
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	如果$f_notify = -1, 返回当前的通知设置, 或者返回1
失败:	返回 0
@@End@@


###Remarks###
IE.au3默认将诊断信息, 警告和错误写入控制台. 该信息可在使用SciTe编辑器运行脚本时被看到.

该函数允许你打开或者关闭这一功能.
