###User Defined Function###
_GUICtrlButton_SetTextMargin

###Description###
设置文本距按钮四边的边距

###Syntax###
#Include <GuiButton.au3>
_GUICtrlButton_SetTextMargin( $hWnd[, $iLeft = 1[, $iTop = 1[, $iRight = 1[, $iBottom = 1]]]] )

###Parameters###
@@ParamTable@@
 $hWnd  控件句柄
 $iLeft  文本的左边距
 $iTop  文本的上边距
 $iRight  文本的右边距
 $iBottom  文本的底边距
@@End@@

###ReturnValue###
@@ReturnTable@@
成功: 真
失败: 假
@@End@@

###Remarks###
最低操作系统: Windows XP
