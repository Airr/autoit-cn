###User Defined Function###
_GUICtrlToolbar_SetIndeterminate

###Description###
设置或清除按钮的不确定状态

###Syntax###
#Include <GuiToolBar.au3>
_GUICtrlToolbar_SetIndeterminate($hWnd, $iCommandID[, $fState = True])

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
$iCommandID
	按钮命令 ID
$fState
	[可选参数] 如为 True,则不确定, 否则为 False
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 True
失败:	返回 False
@@End@@
