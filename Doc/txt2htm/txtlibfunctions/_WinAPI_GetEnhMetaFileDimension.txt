###Function###
_WinAPI_GetEnhMetaFileDimension

###Description###
Retrieves a dimension for the specified enhanced-format metafile.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_GetEnhMetaFileHeader ( $hEmf )

###Parameters###
@@ParamTable@@
$hEmf
	Handle to the enhanced metafile for which a dimension is to be retrieved.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	$tagSIZE structure that contains the enhanced metafile dimension.
Failure	0 and sets the @error flag to non-zero.
@@End@@

None
