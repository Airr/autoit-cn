###User Defined Function###
_GUICtrlMenu_SetItemBitmaps

###Description###
关联指定位图与一个菜单项

###Syntax###
#Include <GuiMenu.au3>
_GUICtrlMenu_SetItemBitmaps($hMenu, $iItem, $hChecked, $hUnChecked[, $fByPos = True])

###Parameters###
@@ParamTable@@
$hMenu
	菜单句柄
$iItem
	菜单项的标识符或位置,由 $fByPos 参数决定.
$hChecked
	显示菜单项被选中的位图句柄
$hUnChecked
	显示菜单项未选中的位图句柄
$fByPos
	[可选参数] 菜单识别标志:
		True - $iItem 为项目的 0 基位置
		False - $iItem 为菜单项的标识符
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 True
失败:	返回 False
@@End@@

###Remarks###
 如果任一 hBitmapUnchecked 或 hBitmapChecked 参数为 0, 系统不会显示菜单项的相应选择状态.
 如果这两个参数都是 0,项目选中时,系统会显示默认复选标记位图,并删除该项目的未选中位图.
