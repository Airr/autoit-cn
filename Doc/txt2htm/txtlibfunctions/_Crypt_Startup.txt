###User Defined Function###
_Crypt_Startup

###Description###
初始化加密库

###Syntax###
#Include <Crypt.au3>
_Crypt_Startup()

###Parameters###
None.


###ReturnValue###
@@ReturnTable@@
成功:	返回 true
	设置 @error 为 0
失败:	返回 False 并设置 @error:
	1 - 打开 Advapi32.dll 失败
	2 - 无法获取加密环境
@@End@@

###Remarks###
 调用这个函数是可选的.
 为了提供向后兼容性, 对于 Windows 2000 将使用 PROV_RSA_FULL provider  对于 Win2000 和 windows xp 及更高版本使用 PROV_RSA_AES
