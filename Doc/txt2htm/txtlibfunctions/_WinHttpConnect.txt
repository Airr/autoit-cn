###User Defined Function###
_WinHttpConnect

###Description###
指定一个HTTP请求的初始目标服务器并返回一个指向初始服务器的HTTP会话连接句柄.

###Syntax###
#Include <WinHttp.au3>
_WinHttpConnect($hSession, $sServerName [, $iServerPort = Default ])

###Parameters###
@@ParamTable@@
$hSession
	由以前调用 WinHttpOpen 函数返回的有效的 WinHTTP 会话句柄.
$sServerName
	HTTP 服务器的主机名.
$iServerPort
	[可选参数] 指定建立连接的服务器上的TCP/IP端口 (默认为 $INTERNET_DEFAULT_PORT)
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回一个指向HTTP会话的有效连接句柄.
失败:	返回 0 并设置 @error:
	1 - DllCall 失败
@@End@@

###Remarks###
$iServerPort 可通过定义全局变量$INTERNET_DEFAULT_PORT, $INTERNET_DEFAULT_HTTP_PORT 或 $INTERNET_DEFAULT_HTTPS_PORT
