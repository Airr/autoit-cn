###User Defined Function###
_WinAPI_IsWindowUnicode

###Description###
确定指定窗口是否为本地 Unicode 窗口.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_IsWindowUnicode ( $hWnd )

###Parameters###
@@ParamTable@@
$hWnd
	窗口句柄.
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 1 - 是.
	返回 0 - 不是.
失败:	返回 0,设置@error:标志为非 0 值.
@@End@@
