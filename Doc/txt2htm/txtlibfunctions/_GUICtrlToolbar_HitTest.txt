###User Defined Function###
_GUICtrlToolbar_HitTest

###Description###
确定测试点的位置

###Syntax###
#Include <GuiToolBar.au3>
_GUICtrlToolbar_HitTest($hWnd, $iX, $iY)

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
$iX
	测试点 X 位置
$iY
	测试点 Y 位置
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回值如为 0 或正数,则为该点所在非分隔符项目的 0 基索引
	in which the point lies.
失败:	返回负数,该点所在不是按钮
@@End@@

###Remarks###
 返回值的绝对值是分隔符项目或就近非分隔项目的索引
