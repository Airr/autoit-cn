###用户自定义函数###
_IEFormElementOptionSelect

###描述###
设置指定表单元素的值

###语法###
#Include <IE.au3>
_IEFormElementOptionSelect ( ByRef $o_object, $s_string [, $f_select = 1 [, $s_mode = "byValue" [, $f_fireEvent = 1]]] )

###参数###
@@ParamTable@@
 $o_object  "Select Option"的表单元素对象
 $s_string  匹配元素的值--按$s_mode设定的模式
 $f_select  可选: 定义元素是否需要被选择或者不选择
-1 = 返回选择的状态
0 = 不选中元素
1 = (默认)
  选中元素
 $s_mode  可选: 定义搜索模式
byValue = (默认) 想要选择的option值
byText =
      想要选择option的文字
byIndex = 想要选择option的索引值(基于 0)
 $f_fireEvent  可选:定义在改变值时是否触发OnChange和OnClick的事件
0 =
      在设置值后不触发OnChange或者OnClick事件
1 = (默认)
  在设置值后触发OnChange或者OnClick事件
@@End@@

###返回值###
@@ReturnTable@@
成功: 如果参数$f_select为-1 , 返回当前选择的状态 , 否则返回1
失败: 返回0并设置@ERROR为:
    0 ($_IEStatus_Success) = 无错误
    3 ($_IEStatus_InvalidDataType) = 无效数据类型
    4 ($_IEStatus_InvalidObjectType) = 无效对象类型
    5 ($_IEStatus_InvalidValue) = 无效值
    7 ($_IEStatus_NoMatch) = 不匹配
    @Extended: 包含无效参数数量
@@End@@

###备注###
$f_fireEvent参数只有当表单元素发生了和它相关的onChange事件后才有效.
