###Function###
_WinAPI_PathIsSameRoot

###Description###
Compares two paths to determine if they have a common root component.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_PathIsSameRoot ( $sPath1, $sPath2 )

###Parameters###
@@ParamTable@@
$sPath1
	The first path to be compared.
$sPath2
	The second path to be compared.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	1 - Both strings have the same root component.
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;0	Otherwise.
Failure	0 and sets the @error flag to non-zero.
@@End@@
