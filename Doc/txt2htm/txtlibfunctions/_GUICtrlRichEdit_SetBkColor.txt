###User Defined Function###
_GUICtrlRichEdit_SetBkColor

###Description###
设置控件的背景颜色

###Syntax###
#Include <GuiRichEdit.au3>
_GUICtrlRichEdit_SetBkColor($hWnd [, $iColor = Default])

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
$iColor
	[可选参数] 滚动类型:
		COLORREF 值
		关键字 Default - 系统颜色（默认）
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 True
失败:	返回 False,设置@error
@error:	101 - $hWnd 参数值不是句柄
	1022 - $iColor 无效
@@End@@

###Remarks###
If you need to create a COLORREF value from an color array use _ColorSetCOLORREF() not _ColorSetRGB().
