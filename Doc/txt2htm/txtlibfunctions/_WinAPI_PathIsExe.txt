###Function###
_WinAPI_PathIsExe

###Description###
Determines whether a file is an executable by examining the file extension.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_PathIsExe ( $sPath )

###Parameters###
@@ParamTable@@
$sPath
	The path to be searched.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	1 - The file extension is .cmd, .bat, .pif, .scf, .exe, .com, or .scr.
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;0	Otherwise.
Failure	0 and sets the @error flag to non-zero.
@@End@@
