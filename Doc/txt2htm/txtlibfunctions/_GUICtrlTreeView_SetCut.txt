###User Defined Function###
_GUICtrlTreeView_SetCut

###Description###
设置项目的剪切和粘贴操作

###Syntax###
#Include <GuiTreeView.au3>
_GUICtrlTreeView_SetCut($hWnd, $hItem[, $fFlag = True])

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
$hItem
	项目句柄
$fFlag
	[可选参数] 设置标志:
		True - 项目被剪切
		False - 项目未被剪切
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 True
失败:	返回 False
@@End@@

###Remarks###
None.
