###User Defined Function###
_WinAPI_VerQueryRoot

###Description###
Retrieves fixed (code page independent) version information from the specified file.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_VerQueryRoot ( $sFile )

###Parameters###
@@ParamTable@@
$sFile
	The name of the file.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	$tagVS_FIXEDFILEINFO structure that contains the version information.
Failure	0 and sets the @error flag to non-zero.
@@End@@

###Remarks###
None
