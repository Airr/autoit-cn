###User Defined Function###
_WinAPI_PathIsRoot

###Description###
Parses a path to determine if it is a directory root.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_PathIsRoot ( $sPath )

###Parameters###
@@ParamTable@@
$sPath
	The path to be validated.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	1 - The path is a root.
0	Otherwise.
失败:	返回 0 并设置 @error 标志为非 0 值.
@@End@@
