###User Defined Function###
_GUICtrlListBox_FindString

###Description###
查找字符串

###Syntax###
#Include <GuiListBox.au3>
_GUICtrlListBox_FindString($hWnd, $sText[, $fExact = False])

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
$sText
	要搜索的字符串
$fExact
	[可选参数] 是否精确匹配
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回项目的 0 基索引
失败:	返回 -1
@@End@@

###Remarks###
 查找列表框匹配指定字符串的第一个字符串.
 (译注:以下依据示例脚本翻译,与原文字面意义有区别!原文留在下面)
 如果 $fExact 指定为 True,则查找列表框中完全匹配指定字符串的第一个项目.
 搜索不区分大小写.
 If exact is specified find the first list box string that exactly matches the specified string,
 except that the search is not case sensitive

###Related###
_GUICtrlListBox_FindInText, _GUICtrlListBox_SelectString
