###用户自定义函数###
_GUICtrlListBox_FindString

###描述###
搜索字符串

###语法###
#Include<GuiListBox.au3>
_GUICtrlListBox_FindString($hWnd, $sText[, $fExact =False])

###参数###
@@ParamTable@@
 $hWnd  控件句柄
 $sText  要搜索的字符串
 $fExact  是否精确匹配
@@End@@

###返回值###
@@ReturnTable@@
成功: 项目的0基索引
失败: -1
@@End@@

###备注###
查找列表箱中从指定字符串开始的第一个字符串.
If exact is specified find the
first list box string that exactly matches the specified string,
except that
the search is not case sensitive

###相关###
_GUICtrlListBox_FindInText , _GUICtrlListBox_SelectString
