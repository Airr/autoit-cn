###User Defined Function###
_WinAPI_FillMemory

###Description###
Fills a cblock of memory with the given value.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_FillMemory ( $pDestination, $iLenght [, $iValue] )

###Parameters###
@@ParamTable@@
$pDestination
	A pointer to the memory to be filled.
$iLenght
	The number of bytes to be filled.
$iValue
	[可选参数] The value to fill the memory.
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 1.
失败:	返回 0 并设置 @error 标志为非 0 值.
@@End@@

None
