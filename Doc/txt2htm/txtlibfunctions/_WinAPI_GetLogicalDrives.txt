###User Defined Function###
_WinAPI_GetLogicalDrives

###Description###
检索当前可用磁盘驱动器的掩码.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_GetLogicalDrives ( )

###Parameters###
@@ParamTable@@
None.
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回当前可用磁盘驱动器的掩码.
	位置 0 (最低有效位)为驱动器 A, 位置 1 为驱动器 B, 位置 2 为驱动器 C, 等等.
失败:	返回 0,设置 the @error 为非 0 值.
@@End@@
