###User Defined Function###
_WinAPI_IsChild

###Description###
Tests whether a window is a child window of a specified parent window.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_IsChild ( $hWnd, $hWndParent )

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the window to be tested.
$hWndParent
	Handle to the parent window.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	1 - The window is a child window of the specified parent window.
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;0	Otherwise.
Failure	0 and sets the @error flag to non-zero.
@@End@@
