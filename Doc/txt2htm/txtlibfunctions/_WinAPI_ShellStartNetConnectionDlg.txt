###User Defined Function###
_WinAPI_ShellStartNetConnectionDlg

###Description###
Displays a general browsing dialog box for a network resource connection.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_ShellStartNetConnectionDlg ( [$sRemote [, $iFlags [, $hParent]]] )

###Parameters###
@@ParamTable@@
$sRemote
	[可选参数] The remote network name.
$iFlags
	[可选参数] The flags that identify the type of resource that the dialog is set to find. This parameter can be
	one or more of the following values.
	
	
$hParent
	[可选参数] Handle of the parent window.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	1.
Failure	0 and sets the @error flag to non-zero.
@@End@@

None
