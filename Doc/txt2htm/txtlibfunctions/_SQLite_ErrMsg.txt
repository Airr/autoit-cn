###User Defined Function###
_SQLite_ErrMsg

###Description###
返回最近调用 sqlite3_* API 的英文错误条件描述的字符串

###Syntax###
#include <SQLite.au3>
_SQLite_ErrMsg ( [ $hDB ] )


###Parameters###
@@ParamTable@@
$hDB
	[可选参数] 打开的数据库,默认为最后打开的数据库
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回错误信息
失败:	@error = 1 - 错误调用 SQLite API 'sqlite3_errmsg16'
	2 - 调用被安全模式阻止
@@End@@

###Remarks###
None.
