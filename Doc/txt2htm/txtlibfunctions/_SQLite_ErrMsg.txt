###User Defined Function###
_SQLite_ErrMsg

###Description###
返回用英语描述的包含最近调用sqlite3_* API出错状态的字符串

###Syntax###
#Include <SQLite.au3>
_SQLite_ErrMsg ( [ $hDB ] )

###Parameters###
@@ParamTable@@
 $hDB  [可选] 打开的数据库, 默认为最近打开的数据库
@@End@@

###ReturnValue###
@@ReturnTable@@
成功: 返回出错消息
失败: @error = 1 - 调用SQLite API 'sqlite3_errmsg16'出错
@@End@@
