###User Defined Function###
_WinHttpOpen

###Description###
初始化WinHTTP函数的使用并返回WinHTTP会话句柄

###Syntax###
#Include <WinHttp.au3>
_WinHttpOpen([$sUserAgent = Default [, $iAccessType = Default [, $sProxyName = Default [, $sProxyBypass = Default [, $iFlag = Default ]]]]])

###Parameters###
@@ParamTable@@
$sUserAgent
	[可选参数] 包含应用程序或调用WinHTTP函数的实体的名称的字符串
$iAccessType
	[可选参数] 需要的权限类型
$sProxyName
	[可选参数] 包含当代理权限通过设置$iAccessType被指定为$WINHTTP_ACCESS_TYPE_NAMED_PROXY时使用的代理服务器的字符串
$sProxyBypass
	[可选参数] 包含代理的主机名或IP地址或两者的可选列表的字符串, 该代理无法在$iAccessType被设置为$WINHTTP_ACCESS_TYPE_NAMED_PROXY时被路由
$iFlag
	[可选参数] 包含指定影响函数结果的选项的标记的整数
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回有效会话句柄设置@error 0
失败:	返回0设置@error 1表示DllCall失败
@@End@@

###Remarks###
For asynchronous mode set $iFlag to $WINHTTP_FLAG_ASYNC
