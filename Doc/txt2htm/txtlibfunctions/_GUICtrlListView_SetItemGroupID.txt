###User Defined Function###
_GUICtrlListView_SetItemGroupID

###Description###
设置项目组 ID

###Syntax###
#Include <GuiListView.au3>
_GUICtrlListView_SetItemGroupID($hWnd, $iIndex, $iGroupID)

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
$iIndex
	项目的 0 基索引
$iGroupID
	组 ID
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 True
失败:	返回 False
@@End@@

###Remarks###
 最低操作系统 - Windows XP.
