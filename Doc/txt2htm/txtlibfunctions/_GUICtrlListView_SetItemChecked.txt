###User Defined Function###
_GUICtrlListView_SetItemChecked

###Description###
设置选中状态

###Syntax###
#Include <GuiListView.au3>
_GUICtrlListView_SetItemChecked($hWnd, $iIndex[, $fCheck = True])

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
$iIndex
	项目的 0 基索引;-1,应用全部项目
$fCheck
	[可选参数] 选中状态标志值:
		True - 选中
		False - 非选中
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 True
失败:	返回 False
@@End@@

###Remarks###
 仅在控件有 $LVS_EX_CHECKBOXES 扩展样式时有效
