###Function###
_WinAPI_GetProcessHandleCount

###Description###
Retrieves the number of open handles that belong to the specified process.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_GetProcessHandleCount ( [$PID] )

###Parameters###
@@ParamTable@@
$PID
	The PID of the process. Default (0) is the current process.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	The number of open handles.
Failure	0 and sets the @error flag to non-zero.
@@End@@

###Remarks###
None
