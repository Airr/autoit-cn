###User Defined Function###
_GUICtrlRebar_SetColorScheme

###Description###
Retrieves a rebar control's default background color

###Syntax###
#Include <GuiRebar.au3>
_GUICtrlRebar_SetColorScheme($hWnd, $BtnHighlight, $BtnShadow)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to rebar control
$BtnHighlight
	COLORREF value that represents the highlight color of the buttons
$BtnShadow
	COLORREF value that represents the shadow color of the buttons
@@End@@

###ReturnValue###
None.

###Remarks###
None.
