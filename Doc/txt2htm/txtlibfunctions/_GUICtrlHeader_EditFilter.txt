###User Defined Function###
_GUICtrlHeader_EditFilter

###Description###
启动编辑指定的过滤器

###Syntax###
#include <GuiHeader.au3>
_GUICtrlHeader_EditFilter($hWnd, $iIndex [, $fDiscard = True])

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
$iIndex
	项目的 0 基索引
$fDiscard
	[可选参数] 编辑过程中发送消息到过滤器时,
		指定如何处理用户的编辑修改:
		 True - 放弃用户修改
		False - 接受用户修改
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 True
失败:	返回 False
@@End@@

###Remarks###
None.
