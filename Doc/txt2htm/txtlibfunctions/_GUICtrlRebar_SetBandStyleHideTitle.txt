###User Defined Function###
_GUICtrlRebar_SetBandStyleHideTitle

###Description###
设置带区是否隐藏标题

###Syntax###
#Include <GuiRebar.au3>
_GUICtrlRebar_SetBandStyleHideTitle($hWnd, $iIndex[, $fEnabled = True])

###Parameters###
@@ParamTable@@
$hWnd
	伸缩条控件句柄
$iIndex
	带区的 0 基索引
$fEnabled
	[可选参数] 设为 True, 则是,设为 False 则不是
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 True
失败:	返回 False
@@End@@

###Remarks###
None.
