###User Defined Function###
_WinAPI_GetSystemDEPPolicy

###Description###
获取系统的数据执行保护策略(DEP)设置

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_GetSystemDEPPolicy( )

###Parameters###
@@ParamTable@@
    无
@@End@@

###ReturnValue###
@@ReturnTable@@
成功: 返回系统的数据执行保护策略(DEP)设置. 可为下列值之一:
    0 - 禁用系统各部分的数据执行保护, 无论硬件是否支持.
    1 - 对系统各部分启用数据执行保护, 所有进程均运行在DEP启用状态下. DEP不能被明确选定的应用程序禁用. 系统兼容性修补程序将被忽略.
    2 - 对运行于可硬件实现DEP功能的处理器上的系统, DEP只对操作系统组件自动启用. Windows客户端版本的默认设置. DEP可对选定的应用程序或当前进程启用.
    3 - 对操作系统组件和所有进程自动启用DEP. Windows Server版本的默认设置.
失败: 返回0并设置@error非0
@@End@@
