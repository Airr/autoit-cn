###Keyword###
Struct...EndStruct

###Description###
Declare a variable with typed subelements.

###Syntax###
[作用范围] <b>Struct</b> $structvar <b>[</b>子脚本<b>]</b>
	<b>type</b> [<i>元素名称</i>];
	...
<b>EndStruct</b>


###Parameters###
@@ParamTable@@
作用范围
	[可选参数] scope of the struct variable can be Local or Global
structvar
	数据结构变量名称.
子脚本
	[可选参数] to define a monodimensional array
type
	See DllStructCreate for definition
元素名称
	name of the element  to be use to reference sub element by "$structvar.elementname"
@@End@@


###Remarks###
Using Struct Keyword is equivalent to use $structvar=DllStructCreate("elements")
See DllStructCreate definition for complete definition of types.
Array element can be used "$structvar.elementname[n]". It differ from DllStruct... function as the first element index is 0.
If a monodimensional struct array as been defined it will be reference as "$structvar[n].elementname[m]"
