###Function###
FileFlush

###Description###
刷新文件缓冲器中的数据到磁盘.

###Syntax###
FileFlush ( 句柄 )


###Parameters###
@@ParamTable@@
句柄
	由前面使用 FileOpen() 函数返回的句柄.
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 true ,如果缓冲器已被清空(或者不需要清空).
失败:	返回 false.
@@End@@


###Remarks###
A file is flushed when it's handle is closed or when Windows internal buffer is full.  This function forces an immediate flushing of the buffer.
这个函数只能使用 FileOpen() 函数返回的句柄.

###Related###
FileClose, FileOpen, FileWrite, FileWriteLine, FileSetPos
