###Function###
GUICtrlRegisterListViewSort

###Description###
为内部listview排序回调函数注册一个用户自定义函数.

###Syntax###
GUICtrlRegisterListViewSort ( 控件ID, "函数" )

###Parameters###
@@ParamTable@@
控件ID
	要让用户函数处理的 listview 控件ID.
函数
	当排序回调运行时调用的用户函数名称.
@@End@@


###ReturnValue###
@@ReturnTable@@
成功:	1
失败:	0
@@End@@


###Remarks###
!!! 请保证您的用户函数可以正常工作于 <strong>最大 4 个函数参数</strong> ,反之就不会调用它 !!!
i.e:
Func MySortFunction($nListViewID, $LParam1, $LParam2, $nColumn)
...
EndFunc

或者

Func MySortFunction($nListViewID, $LParam1, $LParam2)
...
EndFunc


当用户函数被调用时,4个函数参数为下列值:

@@StandardTable1@@
<b>序号</b>	<b>参数</b>	<b>意思</b>
1	控件ID	The controlID of the listview control for which the callback function is used.
2	lParam1	第一个项目的 lParam 值(默认为项目控件ID).
3	lParam2	第二个项目的 lParam 值(默认为项目控件ID).
4	column	The column that was clicked for sorting (the first column number is 0).
@@End@@

The following values have to be <strong>Return</strong>ed to change the behaviour of the sorting callback:

@@StandardTable1@@
<b>返回值</b>	<b>意思</b>
-1	1st item should precede the 2nd.
0	No Change.
1	1st item should follow the 2nd.
@@End@@

See also examples for sorting with selfcreated GUI listview items.
