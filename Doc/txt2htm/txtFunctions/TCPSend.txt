###Function###
TCPSend

###Description###
在已经连接的套接字(socket)上面发送数据.

###Syntax###
TCPSend ( 主套接字, 数据 )


###Parameters###
@@ParamTable@@
主套接字
	一个已经连接的套接字标识 (SocketID),返回于 <a href="TCPConnect.htm">TCPConnect</a> 函数.
数据
	要发送给已连接的套接字的二进制数据/字符串.   
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回发送到已经连接的套接字的字节数量.   
失败:	返回 0 并设置 @error 的值.
@error:	windows API WSAGetError 返回值 (参考 <a href="http://msdn.microsoft.com/en-us/library/ms740668.aspx">MSDN</a>).
@@End@@


###Remarks###
如果要传输一个 Unicode 字符串,请使用 StringToBinary()/BinaryToString() 进行编码解码.


###Related###
TCPRecv, TCPStartup, TCPConnect, TCPListen, TCPTimeout (Option), StringToBinary
