###Function###
StringSplit

###Description###
以指定分隔符把字符串拆分成若干子字符串.

###Syntax###
StringSplit ( "字符串", "分隔符" [, 标志] )


###Parameters###
@@ParamTable@@
字符串
	要进行分割的字符串.
分隔符
	一个或多个作为分隔符的字符(区分大小写).   
标志
	[可选参数] 设置如何分割字符串,如果需要可以添加由多个标志组成的组合:
	  标志 = 0 (默认), 则(分隔符)字符串中的每个字符都将用于拆分字符串.
	  标志 = 1, 则只能以整个分隔符字符串来拆分字符串.
	  标志 = 2, 关闭第一个元素中的返回数量 - 方便使用 UBound() 得到此基于0开始的数组.
@@End@@


###ReturnValue###
返回一个数组,默认第一个元素($aArray[0])保存拆分后子字符串的数量,其余元素($aArray[1],$aArray[2] ,等等)则保存着拆分后的每个字符串. 如果使用了标志=2,则数量不会返回到第一个元素.

若在要进行分割的字符串中未发现分隔符则将 @error 设置为1,子字符串数量($aArray[0])设置为1,并且函数将返回整个字符串到第二个元素($aArray[1]). 


###Remarks###
若指定的分隔符是个空字符串 "",则每个字符将作为数组元素返回.

若分隔符是一个字符串而不是单个字符,请参考下面的第二个示例.

StringSplit 有时候可以作为代替 StringInStr() 函数的更好选择,因为它可以组建数组. 

小心:如果您使用宏 @CRLF ,可能会出现额外的空白行.


###Related###
