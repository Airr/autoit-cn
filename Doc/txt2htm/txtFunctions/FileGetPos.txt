###Function###
FileGetPos

###Description###
返回当前文件坐标.

###Syntax###
FileGetPos ( 文件句柄 )


###Parameters###
@@ParamTable@@
文件句柄
	一个文件句柄,由前面使用 FileOpen() 函数返回的句柄.
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回相对于文件开始处的坐标偏移量(第一个索引为 0).
失败:	返回 0 并设置 @error.
@@End@@


###Remarks###
失败将返回 0 但是 0 同样在坐标内有效(文件开始处),所以请检查 @error 以确定函数是否成功执行.

###Related###
FileSetPos, FileRead, FileReadLine, FileWrite, FileWriteLine, FileOpen
