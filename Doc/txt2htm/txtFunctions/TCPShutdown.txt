###Function###
TCPShutdown, UDPShutdown

###Description###
停止 TCP/UDP 服务. 


###Syntax###
TCPShutdown ( )
UDPShutdown ( )


###Parameters###
None.


###ReturnValue###
@@ReturnTable@@
成功:	返回 1.
失败:	返回 0 并设置 @error.
@error:	windows API WSACleanup 返回值(参考 <a href="http://msdn.microsoft.com/en-us/library/ms741549.aspx">MSDN</a>).    
@@End@@


###Remarks###
每当成功完成了一个TCPStartup()调用,必须调用一个TCPShutdown()来关闭初始化的网络.
UDPShutdown() 是 TCPShutdown()的一个别名.

###Related###
TCPStartup, TCPListen, TCPCloseSocket, UDPCloseSocket
