论坛自定义函数 参考

下面是 论坛自定义函数 的详细列表.  点击 论坛自定义函数 名称得到详细信息.

 

论坛自定义函数 详细信息
_accessAddRecord 在现有的 MS Access 数据库表添加新记录(单个或多个字段)
_accessClearTable 清除 MS Access 数据库指定的表内所有记录
_accessCompactDB 压缩 MS Access 数据库文件(*. mdb)
_accessCountFields 读取 MS Access 数据库表中的字段数
_accessCountRecords 返回 MS Access 数据库表中的记录数
_accessCreateDB 创建一个 MS Access 数据库(*. mdb)文件
_accessCreateTable 在现有的 MS Access 数据库中建立表
_accessDeleteRecord 搜索数据库字符串，并将搜索到的字符串(整列)数据删除
_accessDeleteTable 从 MS Access 数据库(*. mdb)文件中删除表
_accessGetVal 返回指定字段的第一个值
_accessListFields 返回 MS Access 数据库中所指定的表中的所有字段名称
_accessListTables 返回 MS Access 数据库中所有的表名称
_accessQueryLike 搜索数据库中指定的表内的字段所包含指定的字符串
_accessQueryStr 搜索数据库中的指定字符串,并返回搜索结果
_accessSaveXML 将 MS Access 数据库中指定的表将导出为XML文件
_accessTableCount 统计在 MS Access 数据库中有多少个表
_accessUpdateRecord 搜索MS Access数据库中的表，并更新表中的数据记录
_API_Get_NetworkAdapterMAC 根据IP地址使用API得到网卡MAC地址.
_Base64Decode Base64解码
_Base64Encode Base64编码
_ChangeScreenRes 修改 屏幕分辨率,刷新率.
_CMD_SetLocalIP 命令行方式设置IP地址.
_CRC32 CRC32编码
_DateDayOfWeekC 返回今天是星期几.
_DateMonthOfYear 返回这个月是几月份.
_DecryptFile 解密文件,NTFS磁盘系统自带的EFS加密.
_EncryptFile 加密文件,NTFS磁盘系统自带的EFS加密.
_Flashget 调用网际快车下载.
_HideSystemFolder 隐藏系统特殊文件夹.
_MD5 MD5编码
_MD5Init MD5初始化
_MD5Input MD5输入
_MD5Result MD5结果
_MouseClickPlus 发送一个鼠标动作到指定的窗口,不是非常准确,但是可以对最小化的窗口生效.
_NetworkAdapterInfo 获得网卡信息.
_NetworkGetInternetIP 得到公网IP地址.
_RC4 RC4编码
_RefreshIcon 刷新系统图标.
_SHA1 SHA1编码
_SHA1Init SHA1初始化
_SHA1Input SHA1输入
_SHA1Result SHA1结果
_Thunder 调用迅雷下载.
_WMI_SetNetworkAdapterInfo WMI方式设置IP地址.
_XXTEA_Decrypt XXTEA解码
_XXTEA_Decrypt_Pad XXTEA解码算法
_XXTEA_Encrypt XXTEA编码
_XXTEA_Encrypt_Pad XXTEA编码算法
