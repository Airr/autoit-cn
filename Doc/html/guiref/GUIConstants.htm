GUI 常量包含文件

下方列表显示了可用于autoit的GUI常量的完整列表.

文件 相关控件
#include <AVIConstants.au3> 视频播放(AVI)
#include <ButtonConstants.au3> 按钮(Button), 组(Group), 单选(Radio), 检查框(Checkbox)
#include <ComboConstants.au3> 组合列表框(Combo)
#include <Constants.au3> 托盘(Tray), 运行(Run), 标准输入输出(StdIO),
#include <DateTimeConstants.au3> 日期(Date), 月历(MonthCal)
#include <EditConstants.au3> 编辑框(Edit), 输入框(Input)
#include <GUIConstantsEx.au3> GUI 创建, 事件
#include <ListboxConstants.au3> 列表(List)
#include <ListviewConstants.au3> 列表视图(Listview)
#include <ProgressConstants.au3> 进度条(Progress)
#include <SliderConstants.au3> 滑动条(Slider)
#include <StaticConstants.au3> 标签(Label), 图片(Pic), 图标(Icon)
#include <TabConstants.au3> 标签页(Tab)
#include <TreeviewConstants.au3> 树形视图(Treeview)
#include <UpdownConstants.au3> 上下按钮(Updown)
#include <WindowsConstants.au3> Gui 扩展样式
