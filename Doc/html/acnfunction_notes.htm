ACN论坛定义的函数说明(ACN Defined Function Notes)

一直以来,我(thesnow)在汉化版本中就有收集的一些脚本,主要来源于官方论坛和ACN论坛.其中一部分是自己写的.现在为了规范点,就做了个文档,暂时叫做 ACN Defined Functions 吧..
ADF 其实也是一个脚本, 使用 AutoIt3 编写,您可以使用 #include 包含进你的脚本中.例如:

例子:

#include <ACN_NET.au3>
MsgBox(0,"Your MAC address","Your MAC address is " & _API_Get_NetworkAdapterMAC (@IPAddress1))

这个脚本将会显示您的网卡MAC地址
