###User Defined Function###
_GUICtrlButton_SetNote

###Description###
设置与命令按钮有关的提示文本 （必须 windows vista 支持 ） ( 译注：好像不支持中文提示文本 )

###Syntax###
#Include <GuiButton.au3>
_GUICtrlButton_SetNote($hWnd, $sNote)

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
$sNote
	提示文本
@@End@@

###ReturnValue###
成功: 布尔真 True
失败: 布尔假 False

###Remarks###
最低操作系统: Windows Vista
