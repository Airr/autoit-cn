###User Defined Function###
_IE_Introduction

###Description###
在新浏览器窗口中显示一个关于 IE.au3 的介绍信息

###Syntax###
#include <IE.au3> 
_IE_Introduction ( [$s_module = "basic"] )

###Parameters###
@@ParamTable@@
$s_module
	[可选参数] 指定要运行的模块
	basic = (默认) 基本介绍.
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回一个指向 InternetExplorer.Application 对象的对象变量
失败:	返回 0 并设置 @ERROR
@Error:	0 ($_IEStatus_Success) = 没有错误
	5 ($_IEStatus_InvalidValue) = 无效值
@Extended:	包含无效参数的数量.
@@End@@


###Remarks###
Autoit帮助文件不允许UDFs创建包含一系列函数的单一文档,但是相反的每个UDF文档集成到独立的函数中.本函数目的是提供通用记录.它会不断的改进.
