###User Defined Function###
_GUICtrlToolbar_SetBitmapSize

###Description###
Sets the size of the bitmapped images to be added to a toolbar

###Syntax###
#Include <GuiToolBar.au3>
_GUICtrlToolbar_SetBitmapSize($hWnd, $iWidth, $iHeight)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$iWidth
	Width, in pixels, of the bitmapped images
$iHeight
	Height, in pixels, of the bitmapped images
@@End@@

###ReturnValue###
Success: True
Failure: False

###Remarks###
The size can be set only before adding any bitmaps to the toolbar. If an application does not explicitly set
the bitmap size, the size defaults to 16 by 15 pixels.
