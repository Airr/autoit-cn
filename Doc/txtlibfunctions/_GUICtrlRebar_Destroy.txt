###User Defined Function###
_GUICtrlRebar_Destroy

###Description###
Delete the control

###Syntax###
#Include <GuiRebar.au3>
_GUICtrlRebar_Destroy(ByRef $hWnd)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
@@End@@

###ReturnValue###
成功: True, 句柄设置为 0
失败: False

###Remarks###
Restricted to only be used on control created with _GUICtrlRebar_Create
