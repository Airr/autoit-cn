###User Defined Function###
_GDIPlus_Startup

###Description###
Initialize Microsoft Windows GDI+

###Syntax###
#Include <GDIPlus.au3>
_GDIPlus_Startup()

###Parameters###
@@ParamTable@@
None.

@@End@@

###ReturnValue###
Success: True
Failure: False

###Remarks###
Call _GDIPlus_Startup before you create any GDI+ objects. GDI+ requires a redistributable for applications that
run on the Microsoft Windows NT 4.0 SP6, Windows 2000, Windows 98, and Windows Me operating systems.
