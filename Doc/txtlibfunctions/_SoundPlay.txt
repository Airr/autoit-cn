###User Defined Function###
_SoundPlay

###Description###
播放一个声音文件.

###Syntax###
#include <Sound.au3>
_SoundPlay ( $sSnd_id [, $fWait ] )


###Parameters###
@@ParamTable@@
$sSnd_id
	声音 ID ('别名') 为 一个 _SoundOpen 返回的结果或者一个文件.
$fWait
	[可选参数] 这个标志定义脚本是否等待声音播放完成后再继续运行脚本:
	1 = 等待声音播放完成再继续运行脚本.
	0 = 当声音开始播放,脚本继续运行.(默认)
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	1
失败:	0
@Error:	0 = 没有错误
	1 = 播放失败
	2 = $fWait 参数无效
	3 = 无效声音ID. 可能需要 _SoundOpen 使用别名.
@@End@@
