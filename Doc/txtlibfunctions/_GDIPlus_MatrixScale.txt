###User Defined Function###
_GDIPlus_MatrixScale

###Description###
Updates a matrix with the product of itself and a scaling matrix

###Syntax###
#Include <GDIPlus.au3>
_GDIPlus_MatrixScale($hMatrix, $nScaleX, $nScaleY[, $fAppend = False])

###Parameters###
@@ParamTable@@
$hMatrix
	Handle to a Matrix object
$nScaleX
	Multiplyier to scale the x-axis
$nScaleY
	Multiplyier to scale the y-axis
$fAppend
	Specifies the order of the multiplication:
		True  - Specifies that the scaling matrix is on the left
		False - Specifies that the scaling matrix is on the right
@@End@@

###ReturnValue###
Success: True
Failure: False

###Remarks###
None.
