###User Defined Function###
_ClipBoard_Close

###Description###
关闭剪切板

###Syntax###
#Include <Clipboard.au3>
_ClipBoard_Close()

###Parameters###
@@ParamTable@@
None.

@@End@@

###ReturnValue###
Success: True
Failure: False

###Remarks###
When the window has finished examining or changing the clipboard close the clipboard by calling this function.
This enables other windows to access the clipboard. Do not place an object on the clipboard after calling
this function.
