###User Defined Function###
_GUICtrlRebar_SetBandStyleHidden

###Description###
Set whether the band will not be visible

###Syntax###
#Include <GuiRebar.au3>
_GUICtrlRebar_SetBandStyleHidden($hWnd, $iIndex[, $fEnabled = True])

###Parameters###
@@ParamTable@@
$hWnd
	Handle to rebar control
$iIndex
	Zero-based index of the band
$fEnabled
	If True the item state is set, otherwise it is not set
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True
Failure:	False
@@End@@

###Remarks###
None.
