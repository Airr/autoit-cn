###User Defined Function###
_GUICtrlToolbar_GetButtonText

###Description###
Retrieves the display text of a button

###Syntax###
#Include <GuiToolBar.au3>
_GUICtrlToolbar_GetButtonText($hWnd, $iCommandID)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$iCommandID
	Button command ID
@@End@@

###ReturnValue###
Success: Button text

###Remarks###
None.
