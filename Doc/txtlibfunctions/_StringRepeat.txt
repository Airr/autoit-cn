###User Defined Function###
_StringRepeat

###Description###
Repeats a string a specified number of times.

###Syntax###
#Include <String.au3>
_StringRepeat($sString, $iRepeatCount)

###Parameters###
@@ParamTable@@
$sString
	String to repeat
$iRepeatCount
	Number of times to repeat the string
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	Returns string with specified number of repeats
失败:	Returns an empty string and sets @error
@Error:	0 - 没有错误
	1 - One of the parameters is invalid
@@End@@
