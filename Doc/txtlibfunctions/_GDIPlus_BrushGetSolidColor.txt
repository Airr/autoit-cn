###User Defined Function###
_GDIPlus_BrushGetSolidColor

###Description###
Get the color of a Solid Brush object

###Syntax###
#Include <GDIPlus.au3>
_GDIPlus_BrushGetSolidColor($hBrush, [$iARGB = 0xFF000000])

###Parameters###
@@ParamTable@@
$hBrush
	Handle to a Brush object
$iARGB
	The color of the brush.
@@End@@

###ReturnValue###
Success: Brush color
Failure: -1 and @error is set

###Remarks###
None.
