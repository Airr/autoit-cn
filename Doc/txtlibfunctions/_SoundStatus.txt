###User Defined Function###
_SoundStatus

###Description###
Returns the status of the sound 

###Syntax###
#include <Sound.au3>
_SoundStatus ( $sSnd_id )


###Parameters###
@@ParamTable@@
$sSnd_id
	Sound ID (the 'alias') as returned by _SoundOpen or a file
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	Sound Status
Failure:	0
@Error:	0 = No Error
	3 = Invalid Sound ID. May be _SoundOpen with alias needed.
@@End@@

###Remarks###
From MSDN:
All devices can return the "not ready", "paused", "playing", and "stopped" values. 
Some devices can return the additional "open", "parked", "recording", and "seeking" values.
