###User Defined Function###
_GUICtrlToolbar_MapAccelerator

###Description###
Determines the ID of the button that corresponds to the specified accelerator

###Syntax###
#Include <GuiToolBar.au3>
_GUICtrlToolbar_MapAccelerator($hWnd, $cAccel)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$cAccel
	Accelerator character
@@End@@

###ReturnValue###
Success: Command ID of the button that has cAccel as its accelerator character

###Remarks###
None.
