###User Defined Function###
_WinAPI_EnumWindowsPopup

###Description###
Enumerates popup windows

###Syntax###
#Include <WinAPI.au3>
_WinAPI_EnumWindowsPopup()

###Parameters###
None.


###ReturnValue###
@@ReturnTable@@
Success	Array with the following format:
	[0][0] - Number of rows in array (n)
	[1][0] - Window handle
	[1][1] - Window class name
	[n][0] - Window handle
	[n][1] - Window class name
@@End@@

None.
