###User Defined Function###
_GUICtrlComboBoxEx_SetUnicode

###Description###
Sets if control is using Unicode

###Syntax###
#Include <GuiComboBoxEx.au3>
_GUICtrlComboBoxEx_SetUnicode($hWnd[, $fUnicode = True])

###Parameters###
@@ParamTable@@
$hWnd
	Handle to control
$fUnicode
	[optional] May be one of the following
		True - Turn on Unicode
		False - Turn off Unicode
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True
Faliure:	False
@@End@@

###Remarks###
None.
