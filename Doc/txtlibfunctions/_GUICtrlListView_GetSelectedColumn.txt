###User Defined Function###
_GUICtrlListView_GetSelectedColumn

###Description###
Retrieves the index of the selected column

###Syntax###
#Include <GuiListView.au3>
_GUICtrlListView_GetSelectedColumn($hWnd)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	The zero based index of the selected column
Failure:	-1
@@End@@

###Remarks###
Minimum OS - Windows XP.
