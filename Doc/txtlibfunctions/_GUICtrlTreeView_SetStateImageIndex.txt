###User Defined Function###
_GUICtrlTreeView_SetStateImageIndex

###Description###

###Syntax###
#Include <GuiTreeView.au3>
_GUICtrlTreeView_SetStateImageIndex($hWnd, $hItem, $iIndex)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$hItem
	Handle to the item
$iIndex
	Image list index
@@End@@

###ReturnValue###
成功: True
失败: False

###Remarks###
This is a 1 based index list
