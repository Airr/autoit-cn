###User Defined Function###
_GUICtrlRebar_SetBandStyleTopAlign

###Description###
Set whether to keep band in top row

###Syntax###
#Include <GuiRebar.au3>
_GUICtrlRebar_SetBandStyleTopAlign($hWnd, $iIndex[, $fEnabled = True])

###Parameters###
@@ParamTable@@
$hWnd
	Handle to rebar control
$iIndex
	Zero-based index of the band
$fEnabled
	If True the item state is set, otherwise it is not set
@@End@@

###ReturnValue###
成功: True
失败: False

###Remarks###
None.
