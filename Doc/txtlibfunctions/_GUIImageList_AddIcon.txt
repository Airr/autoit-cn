###User Defined Function###
_GUIImageList_AddIcon

###Description###
Adds an icon to an image list

###Syntax###
#Include <GuiImageList.au3>
_GUIImageList_AddIcon($hWnd, $sFile[, $iIndex=0[, $fLarge = False])

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$sFile
	Path to the icon that contains the image
$iIndex
	Specifies the zero-based index of the icon to extract
$fLarge
	Extract Large Icon
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	The index of the image
Failure:	-1
@@End@@

###Remarks###
None.
