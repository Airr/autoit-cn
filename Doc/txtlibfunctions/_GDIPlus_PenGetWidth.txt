###User Defined Function###
_GDIPlus_PenGetWidth

###Description###
Retrieve the width of a pen

###Syntax###
#Include <GDIPlus.au3>
_GDIPlus_PenGetWidth($hPen)

###Parameters###
@@ParamTable@@
$hPen
	Handle to a pen object
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	Width of pen
Failure:	-1 and @error is set
@@End@@
