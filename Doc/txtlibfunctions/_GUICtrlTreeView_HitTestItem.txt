###User Defined Function###
_GUICtrlTreeView_HitTestItem

###Description###
Returns the item at the specified coordinates

###Syntax###
#Include <GuiTreeView.au3>
_GUICtrlTreeView_HitTestItem($hWnd, $iX, $iY)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$iX
	X position to test
$iY
	Y position to test
@@End@@

###ReturnValue###
Success: Handle to the item at the specified point or 0 if no item occupies the point

###Remarks###
None.
