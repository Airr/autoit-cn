###User Defined Function###
_GUICtrlComboBox_Destroy

###Description###
Delete the control

###Syntax###
#Include <GuiComboBox.au3>
_GUICtrlComboBox_Destroy(ByRef $hWnd)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
@@End@@

###ReturnValue###
Success: True, Handle is set to 0
Failure: False

###Remarks###
Restricted to only be used on Listbox created with _GUICtrlComboBox_Create
