###User Defined Function###
_WordDocSave

###Description###
保存一个已经打开的文档.

###Syntax###
#include <Word.au3> 
_WordDocSave ( $o_object )

###Parameters###
@@ParamTable@@
$o_object
	一个 Word.Application 对象变量, 文档对象.
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回 1
失败:	返回 0 并设置 @ERROR
@Error:	0 ($_WordStatus_Success) = 没有错误
	1 ($_WordStatus_GeneralError) = 常规错误
	3 ($_WordStatus_InvalidDataType) = 无效数据类型
	4 ($_WordStatus_InvalidObjectType) = 无效对象类型
@Extended:	包含无效参数数
@@End@@


###Remarks###
If a document hasn't been saved before, the Save As dialog box prompts the user for a file name.


###Related###
_WordDocAdd, _WordDocClose, _WordDocOpen, _WordDocSaveAs
