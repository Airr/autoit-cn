###User Defined Function###
_WordDocSave

###Description###
Saves a previously opened document

###Syntax###
#include <Word.au3> 
_WordDocSave ( $o_object )

###Parameters###
@@ParamTable@@
$o_object
	Object variable of a Word.Application, document object
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	Returns 1
Failure:	Returns 0 and sets @ERROR
@Error:	0 ($_WordStatus_Success) = No Error
	1 ($_WordStatus_GeneralError) = General Error
	3 ($_WordStatus_InvalidDataType) = Invalid Data Type
	4 ($_WordStatus_InvalidObjectType) = Invalid Object Type
@Extended:	Contains invalid parameter number
@@End@@


###Remarks###
If a document hasn't been saved before, the Save As dialog box prompts the user for a file name.


###Related###
_WordDocAdd, _WordDocClose, _WordDocOpen, _WordDocSaveAs
