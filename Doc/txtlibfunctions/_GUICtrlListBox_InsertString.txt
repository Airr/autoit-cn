###User Defined Function###
_GUICtrlListBox_InsertString

###Description###
Insert a string into the list

###Syntax###
#Include <GuiListBox.au3>
_GUICtrlListBox_InsertString($hWnd, $sText[, $iIndex = -1])

###Parameters###
@@ParamTable@@
$hWnd
	Handle to control
$sText
	Text string to be inserted
$iIndex
	Specifies the zero based index of the position at which to insert the string. If this parameter
		is -1 the string is added to the end of the list.
@@End@@

###ReturnValue###
Success: Zero based index of the item position
Failure: -1

###Remarks###
If $iIndex is -1 then string is added to end of list. Unlike the _GUICtrlListBox_AddString,
this function does not cause a list with the $LBS_SORT style to be sorted.
