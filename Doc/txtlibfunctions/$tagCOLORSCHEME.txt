###Structure Name###
$tagCOLORSCHEME

###Description###
Contains information for the drawing of buttons in a toolbar or rebar

###Syntax###
Global Const $tagCOLORSCHEME = "int Size;int BtnHighlight;int BtnShadow"

###Fields###
@@ParamTable@@
Size
	Size of this structure, in bytes
BtnHighlight
	The COLORREF value that represents the highlight color of the buttons. Use $CLR_DEFAULT for the
	default highlight color.
BtnShadow
	The COLORREF value that represents the shadow color of the buttons. Use $CLR_DEFAULT for the
	default shadow color.
@@End@@

###Remarks###
None.
