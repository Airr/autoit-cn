###User Defined Function###
_GUICtrlListView_GetViewList

###Description###
Determines whether the view mode is in list mode

###Syntax###
#Include <GuiListView.au3>
_GUICtrlListView_GetViewList($hWnd)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
@@End@@

###ReturnValue###
@@ReturnTable@@
True:	View is in list mode
False:	View is not in list mode
@@End@@

###Remarks###
Minimum OS - Windows XP.
