###User Defined Function###
_GUICtrlRebar_GetBandChildHandle

###Description###
Retrieves the Handle to the child window contained in the band, if any

###Syntax###
#Include <GuiRebar.au3>
_GUICtrlRebar_GetBandChildHandle($hWnd, $iIndex)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to rebar control
$iIndex
	Zero-based index of the band for which the borders will be retrieved
@@End@@

###ReturnValue###
Success: Handle to the child window contained in the band

###Remarks###
None.
