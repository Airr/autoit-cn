###User Defined Function###
_SQLite_Changes

###Description###
Returns number of changes (excluding triggers) of the last transaction

###Syntax###
#include <SQLite.au3>
_SQLite_Changes ( [ $hDB ] )


###Parameters###
@@ParamTable@@
$hDB
	[optional] An open database, default is the last opened database
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	Returns number of changes
Failure:	Returns 0
@error	1 - Error calling SQLite API 'sqlite3_changes'
	2 - Call prevented by SafeMode
@@End@@
