###User Defined Function###
_IEBodyReadHTML

###Description###
返回文档中 <body> 标记中的HTML.

###Syntax###
#include <IE.au3> 
_IEBodyReadHTML ( ByRef $o_object )

###Parameters###
@@ParamTable@@
$o_object
	一个 InternetExplorer.Application, 窗口或者框架(Frame)对象的对象变量
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回文档中 &lt;body&gt; 标记中的HTML.
失败:	返回 0 并设置 @ERROR
@Error:	0 ($_IEStatus_Success) = 没有错误
	3 ($_IEStatus_InvalidDataType) = 无效数据类型
@Extended:	包含错误参数的数量
@@End@@


###Remarks###
这个函数使用文档主体(body)的 .innerHTML属性. .innerHTML 可用于几乎所有使用_IEPropertyGet的DOM元素.
