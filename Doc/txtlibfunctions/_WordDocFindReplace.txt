###User Defined Function###
_WordDocFindReplace

###Description###
Runs the specified find and replace operation.

###Syntax###
#include <Word.au3> 
_WordDocFindReplace ( ByRef $o_object [, $s_FindText = "" [, $s_ReplaceWith = "" [, $i_Replace = 2 [, $v_SearchRange = 0 [, $b_MatchCase = 0 [, $b_MatchWholeWord = 0 [, $b_MatchWildcards = 0 [, $b_MatchSoundsLike = 0 [, $b_MatchAllWordForms = 0 [, $b_Forward = 1 [, $i_Wrap = 1 [, $b_Format = 0]]]]]]]]]]]] )

###Parameters###
@@ParamTable@@
$o_object
	Object variable of a Word.Application, document object
$s_FindText
	<b>Optional:</b> The text to be searched for. (See Remarks)
	"" = (Default) Used to search for formatting only.
$s_ReplaceWith
	<b>Optional:</b> The replacement text. (See Remarks)
	"" = (Default) Delete the text specified by $s_FindText
$i_Replace
	<b>Optional:</b> Specifies how many replacements are to be made.
	0 = Replace no occurrences
	1 = Replace the first occurrence encountered
	2 = (Default) Replace all occurrences
$v_SearchRange
	<b>Optional:</b> Specifies the Selection or Range to search.
	-1 = Specifies the current selection
	0 = (Default) Specifies the entire document
	Any range object
$b_MatchCase
	<b>Optional:</b> Specifies whether the find text will be case-sensitive.
	0 = (Default) Not case-sensitive
	1 = Case-sensitive
$b_MatchWholeWord
	<b>Optional:</b> Specifies whether to have the find operation locate only entire words, not text that's part of a larger word.
	0 = (Default) Match partial words
	1 = Only match entire words
$b_MatchWildcards
	<b>Optional:</b> Specifies whether to have $s_FindText be a special search operator.
	0 = (Default) Not a special search operator
	1 = Special search operator
$b_MatchSoundsLike
	<b>Optional:</b> Specifies whether to have the find operation locate words that sound similar to $s_FindText.
	0 = (Default) Do not find similar sounding words
	1 = Find similar sounding words
$b_MatchAllWordForms
	<b>Optional:</b> Specifies whether to have the find operation locate all forms of the find text (for example, "sit" locates "sitting" and "sat").
	0 = (Default) Do not match other word forms
	1 = Match all word forms
$b_Forward
	<b>Optional:</b> Specifies which direction to search.
	0 = Search backward (toward the start of the document)
	1 = (Default) Search forward (toward the end of the document)
$i_Wrap
	<b>Optional:</b> Controls what happens if the search begins at a point other than the beginning of the document and the end of the document is reached (or vice versa if $b_Forward is set to 0).
	0 = The find operation ends if the beginning or end of the search range is reached
	1 = (Default) The find operation continues if the beginning or end of the search range is reached
$b_Format
	<b>Optional:</b> Specifies whether to have the find operation locate formatting in addition to or instead of the find text.
	0 = (Default) Do not locate formatting
	1 = Locate formatting
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	Returns 1
Failure:	Retuns 0  and sets @ERROR = 1
@Error:	0 ($_WordStatus_Success) = No Error
	3 ($_WordStatus_InvalidDataType) = Invalid Data Type
	4 ($_WordStatus_InvalidObjectType) = Invalid Object Type
	5 ($_WordStatus_InvalidValue) = Invalid Value
	7 ($_WordStatus_NoMatch) = No Match
@Extended:	Contains invalid parameter number
@@End@@


###Remarks###
You can search for special characters by specifying appropriate character codes. For example, "^p" corresponds to a paragraph mark and "^t" corresponds to a tab character.

You specify special characters and advanced search criteria just as you do for the Find argument. To specify a graphic object or other non-text item as the replacement, move the item to the Clipboard and specify "^c" for $s_ReplaceWith.
