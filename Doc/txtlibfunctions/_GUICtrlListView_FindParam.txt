###User Defined Function###
_GUICtrlListView_FindParam

###Description###
Searches for an item with the specified lParam

###Syntax###
#Include <GuiListView.au3>
_GUICtrlListView_FindParam($hWnd, $iParam[, $iStart = -1])

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$iParam
	Param value to search for
$iStart
	Zero based index of the item to begin the search with or -1 to start from the beginning. The
		specified item is itself excluded from the search.
@@End@@

###ReturnValue###
Success: The zero based index of the item
Failure: -1

###Remarks###
None.
