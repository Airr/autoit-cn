###User Defined Function###
_GUIImageList_Add

###Description###
Adds an image or images to an image list

###Syntax###
#Include <GuiImageList.au3>
_GUIImageList_Add($hWnd, $hImage[, $hMask=0])

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$hImage
	Handle to the bitmap that contains the image or images. The number of images is inferred from
		the width of the bitmap.
$hMask
	Handle to the bitmap that contains the mask
@@End@@

###ReturnValue###
Success: The index of the first new image
Failure: -1

###Remarks###
This function copies the bitmap to an internal data structure. Be sure to use the _WinAPI_DeleteObject function
to delete hImage and hMask after the function returns.
