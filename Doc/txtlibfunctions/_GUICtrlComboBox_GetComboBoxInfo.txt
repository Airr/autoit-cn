###User Defined Function###
_GUICtrlComboBox_GetComboBoxInfo

###Description###
得到关于指定组合框(ComboBox)的信息

###Syntax###
#Include <GuiComboBox.au3>
_GUICtrlComboBox_GetComboBoxInfo($hWnd, ByRef $tInfo)

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
$tInfo
	定义于 $tagCOMBOBOXINFO 的信息
@@End@@

###ReturnValue###
成功: True
失败: False

###Remarks###
最低支持的系统: Windows XP
