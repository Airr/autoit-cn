###User Defined Function###
_GUICtrlRebar_GetBandStyleHideTitle

###Description###
Determine if flag is set

###Syntax###
#Include <GuiRebar.au3>
_GUICtrlRebar_GetBandStyleHideTitle($hWnd, $iIndex)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to rebar control
$iIndex
	Zero-based index of the band
@@End@@

###ReturnValue###
True: Flag is set (Keep band title hidden)
False: Flag not set

###Remarks###
None.
