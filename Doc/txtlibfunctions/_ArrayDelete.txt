###User Defined Function###
_ArrayDelete

###Description###
从数组中删除指定的元素.

###Syntax###
#Include <Array.au3>
_ArrayDelete(ByRef $avArray, $iElement)

###Parameters###
@@ParamTable@@
$avArray
	需要修改的数组
$iElement
	需要删除的元素
@@End@@

###ReturnValue###
成功: 数组新的大小
失败: 0, 设置 @error 为:
		1 - $avArray 不是一个数组.
		3 - $avArray 有太多的维度 (最高支持二维数组)
		(2 - 忽略的错误代码)

###Remarks###
如果数组向左(或者二维数组的一行)有一个元素，在_ArrayDelete()执行之后，它将被设定成""
译注：留存原文[If the array has one element left (or one row for 2D arrays), it will be set to "" after _ArrayDelete() is used on it.]
