###User Defined Function###
_SoundLength

###Description###
返回声音文件的长度.

###Syntax###
#include <Sound.au3>
_SoundLength ( $sSnd_id [, $iMode ] )


###Parameters###
@@ParamTable@@
$sSnd_id
	Sound ID (the 'alias') as returned by _SoundOpen or a file
$iMode
	[optional] This flag determines which format the length of the sound is returned in
	1 = hh:mm:ss where h = hours, m = minutes and s = seconds
	2 = milliseconds
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	Sound Length
Failure:	0
@Error:	0 = No Error
	1 = $iMode is invalid
	3 = Invalid Sound ID. May be _SoundOpen with alias needed.
@@End@@

###Remarks###
None.
