###User Defined Function###
_INetGetSource

###Description###
Gets the source from an URL without writing a temp file.

###Syntax###
#include <INet.au3>
_INetGetSource ( $s_URL )


###Parameters###
@@ParamTable@@
$s_URL
	(The URL of the site.) eg 'www.autoitscript.com'
$nOptions
	InetRead() options to use.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	Returns the source code.
Failure:	An empty string.
@@End@@


###Remarks###
None.
