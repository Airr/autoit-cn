###User Defined Function###
_GUICtrlComboBoxEx_SetItemOverlayImage

###Description###
Sets the index of the item's overlay icon in the control's image list

###Syntax###
#Include <GuiComboBoxEx.au3>
_GUICtrlComboBoxEx_SetItemOverlayImage($hWnd, $iIndex, $iImage)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$iIndex
	Zero based index of the item
$iImage
	Zero based index into the control's image list
@@End@@

###ReturnValue###
成功: True
失败: False

###Remarks###
None.
