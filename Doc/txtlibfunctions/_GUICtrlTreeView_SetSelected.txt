###User Defined Function###
_GUICtrlTreeView_SetSelected

###Description###
Sets whether the item appears in the selected state

###Syntax###
#Include <GuiTreeView.au3>
_GUICtrlTreeView_SetSelected($hWnd, $hItem[, $fFlag = True])

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$hItem
	Handle to the item
$fFlag
	True if item is to be selected, otherwise False
@@End@@

###ReturnValue###
成功: True
失败: False

###Remarks###
None.
