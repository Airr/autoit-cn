###User Defined Function###
_WinAPI_FindExecutable

###Description###
Retrieves the name of the executable file associated with the specified file name

###Syntax###
#Include <WinAPI.au3>
_WinAPI_FindExecutable($sFileName[, $sDirectory = ""])

###Parameters###
@@ParamTable@@
$sFileName
	Fully qualified path to existing file
$sDirectory
	Default directory
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	Full path to the executable file started when an "open" by association is run on the file
	specified or blank if no association was found.
@@End@@
