###User Defined Function###
_FTP_DirGetCurrent

###Description###
Get Current Directory on an FTP server.

###Syntax###
#Include <FTPEx.au3>
_FTP_DirGetCurrent($l_FTPSession)

###Parameters###
@@ParamTable@@
$l_FTPSession
	as returned by _FTP_Connect().
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	Directory Name
Failure:	0 and set @error
@@End@@

###Remarks###
None.
