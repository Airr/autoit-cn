###User Defined Function###
_GUICtrlMenu_AddMenuItem

###Description###
Adds a new menu item to the end of the menu

###Syntax###
#Include <GuiMenu.au3>
_GUICtrlMenu_AddMenuItem($hMenu, $sText[, $iCmdID = 0[, $hSubMenu = 0]])

###Parameters###
@@ParamTable@@
$hMenu
	Handle of the menu
$sText
	Menu item text. If blank, a separator will be added.
$iCmdID
	Command ID to assign to the item
$hSubMenu
	Handle to the submenu associated with the menu item
@@End@@

###ReturnValue###
Success: Zero based menu item index
Failure: -1

###Remarks###
None.
