###User Defined Function###
_GDIPlus_BitmapCreateFromFile

###Description###
Create a Bitmap object from file

###Syntax###
#Include <GDIPlus.au3>
_GDIPlus_BitmapCreateFromFile($sFileName)

###Parameters###
@@ParamTable@@
$sFileName
	Path to a bitmap file
@@End@@

###ReturnValue###
Success: Handle to a Bitmap object
Failure: 0

###Remarks###
When you are done with the Bitmap object, call _WinAPI_DeleteObject to release the resources
