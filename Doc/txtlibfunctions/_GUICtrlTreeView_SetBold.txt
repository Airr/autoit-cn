###User Defined Function###
_GUICtrlTreeView_SetBold

###Description###
Sets whether the item is drawn using a bold sytle

###Syntax###
#Include <GuiTreeView.au3>
_GUICtrlTreeView_SetBold($hWnd, $hItem[, $fFlag = True])

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$hItem
	Handle to the item
$fFlag
	True if item is drawn bold, otherwise False
@@End@@

###ReturnValue###
Success: True
Failure: False

###Remarks###
None.
