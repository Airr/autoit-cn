###User Defined Function###
_ExcelWriteFormula

###Description###
Write a formula to a cell on the active worksheet of the specified Excel object.

###Syntax###
#Include <Excel.au3>
_ExcelWriteFormula($oExcel, $sFormula, $sRangeOrRow [, $iColumn = 1])

###Parameters###
@@ParamTable@@
$oExcel
	Excel object opened by a preceding call to _ExcelBookOpen() or _ExcelBookNew()
$sFormula
	Formula to be written
$sRangeOrRow
	Either an A1 range, or an integer row number to write to if using R1C1
$iColumn
	[optional] The column to write to if using R1C1 (default = 1)
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	1
Failure:	0 and sets @error
@error:	0 - No error
	1 - Specified object does not exist
	2 - Parameter out of range
@extended:	0 - Row out of range
	1 - Column out of range
@@End@@
