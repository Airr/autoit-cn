###User Defined Function###
_HexToString

###Description###
Convert a hex string to a string.

###Syntax###
#Include <String.au3>
_HexToString($strHex)

###Parameters###
@@ParamTable@@
$strHex
	an hexadecimal string
@@End@@

###ReturnValue###
Success: Returns a string.
Failure: Returns -1 and sets @error to 1.
