###User Defined Function###
_WinAPI_CloseHandle

###Description###
Closes an open object handle

###Syntax###
#Include <WinAPI.au3>
_WinAPI_CloseHandle($hObject)

###Parameters###
@@ParamTable@@
$hObject
	Handle of object to close
@@End@@

###ReturnValue###
Success: True
Failure: False

###Remarks###
None.
