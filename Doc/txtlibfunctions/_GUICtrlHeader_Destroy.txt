###User Defined Function###
_GUICtrlHeader_Destroy

###Description###
Delete the Header control

###Syntax###
#Include <GuiHeader.au3>
_GUICtrlHeader_Destroy(ByRef $hWnd)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
@@End@@

###ReturnValue###
Success: True, Handle is set to 0
Failure: False

###Remarks###
Restricted to only be used on Edit created with _GUICtrlHeader_Create
