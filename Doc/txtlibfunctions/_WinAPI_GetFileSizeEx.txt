###User Defined Function###
_WinAPI_GetFileSizeEx

###Description###
Retrieves the size of the specified file

###Syntax###
#Include <WinAPI.au3>
_WinAPI_GetFileSizeEx($hFile)

###Parameters###
@@ParamTable@@
$hFile
	Handle to the file whose size is to be returned
@@End@@

###ReturnValue###
Success: File size

###Remarks###
None.
