###User Defined Function###
_Assert

###Description###
Display a message if assertion fails.

###Syntax###
#include <Debug.au3>
_Assert($sCondition [,$bExit = True [,$nCode = 0x7FFFFFFF [,$sLine = @ScriptLineNumber]]])


###Parameters###
@@ParamTable@@
$sCondition
	The condition (expression quoted) that must evaluate to true.
$bExit
	<b>Optional:</b> If true, the script is aborted.
$nCode
	<b>Optional:</b> The exit code to use if the script is aborted.
$sLine
	<b>Optional:</b> Displays the line number where the assertion failed.  If this value is not changed, then the default value will show the correct line.
@@End@@

###ReturnValue###
The result of the condition (Only valid when not exiting).

###Remarks###
@error and @extended are not destroyed on return.  


###Related###
None.
