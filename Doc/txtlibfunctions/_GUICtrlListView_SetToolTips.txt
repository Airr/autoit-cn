###User Defined Function###
_GUICtrlListView_SetToolTips

###Description###
Sets the ToolTip control that the control will use to display ToolTips

###Syntax###
#Include <GuiListView.au3>
_GUICtrlListView_SetToolTips($hWnd, $hToolTip)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$hToolTip
	Handle to the ToolTip control to be set
@@End@@

###ReturnValue###
Returns the handle to the previous ToolTip control

###Remarks###
None.
