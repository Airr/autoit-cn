###User Defined Function###
_GUICtrlMenu_GetItemID

###Description###
Retrieves the menu item ID

###Syntax###
#Include <GuiMenu.au3>
_GUICtrlMenu_GetItemID($hMenu, $iItem[, $fByPos = True])

###Parameters###
@@ParamTable@@
$hMenu
	Handle of the menu
$iItem
	Identifier or position of the menu item
$fByPos
	Menu identifier flag:
		 True - $iItem is a zero based item position
		False - $iItem is a menu item identifier
@@End@@

###ReturnValue###
Success: Application defined value associated with the menu item

###Remarks###
None.
