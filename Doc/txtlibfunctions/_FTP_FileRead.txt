###User Defined Function###
_FTP_FileRead

###Description###
Reads data from a handle opened by _FTP_FileOpen()

###Syntax###
#Include <FTPEx.au3>
_FTP_FileRead($h_File, $dwNumberOfBytesToRead)

###Parameters###
@@ParamTable@@
$h_File
	Handle returned by _FTP_FileOpen() to the ftp file.
$dwNumberOfBytesToRead
	Number of bytes to read.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	Returns the binary/string read.
Failure:	0 and Sets @error = -1 for end-of-file, @error to non-zero for other errors.
@@End@@

###Remarks###
None.

_FTP_FileOpen
