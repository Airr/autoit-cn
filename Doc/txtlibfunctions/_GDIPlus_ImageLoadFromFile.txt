###User Defined Function###
_GDIPlus_ImageLoadFromFile

###Description###
Create an image object based on a file

###Syntax###
#Include <GDIPlus.au3>
_GDIPlus_ImageLoadFromFile($sFileName)

###Parameters###
@@ParamTable@@
$sFileName
	Fully qualified image file name
@@End@@

###ReturnValue###
Success: Handle to the new image object
Failure: -1 and @error is set

###Remarks###
None.
