###User Defined Function###
_ClipBoard_IsFormatAvailable

###Description###
Determines whether the clipboard contains data in the specified format

###Syntax###
#Include <Clipboard.au3>
_ClipBoard_IsFormatAvailable($iFormat)

###Parameters###
@@ParamTable@@
$iFormat
	Specifies a standard or registered clipboard format
@@End@@

###ReturnValue###
True: Clipboard contains data in the specified format
False: Clipboard does not contain data in the specified format

###Remarks###
None.
