###User Defined Function###
_WinAPI_HiWord

###Description###
Returns the high word of a longword value

###Syntax###
#Include <WinAPI.au3>
_WinAPI_HiWord($iLong)

###Parameters###
@@ParamTable@@
$iLong
	Longword value
@@End@@

###ReturnValue###
Returns an High order word of the longword value
