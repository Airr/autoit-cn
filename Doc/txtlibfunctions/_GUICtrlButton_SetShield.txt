###User Defined Function###
_GUICtrlButton_SetShield

###Description###
设置指定按钮的必需状态或与命令相关的图标

###Syntax###
#Include <GuiButton.au3>
_GUICtrlButton_SetShield($hWnd[, $fRequired = True])

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
$fRequired
	需有时则设置布尔真 True, 否则布尔假 False
@@End@@

###ReturnValue###
成功: 布尔真 True
失败: 布尔假 False

###Remarks###
最低操作系统: Windows Vista
