###User Defined Function###
_EventLog__Full

###Description###
Retrieves whether the event log is full

###Syntax###
#Include <EventLog.au3>
_EventLog__Full($hEventLog)

###Parameters###
@@ParamTable@@
$hEventLog
	A handle to the event log
@@End@@

###ReturnValue###
True: Event log is full
False: Event log is not full

###Remarks###
None.
