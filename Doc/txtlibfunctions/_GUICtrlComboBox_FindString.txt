###User Defined Function###
_GUICtrlComboBox_FindString

###Description###
查找字符串

###Syntax###
#Include <GuiComboBox.au3>
_GUICtrlComboBox_FindString($hWnd, $sText[, $iIndex = -1])

###Parameters###
@@ParamTable@@
$hWnd
	Handle to control
$sText
	String to search for
$iIndex
	Zero based index of the item preceding the first item to be searched
@@End@@

###ReturnValue###
Success: Zero based index of the matching item
Failure: -1

###Remarks###
Finds the first string beginning with the characters specified in $sText

When the search reaches the bottom of the ListBox, it continues from the top of the
ListBox back to the item specified by $iIndex.

If $iIndex is –1, the entire ListBox is searched from the beginning.
