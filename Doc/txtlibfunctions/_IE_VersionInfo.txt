###User Defined Function###
_IE_VersionInfo

###Description###
返回一个包含 IE.au3 版本信息的数组

###Syntax###
#include <IE.au3> 
_IE_VersionInfo ( )

###Parameters###
None.

###ReturnValue###
@@ReturnTable@@
成功:	返回一个数组 ($IEAU3VersionInfo)
	$IEAU3VersionInfo[0] = 发布类型 (T=测试 或者 V=正式)
	$IEAU3VersionInfo[1] = 主版本
	$IEAU3VersionInfo[2] = 修正版本
	$IEAU3VersionInfo[3] = 子版本
	$IEAU3VersionInfo[4] = 发布日期 (YYYYMMDD)
	$IEAU3VersionInfo[5] = 显示版本 (例如 V2.1-0)
失败:	None
@@End@@


###Remarks###
The array contents make it easy to determine of the UDF version is higher or lower than a required version.
