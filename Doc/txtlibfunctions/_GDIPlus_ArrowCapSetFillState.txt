###User Defined Function###
_GDIPlus_ArrowCapSetFillState

###Description###
Sets whether the arrow cap is filled

###Syntax###
#Include <GDIPlus.au3>
_GDIPlus_ArrowCapSetFillState($hArrowCap, $fFilled = True)

###Parameters###
@@ParamTable@@
$hArrowCap
	Handle to a ArrowCap object
$fFilled
	Fill state:
		 True - Arrow cap is filled
		False - Arrow cap is not filled
@@End@@

###ReturnValue###
Success: True
Failure: False

###Remarks###
None.
