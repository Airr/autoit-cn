###User Defined Function###
_FTP_FileGetSize

###Description###
Gets filesize of a file on the FTP server.

###Syntax###
#Include <FTPEx.au3>
_FTP_FileGetSize($l_FTPSession, $s_FileName)

###Parameters###
@@ParamTable@@
$l_FTPSession
	as returned by _FTP_Connect().
$s_FileName
	The file name.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	returns filesize as an uint64.
Failure:	sets @error non-zero
@@End@@

###Remarks###
None.

_FTP_Connect
