###User Defined Function###
_SQLite_Display2DResult

###Description###
Returns or prints a 2Dimensional array formated to Console

###Syntax###
#include <SQLite.au3>
_SQLite_Display2DResult ( $aResult [, $iCellWidth = 0 [, $fReturn = False ]] )


###Parameters###
@@ParamTable@@
$aResult
	The array to be displayed
$iCellWidth
	[optional] specifies the size of a data field
$fReturn
	[optional] if true the formated string is returned
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	none or formated string
Failure:	@error = 1 - $aResult is no array or has wrong dimension
@@End@@

###Remarks###
None.
