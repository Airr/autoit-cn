###User Defined Function###
_SQLite_Display2DResult

###Description###
Returns or prints a 2Dimensional array formated to Console

###Syntax###
#include <SQLite.au3>
_SQLite_Display2DResult ( $aResult [, $iCellWidth = 0 [, $nReturn = 0 ]] )


###Parameters###
@@ParamTable@@
$aResult
	The array to be displayed
$iCellWidth
	[optional] specifies the size of a data field
$nReturn
	[optional] if 1 the formated string is returned
	if 0 ANSI formated string send to StdOut by ConsoleWrite
	other see remarks
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	none or formated string
Failure:	@error = 1 - $aResult is no array or has wrong dimension
@@End@@

###Remarks###
If $nReturn is different from 1 and 0 prints UTF8 formated string prints to console.
This allows application as SCiTE to display those characters if configures with output.code.page=65001.
This is the case when running SCiTE under a non-english version as Vista(fr).
