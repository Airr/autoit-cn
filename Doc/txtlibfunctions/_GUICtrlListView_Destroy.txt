###User Defined Function###
_GUICtrlListView_Destroy

###Description###
Delete the listview control

###Syntax###
#Include <GuiListView.au3>
_GUICtrlListView_Destroy(ByRef $hWnd)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
@@End@@

###ReturnValue###
成功: True, 句柄设置为 0
失败: False

###Remarks###
Restricted to only be used on ListViews created with _GUICtrlListView_Create
