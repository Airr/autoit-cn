###User Defined Function###
_GUICtrlMenu_SetItemText

###Description###
设置菜单项目文本

###Syntax###
#Include <GuiMenu.au3>
_GUICtrlMenu_SetItemText($hMenu, $iItem, $sText[, $fByPos = True])

###Parameters###
@@ParamTable@@
$hMenu
	菜单句柄
$iItem
	菜单项目ID或者坐标
$sText
	菜单项目文本
$fByPos
	[可选参数] 菜单ID标志:
		 True - $iItem 为一个基于0开始的项目坐标
		False - $iItem 为一个菜单项目ID(句柄)
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	True
失败:	False
@@End@@

###Remarks###
None.
