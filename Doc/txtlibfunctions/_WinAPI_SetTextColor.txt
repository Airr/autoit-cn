###User Defined Function###
_WinAPI_SetTextColor

###Description###
Sets the current text color to the specified color value

###Syntax###
#Include <WinAPI.au3>
_WinAPI_SetTextColor($hDC, $iColor)

###Parameters###
@@ParamTable@@
$hDC
	Handle to the device context
$iColor
	Specifies the new text color
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	The previous text color
Failure	0xFFFF
@@End@@
