###User Defined Function###
_SoundClose

###Description###
Closes a sound previously opened with _SoundOpen.

###Syntax###
#include <Sound.au3>
_SoundClose ( $aSnd_id )


###Parameters###
@@ParamTable@@
$aSnd_id
	Sound ID array as returned by _SoundOpen
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	1
Failure:	0 and set @error
	1 = Failure
	3 = Invalid Sound ID. Use return array from _SoundOpen.
@@End@@


###Remarks###
None.
