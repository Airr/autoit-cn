###User Defined Function###
_WinAPI_GUIDFromStringEx

###Description###
Converts a string GUID to binary form

###Syntax###
#Include <WinAPI.au3>
_WinAPI_GUIDFromStringEx($sGUID, $pGUID)

###Parameters###
@@ParamTable@@
$sGUID
	GUID in string form
$pGUID
	Pointer to a $tagGUID structure where the GUID will be stored
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True
Failure:	False
@@End@@

###Remarks###
None.
