###User Defined Function###
_GUICtrlButton_SetSplitInfo

###Description###
获取具有拆分标志的按钮控件信息 (需 Windows Vista 支持 ）

###Syntax###
#Include <GuiButton.au3>
_GUICtrlButton_SetSplitInfo($hWnd[, $himlGlyph = -1[, $iSplitStyle = $BCSS_ALIGNLEFT[, $iWidth = 0[, $iHeight = 0]]]])

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
$himlGlyph
	图像列表句柄
$iSplitStyle
	拆分标志按钮样式. 可以是下列标志之一或多项.
		  $BCSS_ALIGNLEFT - 图标或拆分标志水平居中
		  $BCSS_IMAGE - 以ico 图标作为拆分标志
		  $BCSS_NOSPLIT - 没有拆分标志
		  $BCSS_STRETCH - 伸展拆分标志, 但必须保持高宽比
$iWidth
	拆分标志宽度
$iHeight
	拆分标志高度
@@End@@

###ReturnValue###
成功: 布尔真 True
失败: 布尔假 False

###Remarks###
这个函数只能作用于 $BS_SPLITBUTTON and $BS_DEFSPLITBUTTON 样式的按钮

最低操作系统: Windows Vista
