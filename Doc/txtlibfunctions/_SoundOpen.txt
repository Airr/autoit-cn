###User Defined Function###
_SoundOpen

###Description###
打开一个声音文件,用于其它 _Sound 函数.

###Syntax###
#include <Sound.au3>
_SoundOpen ( $hFile [, $sAlias ] )


###Parameters###
@@ParamTable@@
$hFile
	声音文件路径
$sAlias
	可选 - 可以让其它函数调用的声音别名..
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	声音 ID
失败:	返回0
@Error:	0 = 无错误
	2 = 文件不存在
	3 = 别名包含空格或者符号(原帮助只说感叹号)
@Extended:	0 = 无错误
	其它超过 0 的整数 = MCI 错误
@@End@@

None.
