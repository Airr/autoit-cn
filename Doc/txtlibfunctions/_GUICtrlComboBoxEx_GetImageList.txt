###User Defined Function###
_GUICtrlComboBoxEx_GetImageList

###Description###
Retrieves the handle to an image list assigned to a ComboBoxEx control

###Syntax###
#Include <GuiComboBoxEx.au3>
_GUICtrlComboBoxEx_GetImageList($hWnd)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
@@End@@

###ReturnValue###
Success: The handle to the image list
Failure: 0

None.
