###User Defined Function###
_WinAPI_FreeLibrary

###Description###
Decrements the reference count of the loaded dynamic-link library (DLL) module

###Syntax###
#Include <WinAPI.au3>
_WinAPI_FreeLibrary($hModule)

###Parameters###
@@ParamTable@@
$hModule
	Identifies the loaded library module
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	True
Failure	False
@@End@@

None.
