###User Defined Function###
_GUICtrlMenu_DestroyMenu

###Description###
Destroys the specified menu and frees any memory that the menu occupies

###Syntax###
#Include <GuiMenu.au3>
_GUICtrlMenu_DestroyMenu($hMenu)

###Parameters###
@@ParamTable@@
$hMenu
	Menu handle
@@End@@

###ReturnValue###
成功: True
失败: False

###Remarks###
None.
