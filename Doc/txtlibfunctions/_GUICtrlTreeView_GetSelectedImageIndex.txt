###User Defined Function###
_GUICtrlTreeView_GetSelectedImageIndex

###Description###
Retrieves the index in the image list of the image displayed for the item when it is selected

###Syntax###
#Include <GuiTreeView.au3>
_GUICtrlTreeView_GetSelectedImageIndex($hWnd, $hItem)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$hItem
	Handle to the item
@@End@@

###ReturnValue###
Success: Selected list index

###Remarks###
None.
