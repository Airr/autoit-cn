###User Defined Function###
_GUICtrlListView_ClickItem

###Description###
Clicks an item

###Syntax###
#Include <GuiListView.au3>
_GUICtrlListView_ClickItem($hWnd, $iIndex[, $sButton = "left"[, $fMove = False[, $iClicks = 1[, $iSpeed = 1]]]])

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$iIndex
	Zero based index of the item
$sButton
	[optional] Button to click
$fMove
	[optional] If True, the mouse will be moved. If False, the mouse does not move.
$iClicks
	[optional] Number of clicks
$iSpeed
	[optional] Delay between clicks
@@End@@

###ReturnValue###
None.

###Remarks###
None.
