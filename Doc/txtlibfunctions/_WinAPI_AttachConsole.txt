###User Defined Function###
_WinAPI_AttachConsole

###Description###
Attaches the calling process to the console of the specified process

###Syntax###
#Include <WinAPI.au3>
_WinAPI_AttachConsole([$iProcessID = -1])

###Parameters###
@@ParamTable@@
$iProcessID
	Identifier of the process. Set to -1 to attach to the current process.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	True
Failure	False
@@End@@

None.
