###User Defined Function###
_GUICtrlListView_GetColumnWidth

###Description###
Retrieves the width of a column in report or list view

###Syntax###
#Include <GuiListView.au3>
_GUICtrlListView_GetColumnWidth($hWnd, $iCol)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$iCol
	The index of the column. This parameter is ignored in list view.
@@End@@

###ReturnValue###
Success: Column width
Failure: Zero

###Remarks###
If this message is sent to a list-view control with the $LVS_REPORT style
and the specified column doesn't exist, the return value is undefined.
