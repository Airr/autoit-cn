###User Defined Function###
_GUICtrlListBox_SetItemHeight

###Description###
Sets the height, in pixels, of items

###Syntax###
#Include <GuiListBox.au3>
_GUICtrlListBox_SetItemHeight($hWnd, $iHeight[, $iIndex = 0])

###Parameters###
@@ParamTable@@
$hWnd
	Handle to control
$iHeight
	Specifies the height, in pixels, of the item
$iIndex
	Specifies the zero based index of the item in the list box. Use this only if the control has
		the LBS_OWNERDRAWVARIABLE style otherwise, set it to zero.
@@End@@

###ReturnValue###
Success: True
Failure: False

###Remarks###
None.
