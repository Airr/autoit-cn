###User Defined Function###
_GDIPlus_PenGetCustomEndCap

###Description###
Gets the custom end cap for the pen

###Syntax###
#Include <GDIPlus.au3>
_GDIPlus_PenGetCustomEndCap($hPen)

###Parameters###
@@ParamTable@@
$hPen
	Handle to a pen object
@@End@@

###ReturnValue###
Success: Handle to a CustomLineCap object that specifies the pen custom end cap
Failure: 0

###Remarks###
None.
