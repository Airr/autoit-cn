###User Defined Function###
_GUICtrlTreeView_SetIcon

###Description###
Set an item icon

###Syntax###
#Include <GuiTreeView.au3>
_GUICtrlTreeView_SetIcon($hWnd[, $hItem = 0[, $sIconFile =""[, $iIconID = 0[, $iImageMode = 6]]]])

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$hItem
	item ID/handle
$sIconFile
	The file to extract the icon of
$iIconID
	The iconID to extract of the file
$iImageMode
	2=normal image / 4=seletected image to set
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True
Failure:	False
@@End@@
