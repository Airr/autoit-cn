###User Defined Function###
_GUICtrlStatusBar_SetIcon

###Description###
Sets the icon for a part

###Syntax###
#Include <GuiStatusBar.au3>
_GUICtrlStatusBar_SetIcon($hWnd, $iPart[, $hIcon = -1[, $sIconFile = ""]])

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$iPart
	Zero based part index. If the control is in simple mode, this field is ignored.
$hIcon
	Handle to the icon. If this value is -1, the icon is removed.
$sIconFile
	Icon filename to be used.
@@End@@

###ReturnValue###
成功: True
失败: False

###Remarks###
None.
