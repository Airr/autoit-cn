###User Defined Function###
_FTP_ListToArray

###Description###
Get Filenames, Directories or Both of current remote directory.

###Syntax###
#Include <FTPEx.au3>
_Ftp_ListToArray($l_FTPSession [, $Return_Type = 0 [, $l_Flags = 0]])

###Parameters###
@@ParamTable@@
$l_FTPSession
	as returned by _FTP_Connect().
$Return_type
	<b>Optional:</b> 0 = Both Files and Directories, 1 = Directories, 2 = Files.
$l_Flags
	<b>Optional:</b> see _FTP_FindFileFirst().
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	An array containing the names. Array[0] contain the number of found entries.
Failure:	Array[0] = 0
@@End@@

###Remarks###
None.
