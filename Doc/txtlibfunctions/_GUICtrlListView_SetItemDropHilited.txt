###User Defined Function###
_GUICtrlListView_SetItemDropHilited

###Description###
Sets whether the item is highlighted as a drag-and-drop target

###Syntax###
#Include <GuiListView.au3>
_GUICtrlListView_SetItemDropHilited($hWnd, $iIndex[, $fEnabled = True])

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$iIndex
	Zero based index of the item
$fEnabled
	If True the item state is set, otherwise it is not set.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True
Failure:	False
@@End@@

###Remarks###
None.
