###User Defined Function###
_StringAddThousandsSep

###Description###
Returns the original numbered string with the Thousands and or Decimal delimiter(s) inserted.

###Syntax###
#Include <String.au3>
_StringAddThousandsSep($s_string[, $i_convert_lcid = -1[, $i_current_lcid = -1]])

###Parameters###
@@ParamTable@@
$sString
	要转换的字符串.
$sThousands
	Optional: Default or -1 wll be the User default locale else LCID of what you want to convert number to.
$sDecimal
	Optional: Default or -1 wll be the User default locale else LCID number was converted with originally.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success - The string with Thousands delimiter added.
Failure - Returns empty string and sets @error to 1.
@@End@@
