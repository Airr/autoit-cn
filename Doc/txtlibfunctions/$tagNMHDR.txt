###Structure Name###
$tagNMHDR

###Description###
Contains information about a notification message

###Syntax###
Global Const $tagNMHDR = "hwnd hWndFrom;int IDFrom;int Code"

###Fields###
@@ParamTable@@
hWndFrom
	Window handle to the control sending a message
IDFrom
	Identifier of the control sending a message
Code
	Notification code
@@End@@

###Remarks###
None.
