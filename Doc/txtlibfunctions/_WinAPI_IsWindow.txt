###User Defined Function###
_WinAPI_IsWindow

###Description###
Determines whether the specified window handle identifies an existing window

###Syntax###
#Include <WinAPI.au3>
_WinAPI_IsWindow($hWnd)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to be tested
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	Handle is a window
Failure:	Handle is not a window
@@End@@
