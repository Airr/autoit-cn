###User Defined Function###
_FTP_DirSetCurrent

###Description###
Set Current Directory on an FTP server.

###Syntax###
#Include <FTPEx.au3>
_FTP_DirSetCurrent($l_FTPSession, $s_Remote)

###Parameters###
@@ParamTable@@
$l_FTPSession
	as returned by _FTP_Connect().
$s_Remote
	The Directory to be set.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	Directory Name
Failure:	0  and sets @ERROR
@@End@@

###Remarks###
None.

###Related###
_FTP_Connect
