###User Defined Function###
_GUICtrlDTP_Destroy

###Description###
Delete the control

###Syntax###
#Include <GuiDateTimePicker.au3>
_GUICtrlDTP_Destroy(ByRef $hWnd)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
@@End@@

###ReturnValue###
Success: True, Handle is set to 0
Failure: False

###Remarks###
Restricted to only be used on Date Time Picker create with _GUICtrlDTP_Create
