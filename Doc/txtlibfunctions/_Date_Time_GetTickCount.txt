###User Defined Function###
_Date_Time_GetTickCount

###Description###
Retrieves the number of milliseconds that have elapsed since Windows was started

###Syntax###
#Include <Date.au3>
_Date_Time_GetTickCount()

###Parameters###
@@ParamTable@@
None.

@@End@@

###ReturnValue###
Success: Number of milliseconds that have elapsed since Windows was started

###Remarks###
The elapsed time is stored as a DWORD value. The time will wrap around to zero if Windows is run continuously
for 49.7 days.
