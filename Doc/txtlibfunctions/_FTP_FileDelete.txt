###User Defined Function###
_FTP_FileDelete

###Description###
Delete an file from an FTP server.

###Syntax###
#Include <FTPEx.au3>
_FTP_FileDelete($l_FTPSession, $s_RemoteFile)

###Parameters###
@@ParamTable@@
$l_FTPSession
	as returned by _FTP_Connect()
$s_RemoteFile
	The remote Location for the file.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	1
Failure:	0  and sets @ERROR
@@End@@

###Remarks###
None.

###Related###
_FTP_Connect
