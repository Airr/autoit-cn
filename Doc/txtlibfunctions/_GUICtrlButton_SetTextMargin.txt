###User Defined Function###
_GUICtrlButton_SetTextMargin

###Description###
设置按钮文本的页边空白 （译注：示例演示中看不懂获取的到底是什么？ ）

###Syntax###
#Include <GuiButton.au3>
_GUICtrlButton_SetTextMargin($hWnd[, $iLeft = 1[, $iTop = 1[, $iRight = 1[, $iBottom = 1]]]])

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
$iLeft
	文本的左页边空白
$iTop
	文本的上页边空白
$iRight
	文本的右页边空白
$iBottom
	文本的底页边空白
@@End@@

###ReturnValue###
成功: 布尔真 True
失败: 布尔假 False

###Remarks###
最低操作系统: Windows XP
