###User Defined Function###
_GUICtrlButton_SetTextMargin

###Description###
在按钮控件上设置一个空白用于绘制文本.

###Syntax###
#Include <GuiButton.au3>
_GUICtrlButton_SetTextMargin($hWnd[, $iLeft = 1[, $iTop = 1[, $iRight = 1[, $iBottom = 1]]]])

###Parameters###
@@ParamTable@@
$hWnd
	控件句柄
$iLeft
	左方空白用于绘制文本
$iTop
	顶部空白用于绘制文本
$iRight
	右方空白用于绘制文本
$iBottom
	底部空白用于绘制文本
@@End@@

###ReturnValue###
成功: True
失败: False

###Remarks###
最低操作系统: Windows XP
