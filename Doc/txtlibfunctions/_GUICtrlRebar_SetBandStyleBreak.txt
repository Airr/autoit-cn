###User Defined Function###
_GUICtrlRebar_SetBandStyleBreak

###Description###
Set whether the band is on a new line

###Syntax###
#Include <GuiRebar.au3>
_GUICtrlRebar_SetBandStyleBreak($hWnd, $iIndex[, $fEnabled = True])

###Parameters###
@@ParamTable@@
$hWnd
	Handle to rebar control
$iIndex
	Zero-based index of the band
$fEnabled
	If True the item state is set, otherwise it is not set
@@End@@

###ReturnValue###
Success: True
Failure: False

###Remarks###
None.

_GUICtrlRebar_GetBandStyleBreak, _GUICtrlRebar_SetBandStyle
