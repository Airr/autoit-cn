###User Defined Function###
_GUICtrlMenu_SetMenuDefaultItem

###Description###
设置默认菜单项目

###Syntax###
#Include <GuiMenu.au3>
_GUICtrlMenu_SetMenuDefaultItem($hMenu, $iItem[, $fByPos = True])

###Parameters###
@@ParamTable@@
$hMenu
	菜单句柄
$iItem
	默认菜单项目的ID(标识,句柄)或者坐标,-1为没有默认菜单项目.
$fByPos
	菜单ID标志:
		 True - $iItem 为一个基于0开始的项目坐标
		False - $iItem 为一个菜单项目ID(句柄)
@@End@@

###ReturnValue###
成功: True
失败: False

###Remarks###
None.
