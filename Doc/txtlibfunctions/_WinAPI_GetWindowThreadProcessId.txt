###User Defined Function###
_WinAPI_GetWindowThreadProcessId

###Description###
Retrieves the identifier of the thread that created the specified window

###Syntax###
#Include <WinAPI.au3>
_WinAPI_GetWindowThreadProcessId($hWnd, ByRef $iPID)

###Parameters###
@@ParamTable@@
$hWnd
	Window handle
$iPID
	Process ID of the specified window
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	Thread ID of the specified window
@@End@@

None.
