###User Defined Function###
_GUICtrlToolbar_SetIndeterminate

###Description###
Sets or clears the indeterminate state of the specified button

###Syntax###
#Include <GuiToolBar.au3>
_GUICtrlToolbar_SetIndeterminate($hWnd, $iCommandID[, $fState = True])

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$iCommandID
	Button command ID
$fState
	[optional] True if indeterminate, otherwise False
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	True
失败:	False
@@End@@
