###User Defined Function###
_GUICtrlTab_SetItem

###Description###
Sets information about a tab

###Syntax###
#Include <GuiTab.au3>
_GUICtrlTab_SetItem($hWnd, $iIndex[, $sText = -1[, $iState = -1[, $iImage = -1[, $iParam = -1]]]])

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$iIndex
	Zero based item index
$sText
	[optional] Item text
$iState
	[optional] Item state
$iImage
	[optional] Zero based item image
$iParam
	[optional] Application defined data
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	True
失败:	False
@@End@@

###Remarks###
None.
