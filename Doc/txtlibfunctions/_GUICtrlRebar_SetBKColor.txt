###User Defined Function###
_GUICtrlRebar_SetBKColor

###Description###
返回一个 rebar 控件默认背景色

###Syntax###
#Include <GuiRebar.au3>
_GUICtrlRebar_SetBKColor($hWnd, $iColor)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to rebar control
$iColor
	COLORREF value that represents the new default background color
@@End@@

###ReturnValue###
Returns A COLORREF value that represents the previous default background color

###Remarks###
None.
