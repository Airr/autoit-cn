###User Defined Function###
_GUICtrlRebar_SetBandStyleGripperAlways

###Description###
Set whether the band will always have a sizing grip, even if it is the only band in the rebar

###Syntax###
#Include <GuiRebar.au3>
_GUICtrlRebar_SetBandStyleGripperAlways($hWnd, $iIndex[, $fEnabled = True])

###Parameters###
@@ParamTable@@
$hWnd
	Handle to rebar control
$iIndex
	Zero-based index of the band
$fEnabled
	If True the item state is set, otherwise it is not set
@@End@@

###ReturnValue###
Success: True
Failure: False

###Remarks###
None.
