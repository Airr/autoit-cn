###User Defined Function###
_GUICtrlTreeView_GetPrevChild

###Description###
Retrieves the previous child item of a specified item

###Syntax###
#Include <GuiTreeView.au3>
_GUICtrlTreeView_GetPrevChild($hWnd, $hItem)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$hItem
	Handle to the item
@@End@@

###ReturnValue###
Success: Handle to the previous child item
Failure: 0

None.
