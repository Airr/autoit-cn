###User Defined Function###
_GUICtrlTab_InsertItem

###Description###
Inserts a new tab

###Syntax###
#Include <GuiTab.au3>
_GUICtrlTab_InsertItem($hWnd, $iIndex, $sText[, $iImage = -1[, $iParam = 0]])

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$iIndex
	Index of the new tab
$sText
	Tab text
$iImage
	Zero based tab image index
$iParam
	Application defined data
@@End@@

###ReturnValue###
Success: The zero based index of the new tab
Failure: -1

###Remarks###
None.
