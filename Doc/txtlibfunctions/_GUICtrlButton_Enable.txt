###User Defined Function###
_GUICtrlButton_Enable

###Description###
应用或者禁用鼠标和键盘对指定按钮的输入

###Syntax###
#Include <GuiButton.au3>
_GUICtrlButton_Enable($hWnd[, $fEnable = True])

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$fEnable
	[optional] Specifies whether to enable or disable the button:
		 True - The button is enabled
		False - The button is disabled
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True
Failure:	False
@@End@@

None.
