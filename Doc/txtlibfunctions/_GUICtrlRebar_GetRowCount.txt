###User Defined Function###
_GUICtrlRebar_GetRowCount

###Description###
Retrieves the number of rows of bands in a rebar control

###Syntax###
#Include <GuiRebar.au3>
_GUICtrlRebar_GetRowCount($hWnd)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to rebar control
@@End@@

###ReturnValue###
Success: A value that represents the number of band rows in the control
