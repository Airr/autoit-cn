###User Defined Function###
_GUICtrlToolbar_SetStyleToolTips

###Description###
Sets whether the control has tooltips

###Syntax###
#Include <GuiToolBar.au3>
_GUICtrlToolbar_SetStyleToolTips($hWnd, $fState)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$fState
	True to set, false to unset
@@End@@

###ReturnValue###
Success: True
Failure: False

None.
