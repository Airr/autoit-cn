###User Defined Function###
_WinAPI_GetAsyncKeyState

###Description###
Determines whether a key is up or down at the time the function is called

###Syntax###
#Include <WinAPI.au3>
_WinAPI_GetAsyncKeyState($iKey)

###Parameters###
@@ParamTable@@
$iKey
	Key to test for
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	If the most significant bit is set the key is down, and if the least significant bit is set,
	the key was pressed after the previous call to GetAsyncKeyState. The return value is zero if a window in
	another thread or process currently has the keyboard focus.
@@End@@
