###User Defined Function###
_GUICtrlHeader_OrderToIndex

###Description###
Retrieves an index value for an item based on its order

###Syntax###
#Include <GuiHeader.au3>
_GUICtrlHeader_OrderToIndex($hWnd, $iOrder)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to control
$iOrder
	Order in which the item appears within the header control, from left to right
@@End@@

###ReturnValue###
Success: Returns the item index
Failure: iOrder

###Remarks###
None.
