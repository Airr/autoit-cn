###User Defined Function###
_SetTime

###Description###
设置系统的当前时间

###Syntax###
#Include <Date.au3>
_SetTime($iHour, $iMinute[, $iSecond = 0])

###Parameters###
@@ParamTable@@
$iHour
	the hour. Values: 0-23
$iMinute
	the minute. Values: 0-59
$iSecond
	Optional: the seconds. Values: 0-59
@@End@@

###ReturnValue###
Success: 1
Failure: 0
@Error: 0 - No error.
		1 - Failure
@extended: GetLastError()
		Error code(s): http://msdn.microsoft.com/library/default.asp?url=/library/en-us/debug/base/system_error_codes.asp

###Remarks###
If the optional parameter (iSecond) is NOT defined, the function will not change the current value!
