###User Defined Function###
_GDIPlus_MatrixCreate

###Description###
Creates and initializes a Matrix object that represents the identity matrix

###Syntax###
#Include <GDIPlus.au3>
_GDIPlus_MatrixCreate()

###Parameters###
None.


###ReturnValue###
@@ReturnTable@@
Success:	Handle to a Matrix object
Failure:	0
@@End@@

###Remarks###
When you are done with the matrix, call _GDIPlus_MatrixDispose to release the resources
