###User Defined Function###
_GUICtrlTreeView_GetEditControl

###Description###
Retrieves the handle to the edit control being used to edit a item's text

###Syntax###
#Include <GuiTreeView.au3>
_GUICtrlTreeView_GetEditControl($hWnd)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	Handle to the edit control
Failure:	0
@@End@@
