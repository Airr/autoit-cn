###User Defined Function###
_WinAPI_DeregisterShellHookWindow

###Description###
Unregisters a specified Shell window that is registered to receive Shell hook messages.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_DeregisterShellHookWindow ( $hWnd )

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the window to unregister from receiving Shell hook messages.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	1.
Failure	0 and sets the @error flag to non-zero.
@@End@@

None
