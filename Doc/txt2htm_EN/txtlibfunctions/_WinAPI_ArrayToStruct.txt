###User Defined Function###
_WinAPI_ArrayToStruct

###Description###
Converts an array of strings to the structure.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_ArrayToStruct ( Const ByRef $aData [, $iStart [, $iEnd]] )

###Parameters###
@@ParamTable@@
$aData
	The array to convert.
$iStart
	[可选参数] The index of array to start converting at.
$iEnd
	[可选参数] The index of array to stop converting at.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	The structure of the type "string1{0}string2{0}... stringN{0}{0}".
Failure	0 and sets the @error flag to non-zero.
@@End@@

###Remarks###
This function works for Unicode strings only.
