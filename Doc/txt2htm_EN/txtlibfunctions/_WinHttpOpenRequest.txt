###User Defined Function###
_WinHttpOpenRequest

###Description###
Creates an HTTP request handle.

###Syntax###
#Include <WinHttp.au3>
_WinHttpOpenRequest($hConnect [, $sVerb = Default [, $sObjectName = Default [, $sVersion = Default [, $sReferrer = Default [, $sAcceptTypes = Default [, $iFlags = Default ]]]]]])

###Parameters###
@@ParamTable@@
$hConnect
	Handle to an HTTP session returned by _WinHttpConnect().
$sVerb
	[可选参数] HTTP verb to use in the request. Default is "GET".
$sObjectName
	[可选参数] The name of the target resource of the specified HTTP verb.
$sVersion
	[可选参数] HTTP version. Default is "HTTP/1.1"
$sReferrer
	[可选参数] URL of the document from which the URL in the request $sObjectName was obtained. Default is $WINHTTP_NO_REFERER.
$sAcceptTypes
	[可选参数] Media types accepted by the client. Default is $WINHTTP_DEFAULT_ACCEPT_TYPES
$iFlags
	[可选参数] Integer that contains the Internet flag values. Default is $WINHTTP_FLAG_ESCAPE_DISABLE
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	Returns valid session handle.
Failure	Returns 0 and sets @error:
	1 - DllCall failed
@@End@@
