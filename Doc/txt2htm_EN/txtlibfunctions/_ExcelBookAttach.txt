###User Defined Function###
_ExcelBookAttach

###Description###
Attach to the first existing instance of Microsoft Excel where the search string matches based on the selected mode.

###Syntax###
#Include <Excel.au3>
_ExcelBookAttach($s_string [, $s_mode = "FilePath"])

###Parameters###
@@ParamTable@@
$s_string
	要搜索的字符串
$s_mode
	[可选参数] 指定搜索模式:
		FileName - 打开的工作表文件名称
		FilePath - (默认) 打开的工作表的完整路径
		Title - Excel 窗口标题
@@End@@

###ReturnValue###
@@ReturnTable@@
成功:	返回一个指向 Excel.Application, 工作表对象的对象变量
失败:	返回 0 并设置 @ERROR = 1
@@End@@

###Remarks###
None.
