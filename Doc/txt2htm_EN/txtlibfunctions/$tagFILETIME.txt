###Structure Name###
$tagFILETIME

###Description###
Contains the number of 100-nanosecond intervals since January 1, 1601

###Syntax###
Global Const $tagFILETIME = "struct; dword Lo;dword Hi; endstruct"

###Fields###
@@ParamTable@@
Lo
	The low order part of the file time
Hi
	The high order part of the file time
@@End@@

###Remarks###
None.
