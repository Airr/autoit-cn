###User Defined Function###
_GUICtrlMonthCal_GetRangeMinStr

###Description###
Retrieves the lower limit date range in string form

###Syntax###
#Include <GuiMonthCal.au3>
_GUICtrlMonthCal_GetRangeMinStr($hWnd[, $sFormat = "%02d/%02d/%04d"])

###Parameters###
@@ParamTable@@
$hWnd
	Handle to control
$sFormat
	[optional] StringFormat string used to format the date
@@End@@

###ReturnValue###
Success: Date in the specified format

###Remarks###
None.

###Related###
_GUICtrlMonthCal_GetRange, _GUICtrlMonthCal_GetRangeMin
