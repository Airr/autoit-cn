###User Defined Function###
_WinHttpWriteData

###Description###
Writes request data to an HTTP server.

###Syntax###
#Include <WinHttp.au3>
_WinHttpWriteData($hRequest, $vData [, $iMode = Default ])

###Parameters###
@@ParamTable@@
$hRequest
	Valid handle returned by _WinHttpSendRequest().
$vData
	Data to write.
$iMode
	[可选参数] Integer representing writing mode. Default is 0 - write ANSI string.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	Returns 1
Failure	Returns 0 and sets @error:
	1 - DllCall failed
@@End@@

###Remarks###
$vData variable is either string or binary data to write.
$iMode can have these values:
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;0 - to write ANSI string
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;1 - to write binary data
