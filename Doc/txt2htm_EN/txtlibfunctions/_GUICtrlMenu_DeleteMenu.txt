###User Defined Function###
_GUICtrlMenu_DeleteMenu

###Description###
Deletes an item from the specified menu

###Syntax###
#Include <GuiMenu.au3>
_GUICtrlMenu_DeleteMenu($hMenu, $iItem[, $fByPos = True])

###Parameters###
@@ParamTable@@
$hMenu
	Menu handle
$iItem
	Identifier or position of the menu item
$fByPos
	[optional] Menu identifier flag:
		 True - $iItem is a zero based item position
		False - $iItem is a menu item identifier
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True
Failure:	False
@@End@@

None.
