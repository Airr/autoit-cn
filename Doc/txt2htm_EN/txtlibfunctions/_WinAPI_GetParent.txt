###User Defined Function###
_WinAPI_GetParent

###Description###
Retrieves the handle of the specified child window's parent window

###Syntax###
#Include <WinAPI.au3>
_WinAPI_GetParent($hWnd)

###Parameters###
@@ParamTable@@
$hWnd
	Window handle of child window
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	The handle of the parent window
Failure:	0
@@End@@

None.
