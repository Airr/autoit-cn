###User Defined Function###
_GUICtrlTreeView_GetUnicodeFormat

###Description###
Retrieves the Unicode character format flag

###Syntax###
#Include <GuiTreeView.au3>
_GUICtrlTreeView_GetUnicodeFormat($hWnd)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
@@End@@

###ReturnValue###
@@ReturnTable@@
True:	Control is using Unicode characters
False:	Control is using ANSI characters
@@End@@

###Remarks###
None.
