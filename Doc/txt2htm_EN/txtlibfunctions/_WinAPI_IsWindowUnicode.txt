###User Defined Function###
_WinAPI_IsWindowUnicode

###Description###
Determines whether the specified window is a native Unicode window.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_IsWindowUnicode ( $hWnd )

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the window to test.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	1 - The window is a native Unicode window.
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;0	Otherwise.
Failure	0 and sets the @error flag to non-zero.
@@End@@
