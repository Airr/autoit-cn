###User Defined Function###
_WinHttpSendRequest

###Description###
Sends the specified request to the HTTP server.

###Syntax###
#Include <WinHttp.au3>
_WinHttpSendRequest($hRequest [, $sHeaders = Default [, $sOptional = Default [, $iTotalLength = Default [, $iContext = Default ]]]])

###Parameters###
@@ParamTable@@
$hRequest
	Handle returned by _WinHttpOpenRequest().
$sHeaders
	[可选参数] Additional headers to append to the request. Default is $WINHTTP_NO_ADDITIONAL_HEADERS.
$sOptional
	[可选参数] Optional data to send immediately after the request headers. Default is $WINHTTP_NO_REQUEST_DATA.
$iTotalLength
	[可选参数] Length, in bytes, of the total optional data sent. Default is 0.
$iContext
	[可选参数] Application-defined value that is passed, with the request handle, to any callback functions. Default is 0.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	Returns 1.
Failure	Returns 0 and sets @error:
	1 - DllCall failed
@@End@@

###Remarks###
Specifying optional data ($sOptional) will cause $iTotalLength to receive the size of that data if left default value.
