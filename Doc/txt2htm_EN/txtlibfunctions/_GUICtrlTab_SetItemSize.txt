###User Defined Function###
_GUICtrlTab_SetItemSize

###Description###
Sets the width and height of tabs in a fixed width or owner drawn control

###Syntax###
#Include <GuiTab.au3>
_GUICtrlTab_SetItemSize($hWnd, $iWidth, $iHeight)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$iWidth
	New width, in pixels
$iHeight
	New height, in pixels
@@End@@

###ReturnValue###
Success: Returns the old width and height. The width is in the low order word of the return value, and
		the height is in the high-order word.

###Remarks###
None.
