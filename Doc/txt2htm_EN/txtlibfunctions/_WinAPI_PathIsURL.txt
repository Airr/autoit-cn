###User Defined Function###
_WinAPI_PathIsURL

###Description###
Tests a given string to determine if it conforms to a valid URL format.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_PathIsURL ( $sPath )

###Parameters###
@@ParamTable@@
$sPath
	The URL path to validate.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	1 - The path has a valid URL format.
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;0	Otherwise.
Failure	0 and sets the @error flag to non-zero.
@@End@@
