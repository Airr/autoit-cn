###User Defined Function###
_GUICtrlButton_Show

###Description###
Show/Hide button

###Syntax###
#Include <GuiButton.au3>
_GUICtrlButton_Show($hWnd[, $fShow = True])

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$fShow
	[optional] One of the following:
		 True - Show button
		False - Hide button
@@End@@

###ReturnValue###
Success: One of the Following:
		1 - If the button was previously visible
		0 - If the button was previously hidden

###Remarks###
None.
