###User Defined Function###
_WinHttpCrackUrl

###Description###
Separates a URL into its component parts such as host name and path.

###Syntax###
#Include <WinHttp.au3>
_WinHttpCrackUrl($sURL [, $iFlag = Default ])

###Parameters###
@@ParamTable@@
$sURL
	String. Canonical URL to separate.
$iFlag
	[可选参数] Flag that control the operation. Default is $ICU_ESCAPE
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	Returns array with 8 elements:
	$array[0] - is scheme name
	$array[1] - is internet protocol scheme
	$array[2] - is host name
	$array[3] - is port number
	$array[4] - is user name
	$array[5] - is password
	$array[6] - is URL path
	$array[7] - is extra information
Failure	Returns 0 and sets @error:
	1 - DllCall failed
@@End@@

###Remarks###
$iFlag is defined in WinHttpConstants.au3 and can be:
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;$ICU_DECODE - Converts characters that are "escape encoded" (%xx) to their non-escaped form.
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;$ICU_ESCAPE - Escapes certain characters to their escape sequences (%xx).
