###User Defined Function###
_GUICtrlHeader_GetImageList

###Description###
Retrieves the handle to the image list

###Syntax###
#Include <GuiHeader.au3>
_GUICtrlHeader_GetImageList($hWnd)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to control
@@End@@

###ReturnValue###
Success: A handle to the image list set for the header control
Failure: 0

###Remarks###
None.
