###User Defined Function###
_WinAPI_GetDriveGeometryEx

###Description###
Retrieves extended information about the disk's geometry.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_GetDriveGeometryEx ( $iDrive )

###Parameters###
@@ParamTable@@
$iDrive
	The drive letter to retrieve information, in the format D:, E:, etc.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	$tagDISK_GEOMETRY_EX structure that contains the information about the disk's geometry.
Failure	0 and sets the @error flag to non-zero.
@@End@@

###Remarks###
None
