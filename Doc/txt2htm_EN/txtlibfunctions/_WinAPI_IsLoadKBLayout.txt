###User Defined Function###
_WinAPI_IsLoadKBLayout

###Description###
Determines whether the specified input locale loaded into the system.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_IsLoadKBLayout ( $iLanguage )

###Parameters###
@@ParamTable@@
$iLanguage
	The name of the input locale identifier (LCID).
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	1 - The input locale is loaded.
0	Otherwise.
Failure	0 and sets the @error flag to non-zero.
@@End@@
