###User Defined Function###
_WinAPI_IsWindowEnabled

###Description###
Determines whether the specified window is enabled for mouse and keyboard input.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_IsWindowEnabled ( $hWnd )

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the window to test.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	1 - The window is enabled.
&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;0	Otherwise.
Failure	0 and sets the @error flag to non-zero.
@@End@@

###Remarks###
A child window receives input only if it is both enabled and visible.
