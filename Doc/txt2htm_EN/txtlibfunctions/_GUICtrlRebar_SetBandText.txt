###User Defined Function###
_GUICtrlRebar_SetBandText

###Description###
Retrieves the display text for the band

###Syntax###
#Include <GuiRebar.au3>
_GUICtrlRebar_SetBandText($hWnd, $iIndex, $sText)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to rebar control
$iIndex
	Zero-based index of the band for which the information will be set
$sText
	New display text for the band
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True
Failure:	False
@@End@@

###Remarks###
None.
