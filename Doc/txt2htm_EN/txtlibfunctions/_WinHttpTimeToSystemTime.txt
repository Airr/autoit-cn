###User Defined Function###
_WinHttpTimeToSystemTime

###Description###
Takes an HTTP time/date string and converts it to array (SYSTEMTIME structure values).

###Syntax###
#Include <WinHttp.au3>
_WinHttpTimeToSystemTime($sHttpTime)

###Parameters###
@@ParamTable@@
$sHttpTime
	Date/time string to convert.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	Returns array with 8 elements:
	$array[0] - is Year,
	$array[1] - is Month,
	$array[2] - is DayOfWeek,
	$array[3] - is Day,
	$array[4] - is Hour,
	$array[5] - is Minute,
	$array[6] - is Second.,
	$array[7] - is Milliseconds.
Failure	Returns 0 and sets @error:
	1 - DllCall failed
@@End@@
