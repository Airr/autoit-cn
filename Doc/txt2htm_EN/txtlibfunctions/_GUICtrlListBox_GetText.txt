###User Defined Function###
_GUICtrlListBox_GetText

###Description###
Returns the item (string) at the specified index

###Syntax###
#Include <GuiListBox.au3>
_GUICtrlListBox_GetText($hWnd, $iIndex)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to control
$iIndex
	Specifies the zero-based index of the string to retrieve
@@End@@

###ReturnValue###
Success: item (string)

None.
