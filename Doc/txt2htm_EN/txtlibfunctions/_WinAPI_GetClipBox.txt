###User Defined Function###
_WinAPI_GetClipBox

###Description###
Retrieves the dimensions of the bounding rectangle of the visible area.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_GetClipBox ( $hDC, ByRef $tRECT )

###Parameters###
@@ParamTable@@
$hDC
	Handle to the device context.
$tRECT
	$tagRECT structure that receive the rectangle dimensions, in logical units.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	The value that specifies the new clipping region's complexityit can be one of the following values.
Failure	0 and sets the @error flag to non-zero.
@@End@@

None
