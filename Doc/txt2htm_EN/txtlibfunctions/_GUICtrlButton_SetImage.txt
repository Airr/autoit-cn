###User Defined Function###
_GUICtrlButton_SetImage

###Description###
Sets the image of a button

###Syntax###
#Include <GuiButton.au3>
_GUICtrlButton_SetImage($hWnd, $sImageFile[, $nIconId = -1[, $fLarge = False]])

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$sImageFile
	File containing the Image
$nIconId
	[optional] Specifies the zero-based index of the icon to extract
$fLarge
	[optional] Extract Large Icon
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True
Failure:	False
@@End@@

###Remarks###
None.
