###User Defined Function###
_GUICtrlListBox_DeleteString

###Description###
Delete a string

###Syntax###
#Include <GuiListBox.au3>
_GUICtrlListBox_DeleteString($hWnd, $iIndex)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to control
$iIndex
	Zero-based index of the string to be deleted
@@End@@

###ReturnValue###
Success: Count of strings remaining
Failure: -1

###Remarks###
None.
