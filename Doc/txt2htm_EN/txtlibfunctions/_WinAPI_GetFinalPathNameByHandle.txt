###User Defined Function###
_WinAPI_GetFinalPathNameByHandle

###Description###
Retrieves the final path of the specified file.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_GetFinalPathNameByHandle ( $hFile [, $iFlags] )

###Parameters###
@@ParamTable@@
$hFile
	Handle to a file or directory.
$iFlags
	[可选参数] The type of result to return. This parameter can be combination of one $FILE_NAME_* and one
	$VOLUME_NAME_* values.
	
	$FILE_NAME_NORMALIZED
	$FILE_NAME_OPENED
	
	$VOLUME_NAME_DOS
	$VOLUME_NAME_GUID
	$VOLUME_NAME_NONE
	$VOLUME_NAME_NT
	
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	The path of the file.
Failure	Empty string and sets the @error flag to non-zero.
@@End@@

###Remarks###
This function requires Windows Vista or later.
