###User Defined Function###
_GUICtrlListView_FindNearest

###Description###
Finds the item nearest to the position specified

###Syntax###
#Include <GuiListView.au3>
_GUICtrlListView_FindNearest($hWnd, $iX, $iY[, $iDir = 0[, $iStart = -1[, $fWrapOK = True]]])

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$iX
	X position
$iY
	Y position
$iDir
	[optional] Specifies which direction to search:
		0 - Left
		1 - Right
		2 - Up
		3 - Down
		4 - From start
		5 - From end
		6 - From prior item
		7 - From next item
$iStart
	[optional] Zero based index of the item to begin the search with or -1 to start from the beginning. The
		specified item is itself excluded from the search.
$fWrapOK
	[optional] If True, the search will continue with the first item if no match is found
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	The zero based index of the item
Failure:	-1
@@End@@

###Remarks###
This function is supported only by large icon and small icon modes
