###User Defined Function###
_WinAPI_DeleteFile

###Description###
Deletes an existing file.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_DeleteFile ( $sFile )

###Parameters###
@@ParamTable@@
$sFile
	The name of the file to be deleted.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	1.
Failure	0 and sets the @error flag to non-zero.
@@End@@
