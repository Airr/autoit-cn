###User Defined Function###
_GUICtrlComboBox_EndUpdate

###Description###
Enables screen repainting that was turned off with the BeginUpdate function

###Syntax###
#Include <GuiComboBox.au3>
_GUICtrlComboBox_EndUpdate($hWnd)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
@@End@@

###ReturnValue###
Success: True
Failure: False
