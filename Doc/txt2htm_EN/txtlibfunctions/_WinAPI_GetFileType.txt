###User Defined Function###
_WinAPI_GetFileType

###Description###
Retrieves the file type of the specified file.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_GetFileType ( $hFile )

###Parameters###
@@ParamTable@@
$hFile
	Handle to the file.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	The file type ($FILE_TYPE_*)
Failure	(-1) and sets the @error flag to non-zero.
@@End@@

###Remarks###
None
