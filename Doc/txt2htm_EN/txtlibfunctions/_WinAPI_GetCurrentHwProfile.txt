###User Defined Function###
_WinAPI_GetCurrentHwProfile

###Description###
Retrieves information about the current hardware profile for the local computer.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_GetCurrentHwProfile ( )

###Parameters###
@@ParamTable@@
None.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	$tagHW_PROFILE_INFO structure that contains information about the current hardware profile.
Failure	0 and sets the @error flag to non-zero.
@@End@@
