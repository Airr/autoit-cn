###User Defined Function###
_GUICtrlToolbar_IndexToCommand

###Description###
Retrieves the command identifier associated with the button

###Syntax###
#Include <GuiToolBar.au3>
_GUICtrlToolbar_IndexToCommand($hWnd, $iIndex)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$iIndex
	Button index
@@End@@

###ReturnValue###
Success: Button command identifier

###Remarks###
None.
