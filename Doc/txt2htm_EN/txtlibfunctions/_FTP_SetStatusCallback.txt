###User Defined Function###
_FTP_SetStatusCallback

###Description###

Registers callback function that WinINet functions can call as progress is made during an operation.

###Syntax###
#Include <FTPEx.au3>
_FTP_SetStatusCallback ( $l_InternetSession, $sFunctionName)

###Parameters###
@@ParamTable@@
$l_InternetSession
	as returned by _FTP_Open()
$sFunctionName
	the name of the User Defined Function to call.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	Pointer to callback function
Failure:	0  and sets @error to non-zero
@@End@@

###Remarks###
The callback function is called with the following parameters (see InternetStatusCallback Windows API):
$hInternet, $dwContent, $dwInternetStatus, $lpvStatusInformation, $dwStatusInformationLength
