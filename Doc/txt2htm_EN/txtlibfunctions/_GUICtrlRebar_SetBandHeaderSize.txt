###User Defined Function###
_GUICtrlRebar_SetBandHeaderSize

###Description###
Set the size of the band's header, in pixels

###Syntax###
#Include <GuiRebar.au3>
_GUICtrlRebar_SetBandHeaderSize($hWnd, $iIndex, $iNewSize)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to rebar control
$iIndex
	Zero-based index of the band
$iNewSize
	New size of the band's header
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True
Failure:	False
@@End@@

None.
