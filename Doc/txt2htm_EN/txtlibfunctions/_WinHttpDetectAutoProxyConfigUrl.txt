###User Defined Function###
_WinHttpDetectAutoProxyConfigUrl

###Description###
Finds the URL for the Proxy Auto-Configuration (PAC) file.

###Syntax###
#Include <WinHttp.au3>
_WinHttpDetectAutoProxyConfigUrl($iAutoDetectFlags)

###Parameters###
@@ParamTable@@
$iAutoDetectFlags
	Specifies what protocols to use to locate the PAC file.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	Returns URL for the PAC file.
Failure	Returns empty string and sets @error:
	1 - DllCall failed
	2 - Internal failure.
@@End@@

###Remarks###
$iAutoDetectFlags values are defined in WinHttpconstants.au3
