###User Defined Function###
_GUIImageList_Swap

###Description###
Source image is copied to the destination image's index

###Syntax###
#Include <GuiImageList.au3>
_GUIImageList_Swap($hWnd, $iSource, $iDestination)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$iSource
	The zero-based index of the image to be used as the source of the swap operation
$iDestination
	The zero-based index of the image to be used as the destination of the swap operation
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True
Failure:	False
@@End@@

###Remarks###
None.
