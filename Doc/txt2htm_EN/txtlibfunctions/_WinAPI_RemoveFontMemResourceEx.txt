###User Defined Function###
_WinAPI_RemoveFontMemResourceEx

###Description###
Removes the fonts added from a memory image.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_RemoveFontMemResourceEx ( $hFont )

###Parameters###
@@ParamTable@@
$hFont
	Handle to the font-resource. This handle is returned by the _WinAPI_AddFontMemResourceEx() function.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	1.
Failure	0 and sets the @error flag to non-zero.
@@End@@

###Remarks###
None
