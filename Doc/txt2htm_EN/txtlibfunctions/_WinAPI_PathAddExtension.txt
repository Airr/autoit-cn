###User Defined Function###
_WinAPI_PathAddExtension

###Description###
Adds a file name extension to a path string.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_PathAddExtension ( $sPath [, $sExt] )

###Parameters###
@@ParamTable@@
$sPath
	The path to which the file name extension will be appended. If there is already a file name extension
	present, no extension will be added.
$sExt
	[可选参数] The file name extension. If this parameter is empty string, an ".exe" extension will be added.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	The resulting path, @extended will set to 1 if an extension was added, or 0 otherwise.
Failure	Empty string and sets the @error flag to non-zero.
@@End@@
