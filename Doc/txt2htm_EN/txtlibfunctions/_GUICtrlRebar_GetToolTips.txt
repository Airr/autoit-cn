###User Defined Function###
_GUICtrlRebar_GetToolTips

###Description###
Retrieves the handle to any ToolTip control associated with the rebar control

###Syntax###
#Include <GuiRebar.au3>
_GUICtrlRebar_GetToolTips($hWnd)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to rebar control
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	The handle to the ToolTip control associated with the rebar control
Failure:	0 if no ToolTip control is associated with the rebar control
@@End@@

###Remarks###
None.
