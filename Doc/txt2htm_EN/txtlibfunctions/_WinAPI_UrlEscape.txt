###User Defined Function###
_WinAPI_UrlEscape

###Description###
Converts characters in a URL that might be altered during transport across the Internet into their corresponding escape sequences.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_UrlEscape ( $sUrl [, $iFlags] )

###Parameters###
@@ParamTable@@
$sUrl
	The URL.
$iFlags
	[可选参数] The flags that indicate which characters in the URL should be converted to their escape sequences
	(an escape sequence has the form "%xy"). It can be a combination of the following values.
	
	$URL_DONT_ESCAPE_EXTRA_INFO
	$URL_ESCAPE_SPACES_ONLY
	$URL_ESCAPE_PERCENT
	$URL_ESCAPE_SEGMENT_ONLY
	*Windows 7 or later
	
	$URL_ESCAPE_AS_UTF8
	
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	The escaped URL.
Failure	Empty string and sets the @error flag to non-zero, @extended flag may contain the system error code.
@@End@@
