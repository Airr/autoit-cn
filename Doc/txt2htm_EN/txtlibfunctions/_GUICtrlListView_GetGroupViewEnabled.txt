###User Defined Function###
_GUICtrlListView_GetGroupViewEnabled

###Description###
Checks whether the control has group view enabled

###Syntax###
#Include <GuiListView.au3>
_GUICtrlListView_GetGroupViewEnabled($hWnd)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
@@End@@

###ReturnValue###
Returns the Group view setting:
		 True - Group view is enabled
		False - Group view is not enabled

###Remarks###
Minimum OS - Windows XP.
