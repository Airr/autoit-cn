###User Defined Function###
_WinAPI_EnumSystemLocales

###Description###
Enumerates the locales that are either installed on or supported by an operating system.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_EnumSystemLocales ( $iFlag )

###Parameters###
@@ParamTable@@
$iFlag
	Flags specifying the locale identifiers to enumerate. This parameter can have one of the
	following values.
	
	$LCID_INSTALLED
	$LCID_SUPPORTED
	
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	The array of the locale identifiers.
Failure	0 and sets the @error flag to non-zero.
@@End@@

None
