###User Defined Function###
_WinAPI_PathCreateFromUrl

###Description###
Converts a file URL to a Microsoft MS-DOS path.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_PathCreateFromUrl ( $sUrl )

###Parameters###
@@ParamTable@@
$sUrl
	The URL.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	The MS-DOS path.
Failure	Empty string and sets the @error flag to non-zero, @extended flag may contain the system error code.
@@End@@
