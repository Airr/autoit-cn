###User Defined Function###
_WinAPI_GetVersion

###Description###
Retrieves version of the current operating system.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_GetVersion ( )

###Parameters###
@@ParamTable@@
None.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	The string containing the current OS version.
"6.1"	Windows 7 / Windows Server 2008 R2
"6.0"	Windows Server 2008 / Windows Vista
"5.2"	Windows Server 2003 R2 / Windows Home Server / Windows Server 2003 / Windows XP Professional x64 Edition
"5.1"	Windows XP
"5.0"	Windows 2000
Failure	Empty string and sets the @error flag to non-zero.
@@End@@
