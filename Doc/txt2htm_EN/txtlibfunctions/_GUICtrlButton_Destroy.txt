###User Defined Function###
_GUICtrlButton_Destroy

###Description###
Delete the Button control

###Syntax###
#Include <GuiButton.au3>
_GUICtrlButton_Destroy(ByRef $hWnd)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
@@End@@

###ReturnValue###
Success: True, Handle is set to 0
Failure: False

###Remarks###
Restricted to only be used on Buttons created with _GUICtrlButton_Create
