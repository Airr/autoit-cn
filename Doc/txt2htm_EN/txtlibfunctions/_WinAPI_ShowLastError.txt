###User Defined Function###
_WinAPI_ShowLastError

###Description###
Shows the last error code and message.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_ShowLastError ( [$sText [, $fAbort [, $iLanguage]]] )

###Parameters###
@@ParamTable@@
$sText
	[可选参数] The user's text to be displayed with the error message.
$fAbort
	[可选参数] Specifies whether to exit after displaying an error message, valid values:
		TRUE  - Exit the program after displaying a message if it indicates an error.
		FALSE - Always return normally. (Default)
$iLanguage
	[可选参数] The language identifier for the message.
@@End@@

###ReturnValue###
