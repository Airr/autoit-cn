###User Defined Function###
_WinAPI_GUIDFromString

###Description###
Converts a string GUID to binary form

###Syntax###
#Include <WinAPI.au3>
_WinAPI_GUIDFromString($sGUID)

###Parameters###
@@ParamTable@@
$sGUID
	GUID in string form
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	$tagGUID structure
Failure:	set @error
@@End@@
