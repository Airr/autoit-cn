###User Defined Function###
_WinAPI_GetObjectEx

###Description###
Retrieves information for the specified graphics object.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_GetObjectEx ( $hObject )

###Parameters###
@@ParamTable@@
$hObject
	Handle to the graphics object of interest. This can be a handle to one of the following: a logical
	bitmap, a brush, a font, a palette, or a pen.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	The structure that depends on the graphics object ($hObject) and @extended flag will contain the value
Failure	0 and sets the @error flag to non-zero.
@@End@@

None
