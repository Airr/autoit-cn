###User Defined Function###
_WinHttpSetStatusCallback

###Description###
Sets up a callback function that WinHttp can call as progress is made during an operation.

###Syntax###
#Include <WinHttp.au3>
_WinHttpSetStatusCallback($hInternet, $hInternetCallback [, $iNotificationFlags = Default ])

###Parameters###
@@ParamTable@@
$hInternet
	Handle for which the callback is to be set.
$hInternetCallback
	Callback function to call when progress is made.
$iNotificationFlags
	[可选参数] Flags to indicate which events activate the callback function. Default is $WINHTTP_CALLBACK_FLAG_ALL_NOTIFICATIONS.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	Returns a pointer to the previously defined status callback function or NULL if there was no previously defined status callback function.
Failure	Returns 0 and sets @error:
	1 - DllCall failed
@@End@@
