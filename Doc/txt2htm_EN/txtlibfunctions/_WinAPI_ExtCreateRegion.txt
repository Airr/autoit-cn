###User Defined Function###
_WinAPI_ExtCreateRegion

###Description###
Creates a region from the specified region and transformation data.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_ExtCreateRegion ( $tRGNDATA [, $tXFORM] )

###Parameters###
@@ParamTable@@
$tRGNDATA
	$tagRGNDATA structure that contains the region data in logical units.
$tXFORM
	[可选参数] $tagXFORM structure that defines the transformation to be performed on the region.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	The handle to the region.
Failure	0 and sets the @error flag to non-zero.
@@End@@

###Remarks###
None
