###User Defined Function###
_WinAPI_OffsetClipRgn

###Description###
Moves the clipping region of a device context by the specified offsets.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_OffsetClipRgn ( $hDC, $iXOffset, $iYOffset )

###Parameters###
@@ParamTable@@
$hDC
	Handle to the device context.
$iXOffset
	The number of logical units to move left or right.
$iYOffset
	The number of logical units to move up or down.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	The value that specifies the new clipping region's complexity&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;it can be one of the following values.
Failure	0 and sets the @error flag to non-zero.
@@End@@
