###User Defined Function###
_ColorSetRGB

###Description###
Returns the RGB color to work with (COLORREF).

###Syntax###
#Include <Color.au3>
_ColorSetRGB($aColor)

###Parameters###
@@ParamTable@@
$aColor
	an array of values in the range 0-255:
	[0] Red   component color
	[1] Green component color
	[2] Blue  component color
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	returns the RGB color to work with.
Failure:	set @error
@error:	1 invalid array
	2 invalid color value
@@End@@

###Remarks###
@extended is preserved.
