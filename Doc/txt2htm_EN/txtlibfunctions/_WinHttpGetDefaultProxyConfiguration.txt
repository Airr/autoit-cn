###User Defined Function###
_WinHttpGetDefaultProxyConfiguration

###Description###
Retrieves the default WinHttp proxy configuration.

###Syntax###
#Include <WinHttp.au3>
_WinHttpGetDefaultProxyConfiguration()

###Parameters###
@@ParamTable@@
None.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	Returns array with 3 elements:
	$array[0] - Integer. Access type.
	$array[1] - String. Proxy server list.
	$array[2] - String. Proxy bypass list.
Failure	Returns 0 and sets @error:
	1 - DllCall failed
@@End@@

###Remarks###
Access types are defined in WinHttpconstants.au3:
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;$WINHTTP_ACCESS_TYPE_DEFAULT_PROXY = 0
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;$WINHTTP_ACCESS_TYPE_NO_PROXY = 1
<br>&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;$WINHTTP_ACCESS_TYPE_NAMED_PROXY = 3
