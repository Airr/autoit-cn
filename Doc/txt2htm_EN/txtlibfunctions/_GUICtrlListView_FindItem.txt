###User Defined Function###
_GUICtrlListView_FindItem

###Description###
Searches for an item with the specified characteristics

###Syntax###
#Include <GuiListView.au3>
_GUICtrlListView_FindItem($hWnd, $iStart, ByRef $tFindInfo[, $sText = ""])

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$iStart
	Zero based index of the item to begin the search with or -1 to start from the beginning. The
		specified item is itself excluded from the search.
$tFindInfo
	$tagLVFINDINFO structure that contains the search information
$sText
	[optional] String to compare with the item text. It is valid if $LVFI_STRING or $LVFI_PARTIAL is set in the
		Flags member
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	The zero based index of the item
Failure:	-1
@@End@@

###Remarks###
None.
