###User Defined Function###
_WinAPI_PickIconDlg

###Description###
Displays a dialog box that allows the user to choose an icon.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_PickIconDlg ( [$sIcon [, $iIndex [, $hParent]]] )

###Parameters###
@@ParamTable@@
$sIcon
	[可选参数] The fully-qualified path of the file that contains the initial icon.
$iIndex
	[可选参数] The index of the initial icon.
$hParent
	[可选参数] Handle of the parent window.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	The array containing the following parameters:
[0]	The path of the file that contains the selected icon.
[1]	The index of the selected icon.
Failure	0 and sets the @error flag to non-zero.
@@End@@

###Remarks###
This function also sets the @error flag to 1 if the icon was not selected.
