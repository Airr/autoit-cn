###User Defined Function###
_GDIPlus_BrushGetType

###Description###
Retrieve the type of Brush object

###Syntax###
#Include <GDIPlus.au3>
_GDIPlus_BrushGetType($hBrush)

###Parameters###
@@ParamTable@@
$hBrush
	Handle to a Brush object
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	Brush type:
	0 - Solid color
	1 - Hatch fill
	2 - Texture fill
	3 - Path gradient
	4 - Linear gradient
Failure:	-1 and @error is set
@@End@@

None.
