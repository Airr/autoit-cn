###User Defined Function###
_WinAPI_GetObjectNameByHandle

###Description###
Retrieves a name of the specified object.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_GetObjectNameByHandle ( $hObject )

###Parameters###
@@ParamTable@@
$hObject
	Handle to the object to obtain information about.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	The name of the object.
Failure	Empty string and sets the @error flag to non-zero, @extended flag may contain the NTSTATUS code.
@@End@@

None
