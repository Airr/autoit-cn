###User Defined Function###
_GUICtrlTreeView_Destroy

###Description###
Delete the control

###Syntax###
#Include <GuiTreeView.au3>
_GUICtrlTreeView_Destroy(ByRef $hWnd)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	True, Handle is set to 0
Failure:	False
@@End@@

###Remarks###
Restricted to only be used on TreeViews created with _GUICtrlTreeView_Create
