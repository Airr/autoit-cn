###User Defined Function###
_WinHttpBinaryConcat

###Description###
Concatenates two binary data returned by _WinHttpReadData() in binary mode.

###Syntax###
#Include <WinHttp.au3>
_WinHttpBinaryConcat(ByRef $bBinary1, ByRef $bBinary2)

###Parameters###
@@ParamTable@@
$bBinary1
	Binary data that is to be concatenated.
$bBinary2
	Binary data to concatenate.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	Returns concatenated binary data.
Failure	Returns empty binary and sets @error:
	1 - Invalid input.
@@End@@

###Remarks###
None.
