###User Defined Function###
_WinAPI_ShellAddToRecentDocs

###Description###
Adds a file to the most recently and frequently item list.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_ShellAddToRecentDocs ( $sFile )

###Parameters###
@@ParamTable@@
$sFile
	The name of the file to be added. Set this parameter to empty string to clear all usage data on all items.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	1.
Failure	0 and sets the @error flag to non-zero.
@@End@@

None
