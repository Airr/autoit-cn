###User Defined Function###
_StringToHex

###Description###
Convert a string to a hex string.

###Syntax###
#Include <String.au3>
_StringToHex($strChar)

###Parameters###
@@ParamTable@@
$strChar
	string to be converted.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	Returns an hex string
Failure:	Returns -1 and sets @error to 1
@@End@@

None.
