###User Defined Function###
_WinAPI_OffsetRgn

###Description###
Moves a region by the specified offsets.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_OffsetRgn ( $hRgn, $iXOffset, $iYOffset )

###Parameters###
@@ParamTable@@
$hRgn
	Handle to the region to be moved.
$iXOffset
	The number of logical units to move left or right.
$iYOffset
	The number of logical units to move up or down.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	The value that specifies the new clipping region's complexityit can be one of the following values.
Failure	0 and sets the @error flag to non-zero.
@@End@@

None
