###User Defined Function###
_WinAPI_PathFindOnPath

###Description###
Searches for a file.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_PathFindOnPath ( $sFile [, $aDirs [, $iStart [, $iEnd]]] )

###Parameters###
@@ParamTable@@
$sFile
	The file name for which to search.
$aDirs
	[可选参数] The array of directories to be searched first. If this parameter is not an array, function attempts to
	find the file by searching standard directories such as System32 and the directories specified in
	the PATH environment variable.
$iStart
	[可选参数] The index of array to start searching at.
$iEnd
	[可选参数] The index of array to stop searching at.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	The fully qualified path.
Failure	The original $sFile parameter and sets the @error flag to non-zero.
@@End@@
