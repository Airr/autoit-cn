###Structure Name###
$tagNMHDR

###Description###
Contains information about a notification message

###Syntax###
Global Const $tagNMHDR = "hwnd hWndFrom;uint_ptr IDFrom;INT Code"

###Fields###
@@ParamTable@@
hWndFrom
	Window handle to the control sending a message
IDFrom
	Identifier of the control sending a message
Code
	Notification code
@@End@@

###Remarks###
None.
