###User Defined Function###
_WinAPI_UrlIs

###Description###
Tests whether or not a URL is a specified type.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_UrlIs ( $sUrl [, $iType] )

###Parameters###
@@ParamTable@@
$sUrl
	The URL.
$iType
	[可选参数] The type of URL to be tested for. It can be one of the following values.
	
	$URLIS_APPLIABLE
	$URLIS_DIRECTORY
	$URLIS_FILEURL
	$URLIS_HASQUERY
	$URLIS_NOHISTORY
	$URLIS_OPAQUE
	$URLIS_URL
	
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	1 - The URL is the specified type.
0	Otherwise.
Failure	0 and sets the @error flag to non-zero.
@@End@@
