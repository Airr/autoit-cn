###User Defined Function###
_WinAPI_GetModuleFileName

###Description###
Retrieves the fully-qualified path for the file that contains the specified module.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_GetModuleFileName( $hModule )

###Parameters###
@@ParamTable@@
$hModule
	Handle to the loaded module whose path is being requested. If this parameter is 0, function retrieves
	the path of the executable file of the current process.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	The fully-qualified path of the module.
Failure	Empty string and sets the @error flag to non-zero.
@@End@@

None
