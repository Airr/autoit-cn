###User Defined Function###
_WinAPI_EnumProcessThreads

###Description###
Enumerates the threads that belong to the specified process.

###Syntax###
#Include <WinAPIEx.au3>
_WinAPI_EnumProcessThreads ( [$PID] )

###Parameters###
@@ParamTable@@
$PID
	[可选参数] The PID of the process. Default (0) is the current process.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success	The array of threads identifiers (ID). The zeroth array element contains the number of threads.
Failure	0 and sets the @error flag to non-zero.
@@End@@
