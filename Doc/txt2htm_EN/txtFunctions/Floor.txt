###Function###
Floor

###Description###
Returns a number rounded down to the closest integer.

###Syntax###
Floor ( expression )

###Parameters###
@@ParamTable@@
expression
	Any valid numeric expression. 
@@End@@

###ReturnValue###
Returns the rounded number.
