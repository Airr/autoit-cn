###Function###
GUIDelete

###Description###
Deletes a GUI window and all controls that it contains.

###Syntax###
GUIDelete ( [winhandle] )


###Parameters###
@@ParamTable@@
winhandle
	[optional] Windows handle as returned by GUICreate (default is the previous used).
@@End@@


###ReturnValue###
@@ReturnTable@@
Success:	Returns 1.
Failure:	Returns 0.
@@End@@


###Remarks###
None.


###Related###
GUICreate, GUISwitch
