###Function###
TraySetPauseIcon

###Description###
Loads/Sets a specified tray pause icon.

###Syntax###
TraySetPauseIcon ( [filename [, iconID] )


###Parameters###
@@ParamTable@@
filename
	[optional] The filename of the icon to be display as pause icon in the tray.
iconID
	[optional] Icon identifier if the file contain multiple icons.
@@End@@

###ReturnValue###
None.

###Remarks###
To reset the pause icon to the default (red cross), use the function with no parameters:
TraySetPauseIcon().

Passing a positive number will reference the string equivalent icon name.
Passing a negative number causes 1-based "index" behaviour. Some Dll can have icon extracted 

just with negative numbers.


###Related###
TraySetIcon, TraySetState
