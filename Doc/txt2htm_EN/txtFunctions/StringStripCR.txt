###Function###
StringStripCR

###Description###
Removes all carriage return values ( Chr(13) ) from a string.

###Syntax###
StringStripCR ( "string" )


###Parameters###
@@ParamTable@@
string
	The string to convert.
@@End@@


###ReturnValue###
Returns the string with all instances of the @CR character (Chr(13)) removed.


###Remarks###
None.
