###Function###
EnvGet

###Description###
Retrieves an environment variable.

###Syntax###
EnvGet ( "envvariable" )



###Parameters###
@@ParamTable@@
envvariable
	Name of the environment variable to get such as "TEMP" or "PATH".
@@End@@

###ReturnValue###
Returns the requested variable (or a blank string if the variable does not exist).


###Remarks###
None.

###Related###
EnvSet, EnvUpdate
