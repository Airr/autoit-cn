###User Defined Function###
_GUICtrlTab_SetItem

###Description###
Sets information about a tab

###Syntax###
#Include <GuiTab.au3>
_GUICtrlTab_SetItem($hWnd, $iIndex[, $sText = -1[, $iState = -1[, $iImage = -1[, $iParam = -1]]]])

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$iIndex
	Zero based item index
$sText
	Item text
$iState
	Item state
$iImage
	Zero based item image
$iParam
	Application defined data
@@End@@

###ReturnValue###
Success: True
Failure: False

###Remarks###
None.
