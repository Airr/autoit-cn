###User Defined Function###
_SQLite_Startup

###Description###
Loads SQLite.dll

###Syntax###
#include <SQLite.au3>
_SQLite_Startup ( [ $sDll_Filename = "sqlite3.dll" ] )


###Parameters###
@@ParamTable@@
$sDll_Filename
	[optional] Dll filename
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	Returns path to SQLite3.dll
Failure:	@error = 1
@@End@@

###Remarks###
Requieres SQLite3.dll in @ScriptDir or @SystemDir.

If SQLite3.dll.au3 is included the dll will be created in @SystemDir.
If not successfull a temporary file will be created that will be destroyed at _SQLite_Shutdown().
