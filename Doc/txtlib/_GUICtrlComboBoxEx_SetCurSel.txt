###User Defined Function###
_GUICtrlComboBoxEx_SetCurSel

###Description###
Select a string in the list of a ComboBox

###Syntax###
#Include <GuiComboBoxEx.au3>
_GUICtrlComboBoxEx_SetCurSel($hWnd[, $iIndex = -1])

###Parameters###
@@ParamTable@@
$hWnd
	Handle to control
$iIndex
	Specifies the zero-based index of the string to select
@@End@@

###ReturnValue###
Success: The index of the item selected
Failure: -1

###Remarks###
If $iIndex is ?, any current selection in the list is removed and the edit control is cleared.

If $iIndex is greater than the number of items in the list or if $iIndex is ?, the return value
is -1 and the selection is cleared.
