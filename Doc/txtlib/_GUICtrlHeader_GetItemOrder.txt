###User Defined Function###
_GUICtrlHeader_GetItemOrder

###Description###
Retrieves the order in which the item appears

###Syntax###
#Include <GuiHeader.au3>
_GUICtrlHeader_GetItemOrder($hWnd, $iIndex)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to control
$iIndex
	Zero based item index
@@End@@

###ReturnValue###
Success: Zero based item order

###Remarks###
None.
