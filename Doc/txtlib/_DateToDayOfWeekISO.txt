###User Defined Function###
_DateToDayOfWeekISO

###Description###
Returns the ISO weekdaynumber for a given date.

###Syntax###
#Include <Date.au3>
_DateToDayOfWeekISO($iYear, $iMonth, $iDay)

###Parameters###
@@ParamTable@@
$iYear
	A valid year in format YYYY
$iMonth
	A valid month in format MM
$iDay
	A valid day in format DD
@@End@@

###ReturnValue###
Success: Returns Day of the Week Range is 0 to 6 where 0=Monday.
Failure: 0 and Set @ERROR to:
		0 - No error.
		1 - Invalid Input Date

###Remarks###
None.
