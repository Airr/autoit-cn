###User Defined Function###
_GUICtrlButton_SetImage

###Description###
Sets the check state of a radio button or check box

###Syntax###
#Include <GuiButton.au3>
_GUICtrlButton_SetImage($hWnd, $sImageFile[, $nIconId = -1[, $fLarge = False]])

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$sImageFile
	File containing the Image
$nIconId
	Specifies the zero-based index of the icon to extract
$fLarge
	Extract Large Icon
@@End@@

###ReturnValue###
成功: True
失败: False

###Remarks###
None.
