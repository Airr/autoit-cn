###User Defined Function###
_GUICtrlAVI_Seek

###Description###
Directs an AVI control to display a particular frame of an AVI clip

###Syntax###
#Include <GuiAVI.au3>
_GUICtrlAVI_Seek($hWnd, $iFrame)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$iFrame
	Zero based index of the frame to display
@@End@@

###ReturnValue###
Success: True
Failure: False

###Remarks###
None.
