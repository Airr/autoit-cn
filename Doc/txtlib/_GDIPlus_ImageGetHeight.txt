###User Defined Function###
_GDIPlus_ImageGetHeight

###Description###
Get the image height

###Syntax###
#Include <GDIPlus.au3>
_GDIPlus_ImageGetHeight($hImage)

###Parameters###
@@ParamTable@@
$hImage
	Handle to an image object
@@End@@

###ReturnValue###
Success: Image height, in pixels
Failure: -1 and @error is set
