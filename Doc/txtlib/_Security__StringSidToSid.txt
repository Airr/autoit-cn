###User Defined Function###
_Security__StringSidToSid

###Description###
Converts a String SID to a binary SID

###Syntax###
#Include <Security.au3>
_Security__StringSidToSid($sSID)

###Parameters###
@@ParamTable@@
$sSID
	String SID to be converted
@@End@@

###ReturnValue###
Success: SID in a byte structure
Failure: 0

###Remarks###
None.
