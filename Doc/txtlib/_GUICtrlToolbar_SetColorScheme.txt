###User Defined Function###
_GUICtrlToolbar_SetColorScheme

###Description###
Sets the color scheme information

###Syntax###
#Include <GuiToolBar.au3>
_GUICtrlToolbar_SetColorScheme($hWnd, $iHighlight, $iShadow)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$iHighlight
	Highlight color
$iShadow
	Shadow color
@@End@@

###ReturnValue###
None.

###Remarks###
The control uses the color scheme information when drawing the 3-D elements in the control
