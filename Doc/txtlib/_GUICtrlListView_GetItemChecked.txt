###User Defined Function###
_GUICtrlListView_GetItemChecked

###Description###
Returns the check state for a list-view control item

###Syntax###
#Include <GuiListView.au3>
_GUICtrlListView_GetItemChecked($hWnd, $iIndex)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$iIndex
	Zero based item index to retrieve item check state from
@@End@@

###ReturnValue###
成功: True
失败: False
