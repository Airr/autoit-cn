###User Defined Function###
_GUICtrlListView_GetItemSpacingX

###Description###
Determines the horizontal spacing between items in the control

###Syntax###
#Include <GuiListView.au3>
_GUICtrlListView_GetItemSpacingX($hWnd[, $fSmall = False])

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$fSmall
	View for which to retrieve the item spacing:
		 True - Small icon view
		False - Icon view
@@End@@

###ReturnValue###
Success: Horizontal spacing

###Remarks###
None.
