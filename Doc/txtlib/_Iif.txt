###User Defined Function###
_Iif

###Description###
Perform a boolean test within an expression.

###Syntax###
#Include <Misc.au3>
_Iif($fTest, $vTrueVal, $vFalseVal)

###Parameters###
@@ParamTable@@
$fTest
	Boolean test.
$vTrueVal
	Value to return if $fTest is true.
$vFalseVal
	Value to return if $fTest is false.
@@End@@

###ReturnValue###
True: $vTrueVal
False: $vFalseVal

None.
