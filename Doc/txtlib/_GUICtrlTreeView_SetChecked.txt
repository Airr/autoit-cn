###User Defined Function###
_GUICtrlTreeView_SetChecked

###Description###
Sets whether a item has it's checkbox checked or not

###Syntax###
#Include <GuiTreeView.au3>
_GUICtrlTreeView_SetChecked($hWnd, $hItem[, $fCheck = True])

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$hItem
	Handle to the item
$fCheck
	Value to set checked state to:
		 True - Checked
		False - Not checked
@@End@@

###ReturnValue###
Success: True
Failure: False

###Remarks###
None.
