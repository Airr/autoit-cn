###User Defined Function###
_GUICtrlAVI_Destroy

###Description###
Delete the control

###Syntax###
#Include <GuiAVI.au3>
_GUICtrlAVI_Destroy(ByRef $hWnd)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
@@End@@

###ReturnValue###
成功: True, 句柄设置为 0
失败: False

###Remarks###
Restricted to only be used on AVI Control created with _GUICtrlAVI_Create
