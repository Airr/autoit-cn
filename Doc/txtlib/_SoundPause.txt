###User Defined Function###
_SoundPause

###Description###
Pause a playing sound.

###Syntax###
#include <Sound.au3>
_SoundPause ( $sSnd_id )


###Parameters###
@@ParamTable@@
$sSnd_id
	Sound ID (the 'alias') as returned by _SoundOpen or a file
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	1
Failure:	0
@Error:	0 = No Error
	1 = Pause Failed
	3 = Invalid Sound ID. May be _SoundOpen with alias needed.
@@End@@

###Remarks###
None.

###Related###
_SoundResume, _SoundOpen
