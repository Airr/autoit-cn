###User Defined Function###
_SoundResume

###Description###
Resume a paused sound.

###Syntax###
#include <Sound.au3>
_SoundResume ( $sSnd_id )


###Parameters###
@@ParamTable@@
$sSnd_id
	Sound ID (the 'alias') as returned by _SoundOpen or a file
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	1
Failure:	0
@Error:	0 = No Error
	1 = Pause Failed
	3 = Invalid Sound ID. May be _SoundOpen with alias needed.
@@End@@

###Remarks###
_SoundPlay will also resume playing after a sound has been paused in some cases.

###Related###
_SoundPause, _SoundPlay, _SoundOpen
