###User Defined Function###
_GUICtrlTab_SetItemText

###Description###
Sets the text of a tab

###Syntax###
#Include <GuiTab.au3>
_GUICtrlTab_SetItemText($hWnd, $iIndex, $sText)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$iIndex
	Zero based item index
$sText
	Item text
@@End@@

###ReturnValue###
Success: True
Failure: False

###Remarks###
None.
