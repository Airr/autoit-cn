###User Defined Function###
_GUICtrlComboBoxEx_Destroy

###Description###
Delete the control

###Syntax###
#Include <GuiComboBoxEx.au3>
_GUICtrlComboBoxEx_Destroy(ByRef $hWnd)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
@@End@@

###ReturnValue###
成功: True, 句柄设置为 0
失败: False

###Remarks###
Restricted to only be used on ComboBox created with _GUICtrlComboBoxEx_Create
