###User Defined Function###
_GUICtrlTreeView_SetInsertMark

###Description###
Sets the insertion mark

###Syntax###
#Include <GuiTreeView.au3>
_GUICtrlTreeView_SetInsertMark($hWnd, $hItem[, $fAfter = True])

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$hItem
	Specifies at which item the insertion mark will be placed. If this is 0, the insertion mark is
		removed.
$fAfter
	Specifies if the insertion mark is placed before or after the item. If this is True, the
		insertion mark will be placed after the item. If this is False, the insertion mark will be placed before the
		item.
@@End@@

###ReturnValue###
Success: True
Failure: False

###Remarks###
None.
