###User Defined Function###
_RunDOS

###Description###
Executes a DOS command in a hidden command window.

###Syntax###
#include <Process.au3>
_RunDOS ( $sCommand )


###Parameters###
@@ParamTable@@
$sCommand
	Command to execute
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	Returns the exit code of the command
Failure:	Returns 0 and sets @error to non-zero.
@@End@@
