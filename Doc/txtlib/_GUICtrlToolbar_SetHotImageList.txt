###User Defined Function###
_GUICtrlToolbar_SetHotImageList

###Description###
Sets the hot button image list

###Syntax###
#Include <GuiToolBar.au3>
_GUICtrlToolbar_SetHotImageList($hWnd, $hImageList)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$hImageList
	Handle to the image list that will be set
@@End@@

###ReturnValue###
Success: The handle to the image list previously used to display hot buttons
Failure: 0

###Remarks###
None.
