###User Defined Function###
_GUICtrlHeader_SetItemImage

###Description###

###Syntax###
#Include <GuiHeader.au3>
_GUICtrlHeader_SetItemImage($hWnd, $iIndex, $iImage)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to control
$iIndex
	Zero based item index
$iImage
	Zero based image index
@@End@@

###ReturnValue###
Success: True
Failure: False
