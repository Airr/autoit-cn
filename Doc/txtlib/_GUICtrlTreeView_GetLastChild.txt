###User Defined Function###
_GUICtrlTreeView_GetLastChild

###Description###
Retrieves the last child item of the specified item

###Syntax###
#Include <GuiTreeView.au3>
_GUICtrlTreeView_GetLastChild($hWnd, $hItem)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$hItem
	Handle to the item
@@End@@

###ReturnValue###
Success: The handle of the last child item
Failure: 0

###Remarks###
None.
