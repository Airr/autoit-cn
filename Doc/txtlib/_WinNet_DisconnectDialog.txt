###User Defined Function###
_WinNet_DisconnectDialog

###Description###
Starts a general browsing dialog box for disconnecting from network resources

###Syntax###
#Include <WinNet.au3>
_WinNet_DisconnectDialog($hWnd)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the owner window for the dialog box
@@End@@

###ReturnValue###
Success: True
Failure: False

###Remarks###
None.
