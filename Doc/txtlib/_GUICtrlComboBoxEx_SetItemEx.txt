###User Defined Function###
_GUICtrlComboBoxEx_SetItemEx

###Description###
Sets some or all of a item's attributes

###Syntax###
#Include <GuiComboBoxEx.au3>
_GUICtrlComboBoxEx_SetItemEx($hWnd, ByRef $tItem)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$tItem
	$tagCOMBOBOXEXITEM structure
@@End@@

###ReturnValue###
Success: True
Failure: False

###Remarks###
To set the attributes of an item set the Item member of the $tagCOMBOBOXEXITEM structure to the index of the item.
For an item, you can set the Image, SelectedImage, OverlayImage, Ident, and Param members of the
$tagCOMBOBOXEXITEM structure.

###Related###
_GUICtrlComboBoxEx_GetItemEx, _GUICtrlComboBoxEx_SetItem, $tagCOMBOBOXEXITEM
