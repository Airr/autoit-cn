###User Defined Function###
_GUICtrlMenu_SetItemText

###Description###
Sets the text for a menu item

###Syntax###
#Include <GuiMenu.au3>
_GUICtrlMenu_SetItemText($hMenu, $iItem, $sText[, $fByPos = True])

###Parameters###
@@ParamTable@@
$hMenu
	Menu handle
$iItem
	Identifier or position of the menu item
$sText
	Menu item text
$fByPos
	Menu identifier flag:
		 True - $iItem is a zero based item position
		False - $iItem is a menu item identifier
@@End@@

###ReturnValue###
Success: True
Failure: False

###Remarks###
None.
