###User Defined Function###
_SQLite_Escape

###Description###
Escapes a string

###Syntax###
#include <SQLite.au3>
_SQLite_Escape ( $sString [, $iBuffSize ] )


###Parameters###
@@ParamTable@@
$sString
	string to escape
$iBuffSize 
	[optional] the result will be truncated if its longer
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	Returns escaped string
Failure:	Returns empty string and sets @error to 1
@@End@@

###Remarks###
The escaped string is already wrapped with single quotes.
For example "It's a fine day" would look like "It''s a fine day"
For binary data use _SQLite_Encode()
