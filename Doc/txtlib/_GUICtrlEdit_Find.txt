###User Defined Function###
_GUICtrlEdit_Find

###Description###
Initiates a find dialog

###Syntax###
#Include <GuiEdit.au3>
_GUICtrlEdit_Find($hWnd[, $fReplace = False])

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$fReplace
	Replace Option:
		 True - Show option
		False - Hide option
@@End@@

###ReturnValue###
None.

###Remarks###
If you use text from the edit control and that text gets replaced the function will no
longer function correctly
