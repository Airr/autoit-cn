###User Defined Function###
_GUIToolTip_TrackActivate

###Description###
Activates or deactivates a tracking ToolTip

###Syntax###
#Include <GuiToolTip.au3>
_GUIToolTip_TrackActivate($hWnd[, $fActivate = True[, $hTool = 0[, $iID = 0]]])

###Parameters###
@@ParamTable@@
$hWnd
	Handle to control
$fActivate
	True to activate, False to deactivate
$hTool
	Handle to the window that contains the tool
$iID
	Application-defined identifier of the tool
@@End@@

###ReturnValue###
None.

###Remarks###
None.
