###User Defined Function###
_GUICtrlMenu_GetItemGrayed

###Description###
Retrieves the status of the menu item grayed state

###Syntax###
#Include <GuiMenu.au3>
_GUICtrlMenu_GetItemGrayed($hMenu, $iItem[, $fByPos = True])

###Parameters###
@@ParamTable@@
$hMenu
	Handle of the menu
$iItem
	Identifier or position of the menu item
$fByPos
	Menu identifier flag:
		 True - $iItem is a zero based item position
		False - $iItem is a menu item identifier
@@End@@

###ReturnValue###
True: Item is grayed
False: Item is not grayed

###Remarks###
None.
