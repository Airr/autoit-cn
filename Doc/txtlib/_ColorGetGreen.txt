###User Defined Function###
_ColorGetGreen

###Description###
Returns the green component of a given color.

###Syntax###
#Include <Color.au3>
_ColorGetGreen($nColor)

###Parameters###
@@ParamTable@@
$nColor
	The RGB color to work with (hexadecimal code).
@@End@@

###ReturnValue###
Success: The component color in the range 0-255

###Remarks###
None.
