###User Defined Function###
_GUICtrlRebar_SetBandStyleFixedSize

###Description###
Set whether the band can't be sized. With this style, the sizing grip is not displayed on the band

###Syntax###
#Include <GuiRebar.au3>
_GUICtrlRebar_SetBandStyleFixedSize($hWnd, $iIndex[, $fEnabled = True])

###Parameters###
@@ParamTable@@
$hWnd
	Handle to rebar control
$iIndex
	Zero-based index of the band
$fEnabled
	If True the item state is set, otherwise it is not set
@@End@@

###ReturnValue###
Success: True
Failure: False

###Remarks###
None.

_GUICtrlRebar_GetBandStyleFixedSize, _GUICtrlRebar_SetBandStyle
