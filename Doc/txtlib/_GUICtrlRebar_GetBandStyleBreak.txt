###User Defined Function###
_GUICtrlRebar_GetBandStyleBreak

###Description###
Determine if flag is set

###Syntax###
#Include <GuiRebar.au3>
_GUICtrlRebar_GetBandStyleBreak($hWnd, $iIndex)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to rebar control
$iIndex
	Zero-based index of the band
@@End@@

###ReturnValue###
True: Flag is set (The band is on a new line)
False: Flag not set

###Remarks###
None.
