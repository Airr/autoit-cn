###User Defined Function###
_GUICtrlListView_GetUnicodeFormat

###Description###
Retrieves the UNICODE character format flag

###Syntax###
#Include <GuiListView.au3>
_GUICtrlListView_GetUnicodeFormat($hWnd)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
@@End@@

###ReturnValue###
True: Using Unicode characters
False: Using ANSI characters

###Remarks###
None.
