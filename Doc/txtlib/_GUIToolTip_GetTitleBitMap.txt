###User Defined Function###
_GUIToolTip_GetTitleBitMap

###Description###
Retrieves the title bitmap icon

###Syntax###
#Include <GuiToolTip.au3>
_GUIToolTip_GetTitleBitMap($hWnd)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to control
@@End@@

###ReturnValue###
Success: Handle to the ToolTip icon

###Remarks###
Only available on Windows XP
