###User Defined Function###
_SQLite_SaveMode

###Description###
Disable or Enable Save mode

###Syntax###
#include <SQLite.au3>
_SQLite_SaveMode ( $fSaveModeState )


###Parameters###
@@ParamTable@@
$fSaveModeState
	True or False to enable or disable SafeMode
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	Returns $SQLITE_OK
Failure:	Returns $SQLITE_MISUSE and sets @error = 1
@@End@@

###Remarks###
Save mode prevents SQLite from crashing which can be caused by passing invalid query or database handles.
It's enabled by default.
