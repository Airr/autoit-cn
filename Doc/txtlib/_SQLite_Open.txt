###User Defined Function###
_SQLite_Open

###Description###
Opens/creates a SQLite database.

###Syntax###
#include <SQLite.au3>
_SQLite_Open ( [ $sDatabase_Filename = ":memory:" ] )


###Parameters###
@@ParamTable@@
$sDatabase_Filename
	[optional] Database filename, by default will opens an in memory database.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	Returns database handle.
Failure:	Returns 0.
@extended	value can be compared against $SQLITE_* constants
@error	1 - Error calling SQLite API 'sqlite3_open'
	-1 - SQLite reported an error (Check @extended value)
@@End@@

###Remarks###
There is no need to store the database handle.
Functions that use the handle are using the last opened by default.
