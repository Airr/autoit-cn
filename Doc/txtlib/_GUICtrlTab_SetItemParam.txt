###User Defined Function###
_GUICtrlTab_SetItemParam

###Description###
Sets the param data of a tab

###Syntax###
#Include <GuiTab.au3>
_GUICtrlTab_SetItemParam($hWnd, $iIndex, $iParam)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$iIndex
	Zero based item index
$iParam
	Application defined data
@@End@@

###ReturnValue###
Success: True
Failure: False

###Remarks###
None.
