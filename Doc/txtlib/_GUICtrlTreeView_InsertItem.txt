###User Defined Function###
_GUICtrlTreeView_InsertItem

###Description###
Insert an item

###Syntax###
#Include <GuiTreeView.au3>
_GUICtrlTreeView_InsertItem($hWnd, $sItem_Text[, $hItem_Parent = 0[, $hItem_After = 0[, $iImage = -1[, $iSelImage = -1]]]])

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$sItem_Text
	Text of new item
$hItem_Parent
	parent item ID/handle/item
$hItem_After
	item ID/handle/flag to insert new item after
$iImage
	Zero based index of the item's icon in the control's image list
$iSelImage
	Zero based index of the item's icon in the control's image list
@@End@@

###ReturnValue###
Success: The new item handle
Failure: 0

###Remarks###
None.
