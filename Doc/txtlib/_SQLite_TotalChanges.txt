###User Defined Function###
_SQLite_TotalChanges

###Description###
Returns number of all changes (including triggers) of all transactions

###Syntax###
#include <SQLite.au3>
_SQLite_TotalChanges ( [ $hDB ] )


###Parameters###
@@ParamTable@@
$hDB
	[optional] An open database, default is the last opened database
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	Returns number of changes
Failure:	Returns 0.
@error	1 - Error calling SQLite API 'sqlite3_changes'
	2 - Call prevented by SaveMode
@@End@@
