###User Defined Function###
_GUICtrlIpAddress_SetFont

###Description###
Set font of the control

###Syntax###
#Include <GuiIPAddress.au3>
_GUICtrlIpAddress_SetFont($hWnd[, $sFaceName = "Arial"[, $iFontSize = 12[, $iFontWeight = 400[, $fFontItalic = False]]]])

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$sFaceName
	The Font Name of the font to use
$iFontSize
	The Font Size
$iFontWeight
	The Font Weight
$fFontItalic
	Use Italic Attribute
@@End@@

###ReturnValue###
None.

None.
