###User Defined Function###
_WinAPI_SetFont

###Description###

###Syntax###
#Include <WinAPI.au3>
_WinAPI_SetFont($hWnd, $hFont[, $fRedraw = True])

###Parameters###
@@ParamTable@@
$hWnd
	Window handle
$hFont
	Font handle
$fRedraw
	True to redraw the control
@@End@@

###ReturnValue###
None.
