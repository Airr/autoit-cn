###User Defined Function###
_GUICtrlTreeView_FindItem

###Description###
Retrieves a item based on it's text

###Syntax###
#Include <GuiTreeView.au3>
_GUICtrlTreeView_FindItem($hWnd, $sText[, $fInStr = False[, $hStart = 0]])

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$sText
	Text to search for
$fInStr
	If True, the text can be anywhere in the item's text.
$hStart
	Item to start searching from. If 0, the root item is used.
@@End@@

###ReturnValue###
Success: The handle of the first item that contains the item text
Failure: 0

###Remarks###
The search is case insensitive
