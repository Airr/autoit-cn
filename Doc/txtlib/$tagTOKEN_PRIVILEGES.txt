###Structure Name###
$tagTOKEN_PRIVILEGES

###Description###
Contains information about a set of privileges for an access token

###Syntax###
Global Const $tagTOKEN_PRIVILEGES = "int Count;int64 LUID;int Attributes"

###Fields###
@@ParamTable@@
Count
	Specifies the number of entries
LUID
	Specifies a LUID value
Attributes
	Specifies attributes of the LUID
@@End@@

###Remarks###
None.
