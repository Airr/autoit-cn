###User Defined Function###
_GUICtrlEdit_Destroy

###Description###
Delete the Edit control

###Syntax###
#Include <GuiEdit.au3>
_GUICtrlEdit_Destroy(ByRef $hWnd)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
@@End@@

###ReturnValue###
Success: True, Handle is set to 0
Failure: False

###Remarks###
Restricted to only be used on Edit created with _GUICtrlEdit_Create
