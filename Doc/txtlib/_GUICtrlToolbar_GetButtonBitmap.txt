###User Defined Function###
_GUICtrlToolbar_GetButtonBitmap

###Description###
Retrieves the index of the bitmap associated with a button

###Syntax###
#Include <GuiToolBar.au3>
_GUICtrlToolbar_GetButtonBitmap($hWnd, $iCommandID)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$iCommandID
	Button command ID
@@End@@

###ReturnValue###
Success: Zero based button bitmap index

###Remarks###
None.
