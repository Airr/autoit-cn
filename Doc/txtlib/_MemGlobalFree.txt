###User Defined Function###
_MemGlobalFree

###Description###
Frees the specified global memory object and invalidates its handle

###Syntax###
#Include <Memory.au3>
_MemGlobalFree($hMem)

###Parameters###
@@ParamTable@@
$hMem
	Handle to the global memory object
@@End@@

###ReturnValue###
Success: True
Failure: False

###Remarks###
None.
