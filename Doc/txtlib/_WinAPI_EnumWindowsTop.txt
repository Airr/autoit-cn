###User Defined Function###
_WinAPI_EnumWindowsTop

###Description###
Enumerates all top level windows

###Syntax###
#Include <WinAPI.au3>
_WinAPI_EnumWindowsTop()

###Parameters###
@@ParamTable@@
None.

@@End@@

###ReturnValue###
Success: Array with the following format:
		[0][0] - Number of rows in array (n)
		[1][0] - Window handle
		[1][1] - Window class name
		[n][0] - Window handle
		[n][1] - Window class name

###Remarks###
None.
