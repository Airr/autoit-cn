###User Defined Function###
_WinAPI_GetClassName

###Description###
Retrieves the name of the class to which the specified window belongs

###Syntax###
#Include <WinAPI.au3>
_WinAPI_GetClassName($hWnd)

###Parameters###
@@ParamTable@@
$hWnd
	Handle of window
@@End@@

###ReturnValue###
Success: The window class name
