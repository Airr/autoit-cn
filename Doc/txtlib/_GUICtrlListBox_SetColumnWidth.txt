###User Defined Function###
_GUICtrlListBox_SetColumnWidth

###Description###
Set the width, in pixels, of all columns

###Syntax###
#Include <GuiListBox.au3>
_GUICtrlListBox_SetColumnWidth($hWnd, $iWidth)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to control
$iWidth
	Specifies the width, in pixels, of all columns
@@End@@

###ReturnValue###
None.

###Remarks###
For use on controls created with the $LBS_MULTICOLUMN style
