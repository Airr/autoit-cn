###User Defined Function###
_StringProper

###Description###
Changes a string to proper case, same a =Proper function in Excel

###Syntax###
#Include <String.au3>
_StringProper($s_Str)

###Parameters###
@@ParamTable@@
$s_Str
	Input string
@@End@@

###ReturnValue###
Success: Returns proper string.
Failure: Returns "".

###Remarks###
This function will capitalize every character following a None Apha character.
