###User Defined Function###
_GUICtrlComboBoxEx_SetImageList

###Description###
Sets an image list for a ComboBoxEx control

###Syntax###
#Include <GuiComboBoxEx.au3>
_GUICtrlComboBoxEx_SetImageList($hWnd, $hHandle)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$hHandle
	A handle to the image list to be set for the control
@@End@@

###ReturnValue###
Success: handle to the image list previously associated with the control,
		or returns NULL if no image list was previously set

###Remarks###
None.
