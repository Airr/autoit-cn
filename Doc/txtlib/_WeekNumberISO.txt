###User Defined Function###
_WeekNumberISO

###Description###
Calculate the weeknumber of a given date.

###Syntax###
#Include <Date.au3>
_WeekNumberISO([$iYear = @YEAR[, $iMonth = @MON[, $iDay = @MDAY]]])

###Parameters###
@@ParamTable@@
$iYear
	Year value (default = current year)
$iMonth
	Month value (default = current month)
$iDay
	Day value (default = current day)
@@End@@

###ReturnValue###
Success: Returns week number of given date
Failure: 0
@Error: 0 - No error.
		 1 - faulty parameters values
		99 - On non-acceptable weekstart value

None.
