###User Defined Function###
_WinAPI_DestroyIcon

###Description###
Destroys an icon and frees any memory the icon occupied

###Syntax###
#Include <WinAPI.au3>
_WinAPI_DestroyIcon($hIcon)

###Parameters###
@@ParamTable@@
$hIcon
	Handle to the icon to be destroyed. The icon must not be in use.
@@End@@

###ReturnValue###
Success: True
Failure: False

###Remarks###
None.
