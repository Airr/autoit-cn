###User Defined Function###
_GUICtrlMonthCal_Destroy

###Description###
Delete the MonthCal control

###Syntax###
#Include <GuiMonthCal.au3>
_GUICtrlMonthCal_Destroy(ByRef $hWnd)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
@@End@@

###ReturnValue###
Success: True, Handle is set to 0
Failure: False

###Remarks###
Restricted to only be used on Month Calendars created with _GUICtrlMonthCal_Create
