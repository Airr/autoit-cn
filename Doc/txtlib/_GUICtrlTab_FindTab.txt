###User Defined Function###
_GUICtrlTab_FindTab

###Description###
Searches for a tab with the specific text

###Syntax###
#Include <GuiTab.au3>
_GUICtrlTab_FindTab($hWnd, $sText[, $fInStr = False[, $iStart = 0]])

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$sText
	Text to search for
$fInStr
	If True, the text can be anywhere in the tab's text.
$iStart
	Zero based index of the tab to start searching from
@@End@@

###ReturnValue###
Success: The zero based tab index that matches the search
Failure: -1

###Remarks###
The search is case insensitive
