###User Defined Function###
_GUICtrlTab_HighlightItem

###Description###
Sets the highlight state of a tab item

###Syntax###
#Include <GuiTab.au3>
_GUICtrlTab_HighlightItem($hWnd, $iIndex[, $fHighlight = True])

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$iIndex
	Zero based item index
$fHighlight
	If True, the tab is highlighted, otherwise the tab is reset
@@End@@

###ReturnValue###
Success: True
Failure: False

###Remarks###
None.
