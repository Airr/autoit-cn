###User Defined Function###
_GUICtrlListView_FindText

###Description###
Searches for an item with the specified text

###Syntax###
#Include <GuiListView.au3>
_GUICtrlListView_FindText($hWnd, $sText[, $iStart = -1[, $fPartialOK = True[, $fWrapOK = True]]])

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$sText
	Text to match
$iStart
	Zero based index of the item to begin the search with or -1 to start from the beginning. The
		specified item is itself excluded from the search.
$fPartialOK
	If True, a match will occur if the item text begins with the text
$fWrapOK
	If True, the search will continue with the first item if no match is found
@@End@@

###ReturnValue###
Success: The zero based index of the item
Failure: -1

###Remarks###
The search is case insensitive. The search is performed on the item only. Use _GUICtrlListView_FindInText if you want
to search for the text in subitems.
