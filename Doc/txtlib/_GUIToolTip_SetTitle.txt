###User Defined Function###
_GUIToolTip_SetTitle

###Description###
Adds a standard icon and title string

###Syntax###
#Include <GuiToolTip.au3>
_GUIToolTip_SetTitle($hWnd, $sTitle[, $iIcon = 0])

###Parameters###
@@ParamTable@@
$hWnd
	Handle to control
$sTitle
	Title string
$iIcon
	Set to one of the values below:.
		0 - No icon
		1 - Information icon
		2 - Warning icon
		3 - Error Icon
@@End@@

###ReturnValue###
成功: True
失败: False

###Remarks###
As of Windows XP SP2 and later, $iIcon can contain an HICON value. Any value greater than 3 is assumed to be
an HICON.
