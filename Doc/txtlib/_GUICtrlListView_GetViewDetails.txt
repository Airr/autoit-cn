###User Defined Function###
_GUICtrlListView_GetViewDetails

###Description###
Determines whether the view mode is in detail mode

###Syntax###
#Include <GuiListView.au3>
_GUICtrlListView_GetViewDetails($hWnd)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
@@End@@

###ReturnValue###
True: View is in detail mode
False: View is not in detail mode

###Remarks###
Minimum operating systems Windows XP.
