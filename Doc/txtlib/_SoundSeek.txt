###User Defined Function###
_SoundSeek

###Description###
Seeks the sound to the specified position 

###Syntax###
#include <Sound.au3>
_SoundSeek ( $sSnd_id, $iHour, $iMin, $iSec )

###Parameters###
@@ParamTable@@
$sSnd_id
	Sound ID (the 'alias') as returned by _SoundOpen
$iHour
	Hour to seek to
$iMin
	Minute to seek to
$iSec
	Second to seek to
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	1
Failure:	0
@Error:	0 = No Error
	1 = Failure
	3 = Invalid Sound ID
@@End@@

###Remarks###
After using _SoundSeek on a sound, you must use _SoundPlay to resume playing
