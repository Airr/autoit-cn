###User Defined Function###
_PathFull

###Description###
Creates a path based on the relative path you provide. The newly created absolute path is returned

###Syntax###
#Include <File.au3>
_PathFull($sRelativePath)

###Parameters###
@@ParamTable@@
$szRelPath
	The relative path to be created
@@End@@

###ReturnValue###
Success: Returns the newly created absolute path.

###Remarks###
None.
