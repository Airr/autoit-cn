###User Defined Function###
_ClipBoard_Empty

###Description###
Empties the clipboard and frees handles to data in the clipboard

###Syntax###
#Include <Clipboard.au3>
_ClipBoard_Empty()

###Parameters###
@@ParamTable@@
None.

@@End@@

###ReturnValue###
Success: True
Failure: False

###Remarks###
Before calling this function, you must open the clipboard by using the _ClipBoard_Open function. If you specified
a NULL window handle when opening the clipboard, this function succeeds but sets the clipboard owner to NULL.
Note that this causes _ClipBoard_SetData to fail.

###Related###
_ClipBoard_Open, _ClipBoard_SetData
