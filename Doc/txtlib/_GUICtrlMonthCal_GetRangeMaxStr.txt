###User Defined Function###
_GUICtrlMonthCal_GetRangeMaxStr

###Description###
Retrieves the upper limit date range in string format

###Syntax###
#Include <GuiMonthCal.au3>
_GUICtrlMonthCal_GetRangeMaxStr($hWnd[, $sFormat = "%02d/%02d/%04d"])

###Parameters###
@@ParamTable@@
$hWnd
	Handle to control
$sFormat
	StringFormat string used to format the date
@@End@@

###ReturnValue###
Success: Date in the specified format

###Remarks###
None.

_GUICtrlMonthCal_GetRange, _GUICtrlMonthCal_GetRangeMax
