###User Defined Function###
_MemShowError

###Description###
Displays an error message box with an optional exit

###Syntax###
#Include <Memory.au3>
_MemShowError($sText[, $fExit = True])

###Parameters###
@@ParamTable@@
$sText
	Error text to display
$fExit
	Specifies whether to exit after the display:
		 True - Exit program after display
		False - Return normally after display
@@End@@

###ReturnValue###
None.

###Remarks###
None.
