###User Defined Function###
_GUICtrlMenu_GetItemCount

###Description###
Retrieves the number of items in the specified menu

###Syntax###
#Include <GuiMenu.au3>
_GUICtrlMenu_GetItemCount($hMenu)

###Parameters###
@@ParamTable@@
$hMenu
	Handle of the menu
@@End@@

###ReturnValue###
Success: The number of items in the menu
Failure: 0

###Remarks###
None.
