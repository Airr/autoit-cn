###User Defined Function###
_GUICtrlListBox_GetSelItemsText

###Description###
Retrieves the text of selected items

###Syntax###
#Include <GuiListBox.au3>
_GUICtrlListBox_GetSelItemsText($hWnd)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to control
@@End@@

###ReturnValue###
Success: Array with the following format:
		[0] - Total items in array
		[1] - Text of the selected item
		[2] - Text of the selected item
		[n] - Text of the selected item

###Remarks###
None.
