###User Defined Function###
_Date_Time_SystemTimeToFileTime

###Description###
Converts a system time to file time format

###Syntax###
#Include <Date.au3>
_Date_Time_SystemTimeToFileTime($pSystemTime)

###Parameters###
@@ParamTable@@
$pSystemTime
	Pointer to a $tagSYSTEMTIME structure to be converted.
@@End@@

###ReturnValue###
Success: $tagFILETIME array that contains the converted file time

###Remarks###
None.

###Related###
$tagFILETIME, $tagSYSTEMTIME
