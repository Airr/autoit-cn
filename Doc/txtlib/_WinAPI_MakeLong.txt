###User Defined Function###
_WinAPI_MakeLong

###Description###
Returns a longint value from two int values

###Syntax###
#Include <WinAPI.au3>
_WinAPI_MakeLong($iLo, $iHi)

###Parameters###
@@ParamTable@@
$iLo
	Low word
$iHi
	Hi word
@@End@@

###ReturnValue###
Success: Longint value

None.
