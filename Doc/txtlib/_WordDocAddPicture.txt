###User Defined Function###
_WordDocAddPicture

###Description###
Add a picture to the document

###Syntax###
#include <Word.au3> 
_WordDocAddPicture ( ByRef $o_object, $s_FilePath [, $b_LinkToFile = 0 [, $b_SaveWithDocument = 0 [, $o_Range = ""]]] )

###Parameters###
@@ParamTable@@
$o_object
	Object variable of a Word.Application, document object
$s_FilePath
	The path and file name of the picture.
$b_LinkToFile
	<b>Optional:</b> Specifies whether to link the picture to the file from which it was created.
	0 = (Default) Make the picture an independent copy of the file
	1 = Link the picture to the file from which it was created
$b_SaveWithDocument
	<b>Optional:</b> Specifies whether to save the linked picture with the document.
	0 = (Default) Do not save the linked picture with the document
	1 = Save the linked picture with the document
$o_Range
	<b>Optional:</b> The location where the picture will be placed in the text.
	"" = (Default) The picture is placed automatically
	Any range object
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	Returns an object variable pointing to a Word.Application, shape object
Failure:	Retuns 0  and sets @ERROR = 1
@Error:	0 ($_WordStatus_Success) = No Error
	1 ($_WordStatus_GeneralError) = General Error
	2 ($_WordStatus_ComError) = Com Error
	3 ($_WordStatus_InvalidDataType) = Invalid Data Type
	4 ($_WordStatus_InvalidObjectType) = Invalid Object Type
	5 ($_WordStatus_InvalidValue) = Invalid Value
@Extended:	Contains invalid parameter number
@@End@@
