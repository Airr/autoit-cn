###User Defined Function###
_GUICtrlAVI_Play

###Description###
Plays an AVI clip in an animation control

###Syntax###
#Include <GuiAVI.au3>
_GUICtrlAVI_Play($hWnd[, $iFrom = 0[, $iTo = -1[, $iRepeat = -1]]])

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$iFrom
	Zero based index of the frame where playing begins. The value must be less than 65,536. A value
		of 0 means begin with the first frame in the clip.
$iTo
	Zero based index of the frame where playing ends. The value must be less than 65,536. A value
		of -1 means end with the last frame in the clip.
$iRepeat
	Number of times to replay the AVI clip. A value of -1 means replay the clip indefinitely.
@@End@@

###ReturnValue###
成功: True
失败: False

###Remarks###
The control plays the clip in the background while the thread continues executing
