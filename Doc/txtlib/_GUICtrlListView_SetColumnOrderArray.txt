###User Defined Function###
_GUICtrlListView_SetColumnOrderArray

###Description###
Sets the left-to-right order of columns in the control

###Syntax###
#Include <GuiListView.au3>
_GUICtrlListView_SetColumnOrderArray($hWnd, $aOrder)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$aOrder
	Array of integers that holds the index values of the columsn in the control. The array must be
		formatted as follows:
		[0] - Number of items in array
		[1] - First column index
		[2] - Second column index
		[n] - Last column index
@@End@@

###ReturnValue###
Success: True
Failure: False

###Remarks###
None.
