###User Defined Function###
_WordQuit

###Description###
Close the window and remove the object reference to it

###Syntax###
#include <Word.au3> 
_WordQuit( ByRef $o_object [, $i_SaveChanges = -2 [, $i_OriginalFormat = 1 [, $b_RouteDocument = 0]]] )

###Parameters###
@@ParamTable@@
$o_object
	Object variable of a Word.Application
$i_SaveChanges
	<b>Optional:</b> specifies the save action for the document
	0 = Do not save changes
	-1 = Save changes
	-2 = (Default) Prompt to save changes
$i_OriginalFormat
	<b>Optional:</b> specifies the save format for the document
	0 = Word Document
	1 = (Default) Original Document Format
	2 = Prompt User
$b_RouteDocument
	<b>Optional:</b> specifies whether to route the document to the next recipient
	0 = (Default) do not route
	1 = route to next recipient
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	Returns 1
Failure:	Returns 0 and sets @ERROR
@Error:	0 ($_WordStatus_Success) = No Error
	1 ($_WordStatus_GeneralError) = General Error
	3 ($_WordStatus_InvalidDataType) = Invalid Data Type
	4 ($_WordStatus_InvalidObjectType) = Invalid Object Type
@Extended:	Contains invalid parameter number
@@End@@
