###User Defined Function###
_GUIToolTip_Destroy

###Description###
Delete the ToolTip control

###Syntax###
#Include <GuiToolTip.au3>
_GUIToolTip_Destroy(ByRef $hWnd)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
@@End@@

###ReturnValue###
成功: True, 句柄设置为 0
失败: False

###Remarks###
Restricted to only be used on ToolTip created with _GUIToolTip_Create

_GUIToolTip_Create
