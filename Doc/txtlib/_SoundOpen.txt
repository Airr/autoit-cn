###User Defined Function###
_SoundOpen

###Description###
Opens a sound file for use with other _Sound functions.

###Syntax###
#include <Sound.au3>
_SoundOpen ( $hFile [, $sAlias ] )


###Parameters###
@@ParamTable@@
$hFile
	Path to sound file
$sAlias
	Optional - A name given to the sound which is used in other functions.
@@End@@

###ReturnValue###
@@ReturnTable@@
Success:	Sound ID
Failure:	0
@Error:	0 = No Error
	2 = File does not exist
	3 = alias contains whitespace or !
@Extended:	0 = No Error
	Integer other than 0 = MCI Error
@@End@@

###Remarks###
None.
