###User Defined Function###
_GUICtrlHeader_GetItemBitmap

###Description###
Retrieves the item bitmap handle

###Syntax###
#Include <GuiHeader.au3>
_GUICtrlHeader_GetItemBitmap($hWnd, $iIndex)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to control
$iIndex
	Zero based item index
@@End@@

###ReturnValue###
Success: Item bitmap handle
Failure: 0

###Remarks###
None.
