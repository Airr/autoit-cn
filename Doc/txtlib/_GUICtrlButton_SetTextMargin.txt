###User Defined Function###
_GUICtrlButton_SetTextMargin

###Description###
Sets the margins for drawing text in a button control

###Syntax###
#Include <GuiButton.au3>
_GUICtrlButton_SetTextMargin($hWnd[, $iLeft = 1[, $iTop = 1[, $iRight = 1[, $iBottom = 1]]]])

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
$iLeft
	Left margin to use for drawing text
$iTop
	Top margin to use for drawing text
$iRight
	Right margin to use for drawing text
$iBottom
	Bottom margin to use for drawing text
@@End@@

###ReturnValue###
Success: True
Failure: False

###Remarks###
Minimum Operating Systems: Windows XP
