###User Defined Function###
_GUICtrlTab_Destroy

###Description###
Delete the control

###Syntax###
#Include <GuiTab.au3>
_GUICtrlTab_Destroy(ByRef $hWnd)

###Parameters###
@@ParamTable@@
$hWnd
	Handle to the control
@@End@@

###ReturnValue###
成功: True, 句柄设置为 0
失败: False

###Remarks###
Restricted to only be used on Tab created with _GUICtrlTab_Create
