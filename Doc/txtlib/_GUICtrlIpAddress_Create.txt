###User Defined Function###
_GUICtrlIpAddress_Create

###Description###
Create a GUI IP Address Control

###Syntax###
#Include <GuiIPAddress.au3>
_GUICtrlIpAddress_Create($hWnd, $iX, $iY[, $iWidth = 125[, $iHeigth = 25[, $iStyles = 0x00000000[, $iExstyles = 0x00000000]]]])

###Parameters###
@@ParamTable@@
$hWnd
	Handle to parent or owner window
$iX
	Horizontal position of the control
$iY
	Vertical position of the control
$iWidth
	Control width
$iHeight
	Control height
$iStyle
	Control styles:
		Forced : $WS_CHILD, $WS_VISIBLE, $WS_TABSTOP
$iExStyle
	Extended control styles
@@End@@

###ReturnValue###
Success: Handle to the IP Address control
Failure: 0

###Remarks###
None.
