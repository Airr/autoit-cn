###Keyword###
#RequireAdmin

###Description###
指定当前脚本需要拥有完整的管理员权限才能正确运行.

###Syntax###
<b>#RequireAdmin</b>


###Parameters###
None.


###Remarks###
This function was primarily aimed at allowing AutoIt scripts to work correctly with Windows Vista User Account Control (UAC) (However, will also work on Windows 2000 and Windows XP).<br>
更多信息请参考 <a href="../intro/autoit_on_vista.htm">AutoIt 运行于 Windows Vista</a>.

如果使用了这个函数后,调用了一个新进程,新进程也将拥有管理员权限.但是一些函数[如: Consolewrite()]将不能被捕捉 (Scite 不会显示任何东西).
