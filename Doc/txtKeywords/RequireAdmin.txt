###Keyword###
#RequireAdmin

###Description###
指定当前脚本需要拥有完整的管理员权限才能正确运行.

###Syntax###
<b>#RequireAdmin</b>


###Parameters###
None.


###Remarks###
这个函数主要针对在 Windows Vista <b>User Account Control (UAC)</b>下面对脚本的权限限制. (当然, 它同样能工作于 on Windows 2000 和 Windows XP).<br>
更多信息请参考 <a href="../intro/autoit_on_vista.htm">AutoIt 运行于 Windows Vista</a>.

如果使用了这个函数后,调用了一个新进程,新进程也将拥有管理员权限.但是一些函数[如: Consolewrite() 将不能被捕捉 (Scite 不会显示任何东西).
