Getting started

On start you will see the following Screen:

 

 

The Buttons mean the following:

 

"Clear Window":         to clear the textBox, to start a new Script

"Start Macro":           Starts the Record-mode of the Macro-Programm

"Pause Macro":          Pause the record

"Stop Macro":           Stopps the record, to end this Script

"Save Macro":           saves the Text File (alternativ you can copy and paste)

"Window on Top":      let the Window stay on Top - or not

"About..."                 some Information about this Programm

"execute a Run()":     use this Button to Open a File Requester, and choose a Programm

                              the Programm will be started, and the "Run()" command will be added to the Textbox.

"execute a Command":Type in the Command Line you want execute

 

To automate an Install, press the "Start Macro"-Button:

 

 

Then press the "execute a Run()"-Button or "execute a Command", and your Programm starts.

Click through the Installation, and on the end press the "Stop Macro".
